<Callout type="info" title="¿Buscas un instalador de escritorio?">

LibreChat es una aplicación web autohospedada, no una aplicación nativa de Windows ni un AppImage de Linux. En Windows, utiliza [Docker Desktop](/docs/local/docker) para la configuración local más sencilla. Para escritorios o servidores Linux, utiliza [Docker](/docs/local/docker), [npm](/docs/local/npm) o una [guía de alojamiento remoto](/docs/remote).

</Callout>
