<Callout type="info" title="Cerchi un programma di installazione desktop?">

LibreChat è un'applicazione web self-hosted, non un'app nativa Windows o un AppImage Linux. Su Windows, utilizza [Docker Desktop](/docs/local/docker) per la configurazione locale più semplice. Per desktop o server Linux, utilizza [Docker](/docs/local/docker), [npm](/docs/local/npm) o una [guida all'hosting remoto](/docs/remote).

</Callout>
