<Callout type="info" title="正在寻找桌面安装程序？">

LibreChat 是一个自托管的 Web 应用程序，而非原生的 Windows 应用或 Linux AppImage。在 Windows 上，请使用 [Docker Desktop](/docs/local/docker) 进行最简单的本地设置。对于 Linux 桌面或服务器，请使用 [Docker](/docs/local/docker)、[npm](/docs/local/npm) 或参考[远程托管指南](/docs/remote)。

</Callout>
