<Callout type="info" title="Procurando por um instalador para desktop?">

LibreChat é uma aplicação web auto-hospedada, não um aplicativo nativo do Windows ou AppImage do Linux. No Windows, use o [Docker Desktop](/docs/local/docker) para a configuração local mais simples. Para desktops ou servidores Linux, use [Docker](/docs/local/docker), [npm](/docs/local/npm) ou um [guia de hospedagem remota](/docs/remote).

</Callout>
