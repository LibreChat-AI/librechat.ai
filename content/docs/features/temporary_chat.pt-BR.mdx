Chats temporários permitem que você pergunte algo sem mantê-lo. Use-os para tópicos sensíveis, experimentos rápidos ou qualquer coisa que você não precise salvar. Um chat temporário fica fora da sua barra lateral de histórico, nunca aparece na pesquisa, não pode ser marcado como favorito e é excluído automaticamente assim que seu período de retenção termina (30 dias por padrão).

## Iniciar um Chat Temporário [#start-a-temporary-chat]

<Steps>

<Step>

**Abra o menu de modelos.** Selecione o menu suspenso de modelos no canto superior esquerdo da visualização do chat.

</Step>

<Step>

**Ative o Chat Temporário.** Alterne o interruptor **Temporary Chat** para a posição ON.

<Step>

**Confirme se está ativo.** Um banner aparece acima da caixa de mensagem enquanto o Temporary Chat está ativado. Selecione o fechar (x) no banner para desativá-lo e retornar a uma conversa normal.

## O que um Chat Temporário faz [#what-a-temporary-chat-does]

- Não aparece na barra lateral do histórico de chat.
- É excluído dos resultados de pesquisa.
- Não pode ser marcado como favorito.
- É armazenado no banco de dados até que seu período de retenção termine, sendo então excluído automaticamente.

<Callout type="info" title="Período de retenção">

O período de retenção padrão é de 30 dias. Administradores podem alterá-lo. Veja [Configuration](#configuration-administrators) abaixo.

</Callout>

## Configuração (administradores) [#configuration-administrators]

O Temporary Chat está disponível para os usuários por padrão. Os administradores controlam se o recurso é oferecido e por quanto tempo os chats temporários são mantidos.

A **disponibilidade** é regida pela permissão de função `TEMPORARY_CHAT`. Gerencie-a para cada função a partir do [Admin Panel](/docs/features/admin_panel). A opção `interface.temporaryChat` no `librechat.yaml` apenas define essa permissão para a função `USER` padrão na inicialização e está obsoleta para o gerenciamento de permissões.

**Retention** é definido com `interface.temporaryChatRetention` (em horas). O mínimo é 1 hora, o máximo é 8760 (1 ano) e o padrão é 720 (30 dias).

<Tabs items={['librechat.yaml (recomendado)', 'Variável de ambiente (obsoleto)']}>

<Tabs.Tab>

```yaml filename="librechat.yaml"
interface:
  temporaryChat: true
  temporaryChatRetention: 168 # retain temporary chats for 7 days
```

</Tabs.Tab>

<Tabs.Tab>

```bash filename=".env"
# Hours to retain temporary chats (default: 720 = 30 days)
TEMP_CHAT_RETENTION_HOURS=168
```

<Callout type="warning" title="Obsoleto">

`TEMP_CHAT_RETENTION_HOURS` está obsoleto. Prefira `interface.temporaryChatRetention` no `librechat.yaml`, que tem precedência sobre a variável de ambiente.

</Callout>

</Tabs.Tab>

</Tabs>

Valores comuns de retenção:

| Valor | Período |
| --- | --- |
| `1` | 1 hora (mínimo) |
| `24` | 1 dia |
| `168` | 1 semana |
| `720` | 30 dias (padrão) |
| `8760` | 1 ano (máximo) |

Para o conjunto completo de opções, incluindo `retentionMode` e `retainAgentFiles`, consulte a [referência da interface](/docs/configuration/librechat_yaml/object_structure/interface#temporarychatretention).
