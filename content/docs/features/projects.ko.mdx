Projects를 사용하면 각 사용자가 관련 대화를 이름이 지정된 작업 공간으로 구성할 수 있습니다. 이는 장기적인 작업 흐름, 팀, 클라이언트, 수업 또는 검색에만 의존하지 않고 집중적인 대화 세트를 유지하려는 모든 주제에 유용합니다.

## 프로젝트의 역할 [#what-projects-do]

- 프로젝트별로 대화 그룹화
- 프로젝트 내에서 직접 새 채팅 시작하기
- 기존 대화를 프로젝트로 이동하거나 할당되지 않음으로 되돌리기
- 프로젝트 작업 공간에서 프로젝트 채팅 탐색
- 프로젝트 목록 검색 및 정렬
- 사이드바에서 프로젝트 이름 변경 또는 삭제

프로젝트는 이를 생성한 사용자 개인에게 귀속됩니다. 다른 사용자는 귀하의 프로젝트 목록을 볼 수 없습니다.

## 프로젝트 생성 [#create-a-project]

사이드바에서 **Projects**를 열고 **New project**를 선택하세요. 프로젝트 이름을 지정하면 LibreChat이 프로젝트 작업 공간을 엽니다.

프로젝트 작업 공간에서 **New chat in project**를 사용하여 해당 프로젝트에 자동으로 할당되는 대화를 시작하세요.

## 기존 채팅 이동하기 [#move-existing-chats]

대화 메뉴를 열고 프로젝트 옵션을 선택하세요. 대화를 할당할 프로젝트를 선택하거나, **Unassigned**를 선택하여 프로젝트에서 제거하세요.

프로젝트를 열면 LibreChat은 해당 프로젝트에 할당된 채팅만 표시합니다. 기본 대화 목록에서는 **모든 프로젝트**를 통해 여전히 모든 채팅을 확인할 수 있습니다.

## 프로젝트 관리 [#manage-projects]

각 프로젝트는 사이드바에 표시됩니다. 프로젝트 메뉴를 사용하여 다음을 수행할 수 있습니다:

- 프로젝트 작업 공간을 엽니다
- 프로젝트 이름 변경
- 프로젝트 삭제

프로젝트를 삭제하면 해당 채팅에서 프로젝트 할당이 제거됩니다. 대화 자체는 삭제되지 않습니다.

## 참고 사항 [#notes]

- 프로젝트 작업 공간(Project workspaces)은 고유한 채팅 목록과 정렬 제어 기능을 갖추고 있습니다.
- 프로젝트에서 시작된 새 채팅은 해당 프로젝트 할당을 유지합니다.
- 프로젝트를 보고 있는 도중에 해당 프로젝트가 삭제되면, LibreChat은 프로젝트 목록으로 돌아갑니다.
