O LibreChat integrou o **Meilisearch** para aprimorar a experiência do usuário, fornecendo uma maneira rápida e eficiente de pesquisar em conversas anteriores. O Meilisearch é um mecanismo de busca poderoso e de código aberto, conhecido por sua velocidade e facilidade de uso, tornando-o uma excelente escolha para aplicações como o LibreChat que exigem acesso rápido a um grande volume de dados.

![Pesquisando conversas por uma banana](https://github.com/danny-avila/LibreChat/assets/32828263/60ad41b0-1869-4ee9-848b-502b3a5557b5)


A integração permite que os usuários:
- Realize **pesquisas de texto completo** no histórico de conversas.
- Utilize **typo-tolerance** para consultas de pesquisa mais flexíveis.
- Experimente **resultados instantâneos** conforme você digita, graças aos recursos de busca *as-you-type* do Meilisearch.

Este recurso melhora significativamente a funcionalidade do LibreChat, tornando mais fácil para os usuários encontrarem mensagens ou tópicos específicos dentro do seu histórico de chat. É um testemunho do compromisso da plataforma em proporcionar uma experiência integrada e fácil de usar. Para aqueles interessados nos detalhes técnicos ou na implementação, mais informações podem ser encontradas no [Meilisearch Configuration Guide](/docs/configuration/meilisearch).
