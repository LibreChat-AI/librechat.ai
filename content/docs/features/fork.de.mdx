Das Forking erstellt eine neue Konversation, die von einer bestimmten Nachricht in einer bestehenden Konversation abzweigt. Der neue Chat kopiert die von Ihnen ausgewählten Nachrichten, sodass Sie eine alternative Richtung erkunden, einen anderen Prompt testen oder einen langen Thread in fokussierte Themen aufteilen können, ohne das Original zu verändern.

## So forken Sie eine Konversation [#how-to-fork-a-conversation]

<Steps>
<Step>

**Fahre mit der Maus über eine Nachricht** und öffne deren Menü, wähle dann das Fork-Symbol aus. Die Nachricht, bei der du das Menü öffnest, wird zur Zielnachricht für den Fork.

</Step>
<Step>

**Wählen Sie eine Fork-Option**, um zu steuern, welche Nachrichten kopiert werden (siehe unten).

</Step>
<Step>

**Bestätigen Sie den Fork.** LibreChat öffnet die neue Konversation mit den kopierten Nachrichten, wobei das Original unverändert bleibt.

</Step>
</Steps>

Die *Zielnachricht* ist die Nachricht, von der aus Sie das Menü geöffnet haben. Wenn Sie **Start fork here** aktivieren, wird stattdessen die neueste Nachricht in der Konversation zur Zielnachricht, und das Forking erfolgt ab der ausgewählten Nachricht vorwärts.

## Fork-Optionen [#fork-options]

Jede Option kopiert einen anderen Satz von Nachrichten in Bezug auf das Ziel.

### Nur sichtbare Nachrichten [#visible-messages-only]

Kopiert nur die sichtbaren Nachrichten: den direkten Pfad zur Zielnachricht, unter Ausschluss aller Verzweigungen.

### Zugehörige Branches einbeziehen [#include-related-branches]

Kopiert den direkten Pfad zur Zielnachricht zusammen mit allen Verzweigungen, die entlang dieses Pfades liegen.

<ThemeImage
  light="https://github.com/danny-avila/LibreChat/assets/32828263/e633f701-acf5-4878-bdd1-29abacb3e3e7"
  dark="https://github.com/danny-avila/LibreChat/assets/32828263/0c297451-990b-4ab2-9ff2-bc3958ab7129"
  alt="Option „Zugehörige Branches forken“ einbeziehen"
/>

### Alles von/bis hier einbeziehen [#include-all-tofrom-here]

Die Standardoption. Kopiert jede Nachricht, die zum Ziel führt, einschließlich benachbarter Zweige, unabhängig davon, ob diese sichtbar sind oder auf demselben Pfad liegen.

## Zusätzliche Einstellungen [#additional-settings]

**Start fork here** erstellt einen Fork von der ausgewählten Nachricht bis zur neuesten Nachricht in der Unterhaltung, wobei die oben gewählte Option verwendet wird.

<ThemeImage
  light="https://github.com/danny-avila/LibreChat/assets/32828263/801e50e4-749a-42f3-83bd-a3fc06c6e189"

/>

**Remember** speichert die von Ihnen ausgewählten Optionen und wendet sie auf zukünftige Forks an, sodass Sie diese nicht jedes Mal neu festlegen müssen.
