## Warum Konversationen forken? [#why-fork-conversations]

Stellen Sie sich Forking wie das Erstellen eines neuen Pfads in Ihrem Chat vor – es ist äußerst praktisch, wenn Sie:

### Alles auf Kurs halten [#keep-things-on-track]
Manchmal stößt man auf ein interessantes Nebenthema, möchte aber das Hauptgespräch nicht unterbrechen. Mit „Forking“ können Sie in dieses Thema eintauchen, während Ihr ursprünglicher Chat übersichtlich bleibt.

### „Was wäre wenn“-Szenarien spielen [#play-what-if]
Möchten Sie sehen, wie sich verschiedene Ansätze entwickeln könnten? Erstellen Sie einen Fork des Chats und probieren Sie verschiedene Blickwinkel aus. Es ist, als hätten Sie mehrere parallele Unterhaltungen, von denen jede eine andere Möglichkeit erkundet.

### Wichtigen Kontext speichern [#save-important-context]
Wenn Sie einen Fork erstellen, können Sie so viel oder so wenig vom vorherigen Chat mitnehmen, wie Sie benötigen. Auf diese Weise fangen Sie nicht komplett bei Null an und die KI verfügt weiterhin über den benötigten Kontext.

### Bestimmte Teile teilen [#share-specific-parts]
Müssen Sie jemandem nur einen Teil Ihrer Unterhaltung zeigen? Anstatt Ihren gesamten Chatverlauf zu kopieren, einzufügen oder zu teilen, können Sie einfach den relevanten Teil abspalten (forken) und diesen teilen.

### Ideen organisiert halten [#keep-ideas-organized]
Lange Chats können viele verschiedene Ideen hervorbringen. Anstatt sie miteinander verflechten zu lassen, können Sie jede davon in einen eigenen Thread abspalten (forken) und sie separat weiterentwickeln.

### Verschiedene Ansätze ausprobieren [#try-different-approaches]
Manchmal fragst du dich vielleicht, ob es einen besseren Weg gibt, etwas zu fragen. Mit Forking kannst du mit verschiedenen Arten der Kommunikation mit der KI experimentieren, ohne deine Hauptkonversation zu beeinträchtigen.

## Forking-Optionen [#forking-options]

Verwenden Sie diese Einstellungen, um Nachrichten mit dem gewünschten Verhalten zu forken.

„Forking“ bezieht sich auf das Erstellen einer neuen Konversation, die an bestimmten Nachrichten innerhalb der aktuellen Konversation ansetzt oder endet, wodurch eine Kopie gemäß den ausgewählten Optionen erstellt wird.

Die „Zielnachricht“ bezieht sich entweder auf die Nachricht, von der aus das Popup geöffnet wurde, oder, falls Sie „Start fork here“ aktivieren, auf die letzte Nachricht im Gespräch.


### Nur sichtbare Nachrichten: [#visible-messages-only]
Diese Option verzweigt nur die sichtbaren Nachrichten; mit anderen Worten, den direkten Pfad zur Zielnachricht, ohne jegliche Zweige.

Diese Option verzweigt die sichtbaren Nachrichten zusammen mit den zugehörigen Verzweigungen; mit anderen Worten: den direkten Pfad zur Zielnachricht, einschließlich der Verzweigungen entlang dieses Pfads.

### Alle von/bis hier einschließen (Standard): [#include-all-tofrom-here-default]
Diese Option verzweigt alle Nachrichten, die zu der Zielnachricht führen, einschließlich ihrer Nachbarn; mit anderen Worten: Alle Nachrichtenzweige, unabhängig davon, ob sie sichtbar sind oder auf demselben Pfad liegen, werden einbezogen.
