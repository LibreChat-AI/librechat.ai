Der Import von Konversationen ermöglicht es Benutzern, Konversationen zu importieren, die aus anderen KI-Chat-Anwendungen exportiert wurden. Derzeit unterstützen wir den Import von Konversationen aus [ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/), [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code) und LibreChat selbst.

Die Import-Funktionalität ist im Bereich "Settings" -> "Data Controls" verfügbar.

## Exportieren Sie Ihre Konversationen [#export-your-conversations]

### ChatGPT [#chatgpt]

1. Befolgen Sie die [ChatGPT-Exportanweisungen](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data), um Ihre Konversationen zu exportieren.
2. Sie sollten einen Link zum Herunterladen des Archivs per E-Mail erhalten.
3. Laden Sie das Archiv herunter. Es sollte eine Zip-Datei mit einem zufälligen Namen sein, wie zum Beispiel: _d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip_
4. Entpacken Sie den Inhalt der ZIP-Datei.

### Claude [#claude]

1. Befolgen Sie die [Claude-Exportanweisungen](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data), um Ihre Konversationen zu exportieren.
2. Sie erhalten eine E-Mail mit einem Download-Link für Ihr Export-Archiv.
3. Laden Sie das Archiv herunter und entpacken Sie es.

## Import in LibreChat [#import-into-librechat]

1. Navigieren Sie in LibreChat zu **Settings** → **Data Controls**.
2. Klicken Sie auf die Schaltfläche **Import** und wählen Sie die Datei `conversations.json` aus dem entpackten Archiv aus.
3. In Kürze erhalten Sie eine Benachrichtigung, dass der Import abgeschlossen ist.
