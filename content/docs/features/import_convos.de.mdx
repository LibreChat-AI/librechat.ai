Mit dem Konversationsimport können Sie Konversationen, die aus anderen KI-Chat-Anwendungen exportiert wurden, in LibreChat importieren. Unterstützte Quellen sind [ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/) und [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code).

<Callout type="info" title="Wo Sie es finden">
Der Import befindet sich unter **Settings** → **Data Controls** in LibreChat.
</Callout>

## Exportieren Sie Ihre Unterhaltungen [#export-your-conversations]

Exportieren Sie zunächst Ihre Daten aus der Quellanwendung. Die Schritte unterscheiden sich je nach Quelle.

<Tabs items={["ChatGPT", "Claude", "ChatbotUI v1"]}>
<Tabs.Tab>

<Steps>
<Step>

**Fordern Sie Ihren Export an.** Befolgen Sie die [ChatGPT-Exportanweisungen](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data), um Ihre Konversationen zu exportieren.

</Step>
<Step>

**Laden Sie das Archiv herunter.** Sie erhalten eine E-Mail mit einem Download-Link. Das Archiv ist eine ZIP-Datei mit einem zufälligen Namen wie `d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip`.

</Step>
<Step>

**Entpacken Sie das Archiv**, um auf die darin enthaltene `conversations.json` Datei zuzugreifen.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

<Steps>
<Step>

**Fordern Sie Ihren Export an.** Befolgen Sie die [Claude export instructions](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data), um Ihre Konversationen zu exportieren.

</Step>
<Step>

**Laden Sie das Archiv herunter.** Sie erhalten eine E-Mail mit einem Download-Link für Ihr Export-Archiv.

</Step>
<Step>

**Entpacken Sie das Archiv**, um auf die darin enthaltene `conversations.json` Datei zuzugreifen.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

Exportieren Sie Ihre Konversationen aus [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code) und importieren Sie anschließend die resultierende JSON-Datei direkt. Es ist kein Extraktionsschritt erforderlich.

</Tabs.Tab>
</Tabs>

## Import in LibreChat [#import-into-librechat]

<Steps>
<Step>

**Open Data Controls.** Navigieren Sie in LibreChat zu **Settings** → **Data Controls**.

</Step>
<Step>

**Wählen Sie Ihre Datei aus.** Klicken Sie auf **Import** und wählen Sie die Datei `conversations.json` aus Ihrem entpackten Archiv (oder die exportierte JSON-Datei für ChatbotUI v1) aus.

</Step>
<Step>

**Auf Bestätigung warten.** Eine Benachrichtigung erscheint, sobald der Import abgeschlossen ist.

</Step>
</Steps>
