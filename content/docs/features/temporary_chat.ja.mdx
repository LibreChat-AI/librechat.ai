機密性の高いトピックやちょっとした実験、あるいは永続的に保存する必要のないやり取りには、一時チャットを使用してチャット履歴を整理し、すっきりと保ちましょう。これらのチャットは検索結果から除外され、ブックマークすることはできず、30日後に自動的に削除されます。

## 一時的なチャット（Temporary Chat）の有効化 [#activating-temporary-chat]

1.  LibreChatの左上にあるモデルリストのドロップダウンメニューを開きます。
2.  「Temporary Chat」スライダーをONの位置に切り替えます。

![Temporary Chat Toggle](https://github.com/user-attachments/assets/ebd5370d-7fac-45af-b0d3-2de59df506e9)

3.  チャット入力エリアの上に通知が表示され、Temporary Chatモードが有効であることを確認できます。(x)を押すと、Temporary Chatが無効になります。

![Temporary Chat Notification](https://github.com/user-attachments/assets/ed19485a-99d3-45a2-a148-7824031f9022)


## チャットが一時的（Temporary）としてマークされるとどうなりますか？ [#what-happens-when-a-chat-is-marked-as-temporary]

*   Temporary Chatsは、チャット履歴のサイドバーには表示されません。
*   一時的なチャット（Temporary Chats）は検索結果から除外されます。
*   一時的なチャットはブックマークできません。
*   Temporary Chatsはデータベースに30日間保存され、その後自動的に削除されます。
