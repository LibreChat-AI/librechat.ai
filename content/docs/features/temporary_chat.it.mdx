Le chat temporanee ti consentono di chiedere qualcosa senza conservarla. Usale per argomenti sensibili, esperimenti rapidi o qualsiasi cosa che non hai bisogno di salvare. Una chat temporanea rimane fuori dalla barra laterale della cronologia, non appare mai nella ricerca, non può essere aggiunta ai segnalibri e viene eliminata automaticamente una volta terminato il periodo di conservazione (30 giorni per impostazione predefinita).

## Avvia una chat temporanea [#start-a-temporary-chat]

<Steps>

<Step>

**Apri il menu dei modelli.** Seleziona il menu a tendina dei modelli in alto a sinistra nella vista della chat.

</Step>

<Step>

**Attiva la Chat temporanea.** Sposta l'interruttore **Temporary Chat** sulla posizione ON.

## Cosa fa una chat temporanea [#what-a-temporary-chat-does]

- Non appare nella barra laterale della cronologia chat.
- È escluso dai risultati di ricerca.
- Non può essere aggiunto ai segnalibri.
- Viene archiviato nel database fino al termine del suo periodo di conservazione, dopodiché viene eliminato automaticamente.

<Callout type="info" title="Periodo di conservazione">

Il periodo di conservazione predefinito è di 30 giorni. Gli amministratori possono modificarlo. Vedi [Configurazione](#configuration-administrators) qui sotto.

</Callout>

## Configurazione (amministratori) [#configuration-administrators]

La Temporary Chat è disponibile per gli utenti per impostazione predefinita. Gli amministratori controllano se la funzionalità viene offerta e per quanto tempo vengono conservate le chat temporanee.

La **disponibilità** è regolata dal permesso del ruolo `TEMPORARY_CHAT`. Gestiscila per ogni ruolo dal [Pannello di Amministrazione](/docs/features/admin_panel). L'opzione `interface.temporaryChat` in `librechat.yaml` inizializza solo questo permesso per il ruolo `USER` predefinito all'avvio ed è deprecata per la gestione dei permessi.

**Retention** viene impostato con `interface.temporaryChatRetention` (in ore). Il minimo è 1 ora, il massimo è 8760 (1 anno) e il valore predefinito è 720 (30 giorni).

<Tabs items={['librechat.yaml (consigliato)', 'Variabile d\'ambiente (deprecata)']}>

<Tabs.Tab>

```yaml filename="librechat.yaml"
interface:
  temporaryChat: true
  temporaryChatRetention: 168 # retain temporary chats for 7 days
```

</Tabs.Tab>

<Tabs.Tab>

```bash filename=".env"
# Hours to retain temporary chats (default: 720 = 30 days)
TEMP_CHAT_RETENTION_HOURS=168
```

<Callout type="warning" title="Deprecato">

`TEMP_CHAT_RETENTION_HOURS` è deprecato. È preferibile utilizzare `interface.temporaryChatRetention` in `librechat.yaml`, che ha la precedenza sulla variabile d'ambiente.

</Callout>

</Tabs.Tab>

</Tabs>

Valori di conservazione comuni:

| Valore | Periodo |
| --- | --- |
| `1` | 1 ora (minimo) |
| `24` | 1 giorno |
| `168` | 1 settimana |
| `720` | 30 giorni (predefinito) |
| `8760` | 1 anno (massimo) |

Per l'insieme completo delle opzioni, inclusi `retentionMode` e `retainAgentFiles`, consulta il [riferimento dell'interfaccia](/docs/configuration/librechat_yaml/object_structure/interface#temporarychatretention).
