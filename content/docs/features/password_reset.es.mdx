## Descripción general [#overview]

Esta función habilita la funcionalidad de restablecimiento de contraseña por correo electrónico para su servidor LibreChat. Puede configurarla para que funcione con varios servicios de correo electrónico, incluidos Mailgun API, Gmail y servidores de correo personalizados.

## Características clave [#key-features]

- Admite múltiples servicios de correo electrónico:
  - **Mailgun API** - Recomendado para servidores que bloquean puertos SMTP
  - **Gmail** y otros servicios SMTP predefinidos
  - **Servidores de correo personalizados** con configuración SMTP avanzada
- Permite opciones de configuración básicas y avanzadas
- Habilita la funcionalidad de restablecimiento de contraseña basada en correo electrónico de forma segura para su servidor LibreChat

## Opciones de configuración [#setup-options]

- **Configuración de Mailgun**: Utilice la API de Mailgun para una entrega de correo electrónico confiable, especialmente útil cuando los puertos SMTP están bloqueados
- **Configuración básica de SMTP**: Utilice servicios predefinidos como Gmail con una configuración mínima
- **Configuración avanzada de SMTP**: Configure servicios SMTP genéricos o personalice los ajustes para cualquier proveedor de correo electrónico

**Para obtener instrucciones de configuración detalladas, consulte la guía de configuración: [Email Setup](/docs/configuration/authentication/email)**
