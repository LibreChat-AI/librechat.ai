Los Projects permiten a cada usuario organizar conversaciones relacionadas en espacios de trabajo con nombre. Son útiles para flujos de trabajo de larga duración, equipos, clientes, clases o cualquier tema en el que desee un conjunto enfocado de chats sin depender únicamente de la búsqueda.

## Qué hacen los Projects [#what-projects-do]

- Agrupar conversaciones bajo un proyecto
- Inicia un nuevo chat directamente dentro de un proyecto
- Mover una conversación existente a un proyecto o de vuelta a sin asignar
- Explorar chats del proyecto desde el espacio de trabajo del proyecto
- Buscar y ordenar la lista de proyectos
- Cambiar el nombre o eliminar proyectos desde la barra lateral

Los Projects son personales para el usuario que los crea. Otros usuarios no pueden ver tu lista de proyectos.

## Crear un proyecto [#create-a-project]

Abre **Projects** desde la barra lateral y elige **New project**. Dale un nombre al proyecto y LibreChat abrirá el espacio de trabajo del proyecto.

Desde un espacio de trabajo de proyecto, utiliza **New chat in project** para iniciar una conversación que se asigna automáticamente a ese proyecto.

## Mover chats existentes [#move-existing-chats]

Abre el menú de una conversación y elige la opción de proyecto. Selecciona un proyecto para asignar el chat, o elige **Unassigned** para eliminarlo de un proyecto.

Cuando abres un proyecto, LibreChat muestra solo los chats asignados a ese proyecto. La lista principal de conversaciones aún puede mostrar todos los chats a través de **All projects**.

## Gestionar proyectos [#manage-projects]

Cada proyecto aparece en la barra lateral. Utiliza el menú del proyecto para:

- Abra el espacio de trabajo del proyecto
- Renombrar el proyecto
- Eliminar el proyecto

Eliminar un proyecto elimina la asignación del proyecto de sus chats. Las conversaciones en sí no se eliminan.

## Notas [#notes]

- Los espacios de trabajo del proyecto tienen su propia lista de chat y controles de ordenamiento.
- Los nuevos chats iniciados desde un proyecto mantienen esa asignación de proyecto.
- Si un proyecto se elimina mientras lo estás visualizando, LibreChat te devuelve a la lista de proyectos.
