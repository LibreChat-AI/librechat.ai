## Ikhtisar [#overview]

Fitur ini mengaktifkan fungsionalitas pengaturan ulang kata sandi berbasis email untuk server LibreChat Anda. Anda dapat mengonfigurasinya agar berfungsi dengan berbagai layanan email, termasuk Mailgun API, Gmail, dan server email kustom.

## Fitur Utama [#key-features]

- Mendukung berbagai layanan email:
  - **Mailgun API** - Direkomendasikan untuk server yang memblokir port SMTP
  - **Gmail** dan layanan SMTP bawaan lainnya
  - **Server email kustom** dengan konfigurasi SMTP tingkat lanjut
- Memungkinkan opsi konfigurasi dasar dan lanjutan
- Mengaktifkan fungsionalitas pengaturan ulang kata sandi berbasis email yang aman untuk server LibreChat Anda

## Opsi Pengaturan [#setup-options]

- **Konfigurasi Mailgun**: Gunakan API Mailgun untuk pengiriman email yang andal, sangat berguna ketika port SMTP diblokir
- **Konfigurasi SMTP Dasar**: Gunakan layanan yang telah ditentukan sebelumnya seperti Gmail dengan konfigurasi minimal
- **Konfigurasi SMTP Lanjutan**: Konfigurasikan layanan SMTP generik atau sesuaikan pengaturan untuk penyedia email apa pun

Untuk instruksi pengaturan terperinci, lihat panduan konfigurasi: [Email Setup](/docs/configuration/authentication/email)
