对话导入功能允许用户导入从其他 AI 聊天应用程序导出的对话。目前，我们支持从 [ChatGPT](https://chatgpt.com/)、[Claude](https://claude.ai/)、[ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code) 以及 LibreChat 本身导入对话。

导入功能位于“Settings” -> “Data Controls”部分。

## 导出您的对话 [#export-your-conversations]

### ChatGPT [#chatgpt]

1. 请按照 [ChatGPT 导出说明](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data) 导出您的对话。
2. 您应该会在电子邮件中收到一个下载归档文件的链接。
3. 下载该归档文件。它应该是一个随机命名的 zip 文件，例如：_d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip_
4. 解压 zip 文件。

### Claude [#claude]

1. 请按照 [Claude export instructions](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data) 中的说明导出您的对话。
2. 您将收到一封包含导出存档下载链接的电子邮件。
3. 下载并解压该归档文件。

## 导入到 LibreChat [#import-into-librechat]

1. 导航至 LibreChat **Settings** → **Data Controls**。
2. 点击 **Import** 按钮，并从解压后的归档文件中选择 `conversations.json` 文件。
3. 很快您就会收到导入完成的通知。
