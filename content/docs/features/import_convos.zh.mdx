Conversations Import 允许您将从其他 AI 聊天应用程序导出的对话导入到 LibreChat 中。支持的来源包括 [ChatGPT](https://chatgpt.com/)、[Claude](https://claude.ai/) 和 [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code)。

<Callout type="info" title="在哪里可以找到它">
在 LibreChat 中，导入功能位于 **Settings** → **Data Controls** 下。
</Callout>

## 导出您的对话 [#export-your-conversations]

首先，从源应用程序导出您的数据。具体步骤因源而异。

<Tabs items={["ChatGPT", "Claude", "ChatbotUI v1"]}>
<Tabs.Tab>

<Steps>
<Step>

**请求导出。** 按照 [ChatGPT 导出说明](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data) 来导出您的对话。

</Step>
<Step>

**下载压缩包。** 您将收到一封包含下载链接的电子邮件。该压缩包是一个随机命名的 zip 文件，例如 `d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip`。

</Step>
<Step>

**解压归档文件**以访问其中的 `conversations.json` 文件。

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

<Steps>
<Step>

**请求您的导出。** 请遵循 [Claude 导出说明](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data) 来导出您的对话。

</Step>
<Step>

**下载归档文件。** 您将收到一封包含导出归档文件下载链接的电子邮件。

</Step>
<Step>

**解压归档文件**以访问其中的 `conversations.json` 文件。

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

从 [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code) 导出您的对话，然后直接导入生成的 JSON 文件。无需进行提取步骤。

</Tabs.Tab>
</Tabs>

## 导入到 LibreChat [#import-into-librechat]

<Steps>
<Step>

**开放数据控制。** 在 LibreChat 中，前往 **设置** → **数据控制**。

</Step>
<Step>

**选择您的文件。** 点击 **Import** 并从您解压的归档文件中选择 `conversations.json` 文件（或 ChatbotUI v1 的导出 JSON 文件）。

</Step>
<Step>

**等待确认。** 导入完成后会出现一条通知。

</Step>
</Steps>
