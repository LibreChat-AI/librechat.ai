Met Projects kunnen gebruikers gerelateerde gesprekken organiseren in benoemde werkruimten. Ze zijn nuttig voor langlopende werkstromen, teams, klanten, lessen of elk onderwerp waarvoor je een gerichte set chats wilt hebben zonder alleen op de zoekfunctie te vertrouwen.

## Wat Projecten doen [#what-projects-do]

- Groepeer gesprekken onder een project
- Start een nieuwe chat direct binnen een project
- Verplaats een bestaand gesprek naar een project of terug naar niet-toegewezen
- Blader door projectchats vanuit de projectwerkruimte
- Zoek en sorteer de projectlijst
- Projecten hernoemen of verwijderen vanuit de zijbalk

Projecten zijn persoonlijk voor de gebruiker die ze aanmaakt. Andere gebruikers zien jouw projectenlijst niet.

## Een project aanmaken [#create-a-project]

Open **Projects** vanuit de zijbalk en kies **New project**. Geef het project een naam, waarna LibreChat de projectwerkruimte opent.

Gebruik vanuit een projectwerkruimte **New chat in project** om een gesprek te starten dat automatisch aan dat project wordt toegewezen.

## Bestaande chats verplaatsen [#move-existing-chats]

Open het menu van een conversatie en kies de projectoptie. Selecteer een project om de chat aan toe te wijzen, of kies **Unassigned** om deze uit een project te verwijderen.

Wanneer je een project opent, toont LibreChat alleen de chats die aan dat project zijn toegewezen. De hoofdlijst met gesprekken kan nog steeds alle chats weergeven via **All projects**.

## Projecten beheren [#manage-projects]

Elk project verschijnt in de zijbalk. Gebruik het projectmenu om:

- Open de projectwerkruimte
- De naam van het project wijzigen
- Verwijder het project

Het verwijderen van een project verwijdert de projecttoewijzing van de bijbehorende chats. De gesprekken zelf worden niet verwijderd.

## Opmerkingen [#notes]

- Project-werkruimtes hebben hun eigen chatlijst en sorteerfuncties.
- Nieuwe chats die vanuit een project worden gestart, behouden die projecttoewijzing.
- Als een project wordt verwijderd terwijl je het bekijkt, brengt LibreChat je terug naar de projectenlijst.
