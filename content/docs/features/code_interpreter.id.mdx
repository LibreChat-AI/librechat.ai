## Pendahuluan [#introduction]

API Code Interpreter LibreChat menyediakan cara yang aman dan bebas hambatan untuk mengeksekusi kode dan mengelola file melalui antarmuka API yang sederhana. Baik Anda menggunakannya melalui Agen LibreChat atau mengintegrasikannya langsung ke dalam aplikasi Anda, API ini menawarkan lingkungan sandbox yang canggih untuk menjalankan kode dalam berbagai bahasa pemrograman.

## Memulai [#getting-started]

1. Deploy layanan [code-interpreter](https://github.com/ClickHouse/code-interpreter) (Docker Compose atau Helm, lihat README repositori tersebut)
2. Arahkan LibreChat ke sana dengan variabel lingkungan `LIBRECHAT_CODE_BASEURL` dan `LIBRECHAT_CODE_API_KEY`
3. Mulai jalankan kode dan buat file secara aman melalui Agen LibreChat atau tombol "Run Code"

## Fitur Utama [#key-features]

### Bahasa yang Didukung [#supported-languages]

Jalankan kode dalam berbagai bahasa pemrograman:
- Python, Node.js (JS/TS), Go, C/C++, Java, PHP, Rust, Fortran, Rscript

### Penanganan File yang Mulus [#seamless-file-handling]

- Unggah file untuk diproses
- Unduh output yang dihasilkan
- Manajemen file yang aman
- Pengorganisasian file berbasis sesi

### Keamanan & Kenyamanan [#security--convenience]

- Lingkungan eksekusi sandbox yang aman
- Mode isolasi yang kuat (NsJail atau microVM melalui libkrun)
- Tidak diperlukan pengaturan lokal untuk pengguna akhir
- Penyimpanan file berbasis sesi yang terisolasi

## Menggunakan API [#using-the-api]

### Di LibreChat [#in-librechat]

API memiliki dukungan kelas satu di LibreChat melalui dua metode utama:

1. **[AI Agents](/docs/features/agents#code-interpreter)**: Aktifkan Code Interpreter dalam konfigurasi agen Anda agar agen tersebut dapat menjalankan kode dan memproses file secara otomatis.

2. **Eksekusi Manual**: Gunakan tombol "Run Code" di dalam blok kode di antarmuka obrolan, seperti yang ditunjukkan di sini:

![Code Interpreter in LibreChat](/images/agents/run_code.png)

### Menyiapkan API key [#set-up-api-key]

- **Penyiapan per-pengguna:** masukkan kunci API Anda di LibreChat saat diminta (menggunakan metode di atas)
- **Pengaturan global:** gunakan variabel lingkungan `LIBRECHAT_CODE_API_KEY` di dalam file .env pada proyek Anda (memberikan akses ke semua pengguna)

### Integrasi API Langsung [#direct-api-integration]

Code Interpreter API dapat diintegrasikan ke dalam aplikasi apa pun menggunakan autentikasi API key yang sederhana:

1. Deploy instans [code-interpreter](https://github.com/ClickHouse/code-interpreter) Anda sendiri dan buat API key
2. Sertakan kunci API dalam permintaan Anda menggunakan header `x-api-key`

### URL dasar self-hosted [#self-hosted-base-url]

Atur variabel lingkungan `LIBRECHAT_CODE_BASEURL` untuk mengarahkan LibreChat ke instans [code-interpreter](https://github.com/ClickHouse/code-interpreter) yang Anda host sendiri, beserta kunci API yang dibuat pada deployment tersebut, yang digunakan dengan cara yang sama seperti di atas.

## Fungsionalitas Inti [#core-functionality]

### Eksekusi Kode [#code-execution]

- Jalankan cuplikan kode dalam bahasa yang didukung
- Menerima output stdout/stderr
- Dapatkan statistik eksekusi (penggunaan memori, waktu CPU)
- Menangani argumen program
- Akses status eksekusi dan hasil

### Operasi File [#file-operations]

- Unggah file input
- Unduh output yang dihasilkan
- Daftar file yang tersedia
- Hapus file yang tidak diperlukan
- Kelola sesi file

### Batasan [#limitations]
- Kode tidak dapat mengakses jaringan
- Hanya 10 file yang dapat dibuat per eksekusi
- Batas sumber daya (RAM per eksekusi, ukuran unggahan file, dan kuota permintaan) bergantung pada bagaimana Anda menyediakan dan mengonfigurasi deployment Anda.

## Kasus Penggunaan [#use-cases]

- **Pengujian Kode**: Uji cuplikan kode dalam berbagai bahasa
- **Pemrosesan File**: Mengubah dan menganalisis file secara terprogram
- **Aplikasi AI**: Jalankan kode yang dihasilkan AI dengan aman
- **Alat Pengembangan**: Membangun lingkungan pengkodean interaktif
- **Logika Objektif**: Memverifikasi logika dan kebenaran kode, meningkatkan model AI

## Open Source & Self-Hosting [#open-source--self-hosting]

Layanan Code Interpreter bersifat sumber terbuka di bawah lisensi Apache 2.0 di [ClickHouse/code-interpreter](https://github.com/ClickHouse/code-interpreter). Menjaga eksekusi kode sebagai layanan terpisah membuat aplikasi inti LibreChat tetap ringan: Anda hanya perlu men-deploy infrastruktur sandbox saat membutuhkannya, dan Anda dapat menskalakannya secara independen dari aplikasi chat.

Layanan ini berjalan sebagai sekumpulan komponen yang dapat diskalakan secara independen (gateway API, worker dalam sandbox, dan server file) serta mendukung mode isolasi yang kuat (NsJail atau microVM melalui libkrun) sehingga Anda dapat menjalankan kode yang tidak tepercaya dengan aman. Lihat README repositori untuk instruksi penyebaran Docker Compose dan Helm.

Code Interpreter API menyediakan cara yang aman dan nyaman untuk mengeksekusi kode serta mengelola file di dalam sandbox yang terisolasi. Baik Anda menggunakannya melalui Agents di LibreChat atau mengintegrasikannya langsung ke dalam aplikasi Anda, API ini menawarkan solusi yang tangguh untuk kebutuhan eksekusi kode.

Untuk spesifikasi teknis terperinci, panduan penerapan, dan referensi API, lihat [repositori code-interpreter](https://github.com/ClickHouse/code-interpreter).
