Le « forking » (bifurcation) crée une nouvelle conversation qui se détache d'un message spécifique dans une discussion existante. La nouvelle discussion copie les messages que vous choisissez, ce qui vous permet d'explorer une direction différente, de tester un prompt différent ou de diviser un long fil de discussion en sujets ciblés sans modifier l'original.

## Comment dupliquer une conversation [#how-to-fork-a-conversation]

<Steps>
<Step>

**Survolez un message** et ouvrez son menu, puis sélectionnez l'icône de bifurcation (fork). Le message à partir duquel vous ouvrez le menu devient le message cible pour la bifurcation.

</Step>
<Step>

**Choisissez une option de fork** pour contrôler quels messages sont copiés (voir ci-dessous).

</Step>
<Step>

**Confirmez la bifurcation.** LibreChat ouvre la nouvelle conversation avec les messages copiés, laissant l'original intact.

</Step>
</Steps>

Le *message cible* est le message à partir duquel vous avez ouvert le menu. Si vous activez **Démarrer la bifurcation ici**, la cible devient alors le dernier message de la conversation, et la bifurcation s'effectue à partir du message sélectionné.

## Options de fork [#fork-options]

Chaque option copie un ensemble différent de messages par rapport à la cible.

### Messages visibles uniquement [#visible-messages-only]

Copie uniquement les messages visibles : le chemin direct vers le message cible, en excluant toutes les branches.

/>

### Inclure les branches associées [#include-related-branches]

Copie le chemin direct vers le message cible ainsi que toutes les branches situées le long de ce chemin.

<ThemeImage
  light="https://github.com/danny-avila/LibreChat/assets/32828263/e633f701-acf5-4878-bdd1-29abacb3e3e7"
  dark="https://github.com/danny-avila/LibreChat/assets/32828263/0c297451-990b-4ab2-9ff2-bc3958ab7129"
  alt="Inclure l'option de fork des branches associées"
/>

### Inclure tout depuis/vers ici [#include-all-tofrom-here]

L'option par défaut. Copie chaque message menant à la cible, y compris les branches voisines, qu'elles soient visibles ou sur le même chemin.

## Paramètres supplémentaires [#additional-settings]

**Start fork here** crée une bifurcation à partir du message sélectionné jusqu'au dernier message de la conversation, en utilisant l'option choisie ci-dessus.

<ThemeImage
  light="https://github.com/danny-avila/LibreChat/assets/32828263/801e50e4-749a-42f3-83bd-a3fc06c6e189"

**Remember** enregistre les options que vous sélectionnez et les applique aux futures ramifications (forks), afin que vous n'ayez pas à les définir à chaque fois.
