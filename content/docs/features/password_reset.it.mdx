## Panoramica [#overview]

Questa funzionalità abilita la possibilità di reimpostare la password tramite email per il tuo server LibreChat. Puoi configurarla per funzionare con vari servizi email, inclusi Mailgun API, Gmail e server di posta personalizzati.

## Caratteristiche principali [#key-features]

- Supporta molteplici servizi email:
  - **Mailgun API** - Consigliato per server che bloccano le porte SMTP
  - **Gmail** e altri servizi SMTP predefiniti
  - **Server di posta personalizzati** con configurazione SMTP avanzata
- Consente opzioni di configurazione di base e avanzate
- Abilita la funzionalità di reimpostazione della password basata su email sicura per il tuo server LibreChat

## Opzioni di configurazione [#setup-options]

- **Configurazione Mailgun**: Utilizza l'API di Mailgun per una consegna affidabile delle email, particolarmente utile quando le porte SMTP sono bloccate
- **Configurazione SMTP di base**: Utilizza servizi predefiniti come Gmail con una configurazione minima
- **Configurazione SMTP avanzata**: Configura servizi SMTP generici o personalizza le impostazioni per qualsiasi provider di posta elettronica

**Per istruzioni dettagliate sulla configurazione, fare riferimento alla guida alla configurazione: [Email Setup](/docs/configuration/authentication/email)**
