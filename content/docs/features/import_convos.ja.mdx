Conversations Importを使用すると、他のAIチャットアプリケーションからエクスポートされた会話をLibreChatに取り込むことができます。サポートされているソースは、[ChatGPT](https://chatgpt.com/)、[Claude](https://claude.ai/)、および[ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code)です。

<Callout type="info" title="場所の確認方法">
インポートは、LibreChatの**設定** → **データコントロール**から行えます。
</Callout>

## 会話のエクスポート [#export-your-conversations]

まず、ソースアプリケーションからデータをエクスポートします。手順はソースによって異なります。

<Tabs items={["ChatGPT", "Claude", "ChatbotUI v1"]}>
<Tabs.Tab>

<Steps>
<Step>

**エクスポートをリクエストします。** [ChatGPTのエクスポート手順](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data) に従って、会話をエクスポートしてください。

</Step>
<Step>

**アーカイブをダウンロードします。** ダウンロードリンクが記載されたメールが届きます。アーカイブは、`d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip` のようなランダムな名前のzipファイルです。

</Step>
<Step>

**アーカイブを解凍**して、中にある `conversations.json` ファイルにアクセスします。

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

<Steps>
<Step>

**エクスポートをリクエストします。** [Claudeのエクスポート手順](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data) に従って、会話をエクスポートしてください。

</Step>
<Step>

**アーカイブをダウンロードします。** エクスポートアーカイブのダウンロードリンクが記載されたメールが届きます。

</Step>
<Step>

**アーカイブを解凍**して、中にある `conversations.json` ファイルにアクセスします。

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

[ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code) から会話をエクスポートし、その結果生成されたJSONファイルを直接インポートしてください。抽出ステップは不要です。

</Tabs.Tab>
</Tabs>

## LibreChat にインポートする [#import-into-librechat]

<Steps>
<Step>

**データ制御を開く。** LibreChatで、**設定** → **データ制御** に移動します。

</Step>
<Step>

**ファイルを選択します。** **Import** をクリックし、展開したアーカイブから `conversations.json` ファイル（または ChatbotUI v1 用のエクスポート済み JSON ファイル）を選択します。

</Step>
<Step>

**確認を待機します。** インポートが完了すると通知が表示されます。

</Step>
</Steps>
