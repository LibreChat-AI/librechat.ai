Conversations Import を使用すると、ユーザーは他のAIチャットアプリケーションからエクスポートされた会話をインポートできます。現在、[ChatGPT](https://chatgpt.com/)、[Claude](https://claude.ai/)、[ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code)、および LibreChat 自体からの会話のインポートをサポートしています。

インポート機能は、「Settings」->「Data Controls」セクションから利用できます。

## 会話のエクスポート [#export-your-conversations]

### ChatGPT [#chatgpt]

1. [ChatGPTのエクスポート手順](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data)に従って、会話をエクスポートしてください。
2. アーカイブをダウンロードするためのリンクがメールで届きます。
3. アーカイブをダウンロードします。これは以下のようなランダムな名前のzipファイルであるはずです：_d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip_
4. zipファイルを解凍します。

### Claude [#claude]

1. [Claudeのエクスポート手順](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data)に従って、会話データをエクスポートしてください。
2. エクスポートアーカイブのダウンロードリンクが記載されたメールが送信されます。
3. アーカイブをダウンロードして解凍します。

## LibreChat にインポートする [#import-into-librechat]

1. LibreChatの**設定** → **データコントロール**に移動します。
2. **Import** ボタンをクリックし、展開したアーカイブから `conversations.json` ファイルを選択します。
3. まもなくインポートが完了したという通知が届きます。
