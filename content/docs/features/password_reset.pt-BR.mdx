## Visão geral [#overview]

Este recurso habilita a funcionalidade de redefinição de senha por e-mail para o seu servidor LibreChat. Você pode configurá-lo para funcionar com vários serviços de e-mail, incluindo a API do Mailgun, Gmail e servidores de e-mail personalizados.

## Principais Recursos [#key-features]

- Suporta múltiplos serviços de e-mail:
  - **Mailgun API** - Recomendado para servidores que bloqueiam portas SMTP
  - **Gmail** e outros serviços SMTP predefinidos
  - **Servidores de e-mail personalizados** com configuração SMTP avançada
- Permite opções de configuração básicas e avançadas
- Habilita a funcionalidade de redefinição de senha baseada em e-mail segura para o seu servidor LibreChat

## Opções de Configuração [#setup-options]

- **Configuração do Mailgun**: Use a API do Mailgun para entrega confiável de e-mails, especialmente útil quando as portas SMTP estão bloqueadas
- **Configuração básica de SMTP**: Use serviços predefinidos como o Gmail com configuração mínima
- **Configuração Avançada de SMTP**: Configure serviços SMTP genéricos ou personalize as configurações para qualquer provedor de e-mail

Para instruções detalhadas de configuração, consulte o guia de configuração: [Email Setup](/docs/configuration/authentication/email)
