Projekte ermöglichen es jedem Benutzer, zusammengehörige Konversationen in benannten Arbeitsbereichen zu organisieren. Sie sind nützlich für langfristige Arbeitsabläufe, Teams, Kunden, Kurse oder jedes Thema, bei dem Sie eine fokussierte Auswahl an Chats wünschen, ohne sich allein auf die Suche verlassen zu müssen.

## Was Projekte bewirken [#what-projects-do]

- Gruppieren Sie Unterhaltungen unter einem Projekt
- Starten Sie einen neuen Chat direkt innerhalb eines Projekts
- Verschieben Sie eine bestehende Konversation in ein Projekt oder zurück zu „Nicht zugewiesen“
- Durchsuchen Sie Projekt-Chats aus dem Projekt-Arbeitsbereich
- Projektliste durchsuchen und sortieren
- Projekte in der Seitenleiste umbenennen oder löschen

Projekte sind persönlich für den Benutzer, der sie erstellt. Andere Benutzer sehen Ihre Projektliste nicht.

## Ein Projekt erstellen [#create-a-project]

Öffnen Sie **Projects** in der Seitenleiste und wählen Sie **New project**. Geben Sie dem Projekt einen Namen, dann öffnet LibreChat den Projekt-Arbeitsbereich.

Verwenden Sie in einem Projektarbeitsbereich **Neuer Chat im Projekt**, um eine Unterhaltung zu starten, die automatisch diesem Projekt zugewiesen wird.

## Bestehende Chats verschieben [#move-existing-chats]

Öffnen Sie das Menü einer Unterhaltung und wählen Sie die Projektoption. Wählen Sie ein Projekt aus, um den Chat zuzuweisen, oder wählen Sie **Unassigned**, um ihn aus einem Projekt zu entfernen.

Wenn Sie ein Projekt öffnen, zeigt LibreChat nur die Chats an, die diesem Projekt zugewiesen sind. Die Haupt-Konversationsliste kann über **Alle Projekte** weiterhin alle Chats anzeigen.

## Projekte verwalten [#manage-projects]

Jedes Projekt erscheint in der Seitenleiste. Verwenden Sie das Projektmenü, um:

- Öffnen Sie den Projekt-Arbeitsbereich
- Projekt umbenennen
- Projekt löschen

Das Löschen eines Projekts entfernt die Projektzuweisung aus dessen Chats. Die Unterhaltungen selbst werden nicht gelöscht.

## Hinweise [#notes]

- Projekt-Arbeitsbereiche verfügen über eine eigene Chat-Liste und Sortierfunktionen.
- Neue Chats, die aus einem Projekt heraus gestartet werden, behalten diese Projektzuweisung bei.
- Wenn ein Projekt gelöscht wird, während Sie es gerade betrachten, leitet LibreChat Sie zurück zur Projektliste.
