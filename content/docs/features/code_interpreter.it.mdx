## Introduzione [#introduction]

L'API Code Interpreter di LibreChat fornisce un modo sicuro e semplice per eseguire codice e gestire file attraverso una semplice interfaccia API. Che tu la stia utilizzando tramite gli Agent di LibreChat o integrandola direttamente nelle tue applicazioni, l'API offre un potente ambiente sandbox per l'esecuzione di codice in molteplici linguaggi di programmazione.

## Introduzione [#getting-started]

1. Distribuisci il servizio [code-interpreter](https://github.com/ClickHouse/code-interpreter) (Docker Compose o Helm, consulta il README del repository)
2. Punta LibreChat verso di esso con le variabili d'ambiente `LIBRECHAT_CODE_BASEURL` e `LIBRECHAT_CODE_API_KEY`
3. Inizia a eseguire codice e a generare file in modo sicuro tramite gli Agents di LibreChat o il pulsante "Run Code"

## Caratteristiche principali [#key-features]

### Lingue supportate [#supported-languages]

Esegui codice in molteplici linguaggi di programmazione:
- Python, Node.js (JS/TS), Go, C/C++, Java, PHP, Rust, Fortran, Rscript

### Gestione fluida dei file [#seamless-file-handling]

- Carica file per l'elaborazione
- Scarica gli output generati
- Gestione sicura dei file
- Organizzazione dei file basata sulla sessione

### Sicurezza e Comodità [#security--convenience]

- Ambiente di esecuzione sandbox sicuro
- Modalità di isolamento avanzate (NsJail o microVM tramite libkrun)
- Nessuna configurazione locale richiesta per gli utenti finali
- Archiviazione file isolata basata su sessione

## Utilizzo dell'API [#using-the-api]

### In LibreChat [#in-librechat]

L'API ha un supporto di primo livello in LibreChat attraverso due metodi principali:

1. **[AI Agents](/docs/features/agents#code-interpreter)**: Abilita Code Interpreter nella configurazione del tuo agente per consentirgli di eseguire codice ed elaborare file automaticamente.

2. **Esecuzione manuale**: Utilizza il pulsante "Run Code" nei blocchi di codice all'interno dell'interfaccia di chat, come mostrato qui:

![Code Interpreter in LibreChat](/images/agents/run_code.png)

### Configura la chiave API [#set-up-api-key]

- **Configurazione per utente:** inserisci la tua chiave API in LibreChat quando richiesto (utilizzando i metodi di cui sopra)
- **Configurazione globale:** utilizza la variabile d'ambiente `LIBRECHAT_CODE_API_KEY` nel file .env del tuo progetto (fornisce l'accesso a tutti gli utenti)

### Integrazione API Diretta [#direct-api-integration]

L'API di Code Interpreter può essere integrata in qualsiasi applicazione utilizzando una semplice autenticazione tramite API key:

1. Distribuisci la tua istanza di [code-interpreter](https://github.com/ClickHouse/code-interpreter) e crea una chiave API
2. Includi la chiave API nelle tue richieste utilizzando l'header `x-api-key`

### URL di base self-hosted [#self-hosted-base-url]

Imposta la variabile d'ambiente `LIBRECHAT_CODE_BASEURL` per puntare LibreChat alla tua istanza self-hosted di [code-interpreter](https://github.com/ClickHouse/code-interpreter), insieme a una chiave API generata su quel deployment, utilizzata nello stesso modo descritto sopra.

## Funzionalità principali [#core-functionality]

### Esecuzione del codice [#code-execution]

- Esegui snippet di codice nei linguaggi supportati
- Ricevi l'output stdout/stderr
- Ottieni statistiche di esecuzione (utilizzo della memoria, tempo CPU)
- Gestione degli argomenti del programma
- Accedi allo stato di esecuzione e ai risultati

### Operazioni sui file [#file-operations]

- Caricamento file di input
- Scarica gli output generati
- Elenca i file disponibili
- Elimina i file non necessari
- Gestisci le sessioni dei file

### Limitazioni [#limitations]
- Il codice non può accedere alla rete
- È possibile generare solo 10 file per esecuzione
- I limiti delle risorse (RAM per esecuzione, dimensione del caricamento file e quote di richiesta) dipendono da come si esegue il provisioning e la configurazione del proprio deployment.

## Casi d'uso [#use-cases]

- **Test del codice**: Testa snippet di codice in più linguaggi
- **File Processing**: Trasforma e analizza i file a livello programmatico
- **Applicazioni AI**: Esegui codice generato dall'AI in modo sicuro
- **Strumenti di sviluppo**: Crea ambienti di programmazione interattivi
- **Logica dell'obiettivo**: Verificare la logica e la correttezza del codice, migliorando i modelli di IA

## Open Source & Self-Hosting [#open-source--self-hosting]

Il servizio Code Interpreter è open source sotto licenza Apache 2.0 su [ClickHouse/code-interpreter](https://github.com/ClickHouse/code-interpreter). Mantenere l'esecuzione del codice come servizio separato mantiene l'applicazione core LibreChat leggera: distribuisci l'infrastruttura sandbox solo quando ne hai bisogno e puoi scalarla indipendentemente dall'applicazione di chat.

Il servizio viene eseguito come un insieme di componenti scalabili in modo indipendente (un API gateway, worker in sandbox e un file server) e supporta modalità di isolamento avanzate (NsJail o una microVM tramite libkrun) in modo da poter eseguire codice non attendibile in sicurezza. Consulta il README del repository per le istruzioni di distribuzione con Docker Compose e Helm.
