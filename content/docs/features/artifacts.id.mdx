- Pembuatan tata letak yang dinamis dan responsif dengan mudah

## Content-Security-Policy

Anda mungkin perlu memperbarui Content-Security-Policy pada server web Anda untuk menyertakan `frame-src 'self' https://*.codesandbox.io` agar dapat memuat aplikasi HTML yang dihasilkan di panel Artifacts. Ini adalah dependensi dari pustaka [sandpack](https://sandpack.codesandbox.io/).

## Self-Hosting the Sandpack Bundler [#self-hosting-the-sandpack-bundler]

Artifacts di LibreChat menggunakan pustaka Sandpack dari CodeSandbox untuk merender kode HTML/JS secara aman. Secara default, LibreChat terhubung ke CDN publik CodeSandbox, yang mungkin juga mengirimkan telemetri untuk penggunaannya.

Untuk privasi yang lebih baik, kepatuhan keamanan, atau lingkungan jaringan yang terisolasi, Anda dapat [melakukan self-host pada bundler.](https://sandpack.codesandbox.io/docs/guides/hosting-the-bundler)

### Mengapa Melakukan Self-Host pada Sandpack Bundler? [#why-self-host-the-sandpack-bundler]

[Self-hosting the bundler](https://sandpack.codesandbox.io/docs/guides/hosting-the-bundler) memberikan beberapa keuntungan:

- **Privasi & Keamanan**: Jaga eksekusi kode tetap berada di dalam infrastruktur Anda sendiri
- **Reliabilitas**: Menghapus ketergantungan pada layanan eksternal
- **Performa**: Kurangi latensi dengan menghosting bundler di jaringan Anda
- **Kepatuhan**: Memenuhi persyaratan penanganan data organisasi
- **Kontrol**: Konfigurasikan dan sesuaikan dengan kebutuhan spesifik Anda

### Konfigurasi [#configuration]

Setelah Anda menyiapkan bundler yang di-host sendiri, cukup konfigurasikan LibreChat untuk menggunakannya dengan mengatur variabel lingkungan:

```bash
# `.env` file
SANDPACK_BUNDLER_URL=http://your-bundler-url
```

### Pertimbangan Umum [#general-considerations]

Self-hosting the bundler memperkenalkan beberapa overhead tambahan:

- **Konfigurasi CORS**: Anda perlu mengelola kebijakan cross-origin resource sharing
- **Manajemen Keamanan**: Anda bertanggung jawab atas keamanan layanan bundler
- **Pemeliharaan**: Pembaruan dan patch perlu diterapkan secara manual
- **Persyaratan Sumber Daya**: Sumber daya server tambahan akan diperlukan untuk menampung bundler

Untuk instruksi terperinci mengenai pengaturan dan pemeliharaan bundler yang di-host sendiri, lihat [repositori CodeSandbox yang di-fork](https://github.com/LibreChat-AI/codesandbox-client) yang disesuaikan untuk deployment LibreChat, termasuk penghapusan telemetri dari Sandpack.
