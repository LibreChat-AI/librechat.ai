## Aperçu [#overview]

Cette fonctionnalité permet la réinitialisation de mot de passe par e-mail pour votre serveur LibreChat. Vous pouvez la configurer pour fonctionner avec divers services de messagerie, notamment l'API Mailgun, Gmail et des serveurs de messagerie personnalisés.

## Fonctionnalités clés [#key-features]

- Prend en charge plusieurs services de messagerie :
  - **Mailgun API** - Recommandé pour les serveurs qui bloquent les ports SMTP
  - **Gmail** et d'autres services SMTP prédéfinis
  - **Serveurs de messagerie personnalisés** avec configuration SMTP avancée
- Permet des options de configuration basiques et avancées
- Active la fonctionnalité de réinitialisation de mot de passe par e-mail sécurisée pour votre serveur LibreChat

## Options de configuration [#setup-options]

- **Configuration de Mailgun** : Utilisez l'API de Mailgun pour une distribution fiable des e-mails, particulièrement utile lorsque les ports SMTP sont bloqués
- **Configuration SMTP de base** : Utilisez des services prédéfinis comme Gmail avec une configuration minimale
- **Configuration SMTP avancée** : Configurez des services SMTP génériques ou personnalisez les paramètres pour n'importe quel fournisseur de messagerie

**Pour des instructions de configuration détaillées, consultez le guide de configuration : [Email Setup](/docs/configuration/authentication/email)**
