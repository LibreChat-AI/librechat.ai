## 概述 [#overview]

此功能为您的 LibreChat 服务器启用了基于电子邮件的密码重置功能。您可以将其配置为与各种电子邮件服务配合使用，包括 Mailgun API、Gmail 和自定义邮件服务器。

## 核心功能 [#key-features]

- 支持多种电子邮件服务：
  - **Mailgun API** - 推荐用于屏蔽了 SMTP 端口的服务器
  - **Gmail** 以及其他预定义的 SMTP 服务
  - **自定义邮件服务器**，支持高级 SMTP 配置
- 允许进行基础和高级配置选项
- 为您的 LibreChat 服务器启用基于电子邮件的安全密码重置功能

## 设置选项 [#setup-options]

- **Mailgun 配置**：使用 Mailgun 的 API 进行可靠的电子邮件投递，在 SMTP 端口被封锁时特别有用。
- **基本 SMTP 配置**：使用 Gmail 等预定义服务，只需极简配置
- **高级 SMTP 配置**：配置通用 SMTP 服务或自定义任何邮件提供商的设置

**有关详细的设置说明，请参阅配置指南：[Email Setup](/docs/configuration/authentication/email)**
