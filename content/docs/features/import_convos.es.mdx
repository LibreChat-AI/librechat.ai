La importación de conversaciones permite a los usuarios importar conversaciones exportadas desde otras aplicaciones de chat de IA. Actualmente, admitimos la importación de conversaciones desde [ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/), [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code) y el propio LibreChat.

La funcionalidad de Import está disponible en la sección "Settings" -> "Data Controls".

## Exportar tus conversaciones [#export-your-conversations]

### ChatGPT [#chatgpt]

1. Sigue las [instrucciones de exportación de ChatGPT](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data) para exportar tus conversaciones.
2. Deberías recibir un enlace para descargar el archivo en tu correo electrónico.
3. Descargue el archivo. Debería ser un archivo zip con un nombre aleatorio como: _d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip_
4. Extraiga el contenido del archivo zip.

### Claude [#claude]

1. Sigue las [instrucciones de exportación de Claude](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data) para exportar tus conversaciones.
2. Recibirás un correo electrónico con un enlace de descarga para tu archivo de exportación.
3. Descargue y extraiga el archivo.

## Importar a LibreChat [#import-into-librechat]

1. Navega a **Settings** → **Data Controls** en LibreChat.
2. Haz clic en el botón **Import** y selecciona el archivo `conversations.json` del archivo extraído.
3. En breve recibirás una notificación de que la importación se ha completado.
