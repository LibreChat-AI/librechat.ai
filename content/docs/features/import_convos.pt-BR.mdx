A Importação de Conversas permite que você traga conversas exportadas de outros aplicativos de chat de IA para o LibreChat. As fontes suportadas são [ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/) e [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code).

<Callout type="info" title="Onde encontrar">
A importação encontra-se em **Settings** → **Data Controls** no LibreChat.
</Callout>

## Exporte suas conversas [#export-your-conversations]

Primeiro, exporte seus dados do aplicativo de origem. As etapas variam de acordo com a origem.

<Tabs items={["ChatGPT", "Claude", "ChatbotUI v1"]}>
<Tabs.Tab>

<Steps>
<Step>

**Solicite sua exportação.** Siga as [instruções de exportação do ChatGPT](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data) para exportar suas conversas.

</Step>
<Step>

**Baixe o arquivo.** Você receberá um e-mail com um link de download. O arquivo é um arquivo zip com um nome aleatório, como `d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip`.

</Step>
<Step>

**Extraia o arquivo** para acessar o arquivo `conversations.json` dentro dele.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

<Steps>
<Step>

**Solicite sua exportação.** Siga as [instruções de exportação do Claude](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data) para exportar suas conversas.

</Step>
<Step>

**Baixe o arquivo.** Você receberá um e-mail com um link de download para o seu arquivo de exportação.

</Step>
<Step>

**Extraia o arquivo** para acessar o arquivo `conversations.json` dentro dele.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

Exporte suas conversas do [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code) e, em seguida, importe o arquivo JSON resultante diretamente. Nenhuma etapa de extração é necessária.

</Tabs.Tab>
</Tabs>

## Importar para o LibreChat [#import-into-librechat]

<Steps>
<Step>

**Controles de Dados Abertos.** No LibreChat, vá para **Configurações** → **Controles de Dados**.

</Step>
<Step>

**Selecione seu arquivo.** Clique em **Import** e escolha o arquivo `conversations.json` do seu arquivo extraído (ou o arquivo JSON exportado do ChatbotUI v1).

</Step>
<Step>

**Aguarde a confirmação.** Uma notificação aparece assim que a importação é concluída.

</Step>
</Steps>
