L'importazione delle conversazioni ti consente di trasferire le conversazioni esportate da altre applicazioni di chat AI in LibreChat. Le fonti supportate sono [ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/) e [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code).

<Callout type="info" title="Dove trovarlo">
L'importazione si trova sotto **Settings** → **Data Controls** in LibreChat.
</Callout>

## Esporta le tue conversazioni [#export-your-conversations]

Per prima cosa, esporta i tuoi dati dall'applicazione di origine. I passaggi variano a seconda della fonte.

<Tabs items={["ChatGPT", "Claude", "ChatbotUI v1"]}>
<Tabs.Tab>

<Steps>
<Step>

**Richiedi la tua esportazione.** Segui le [istruzioni per l'esportazione di ChatGPT](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data) per esportare le tue conversazioni.

</Step>
<Step>

**Scarica l'archivio.** Riceverai un'email con un link per il download. L'archivio è un file zip con un nome casuale come `d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip`.

</Step>
<Step>

**Estrai l'archivio** per accedere al file `conversations.json` al suo interno.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

<Steps>
<Step>

**Richiedi la tua esportazione.** Segui le [istruzioni per l'esportazione di Claude](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data) per esportare le tue conversazioni.

</Step>
<Step>

**Scarica l'archivio.** Riceverai un'email con un link per il download del tuo archivio di esportazione.

</Step>
<Step>

**Estrai l'archivio** per accedere al file `conversations.json` al suo interno.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

Esporta le tue conversazioni da [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code), quindi importa direttamente il file JSON risultante. Non è richiesto alcun passaggio di estrazione.

</Tabs.Tab>
</Tabs>

## Importazione in LibreChat [#import-into-librechat]

<Steps>
<Step>

**Controlli dei dati aperti.** In LibreChat, vai su **Impostazioni** → **Controlli dei dati**.

</Step>
<Step>

**Seleziona il tuo file.** Fai clic su **Import** e scegli il file `conversations.json` dal tuo archivio estratto (o il file JSON esportato per ChatbotUI v1).

</Step>
<Step>

**Attendi la conferma.** Una notifica apparirà una volta terminata l'importazione.

</Step>
</Steps>
