Tijdelijke chats laten je iets vragen zonder het te bewaren. Gebruik ze voor gevoelige onderwerpen, snelle experimenten of alles wat je niet hoeft op te slaan. Een tijdelijke chat blijft uit je geschiedenis-zijbalk, verschijnt nooit in zoekopdrachten, kan niet worden toegevoegd aan bladwijzers en wordt automatisch verwijderd zodra de bewaartermijn is verstreken (standaard 30 dagen).

## Een tijdelijke chat starten [#start-a-temporary-chat]

<Steps>

<Step>

**Open het modelmenu.** Selecteer het model-dropdownmenu linksboven in de chatweergave.

</Step>

<Step>

**Schakel Tijdelijke chat in.** Zet de **Temporary Chat**-schakelaar op de AAN-stand.

## Wat een tijdelijke chat doet [#what-a-temporary-chat-does]

- Verschijnt niet in de zijbalk met chathistorie.
- Wordt uitgesloten van zoekresultaten.
- Kan niet worden toegevoegd aan bladwijzers.
- Wordt opgeslagen in de database totdat de bewaartermijn is verstreken en wordt daarna automatisch verwijderd.

<Callout type="info" title="Bewaartermijn">

De standaard bewaartermijn is 30 dagen. Beheerders kunnen dit wijzigen. Zie [Configuration](#configuration-administrators) hieronder.

</Callout>

## Configuratie (beheerders) [#configuration-administrators]

Temporary Chat is standaard beschikbaar voor gebruikers. Beheerders bepalen of de functie wordt aangeboden en hoe lang tijdelijke chats worden bewaard.

**Beschikbaarheid** wordt beheerd door de `TEMPORARY_CHAT` roltoestemming. Beheer dit voor elke rol vanuit het [Admin Panel](/docs/features/admin_panel). De `interface.temporaryChat` optie in `librechat.yaml` voegt deze toestemming alleen toe voor de standaard `USER` rol bij het opstarten en is verouderd voor toestemmingsbeheer.

**Retention** wordt ingesteld met `interface.temporaryChatRetention` (in uren). Het minimum is 1 uur, het maximum is 8760 (1 jaar) en de standaardwaarde is 720 (30 dagen).

<Tabs items={['librechat.yaml (aanbevolen)', 'Omgevingsvariabele (verouderd)']}>

<Tabs.Tab>

```yaml filename="librechat.yaml"
interface:
  temporaryChat: true
  temporaryChatRetention: 168 # retain temporary chats for 7 days
```

</Tabs.Tab>

<Tabs.Tab>

```bash filename=".env"
# Hours to retain temporary chats (default: 720 = 30 days)
TEMP_CHAT_RETENTION_HOURS=168
```

<Callout type="warning" title="Verouderd">

`TEMP_CHAT_RETENTION_HOURS` is verouderd. Gebruik bij voorkeur `interface.temporaryChatRetention` in `librechat.yaml`, welke voorrang heeft op de omgevingsvariabele.

</Callout>

</Tabs.Tab>

</Tabs>

Veelvoorkomende retentiewaarden:

| Waarde | Periode |
| --- | --- |
| `1` | 1 uur (minimum) |
| `24` | 1 dag |
| `168` | 1 week |
| `720` | 30 dagen (standaard) |
| `8760` | 1 jaar (maximum) |

Voor de volledige set opties, inclusief `retentionMode` en `retainAgentFiles`, zie de [interface reference](/docs/configuration/librechat_yaml/object_structure/interface#temporarychatretention).
