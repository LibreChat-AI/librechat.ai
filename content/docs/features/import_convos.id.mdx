Impor Percakapan memungkinkan Anda untuk membawa percakapan yang diekspor dari aplikasi obrolan AI lain ke dalam LibreChat. Sumber yang didukung adalah [ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/), dan [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code).

<Callout type="info" title="Tempat menemukannya">
Impor berada di bawah **Settings** → **Data Controls** di LibreChat.
</Callout>

## Ekspor percakapan Anda [#export-your-conversations]

Pertama, ekspor data Anda dari aplikasi sumber. Langkah-langkahnya berbeda tergantung pada sumbernya.

<Tabs items={["ChatGPT", "Claude", "ChatbotUI v1"]}>
<Tabs.Tab>

<Steps>
<Step>

**Minta ekspor Anda.** Ikuti [instruksi ekspor ChatGPT](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data) untuk mengekspor percakapan Anda.

</Step>
<Step>

**Unduh arsip tersebut.** Anda akan menerima email berisi tautan unduhan. Arsip tersebut adalah file zip dengan nama acak seperti `d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip`.

</Step>
<Step>

**Ekstrak arsip tersebut** untuk mengakses file `conversations.json` di dalamnya.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

<Steps>
<Step>

**Minta ekspor Anda.** Ikuti [instruksi ekspor Claude](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data) untuk mengekspor percakapan Anda.

</Step>
<Step>

**Unduh arsip.** Anda akan menerima email berisi tautan unduhan untuk arsip ekspor Anda.

</Step>
<Step>

**Ekstrak arsip tersebut** untuk mengakses file `conversations.json` di dalamnya.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

Ekspor percakapan Anda dari [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code), lalu impor file JSON yang dihasilkan secara langsung. Tidak diperlukan langkah ekstraksi.

</Tabs.Tab>
</Tabs>

## Impor ke LibreChat [#import-into-librechat]

<Steps>
<Step>

**Kontrol Data Terbuka.** Di LibreChat, buka **Settings** → **Data Controls**.

</Step>
<Step>

**Pilih file Anda.** Klik **Import** dan pilih file `conversations.json` dari arsip yang telah Anda ekstrak (atau file JSON yang diekspor untuk ChatbotUI v1).

</Step>
<Step>

**Tunggu konfirmasi.** Sebuah notifikasi akan muncul setelah proses impor selesai.

</Step>
</Steps>
