Les Projects permettent à chaque utilisateur d'organiser des conversations associées dans des espaces de travail nommés. Ils sont utiles pour les flux de travail de longue durée, les équipes, les clients, les cours ou tout sujet pour lequel vous souhaitez disposer d'un ensemble ciblé de discussions sans dépendre uniquement de la recherche.

## Ce que font les projets [#what-projects-do]

- Regrouper les conversations sous un projet
- Démarrer une nouvelle discussion directement au sein d'un projet
- Déplacer une conversation existante vers un projet ou la remettre dans les éléments non assignés
- Parcourir les discussions du projet depuis l'espace de travail du projet
- Rechercher et trier la liste des projets
- Renommer ou supprimer des projets depuis la barre latérale

Les Projects sont personnels à l'utilisateur qui les crée. Les autres utilisateurs ne voient pas votre liste de projets.

## Créer un projet [#create-a-project]

Ouvrez **Projects** depuis la barre latérale et choisissez **New project**. Donnez un nom au projet, puis LibreChat ouvrira l'espace de travail du projet.

Depuis un espace de travail de projet, utilisez **New chat in project** pour démarrer une conversation qui est automatiquement assignée à ce projet.

## Déplacer les discussions existantes [#move-existing-chats]

Ouvrez le menu d'une conversation et choisissez l'option de projet. Sélectionnez un projet pour assigner la discussion, ou choisissez **Unassigned** pour la retirer d'un projet.

Lorsque vous ouvrez un projet, LibreChat affiche uniquement les conversations assignées à ce projet. La liste principale des conversations peut toujours afficher toutes les conversations via **Tous les projets**.

## Gérer les projets [#manage-projects]

Chaque projet apparaît dans la barre latérale. Utilisez le menu du projet pour :

- Ouvrir l'espace de travail du projet
- Renommer le projet
- Supprimer le projet

La suppression d'un projet retire l'assignation du projet de ses discussions. Les conversations elles-mêmes ne sont pas supprimées.

## Notes [#notes]

- Les espaces de travail de projet disposent de leur propre liste de discussions et de leurs propres contrôles de tri.
- Les nouvelles discussions démarrées à partir d'un projet conservent cette affectation de projet.
- Si un projet est supprimé pendant que vous le consultez, LibreChat vous renvoie à la liste des projets.
