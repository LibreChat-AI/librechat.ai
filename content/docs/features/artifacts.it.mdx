## Content-Security-Policy

Potrebbe essere necessario aggiornare la Content-Security-Policy del proprio server web per includere `frame-src 'self' https://*.codesandbox.io` al fine di caricare le app HTML generate nel pannello Artifacts. Questa è una dipendenza della libreria [sandpack](https://sandpack.codesandbox.io/).

## Self-Hosting del Sandpack Bundler [#self-hosting-the-sandpack-bundler]

Gli Artifacts in LibreChat utilizzano la libreria Sandpack di CodeSandbox per eseguire il rendering sicuro di codice HTML/JS. Per impostazione predefinita, LibreChat si connette alla CDN pubblica di CodeSandbox, che potrebbe anche trasmettere telemetria relativa al suo utilizzo.

Per una maggiore privacy, conformità alla sicurezza o ambienti di rete isolati, è possibile [ospitare autonomamente il bundler.](https://sandpack.codesandbox.io/docs/guides/hosting-the-bundler)

### Perché ospitare autonomamente il Sandpack Bundler? [#why-self-host-the-sandpack-bundler]

[Self-hosting the bundler](https://sandpack.codesandbox.io/docs/guides/hosting-the-bundler) offre diversi vantaggi:

- **Privacy & Security**: Mantieni l'esecuzione del codice all'interno della tua infrastruttura
- **Affidabilità**: Rimuovere la dipendenza da servizi esterni
- **Performance**: Riduci la latenza ospitando il bundler nella tua rete
- **Conformità**: Soddisfa i requisiti organizzativi di gestione dei dati
- **Controllo**: Configura e personalizza in base alle tue esigenze specifiche

### Configurazione [#configuration]

Una volta configurato un bundler self-hosted, è sufficiente configurare LibreChat per utilizzarlo impostando la variabile d'ambiente:

```bash
# `.env` file
SANDPACK_BUNDLER_URL=http://your-bundler-url
```

### Considerazioni generali [#general-considerations]

L'hosting autonomo del bundler introduce un ulteriore sovraccarico:

- **Configurazione CORS**: Dovrai gestire le policy di cross-origin resource sharing
- **Gestione della sicurezza**: Diventi responsabile della sicurezza del servizio bundler
- **Manutenzione**: Gli aggiornamenti e le patch dovranno essere applicati manualmente
- **Requisiti di sistema**: Saranno necessarie risorse server aggiuntive per ospitare il bundler

Per istruzioni dettagliate sulla configurazione e la manutenzione del tuo bundler self-hosted, fai riferimento al [repository forked di CodeSandbox](https://github.com/LibreChat-AI/codesandbox-client) adattato per il deployment di LibreChat, inclusa la rimozione della telemetria da Sandpack.
