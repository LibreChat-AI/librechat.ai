title: 메시지 검색

description: LibreChat의 통합 Meilisearch를 사용하여 과거 대화를 빠르게 검색하세요
---

LibreChat은 과거 대화 내용을 빠르고 효율적으로 검색할 수 있도록 **Meilisearch**를 통합하여 사용자 경험을 향상시켰습니다. Meilisearch는 속도와 사용 편의성으로 잘 알려진 강력한 오픈 소스 검색 엔진으로, 대량의 데이터에 대한 빠른 액세스가 필요한 LibreChat과 같은 애플리케이션에 탁월한 선택입니다.

![바나나에 대한 대화 검색](https://github.com/danny-avila/LibreChat/assets/32828263/60ad41b0-1869-4ee9-848b-502b3a5557b5)


이 통합을 통해 사용자는 다음을 수행할 수 있습니다:
- 대화 기록에 대해 **전체 텍스트 검색**을 수행합니다.
- 더 관대한 검색 쿼리를 위해 **typo-tolerance**를 활용하세요.
- Meilisearch의 as-you-type 검색 기능 덕분에 입력하는 즉시 **즉각적인 결과**를 경험하세요.

이 기능은 LibreChat의 기능을 크게 향상시켜 사용자가 채팅 기록 내에서 특정 메시지나 주제를 더 쉽게 찾을 수 있도록 합니다. 이는 원활하고 사용자 친화적인 경험을 제공하려는 플랫폼의 노력을 잘 보여줍니다. 기술적인 세부 사항이나 구현에 관심이 있는 분들은 [Meilisearch Configuration Guide](/docs/configuration/meilisearch)에서 자세한 정보를 확인하실 수 있습니다.
