Projetos permitem que cada usuário organize conversas relacionadas em espaços de trabalho nomeados. Eles são úteis para fluxos de trabalho de longa duração, equipes, clientes, aulas ou qualquer tópico onde você deseje um conjunto focado de chats sem depender apenas da pesquisa.

## O que os Projects fazem [#what-projects-do]

- Agrupar conversas em um projeto
- Inicie um novo chat diretamente dentro de um projeto
- Mover uma conversa existente para um projeto ou de volta para não atribuído
- Navegue pelos chats do projeto a partir do workspace do projeto
- Pesquisar e ordenar a lista de projetos
- Renomear ou excluir projetos da barra lateral

Os Projects são pessoais para o usuário que os cria. Outros usuários não veem sua lista de projetos.

## Criar um Projeto [#create-a-project]

Abra **Projects** na barra lateral e escolha **New project**. Dê um nome ao projeto e, em seguida, o LibreChat abrirá o espaço de trabalho do projeto.

A partir de um workspace de projeto, use **New chat in project** para iniciar uma conversa que é automaticamente atribuída a esse projeto.

## Mover Chats Existentes [#move-existing-chats]

Abra o menu de uma conversa e escolha a opção de projeto. Selecione um projeto para atribuir o chat ou escolha **Unassigned** para removê-lo de um projeto.

Ao abrir um projeto, o LibreChat mostra apenas os chats atribuídos a esse projeto. A lista principal de conversas ainda pode exibir todos os chats através de **All projects**.

## Gerenciar Projetos [#manage-projects]

Cada projeto aparece na barra lateral. Use o menu do projeto para:

- Abra o workspace do projeto
- Renomear o projeto
- Excluir o projeto

Excluir um projeto remove a atribuição do projeto de seus chats. As conversas em si não são excluídas.

## Notas [#notes]

- Os workspaces de projeto possuem sua própria lista de chat e controles de ordenação.
- Novos chats iniciados a partir de um projeto mantêm essa atribuição de projeto.
- Se um projeto for excluído enquanto você o estiver visualizando, o LibreChat o levará de volta à lista de projetos.
