L'importation de conversations permet aux utilisateurs d'importer des conversations exportées depuis d'autres applications de chat IA. Actuellement, nous prenons en charge l'importation de conversations depuis [ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/), [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code) et LibreChat lui-même.

La fonctionnalité d'importation est disponible dans la section « Settings » -> « Data Controls ».

## Exporter vos conversations [#export-your-conversations]

### ChatGPT [#chatgpt]

1. Suivez les [instructions d'exportation de ChatGPT](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data) pour exporter vos conversations.
2. Vous devriez recevoir un lien pour télécharger l'archive dans votre e-mail.
3. Téléchargez l'archive. Il devrait s'agir d'un fichier zip avec un nom aléatoire tel que : _d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip_
4. Extrayez le contenu du fichier zip.

### Claude [#claude]

1. Suivez les [instructions d'exportation de Claude](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data) pour exporter vos conversations.
2. Vous recevrez un e-mail contenant un lien de téléchargement pour votre archive d'exportation.
3. Téléchargez et extrayez l'archive.

## Importer dans LibreChat [#import-into-librechat]

1. Accédez à **Settings** → **Data Controls** dans LibreChat.
2. Cliquez sur le bouton **Import** et sélectionnez le fichier `conversations.json` depuis l'archive extraite.
3. Vous recevrez bientôt une notification indiquant que l'importation est terminée.
