L'importation de conversations vous permet d'intégrer dans LibreChat des conversations exportées depuis d'autres applications de chat IA. Les sources prises en charge sont [ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/) et [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code).

<Callout type="info" title="Où le trouver">
L'importation se trouve sous **Settings** → **Data Controls** dans LibreChat.
</Callout>

## Exporter vos conversations [#export-your-conversations]

Tout d'abord, exportez vos données depuis l'application source. Les étapes diffèrent selon la source.

<Tabs items={["ChatGPT", "Claude", "ChatbotUI v1"]}>
<Tabs.Tab>

<Steps>
<Step>

**Demandez votre exportation.** Suivez les [instructions d'exportation de ChatGPT](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data) pour exporter vos conversations.

</Step>
<Step>

**Téléchargez l'archive.** Vous recevrez un e-mail contenant un lien de téléchargement. L'archive est un fichier zip avec un nom aléatoire tel que `d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip`.

</Step>
<Step>

**Extrayez l'archive** pour accéder au fichier `conversations.json` à l'intérieur.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

<Steps>
<Step>

**Demandez votre exportation.** Suivez les [instructions d'exportation de Claude](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data) pour exporter vos conversations.

</Step>
<Step>

**Téléchargez l'archive.** Vous recevrez un e-mail contenant un lien de téléchargement pour votre archive d'exportation.

</Step>
<Step>

**Extrayez l'archive** pour accéder au fichier `conversations.json` à l'intérieur.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

Exportez vos conversations depuis [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code), puis importez directement le fichier JSON résultant. Aucune étape d'extraction n'est requise.

</Tabs.Tab>
</Tabs>

## Importer dans LibreChat [#import-into-librechat]

<Steps>
<Step>

**Contrôles des données ouvertes.** Dans LibreChat, allez dans **Paramètres** → **Contrôles des données**.

</Step>
<Step>

**Sélectionnez votre fichier.** Cliquez sur **Import** et choisissez le fichier `conversations.json` depuis votre archive extraite (ou le fichier JSON exporté pour ChatbotUI v1).

</Step>
<Step>

**Attendez la confirmation.** Une notification apparaît une fois l'importation terminée.

</Step>
</Steps>
