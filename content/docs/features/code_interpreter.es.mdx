## Introducción [#introduction]

La API de Code Interpreter de LibreChat proporciona una forma segura y sencilla de ejecutar código y gestionar archivos a través de una interfaz de API simple. Ya sea que la utilices a través de los Agents de LibreChat o la integres directamente en tus aplicaciones, la API ofrece un potente entorno de sandbox para ejecutar código en múltiples lenguajes de programación.

## Primeros pasos [#getting-started]

1. Visita [code.librechat.ai](https://code.librechat.ai/pricing) para obtener tu API key
2. Integre la API en su aplicación o utilícela a través de LibreChat
3. Comience a ejecutar código y generar archivos de forma segura

## Características clave [#key-features]

### Idiomas compatibles [#supported-languages]

Ejecuta código en múltiples lenguajes de programación:
- Python, Node.js (JS/TS), Go, C/C++, Java, PHP, Rust, Fortran, Rscript

### Manejo fluido de archivos [#seamless-file-handling]

- Subir archivos para su procesamiento
- Descargar resultados generados
- Gestión segura de archivos
- Organización de archivos basada en sesiones

### Seguridad y Conveniencia [#security--convenience]

- Entorno de ejecución aislado y seguro (sandboxed)
- No se requiere configuración local
- No se requiere implementación de servidor
- Sin gestión de configuración

## Uso de la API [#using-the-api]

### En LibreChat [#in-librechat]

La API tiene soporte de primera clase en LibreChat a través de dos métodos principales:

1. **[Agentes de IA](/docs/features/agents#code-interpreter)**: Habilite el Code Interpreter en la configuración de su agente para permitirle ejecutar código y procesar archivos automáticamente.

2. **Ejecución manual**: Utilice el botón "Run Code" en los bloques de código dentro de la interfaz de chat, como se muestra aquí:

![Intérprete de código en LibreChat](/images/agents/run_code.png)

### Configurar la API key [#set-up-api-key]

- **Configuración por usuario:** ingrese su clave de API en LibreChat cuando se le solicite (utilizando los métodos anteriores)
- **Configuración global:** utilice la variable de entorno `LIBRECHAT_CODE_API_KEY` en el archivo .env de su proyecto (proporciona acceso a todos los usuarios)

### Integración directa de API [#direct-api-integration]

La API de Code Interpreter se puede integrar en cualquier aplicación utilizando una autenticación simple mediante clave de API:

1. Obtén tu clave de API en [code.librechat.ai](https://code.librechat.ai/pricing)
2. Incluya la clave de API en sus solicitudes utilizando el encabezado `x-api-key`

### Enterprise [#enterprise]

El plan enterprise requiere el uso de la variable de entorno `LIBRECHAT_CODE_BASEURL` para corresponder con la instancia autohospedada de la API, junto con cualquier clave de API generada desde el panel de control, utilizada de la misma manera que se mencionó anteriormente.

## Funcionalidad principal [#core-functionality]

### Ejecución de código [#code-execution]

- Ejecutar fragmentos de código en lenguajes compatibles
- Recibir salida stdout/stderr
- Obtener estadísticas de ejecución (uso de memoria, tiempo de CPU)
- Gestionar argumentos de programa
- Acceder al estado de ejecución y a los resultados

### Operaciones de archivos [#file-operations]

- Subir archivos de entrada
- Descargar resultados generados
- Listar archivos disponibles
- Eliminar archivos innecesarios
- Gestionar sesiones de archivos

### Limitaciones [#limitations]
- El código no puede acceder a la red
- Solo se pueden generar 10 archivos por ejecución
- Los límites de ejecución varían según el plan:
  - **Hobby**:
    - 256 MB de RAM por ejecución
    - 25 MB por carga de archivo
    - 750 solicitudes por mes
  - **Entusiasta**:
    - 512 MB de RAM por ejecución
    - 50 MB por carga de archivo
    - 3,000 solicitudes por mes
  - **Pro**:
    - 512 MB de RAM por ejecución
    - 150 MB por carga de archivo
    - 7,000 solicitudes por mes
- El [Enterprise Plan](https://code.librechat.ai/pricing) proporciona límites y funciones personalizadas

## ¿Por qué una API de pago? [#why-a-paid-api]

Aunque LibreChat sigue siendo gratuito y de código abierto bajo la licencia MIT, la API de Code Interpreter se ofrece como un servicio de pago por varias razones clave:

1. **Sostenibilidad del proyecto**: Suscribirse a un plan de API proporciona apoyo directo al desarrollo del proyecto, incluso de manera más efectiva que [GitHub Sponsors](https://github.com/sponsors/danny-avila). Su suscripción ayuda a garantizar el crecimiento y la mejora continua de LibreChat.

2. **Consideraciones técnicas**: Incluir capacidades de ejecución de código en el proyecto principal añadiría una complejidad y unos requisitos de hardware significativos que no todos los usuarios necesitan. El servicio API elimina estas preocupaciones mientras mantiene una aplicación principal ligera.

3. **Beneficios del servicio gestionado**:
   - Sin configuración compleja
   - Disponibilidad inmediata
   - Actualizaciones y mantenimiento regulares
   - Soporte profesional
   - Entorno seguro y aislado (sandboxed)

4. **Protección de la Propiedad Intelectual**: La arquitectura del Code Interpreter representa una innovación significativa en tecnología de sandbox segura y escalable. Aunque existen soluciones similares, a menudo carecen de las medidas de seguridad integrales y las características de escalabilidad que hacen que esta implementación sea única. Mantener este componente como una API de código cerrado ayuda a proteger estas innovaciones y garantiza que el servicio mantenga sus altos estándares de seguridad y rendimiento.

Incluso si solo utilizas la ejecución de código ocasionalmente, suscribirte ayuda a apoyar el desarrollo de LibreChat mientras mejora tu experiencia con funciones de nivel profesional. Es una situación en la que todos ganan, manteniendo el proyecto principal gratuito mientras se ofrecen capacidades avanzadas opcionales para quienes las necesiten.
