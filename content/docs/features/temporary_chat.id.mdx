Obrolan sementara memungkinkan Anda menanyakan sesuatu tanpa menyimpannya. Gunakan fitur ini untuk topik sensitif, eksperimen cepat, atau apa pun yang tidak perlu Anda simpan. Obrolan sementara tidak akan muncul di bilah sisi riwayat Anda, tidak pernah muncul dalam pencarian, tidak dapat ditandai (bookmark), dan akan dihapus secara otomatis setelah periode retensinya berakhir (30 hari secara default).

## Memulai Obrolan Sementara [#start-a-temporary-chat]

<Steps>

<Step>

**Buka menu model.** Pilih menu drop-down model di kiri atas tampilan obrolan.

</Step>

<Step>

**Aktifkan Obrolan Sementara.** Alihkan sakelar **Temporary Chat** ke posisi ON.

## Apa yang Dilakukan Obrolan Sementara [#what-a-temporary-chat-does]

- Tidak muncul di bilah sisi riwayat obrolan.
- Dikecualikan dari hasil pencarian.
- Tidak dapat ditandai.
- Disimpan di dalam database hingga periode retensinya berakhir, kemudian dihapus secara otomatis.

<Callout type="info" title="Periode retensi">

Periode retensi default adalah 30 hari. Administrator dapat mengubahnya. Lihat [Configuration](#configuration-administrators) di bawah.

</Callout>

## Konfigurasi (administrator) [#configuration-administrators]

Temporary Chat tersedia bagi pengguna secara default. Administrator mengontrol apakah fitur ini ditawarkan dan berapa lama temporary chat disimpan.

**Ketersediaan** diatur oleh izin peran `TEMPORARY_CHAT`. Kelola izin tersebut untuk setiap peran dari [Admin Panel](/docs/features/admin_panel). Opsi `interface.temporaryChat` di dalam `librechat.yaml` hanya memberikan izin ini untuk peran `USER` bawaan saat startup dan sudah tidak digunakan lagi (deprecated) untuk pengelolaan izin.

**Retention** diatur dengan `interface.temporaryChatRetention` (dalam jam). Minimumnya adalah 1 jam, maksimumnya adalah 8760 (1 tahun), dan default-nya adalah 720 (30 hari).

<Tabs items={['librechat.yaml (disarankan)', 'Variabel lingkungan (tidak digunakan lagi)']}>

<Tabs.Tab>

```yaml filename="librechat.yaml"
interface:
  temporaryChat: true
  temporaryChatRetention: 168 # retain temporary chats for 7 days
```

</Tabs.Tab>

<Tabs.Tab>

```bash filename=".env"
# Hours to retain temporary chats (default: 720 = 30 days)
TEMP_CHAT_RETENTION_HOURS=168
```

<Callout type="warning" title="Tidak digunakan lagi">

`TEMP_CHAT_RETENTION_HOURS` sudah tidak digunakan lagi (deprecated). Gunakan `interface.temporaryChatRetention` di dalam `librechat.yaml`, yang memiliki prioritas lebih tinggi daripada variabel lingkungan tersebut.

</Callout>

</Tabs.Tab>

</Tabs>

Nilai retensi umum:

| Nilai | Periode |
| --- | --- |
| `1` | 1 jam (minimum) |
| `24` | 1 hari |
| `168` | 1 minggu |
| `720` | 30 hari (default) |
| `8760` | 1 tahun (maksimum) |

Untuk rangkaian opsi lengkap, termasuk `retentionMode` dan `retainAgentFiles`, lihat [referensi antarmuka](/docs/configuration/librechat_yaml/object_structure/interface#temporarychatretention).
