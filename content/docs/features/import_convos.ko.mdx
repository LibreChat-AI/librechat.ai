Conversations Import 기능을 사용하면 다른 AI 채팅 애플리케이션에서 내보낸 대화를 LibreChat으로 가져올 수 있습니다. 지원되는 소스는 [ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/), [ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code)입니다.

<Callout type="info" title="찾을 수 있는 곳">
가져오기(Import)는 LibreChat의 **설정(Settings)** → **데이터 제어(Data Controls)** 아래에 있습니다.
</Callout>

## 대화 내보내기 [#export-your-conversations]

먼저, 소스 애플리케이션에서 데이터를 내보내세요. 단계는 소스에 따라 다릅니다.

<Tabs items={["ChatGPT", "Claude", "ChatbotUI v1"]}>
<Tabs.Tab>

<Steps>
<Step>

**내보내기를 요청하세요.** [ChatGPT 내보내기 지침](https://help.openai.com/en/articles/7260999-how-do-i-export-my-chatgpt-history-and-data)에 따라 대화 내용을 내보내세요.

</Step>
<Step>

**아카이브를 다운로드하세요.** 다운로드 링크가 포함된 이메일을 받게 됩니다. 아카이브는 `d119d98bb3711aff7a2c73bcc7ea53d96c984650d8f7e033faef78386a9907-2024-01-01-10-30-00.zip`과 같이 무작위 이름으로 된 zip 파일입니다.

</Step>
<Step>

**아카이브의 압축을 풀어** 내부에 있는 `conversations.json` 파일에 액세스하세요.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

<Steps>
<Step>

**내보내기를 요청하세요.** [Claude 내보내기 지침](https://support.claude.com/en/articles/9450526-how-can-i-export-my-claude-data)에 따라 대화 내용을 내보내세요.

</Step>
<Step>

**아카이브를 다운로드하세요.** 내보낸 아카이브에 대한 다운로드 링크가 포함된 이메일을 받게 됩니다.

</Step>
<Step>

**아카이브의 압축을 풀어** 내부에 있는 `conversations.json` 파일에 액세스하세요.

</Step>
</Steps>

</Tabs.Tab>
<Tabs.Tab>

[ChatbotUI v1](https://github.com/mckaywrigley/chatbot-ui/tree/b865b0555f53957e96727bc0bbb369c9eaecd83b?tab=readme-ov-file#legacy-code)에서 대화 내용을 내보낸 다음, 결과로 생성된 JSON 파일을 직접 가져오세요. 별도의 추출 단계는 필요하지 않습니다.

## LibreChat으로 가져오기 [#import-into-librechat]

<Steps>
<Step>

**데이터 제어 열기.** LibreChat에서 **설정(Settings)** → **데이터 제어(Data Controls)**로 이동하세요.

</Step>
<Step>

**파일을 선택하세요.** **Import**를 클릭하고 압축을 푼 아카이브에서 `conversations.json` 파일(또는 ChatbotUI v1용으로 내보낸 JSON 파일)을 선택하세요.

</Step>
<Step>

**확인을 기다리세요.** 가져오기가 완료되면 알림이 표시됩니다.

</Step>
</Steps>
