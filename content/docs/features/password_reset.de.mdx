## Übersicht [#overview]

Diese Funktion ermöglicht eine passwortbasierte Zurücksetzungsfunktion per E-Mail für Ihren LibreChat-Server. Sie können sie so konfigurieren, dass sie mit verschiedenen E-Mail-Diensten funktioniert, einschließlich Mailgun API, Gmail und benutzerdefinierten Mail-Servern.

## Hauptfunktionen [#key-features]

- Unterstützt mehrere E-Mail-Dienste:
  - **Mailgun API** - Empfohlen für Server, die SMTP-Ports blockieren
  - **Gmail** und andere vordefinierte SMTP-Dienste
  - **Benutzerdefinierte Mailserver** mit erweiterter SMTP-Konfiguration
- Ermöglicht grundlegende und erweiterte Konfigurationsoptionen
- Aktiviert eine sichere, E-Mail-basierte Funktion zum Zurücksetzen des Passworts für Ihren LibreChat-Server

## Einrichtungsoptionen [#setup-options]

- **Mailgun-Konfiguration**: Verwenden Sie die API von Mailgun für eine zuverlässige E-Mail-Zustellung, was besonders nützlich ist, wenn SMTP-Ports blockiert sind.
- **Grundlegende SMTP-Konfiguration**: Verwenden Sie vordefinierte Dienste wie Gmail mit minimaler Konfiguration
- **Erweiterte SMTP-Konfiguration**: Konfigurieren Sie allgemeine SMTP-Dienste oder passen Sie die Einstellungen für einen beliebigen E-Mail-Anbieter an

**Für detaillierte Einrichtungsanweisungen siehe die Konfigurationsanleitung: [Email Setup](/docs/configuration/authentication/email)**
