## はじめに [#introduction]

LibreChatのCode Interpreter APIは、シンプルなAPIインターフェースを通じて、安全かつ手軽にコードを実行し、ファイルを管理する方法を提供します。LibreChatのAgentsを通じて使用する場合でも、アプリケーションに直接統合する場合でも、このAPIは複数のプログラミング言語でコードを実行するための強力なサンドボックス環境を提供します。

## はじめに [#getting-started]

1. [code.librechat.ai](https://code.librechat.ai/pricing) にアクセスして、APIキーを取得してください。
2. APIをアプリケーションに統合するか、LibreChatを通じて使用してください。
3. 安全にコードを実行し、ファイルを生成する

## 主な機能 [#key-features]

### サポートされている言語 [#supported-languages]

複数のプログラミング言語でコードを実行する:
- Python, Node.js (JS/TS), Go, C/C++, Java, PHP, Rust, Fortran, Rscript

### シームレスなファイル処理 [#seamless-file-handling]

- 処理のためにファイルをアップロード
- 生成された出力をダウンロード
- 安全なファイル管理
- セッションベースのファイル整理

### セキュリティと利便性 [#security--convenience]

- 安全なサンドボックス化された実行環境
- ローカルでのセットアップは不要です
- サーバーのデプロイは不要です
- 構成管理なし

## APIの使用 [#using-the-api]

### LibreChatにて [#in-librechat]

LibreChatでは、主に2つの方法を通じてAPIがファーストクラスでサポートされています。

1. **[AI Agents](/docs/features/agents#code-interpreter)**: エージェントの設定で Code Interpreter を有効にすると、コードを実行してファイルを自動的に処理できるようになります。

2. **手動実行**: チャットインターフェース内のコードブロックにある「Run Code」ボタンを使用します（以下を参照）。

![LibreChatでのコードインタープリター](/images/agents/run_code.png)

### APIキーの設定 [#set-up-api-key]

- **ユーザーごとのセットアップ:** プロンプトが表示されたら、LibreChatにAPIキーを入力します（上記の方法を使用）。
- **グローバル設定:** プロジェクトの .env ファイルで `LIBRECHAT_CODE_API_KEY` 環境変数を使用します（全ユーザーへのアクセスを提供します）

### 直接的なAPI統合 [#direct-api-integration]

Code Interpreter APIは、シンプルなAPIキー認証を使用して任意のアプリケーションに統合できます：

1. [code.librechat.ai](https://code.librechat.ai/pricing) からAPIキーを取得してください。
2. `x-api-key` ヘッダーを使用して、リクエストに API キーを含めてください。

### Enterprise [#enterprise]

Enterpriseプランでは、`LIBRECHAT_CODE_BASEURL` 環境変数を使用してセルフホストされたAPIインスタンスと連携させる必要があります。また、ダッシュボードから生成されたAPIキーを、上記で説明した方法と同様に使用してください。

## コア機能 [#core-functionality]

### Code Execution [#code-execution]

- サポートされている言語でコードスニペットを実行する
- stdout/stderr出力を受信する
- 実行統計（メモリ使用量、CPU時間）を取得する
- プログラム引数を処理する
- 実行ステータスと結果へのアクセス

### ファイル操作 [#file-operations]

- 入力ファイルをアップロード
- 生成された出力をダウンロード
- 利用可能なファイルの一覧を表示
- 不要なファイルを削除
- ファイルセッションの管理

### 制限事項 [#limitations]
- コードはネットワークにアクセスできません
- 1回の実行につき最大10ファイルまで生成可能です
- 実行制限はプランによって異なります：
  - **趣味**:
    - 実行ごとに 256 MB の RAM
    - ファイルアップロードあたり 25 MB
    - 月間750リクエスト
  - **愛好家**:
    - 実行ごとに 512 MB の RAM
    - ファイルアップロードあたり 50 MB
    - 月間3,000リクエスト
  - **プロ**:
    - 実行ごとに 512 MB の RAM
    - 1ファイルあたり150 MBのアップロード
    - 月間7,000リクエスト
- [Enterprise Plan](https://code.librechat.ai/pricing) は、カスタム制限と機能を提供します

## ユースケース [#use-cases]

- **コードテスト**: 複数の言語でコードスニペットをテストします
- **File Processing**: プログラムでファイルを変換および分析する
- **AI Applications**: AIが生成したコードを安全に実行
- **Development Tools**: インタラクティブなコーディング環境を構築する
- **Objective Logic**: コードのロジックと正確性を検証し、AIモデルを改善する

## なぜ有料のAPIが必要なのか？ [#why-a-paid-api]

LibreChatはMITライセンスの下で無料かつオープンソースであり続けますが、Code Interpreter APIが有料サービスとして提供されるのには、いくつかの重要な理由があります。

1. **プロジェクトの持続可能性**: APIプランへのサブスクリプションは、[GitHub Sponsors](https://github.com/sponsors/danny-avila)よりもさらに効果的に、プロジェクトの開発を直接支援するものです。皆様のサブスクリプションは、LibreChatの継続的な成長と改善を確実にする助けとなります。

2. **技術的な考慮事項**: コアプロジェクトにコード実行機能を含めると、すべてのユーザーが必要とするわけではない大幅な複雑さとハードウェア要件が追加されることになります。APIサービスは、軽量なコアアプリケーションを維持しながら、これらの懸念を解消します。

3. **Managed Service Benefits**:
   - 複雑な設定は不要
   - 即時利用可能
   - 定期的なアップデートとメンテナンス
   - プロフェッショナルサポート
   - 安全でサンドボックス化された環境

4. **知的財産保護**: Code Interpreterのアーキテクチャは、安全でスケーラブルなサンドボックス技術における重要な革新を表しています。類似のソリューションは存在しますが、それらの多くは、この実装をユニークなものにしている包括的なセキュリティ対策やスケーラビリティ機能を欠いています。このコンポーネントをクローズドソースのAPIとして維持することは、これらの革新を保護し、サービスが高いセキュリティとパフォーマンス基準を維持することを確実にするために役立ちます。

コード実行をたまにしか使用しない場合でも、サブスクリプションに登録することでLibreChatの開発を支援し、プロフェッショナルグレードの機能でエクスペリエンスを向上させることができます。これは、コアプロジェクトを無料で維持しつつ、必要とするユーザーにはオプションの高度な機能を提供するという、双方にとって有益な仕組みです。

## 結論 [#conclusion]

Code Interpreter APIは、実行環境のセットアップや維持の手間をかけることなく、コードの実行やファイルの管理を行うための安全で便利な方法を提供します。LibreChatのAgentsを通じて使用する場合でも、アプリケーションに直接統合する場合でも、コード実行のニーズに対して堅牢なソリューションを提供します。
