<Callout type="info" title="Szukasz instalatora na pulpit?">

LibreChat to samodzielnie hostowana aplikacja internetowa, a nie natywna aplikacja dla systemu Windows czy AppImage dla systemu Linux. W systemie Windows użyj [Docker Desktop](/docs/local/docker), aby uzyskać najprostszą konfigurację lokalną. W przypadku komputerów stacjonarnych lub serwerów z systemem Linux użyj [Docker](/docs/local/docker), [npm](/docs/local/npm) lub [przewodnika po zdalnym hostingu](/docs/remote).

</Callout>
