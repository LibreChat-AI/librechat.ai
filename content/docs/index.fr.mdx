<Callout type="info" title="Vous cherchez un installateur pour ordinateur ?">

LibreChat est une application web auto-hébergée, et non une application Windows native ou une AppImage Linux. Sous Windows, utilisez [Docker Desktop](/docs/local/docker) pour une installation locale simplifiée. Pour les ordinateurs de bureau ou serveurs Linux, utilisez [Docker](/docs/local/docker), [npm](/docs/local/npm), ou un [guide d'hébergement distant](/docs/remote).

</Callout>
