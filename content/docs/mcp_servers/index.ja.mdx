これらのガイドを使用して、適切なトランスポート、OAuthコールバック、スコープ、環境変数、およびLibreChat設定を備えた特定のMCPサーバーを構成してください。

<Cards num={3}>
  <Cards.Card title="Google Workspace MCP" href="/docs/mcp_servers/google_workspace" arrow>
    Google OAuthを使用して、Gmail、Drive、Calendar、People、およびChatの各リモートMCPサーバーを設定します。
  </Cards.Card>
  <Cards.Card title="Salesforce MCP" href="/docs/mcp_servers/salesforce" arrow>
    External Client AppとユーザーごとのOAuthを使用して、Salesforceホスト型のMCPサーバーを構成します。
  </Cards.Card>
  <Cards.Card title="MCPの概要" href="/docs/features/mcp" arrow>
    製品固有のサーバーを追加する前に、LibreChatでMCPがどのように機能するかを学びましょう。
  </Cards.Card>
</Cards>
