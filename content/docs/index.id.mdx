<Callout type="info" title="Mencari penginstal desktop?">

LibreChat adalah aplikasi web yang di-host sendiri (self-hosted), bukan aplikasi Windows asli atau AppImage Linux. Di Windows, gunakan [Docker Desktop](/docs/local/docker) untuk pengaturan lokal yang paling sederhana. Untuk desktop atau server Linux, gunakan [Docker](/docs/local/docker), [npm](/docs/local/npm), atau [panduan hosting jarak jauh](/docs/remote).

</Callout>
