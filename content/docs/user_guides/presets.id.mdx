# Pemberitahuan Penghentian Dukungan [#deprecation-notice]

**Fitur ini sudah tidak digunakan lagi (deprecated). Silakan merujuk ke [Panduan Agen](/docs/features/agents), yang berfungsi sebagai pengganti Presets**

Fitur "presets" di aplikasi kami adalah alat canggih yang memungkinkan pengguna untuk menyimpan dan memuat pengaturan yang telah ditentukan sebelumnya untuk percakapan mereka. Pengguna dapat mengimpor dan mengekspor preset ini sebagai file JSON, menetapkan preset default, dan membagikannya dengan orang lain di Discord.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8c39ad89-71ae-42c6-a792-3db52d539fcd)

## Membuat Preset: [#create-a-preset]

- Buka pengaturan model

![image](https://github.com/danny-avila/LibreChat/assets/32828263/2fc883e9-f4a3-47ac-b375-502e82234194)

- Pilih model, beri nama, tambahkan beberapa instruksi khusus, dan sesuaikan parameter jika diperlukan

![image](https://github.com/danny-avila/LibreChat/assets/32828263/090dc065-f9ea-4a43-9380-e6d504e64992)

- Uji coba

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8a383495-0d5e-4ab7-93a7-eca5388c3f6f)

- Kembali ke pengaturan lanjutan model, dan sesuaikan jika diperlukan. Jika Anda sudah puas dengan hasilnya, klik `Save As Preset` (dari pengaturan lanjutan model).

![image](https://github.com/danny-avila/LibreChat/assets/32828263/96fd88ec-b4b6-4de0-a7d7-f156fdace354)

- Berikan nama yang sesuai, lalu klik simpan

![image](https://github.com/danny-avila/LibreChat/assets/32828263/76ad8db4-a949-4633-8a5f-f9e8358d57f3)

- Sekarang Anda dapat memilihnya dari menu preset!

![image](https://github.com/danny-avila/LibreChat/assets/32828263/81271990-2739-4f5c-b1a5-7d7deeaa385c)

## Penjelasan Parameter: [#parameters-explained]

- **Nama Preset:**
  - Di sinilah Anda memberi nama preset Anda agar mudah dikenali.

- **Endpoint:**
  - Pilih endpoint, seperti OpenAI, yang ingin Anda gunakan untuk memproses percakapan.

- **Model:**
  - Pilih model seperti `gpt-3.5-turbo` yang akan digunakan untuk menghasilkan respons.

- **Nama Kustom:**
  - Secara opsional, berikan nama kustom untuk preset Anda. Ini adalah nama yang akan ditampilkan di UI saat menggunakannya.

- **Instruksi Kustom:**
  - Tentukan instruksi atau panduan yang akan ditampilkan sebelum setiap prompt untuk memandu pengguna dalam memberikan input.

- **Temperature:**
  - Sesuaikan parameter ini untuk mengontrol keacakan output model. Nilai yang lebih tinggi membuat output menjadi lebih acak, sementara nilai yang lebih rendah membuatnya lebih terfokus dan deterministik.

- **Top P:**
  - Kontrol parameter nucleus sampling untuk memengaruhi keragaman teks yang dihasilkan. Nilai yang lebih rendah membuat teks lebih terfokus, sementara nilai yang lebih tinggi meningkatkan keragaman.

- **Frequency Penalty:**
  - Gunakan pengaturan ini untuk memberikan penalti pada token yang sering muncul dan meningkatkan keberagaman dalam respons.

- **Presence Penalty:**
  - Sesuaikan parameter ini untuk memberikan penalti pada token baru yang dimasukkan ke dalam respons, guna mengontrol pengulangan dan meningkatkan konsistensi.

## Mengimpor/Mengekspor Preset [#importingexporting-presets]

Anda dapat dengan mudah mengimpor atau mengekspor preset sebagai file JSON dengan mengeklik tombol 'Import' atau 'Export'. Hal ini memungkinkan Anda untuk membagikan pengaturan yang telah disesuaikan kepada orang lain atau beralih antar konfigurasi yang berbeda dengan cepat.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/b9ef56e2-393e-45eb-b72b-8d568a13a015)

Untuk mengekspor preset, pertama-tama buka menu preset, lalu klik tombol untuk mengedit preset yang dipilih.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/3fb065e6-977b-49b4-9fc6-de55b9839031)

Kemudian di bagian bawah pengaturan preset, Anda akan memiliki opsi untuk mengekspornya.

![Menyimpan dan memilih preset](https://github.com/danny-avila/LibreChat/assets/32828263/a624345f-e3e6-4192-8384-293ba6ce54cc)

## Mengatur Preset Default [#setting-default-preset]

Pilih preset sebagai default agar dimuat secara otomatis setiap kali Anda memulai percakapan baru. Ini menghemat waktu jika Anda sering menggunakan pengaturan tertentu.
