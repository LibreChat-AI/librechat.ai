O LibreChat permite que você configure e integre vários provedores de IA, APIs e suas respectivas credenciais. Isso possibilita a utilização de diferentes modelos de IA, configurações e funcionalidades com base em suas necessidades e requisitos.

## Conceitos-chave [#key-concepts]

- **Endpoints**: Um endpoint refere-se a um provedor de IA, configuração ou API que determina os modelos e configurações disponíveis para uma solicitação de chat. Exemplos incluem OpenAI, Google, Plugins, Anthropic e outros.

- **Presets**: Um preset é uma combinação salva de um endpoint, modelo e configurações de conversa. Você pode criar e gerenciar múltiplos presets para atender a diferentes casos de uso.

- **Default Endpoint**: Se você tiver vários endpoints configurados, você pode especificar um endpoint padrão a ser usado ao criar uma nova conversa.

- **Default Preset**: Da mesma forma, você pode definir um preset padrão para ser usado automaticamente ao iniciar uma nova conversa.

## Funcionalidade [#functionality]

1. **Provedores de IA**: Configure vários provedores de IA pré-configurados fornecendo as credenciais e chaves de API necessárias.

2. **Gerenciar endpoints**: Habilite ou desabilite diferentes endpoints com base em seus requisitos.

3. **Criar e Gerenciar Presets**: Defina e salve combinações específicas de endpoints, modelos e configurações de conversa como presets.

4. **Definir Endpoint e Preset Padrão**: Especifique um endpoint e um preset padrão para otimizar o processo de iniciar novas conversas. Aqui está um vídeo para demonstrar: **[Configurando um Preset Padrão](https://github.com/danny-avila/LibreChat/assets/110412045/bbde830f-18d9-4884-88e5-1bd8f7ac585d)**

5. **Personalizar Configurações**: Explore opções avançadas de configuração, como adicionar endpoints personalizados como Ollama, Mistral AI ou OpenRouter.
