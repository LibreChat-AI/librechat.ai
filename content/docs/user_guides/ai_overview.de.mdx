LibreChat ermöglicht es Ihnen, verschiedene KI-Anbieter, APIs und die dazugehörigen Anmeldedaten zu konfigurieren und zu integrieren. Dies ermöglicht es Ihnen, je nach Bedarf und Anforderungen unterschiedliche KI-Modelle, Einstellungen und Funktionalitäten zu nutzen.

## Grundlegende Konzepte [#key-concepts]

- **Endpoints**: Ein Endpoint bezieht sich auf einen KI-Anbieter, eine Konfiguration oder eine API, die die verfügbaren Modelle und Einstellungen für eine Chat-Anfrage bestimmt. Beispiele hierfür sind OpenAI, Google, Plugins, Anthropic und andere.

- **Presets**: Ein Preset ist eine gespeicherte Kombination aus einem endpoint, einem Modell und Konversationseinstellungen. Sie können mehrere Presets erstellen und verwalten, um sie an verschiedene Anwendungsfälle anzupassen.

- **Standard-Endpoint**: Wenn Sie mehrere Endpoints konfiguriert haben, können Sie einen Standard-Endpoint festlegen, der beim Erstellen einer neuen Konversation verwendet werden soll.

- **Standard-Voreinstellung**: Ebenso können Sie eine Standard-Voreinstellung festlegen, die automatisch beim Starten einer neuen Konversation verwendet wird.

## Funktionalität [#functionality]

1. **AI-Anbieter**: Richten Sie verschiedene vorkonfigurierte AI-Anbieter ein, indem Sie die erforderlichen Anmeldedaten und API-Schlüssel bereitstellen.

2. **Endpoints verwalten**: Aktivieren oder deaktivieren Sie verschiedene Endpoints basierend auf Ihren Anforderungen.

3. **Presets erstellen und verwalten**: Definieren und speichern Sie spezifische Kombinationen aus endpoints, Modellen und Konversationseinstellungen als Presets.

4. **Standard-endpoint und -Voreinstellung festlegen**: Geben Sie einen Standard-endpoint und eine Voreinstellung an, um den Prozess zum Starten neuer Konversationen zu optimieren. Hier ist ein Video zur Demonstration: **[Festlegen einer Standard-Voreinstellung](https://github.com/danny-avila/LibreChat/assets/110412045/bbde830f-18d9-4884-88e5-1bd8f7ac585d)**

5. **Konfigurationen anpassen**: Entdecken Sie erweiterte Konfigurationsoptionen, wie das Hinzufügen benutzerdefinierter endpoints wie Ollama, Mistral AI oder OpenRouter.
