# Aviso de obsolescencia [#deprecation-notice]

**Esta función está obsoleta. Por favor, consulte la [Guía de Agentes](/docs/features/agents), que actúa como sucesora de los Presets.**

La función "presets" en nuestra aplicación es una herramienta potente que permite a los usuarios guardar y cargar configuraciones predefinidas para sus conversaciones. Los usuarios pueden importar y exportar estos presets como archivos JSON, establecer un preset predeterminado y compartirlos con otros en Discord.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8c39ad89-71ae-42c6-a792-3db52d539fcd)

## Explicación de los parámetros: [#parameters-explained]

- **Nombre del ajuste preestablecido:**
  - Aquí es donde nombras tu preset para una fácil identificación.

- **Endpoint:**
  - Elija el endpoint, como OpenAI, que desea utilizar para procesar la conversación.

- **Modelo:**
  - Seleccione el modelo como `gpt-3.5-turbo` que se utilizará para generar respuestas.

- **Nombre personalizado:**
  - Opcionalmente, proporcione un nombre personalizado para su preset. Este es el nombre que se mostrará en la UI al utilizarlo.

- **Instrucciones personalizadas:**
  - Defina instrucciones o directrices que se mostrarán antes de cada prompt para guiar al usuario al proporcionar información.

- **Temperatura:**
  - Ajuste este parámetro para controlar la aleatoriedad de la salida del modelo. Un valor más alto hace que la salida sea más aleatoria, mientras que un valor más bajo la hace más enfocada y determinista.

- **Top P:**
  - Controle el parámetro de muestreo de núcleo (nucleus sampling) para influir en la diversidad del texto generado. Los valores más bajos hacen que el texto sea más enfocado, mientras que los valores más altos aumentan la diversidad.

- **Penalización de frecuencia:**
  - Utilice este ajuste para penalizar los tokens que aparecen con frecuencia y promover la diversidad en las respuestas.

- **Penalización de presencia:**
  - Ajuste este parámetro para penalizar los nuevos tokens que se introducen en las respuestas, controlando la repetición y promoviendo la consistencia.

## Importar/Exportar Presets [#importingexporting-presets]

Puede importar o exportar ajustes preestablecidos fácilmente como archivos JSON haciendo clic en los botones 'Import' o 'Export' respectivamente. Esto le permite compartir sus configuraciones personalizadas con otros o cambiar entre diferentes configuraciones rápidamente.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/b9ef56e2-393e-45eb-b72b-8d568a13a015)

Para exportar un preset, primero ve al menú de presets y luego haz clic en el botón para editar el preset seleccionado.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/3fb065e6-977b-49b4-9fc6-de55b9839031)

Luego, en la parte inferior de la configuración del ajuste preestablecido, tendrás la opción de exportarlo.

    <img src="https://github.com/danny-avila/LibreChat/assets/32828263/a624345f-e3e6-4192-8384-293ba6ce54cc" width="50%"/>
</p>

## Configuración del Preset predeterminado [#setting-default-preset]

Elige un preset como predeterminado para que se cargue automáticamente cada vez que inicies una nueva conversación. Esto ahorra tiempo si utilizas configuraciones específicas con frecuencia.
