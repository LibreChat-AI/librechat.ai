# 弃用通知 [#deprecation-notice]

**此功能已弃用。请参阅 [Agents Guide](/docs/features/agents)，它将作为 Presets 的后续替代方案。**

我们应用中的“presets”功能是一个强大的工具，允许用户保存和加载对话的预定义设置。用户可以将这些 presets 导入和导出为 JSON 文件，设置默认 preset，并与 Discord 上的其他人分享。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8c39ad89-71ae-42c6-a792-3db52d539fcd)

## 创建预设 (Preset): [#create-a-preset]

- 进入模型设置

![image](https://github.com/danny-avila/LibreChat/assets/32828263/2fc883e9-f4a3-47ac-b375-502e82234194)

- 选择模型，为其命名，添加一些自定义指令，并根据需要调整参数

![image](https://github.com/danny-avila/LibreChat/assets/32828263/090dc065-f9ea-4a43-9380-e6d504e64992)

- 测试它

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8a383495-0d5e-4ab7-93a7-eca5388c3f6f)

- 回到模型高级设置中，根据需要进行调整。当你对结果满意后，点击 `Save As Preset`（在模型高级设置中）。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/96fd88ec-b4b6-4de0-a7d7-f156fdace354)

- 为其起一个合适的名称，然后点击保存

![image](https://github.com/danny-avila/LibreChat/assets/32828263/76ad8db4-a949-4633-8a5f-f9e8358d57f3)

- 现在你可以从预设菜单中选择它了！

![image](https://github.com/danny-avila/LibreChat/assets/32828263/81271990-2739-4f5c-b1a5-7d7deeaa385c)

## 参数说明： [#parameters-explained]

- **预设名称：**
  - 在此处为您的预设命名，以便于识别。

- **Endpoint:**
  - 选择您想要用于处理对话的 endpoint，例如 openAI。

- **模型：**
  - 选择用于生成响应的模型，例如 `gpt-3.5-turbo`。

- **自定义名称：**
  - 您可以选择为您的预设提供一个自定义名称。这就是在 UI 中使用该预设时显示的名称。

- **自定义指令：**
  - 定义将在每个提示词之前显示的说明或指南，以引导用户提供输入。

- **温度 (Temperature):**
  - 调整此参数以控制模型输出的随机性。较高的值会使输出更具随机性，而较低的值则会使输出更加集中和确定。

- **Top P：**
  - 控制核采样（nucleus sampling）参数以影响生成文本的多样性。较低的值会使文本更聚焦，而较高的值则会增加多样性。

- **频率惩罚 (Frequency Penalty):**
  - 使用此设置可以惩罚频繁出现的 token，并提升响应的多样性。

- **Presence Penalty:**
  - 调整此参数以惩罚引入到响应中的新 token，从而控制重复并提高一致性。

## 导入/导出 Presets [#importingexporting-presets]

您可以通过点击“Import”或“Export”按钮，轻松地将预设导入或导出为 JSON 文件。这使您可以与他人分享您的自定义设置，或在不同的配置之间快速切换。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/b9ef56e2-393e-45eb-b72b-8d568a13a015)

要导出预设，请先进入预设菜单，然后点击按钮以编辑所选预设。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/3fb065e6-977b-49b4-9fc6-de55b9839031)

然后在预设设置的底部，你将看到导出它的选项。

![保存并选择预设](https://github.com/danny-avila/LibreChat/assets/32828263/a624345f-e3e6-4192-8384-293ba6ce54cc)

## 设置默认预设 [#setting-default-preset]

选择一个预设作为默认值，这样每次开始新对话时它都会自动加载。如果您经常使用特定的设置，这可以节省时间。
