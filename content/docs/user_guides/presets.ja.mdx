# 非推奨のお知らせ [#deprecation-notice]

**この機能は非推奨となりました。Presetsの後継である[Agents Guide](/docs/features/agents)を参照してください。**

当アプリの「presets」機能は、会話の定義済み設定を保存および読み込みできる強力なツールです。ユーザーはこれらのpresetsをJSONファイルとしてインポート・エクスポートしたり、デフォルトのpresetを設定したり、Discord上で他のユーザーと共有したりすることができます。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8c39ad89-71ae-42c6-a792-3db52d539fcd)

## プリセットの作成: [#create-a-preset]

- モデル設定に移動します

![image](https://github.com/danny-avila/LibreChat/assets/32828263/2fc883e9-f4a3-47ac-b375-502e82234194)

- モデルを選択し、名前を付け、カスタム指示を入力し、必要に応じてパラメータを調整します。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/090dc065-f9ea-4a43-9380-e6d504e64992)

- テストする

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8a383495-0d5e-4ab7-93a7-eca5388c3f6f)

- モデルの詳細設定に戻り、必要に応じて調整してください。結果に満足したら、（モデルの詳細設定から）`Save As Preset` をクリックします。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/96fd88ec-b4b6-4de0-a7d7-f156fdace354)

- 適切な名前を付け、「保存」をクリックします。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/76ad8db4-a949-4633-8a5f-f9e8358d57f3)

- これでプリセットメニューから選択できるようになりました！

![image](https://github.com/danny-avila/LibreChat/assets/32828263/81271990-2739-4f5c-b1a5-7d7deeaa385c)

## パラメータの解説: [#parameters-explained]

- **プリセット名:**
  - これは、識別しやすいようにプリセットに名前を付ける場所です。

- **Endpoint:**
  - 会話の処理に使用したいendpoint（OpenAIなど）を選択します。

- **モデル:**
  - 応答の生成に使用する `gpt-3.5-turbo` のようなモデルを選択します。

- **カスタム名:**
  - 必要に応じて、プリセットにカスタム名を設定できます。これは、使用時にUIに表示される名前となります。

- **カスタム指示:**
  - 各プロンプトの前に表示され、ユーザーの入力をガイドするための指示やガイドラインを定義します。

- **Temperature:**
  - このパラメータを調整して、モデルの出力のランダム性を制御します。値を高くすると出力はよりランダムになり、値を低くするとより焦点が絞られ、決定論的になります。

- **Top P:**
  - nucleus samplingパラメータを制御して、生成されるテキストの多様性に影響を与えます。値を低くするとテキストはより焦点が絞られ、値を高くすると多様性が増します。

- **Frequency Penalty:**
  - この設定を使用して、頻繁に出現するトークンにペナルティを課し、応答の多様性を促進します。

- **Presence Penalty:**
  - このパラメータを調整することで、応答に導入される新しいトークンにペナルティを課し、繰り返しを制御して一貫性を促進します。

## プリセットのインポート/エクスポート [#importingexporting-presets]

'Import' または 'Export' ボタンをクリックすることで、プリセットをJSONファイルとして簡単にインポートまたはエクスポートできます。これにより、カスタマイズした設定を他のユーザーと共有したり、異なる構成間を素早く切り替えたりすることが可能になります。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/b9ef56e2-393e-45eb-b72b-8d568a13a015)

プリセットをエクスポートするには、まずプリセットメニューに移動し、選択したプリセットを編集するためのボタンをクリックします。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/3fb065e6-977b-49b4-9fc6-de55b9839031)

次に、プリセット設定の下部にエクスポートするオプションが表示されます。

![プリセットの保存と選択](https://github.com/danny-avila/LibreChat/assets/32828263/a624345f-e3e6-4192-8384-293ba6ce54cc)

## デフォルトプリセットの設定 [#setting-default-preset]

プリセットをデフォルトとして選択すると、新しい会話を開始するたびに自動的に読み込まれるようになります。特定の構成を頻繁に使用する場合、時間を節約できます。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/5912650d-49b6-40d3-b9ad-ff2ff7fbe3e7)
![image](https://github.com/danny-avila/LibreChat/assets/32828263/dcfb5e27-f60b-419e-b387-25db85fa6a63)
