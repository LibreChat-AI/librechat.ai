LibreChat 允许您配置和集成各种 AI 提供商、API 及其相应的凭据。这使您能够根据自己的需求和要求使用不同的 AI 模型、设置和功能。

## 关键概念 [#key-concepts]

- **Endpoints**: Endpoint 指的是 AI 提供商、配置或 API，它决定了聊天请求中可用的模型和设置。例如 OpenAI、Google、Plugins、Anthropic 等。

- **Presets**: 预设（Preset）是端点（endpoint）、模型和对话设置的保存组合。您可以创建并管理多个预设，以适应不同的使用场景。

- **默认 endpoint**：如果您配置了多个 endpoint，则可以指定一个在创建新对话时使用的默认 endpoint。

- **默认预设 (Default Preset)**：同样地，您可以设置一个默认预设，以便在开始新对话时自动使用。

## 功能 [#functionality]

1. **AI Providers**: 通过提供必要的凭据和 API 密钥，设置各种预配置的 AI 提供商。

2. **管理 Endpoints**：根据您的需求启用或禁用不同的 Endpoints。

3. **创建和管理预设 (Presets)**：定义并保存 endpoint、模型和对话设置的特定组合作为预设。

4. **设置默认 endpoint 和预设 (Preset)**：指定一个默认的 endpoint 和预设，以简化开始新对话的过程。以下是演示视频：**[设置默认预设](https://github.com/danny-avila/LibreChat/assets/110412045/bbde830f-18d9-4884-88e5-1bd8f7ac585d)**

5. **自定义配置**：探索高级配置选项，例如添加自定义 endpoint，如 Ollama、Mistral AI 或 OpenRouter。
