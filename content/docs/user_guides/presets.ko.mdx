# Deprecation Notice [#deprecation-notice]

**이 기능은 더 이상 사용되지 않습니다. Presets의 후속 기능인 [Agents Guide](/docs/features/agents)를 참조하시기 바랍니다.**

저희 앱의 "presets" 기능은 사용자가 대화에 대한 사전 정의된 설정을 저장하고 불러올 수 있게 해주는 강력한 도구입니다. 사용자는 이러한 presets를 JSON 파일로 가져오거나 내보낼 수 있으며, 기본 preset을 설정하고 Discord에서 다른 사람들과 공유할 수도 있습니다.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8c39ad89-71ae-42c6-a792-3db52d539fcd)

## 매개변수 설명: [#parameters-explained]

- **프리셋 이름:**
  - 여기에 프리셋의 이름을 지정하여 쉽게 식별할 수 있습니다.

- **Endpoint:**
  - 대화 처리에 사용할 OpenAI와 같은 endpoint를 선택하세요.

- **모델:**
  - 응답 생성에 사용할 `gpt-3.5-turbo`와 같은 모델을 선택하세요.

- **사용자 지정 이름:**
  - 선택적으로 프리셋에 사용자 지정 이름을 제공할 수 있습니다. 이 이름은 사용할 때 UI에 표시되는 이름입니다.

- **사용자 지정 지침(Custom Instructions):**
  - 각 프롬프트 이전에 표시되어 사용자가 입력을 제공할 때 지침이 될 수 있는 지시사항이나 가이드라인을 정의하세요.

- **Temperature:**
  - 모델 출력의 무작위성을 제어하려면 이 매개변수를 조정하세요. 값이 높을수록 출력이 더 무작위해지며, 값이 낮을수록 더 집중되고 결정론적인 출력이 생성됩니다.

- **Top P:**
  - 생성된 텍스트의 다양성에 영향을 미치도록 nucleus sampling 파라미터를 제어하세요. 값이 낮을수록 텍스트가 더 집중되고, 값이 높을수록 다양성이 증가합니다.

- **Frequency Penalty:**
  - 이 설정을 사용하여 자주 발생하는 토큰에 페널티를 부여하고 응답의 다양성을 높이세요.

- **Presence Penalty:**
  - 응답에 도입되는 새로운 토큰에 페널티를 부여하여 반복을 제어하고 일관성을 높이려면 이 매개변수를 조정하세요.

## 프리셋 가져오기/내보내기 (Importing/Exporting Presets) [#importingexporting-presets]

'Import' 또는 'Export' 버튼을 클릭하여 프리셋을 JSON 파일로 쉽게 가져오거나 내보낼 수 있습니다. 이를 통해 사용자 지정 설정을 다른 사람과 공유하거나 다양한 구성 간에 빠르게 전환할 수 있습니다.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/b9ef56e2-393e-45eb-b72b-8d568a13a015)

프리셋을 내보내려면 먼저 프리셋 메뉴로 이동한 다음, 선택한 프리셋을 편집하는 버튼을 클릭하세요.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/3fb065e6-977b-49b4-9fc6-de55b9839031)

그런 다음 프리셋 설정 하단에서 이를 내보낼 수 있는 옵션을 확인할 수 있습니다.

![프리셋 저장 및 선택](https://github.com/danny-avila/LibreChat/assets/32828263/a624345f-e3e6-4192-8384-293ba6ce54cc)

## Discord에서 공유하기 [#sharing-on-discord]

[discord](https://discord.librechat.ai)에서 저희와 함께하세요. 전 세계 사용자들이 수천 개의 프리셋을 공유하는 **[#presets ](https://discord.com/channels/1086345563026489514/1093249324797935746)** 채널을 확인해 보세요. 인기 있는 프리셋은 고정된 게시물에서 확인하실 수 있습니다!
