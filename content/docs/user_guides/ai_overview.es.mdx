LibreChat le permite configurar e integrar varios proveedores de IA, APIs y sus credenciales correspondientes. Esto le permite utilizar diferentes modelos de IA, configuraciones y funcionalidades según sus necesidades y requisitos.

## Conceptos clave [#key-concepts]

- **Endpoints**: Un endpoint se refiere a un proveedor de IA, configuración o API que determina los modelos y ajustes disponibles para una solicitud de chat. Algunos ejemplos incluyen OpenAI, Google, Plugins, Anthropic y otros.

- **Presets**: Un preset es una combinación guardada de un endpoint, modelo y ajustes de conversación. Puedes crear y gestionar múltiples presets para adaptarlos a diferentes casos de uso.

- **Default Endpoint**: Si tienes múltiples endpoints configurados, puedes especificar un endpoint predeterminado que se utilizará al crear una nueva conversación.

- **Default Preset**: De manera similar, puedes establecer un preset predeterminado para que se utilice automáticamente al iniciar una nueva conversación.

## Funcionalidad [#functionality]

1. **Proveedores de IA**: Configure varios proveedores de IA preconfigurados proporcionando las credenciales y claves API necesarias.

2. **Gestionar endpoints**: Habilite o deshabilite diferentes endpoints según sus requisitos.

3. **Crear y gestionar Presets**: Define y guarda combinaciones específicas de endpoints, modelos y configuraciones de conversación como presets.

4. **Establecer endpoint y preset predeterminados**: Especifique un endpoint y un preset predeterminados para agilizar el proceso de iniciar nuevas conversaciones. Aquí hay un video para demostrarlo: **[Configuración de un preset predeterminado](https://github.com/danny-avila/LibreChat/assets/110412045/bbde830f-18d9-4884-88e5-1bd8f7ac585d)**

5. **Personalizar configuraciones**: Explore opciones de configuración avanzadas, como añadir endpoints personalizados como Ollama, Mistral AI o OpenRouter.
