LibreChat memungkinkan Anda untuk mengonfigurasi dan mengintegrasikan berbagai penyedia AI, API, serta kredensial terkait. Hal ini memungkinkan Anda untuk memanfaatkan berbagai model AI, pengaturan, dan fungsionalitas berdasarkan kebutuhan dan persyaratan Anda.

## Konsep Utama [#key-concepts]

- **Endpoints**: Endpoint merujuk pada penyedia AI, konfigurasi, atau API yang menentukan model dan pengaturan yang tersedia untuk permintaan obrolan. Contohnya termasuk OpenAI, Google, Plugins, Anthropic, dan lainnya.

- **Presets**: Preset adalah kombinasi tersimpan dari endpoint, model, dan pengaturan percakapan. Anda dapat membuat dan mengelola beberapa preset untuk menyesuaikan dengan berbagai kasus penggunaan.

- **Default Endpoint**: Jika Anda memiliki beberapa endpoint yang dikonfigurasi, Anda dapat menentukan endpoint default yang akan digunakan saat membuat percakapan baru.

- **Default Preset**: Demikian pula, Anda dapat mengatur preset default agar digunakan secara otomatis saat memulai percakapan baru.

## Fungsionalitas [#functionality]

1. **Penyedia AI**: Siapkan berbagai penyedia AI yang telah dikonfigurasi sebelumnya dengan memberikan kredensial dan kunci API yang diperlukan.

2. **Kelola endpoint**: Aktifkan atau nonaktifkan berbagai endpoint berdasarkan kebutuhan Anda.

3. **Membuat dan Mengelola Preset**: Tentukan dan simpan kombinasi spesifik dari endpoint, model, dan pengaturan percakapan sebagai preset.

4. **Atur Endpoint dan Preset Default**: Tentukan endpoint dan preset default untuk menyederhanakan proses memulai percakapan baru. Berikut adalah video untuk mendemonstrasikannya: **[Mengatur Preset Default](https://github.com/danny-avila/LibreChat/assets/110412045/bbde830f-18d9-4884-88e5-1bd8f7ac585d)**

5. **Sesuaikan Konfigurasi**: Jelajahi opsi konfigurasi tingkat lanjut, seperti menambahkan endpoint kustom seperti Ollama, Mistral AI, atau OpenRouter.
