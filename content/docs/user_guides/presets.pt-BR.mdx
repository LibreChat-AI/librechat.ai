# Aviso de Descontinuação [#deprecation-notice]

**Este recurso está obsoleto. Consulte o [Guia de Agentes](/docs/features/agents), que atua como um sucessor para os Presets.**

O recurso de "presets" em nosso aplicativo é uma ferramenta poderosa que permite aos usuários salvar e carregar configurações predefinidas para suas conversas. Os usuários podem importar e exportar esses presets como arquivos JSON, definir um preset padrão e compartilhá-los com outras pessoas no Discord.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8c39ad89-71ae-42c6-a792-3db52d539fcd)

## Criar um Preset: [#create-a-preset]

- Vá nas configurações do modelo

![image](https://github.com/danny-avila/LibreChat/assets/32828263/2fc883e9-f4a3-47ac-b375-502e82234194)

- Escolha o modelo, dê a ele um nome, algumas instruções personalizadas e ajuste os parâmetros, se necessário

![image](https://github.com/danny-avila/LibreChat/assets/32828263/090dc065-f9ea-4a43-9380-e6d504e64992)

- Teste-o

![image](https://github.com/danny-avila/LibreChat/assets/32828263/8a383495-0d5e-4ab7-93a7-eca5388c3f6f)

- Volte nas configurações avançadas do modelo e ajuste-as, se necessário. Quando estiver satisfeito com o resultado, clique em `Save As Preset` (nas configurações avançadas do modelo).

![image](https://github.com/danny-avila/LibreChat/assets/32828263/96fd88ec-b4b6-4de0-a7d7-f156fdace354)

- Dê a ele um nome apropriado e clique em salvar

![image](https://github.com/danny-avila/LibreChat/assets/32828263/76ad8db4-a949-4633-8a5f-f9e8358d57f3)

- Agora você pode selecioná-lo no menu de predefinições!

![image](https://github.com/danny-avila/LibreChat/assets/32828263/81271990-2739-4f5c-b1a5-7d7deeaa385c)

## Parâmetros Explicados: [#parameters-explained]

- **Nome do Preset:**
  - É aqui que você nomeia seu preset para fácil identificação.

- **Endpoint:**
  - Escolha o endpoint, como o OpenAI, que você deseja usar para processar a conversa.

- **Modelo:**
  - Selecione o modelo como `gpt-3.5-turbo` que será usado para gerar respostas.

- **Nome Personalizado:**
  - Opcionalmente, forneça um nome personalizado para o seu preset. Este é o nome que será exibido na UI ao utilizá-lo.

- **Instruções Personalizadas:**
  - Defina instruções ou diretrizes que serão exibidas antes de cada prompt para orientar o usuário no fornecimento de entrada.

- **Temperatura:**
  - Ajuste este parâmetro para controlar a aleatoriedade da saída do modelo. Um valor mais alto torna a saída mais aleatória, enquanto um valor mais baixo a torna mais focada e determinística.

- **Top P:**
  - Controle o parâmetro de amostragem de núcleo (nucleus sampling) para influenciar a diversidade do texto gerado. Valores mais baixos tornam o texto mais focado, enquanto valores mais altos aumentam a diversidade.

- **Penalidade de Frequência:**
  - Use esta configuração para penalizar tokens que ocorrem com frequência e promover a diversidade nas respostas.

- **Presence Penalty:**
  - Ajuste este parâmetro para penalizar novos tokens que são introduzidos nas respostas, controlando a repetição e promovendo a consistência.

## Importando/Exportando Presets [#importingexporting-presets]

Você pode importar ou exportar presets facilmente como arquivos JSON clicando nos botões 'Import' ou 'Export', respectivamente. Isso permite que você compartilhe suas configurações personalizadas com outras pessoas ou alterne entre diferentes configurações rapidamente.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/b9ef56e2-393e-45eb-b72b-8d568a13a015)

Para exportar um preset, primeiro vá ao menu de presets e, em seguida, clique no botão para editar o preset selecionado.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/3fb065e6-977b-49b4-9fc6-de55b9839031)

Então, na parte inferior das configurações de predefinição, você terá a opção de exportá-la.

![Salvando e selecionando um preset](https://github.com/danny-avila/LibreChat/assets/32828263/a624345f-e3e6-4192-8384-293ba6ce54cc)

## Definindo o Preset Padrão [#setting-default-preset]

Escolha um preset como padrão para que ele carregue automaticamente sempre que você iniciar uma nova conversa. Isso economiza tempo se você costuma usar configurações específicas.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/5912650d-49b6-40d3-b9ad-ff2ff7fbe3e7)
