  ```tsx copy filename="carousel"
<Carousel autoplay animationDuration="1500" showBullets perView="3">
  <img src="slide1.jpg" alt="Slide 1" />
  <img src="slide2.jpg" alt="Slide 2" />
  <img src="slide3.jpg" alt="Slide 3" />
</Carousel>
  ```

此示例演示了如何使用带有以下 props 的 `Carousel` 组件：

- `autoplay`：启用轮播图的自动播放功能。
- `animationDuration="1500"`：将动画持续时间设置为 1500 毫秒。
- `showControls`: 显示轮播图的上一个和下一个控件。
- `showBullets`: 显示轮播图的圆点导航。
- `perView="3"`：在轮播图中一次显示三张幻灯片。

在 `Carousel` 组件内部，您可以放置任何想要作为幻灯片显示的内容。在此示例中，我们使用了带有指向 `slide1.jpg`、`slide2.jpg` 和 `slide3.jpg` 源路径的 `<img>` 元素。

## 文件树 [#file-tree]

<FileTree>
  <FileTree.Folder name="src" defaultOpen>
    <FileTree.File name="main.js" active />
    <FileTree.Folder name="components">
      <FileTree.File name="Header.js" />
      <FileTree.File name="Footer.js" />
    </FileTree.Folder>
  </FileTree.Folder>
  <FileTree.File name="package.json" />
</FileTree>

</Tabs.Tab>

<Tabs.Tab>

- `<FileTree.Folder>`：表示一个文件夹。它可以包含嵌套的 Folder 或 File 组件。
- `<FileTree.File>`：表示一个文件。此组件仅需要一个 name 属性，该属性决定了 UI 中显示的文件名。
- `name`: 这决定了在 UI 中显示的文件或文件夹名称。
- `defaultOpen`: 这显示了文件夹的初始状态。如果设置为 true，则文件夹在首次渲染时将处于展开状态。
- `open`：如果将此 prop 传递给 Folder，无论用户如何交互，它都将始终保持打开状态。当你希望保持文件树的某些部分处于展开状态时，这非常有用。
- `active`：当在 File 中使用此 prop 时，文件名将会被高亮显示。当你想要向用户展示当前处于活动状态或被选中的文件时，这非常有用。

```tsx copy filename="file-tree.tsx"
<FileTree>
  <FileTree.Folder name="src" defaultOpen>
    <FileTree.File name="main.js" active />
    <FileTree.Folder name="components">
      <FileTree.File name="Header.js" />
      <FileTree.File name="Footer.js" />
    </FileTree.Folder>
  </FileTree.Folder>
  <FileTree.File name="package.json" />
</FileTree>
```
