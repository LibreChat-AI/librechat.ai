Diese Dokumentation verwendet [Shiki](https://shiki.matsu.io) für Syntax-Highlighting zur Build-Zeit.

## Grundlegende Verwendung [#basic-usage]

Fügen Sie Codeblöcke unter Verwendung von dreifachen Backticks mit einer Sprachkennung hinzu:

````md
```js
console.log('hello, world')
```
````

Wird gerendert als:

```js
console.log('hello, world')
```

## Unterstützte Sprachen [#supported-languages]

Alle von Shiki unterstützten Sprachen sind verfügbar. Zu den gängigen gehören:

- `js` / `javascript`
- `ts` / `typescript`
- `jsx` / `tsx`
- `bash` / `sh`
- `json`
- `yaml`
- `python`
- `go`
- `rust`
- `css`
- `html`
- `markdown` / `md`
- `sql`
- `dockerfile`

## Dateinamen [#filenames]

Sie können Codeblöcken einen Dateinamen hinzufügen:

````md
```js filename="example.js"
console.log('hello')
```
````

## Zeilenhervorhebung [#line-highlighting]

Bestimmte Zeilen hervorheben:

````md
```js {1,3-4}
import React from 'react'

function Hello() {
  return <div>Hello</div>
}
```
````
