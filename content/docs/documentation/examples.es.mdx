  ```tsx copy filename="carousel"
<Carousel autoplay animationDuration="1500" showBullets perView="3">
  <img src="slide1.jpg" alt="Slide 1" />
  <img src="slide2.jpg" alt="Slide 2" />
  <img src="slide3.jpg" alt="Slide 3" />
</Carousel>
  ```

Este ejemplo demuestra cómo utilizar el componente `Carousel` con las siguientes props:

- `autoplay`: Habilita la reproducción automática para el carrusel.
- `animationDuration="1500"`: Establece la duración de la animación en 1500 milisegundos.
- `showControls`: Muestra los controles anterior y siguiente para el carrusel.
- `showBullets`: Muestra la navegación por viñetas para el carrusel.
- `perView="3"`: Muestra tres diapositivas a la vez en el carrusel.

Dentro del componente `Carousel`, puedes colocar cualquier contenido que desees mostrar como diapositivas. En este ejemplo, estamos utilizando elementos `<img>` con rutas de origen a `slide1.jpg`, `slide2.jpg` y `slide3.jpg`.

## Árbol de archivos [#file-tree]

<Tabs items={['Ejemplo','Código']}>
&nbsp;

<Tabs.Tab>

<FileTree>
  <FileTree.Folder name="src" defaultOpen>
    <FileTree.File name="main.js" active />
    <FileTree.Folder name="components">
      <FileTree.File name="Header.js" />
      <FileTree.File name="Footer.js" />
    </FileTree.Folder>
  </FileTree.Folder>
  <FileTree.File name="package.json" />
</FileTree>

</Tabs.Tab>

<Tabs.Tab>

- `<FileTree.Folder>`: Representa una carpeta. Puede contener componentes Folder o File anidados.
- `<FileTree.File>`: Representa un archivo. Este componente solo requiere una prop name, la cual determina el nombre del archivo que se muestra en la interfaz de usuario.
- `name`: Esto determina el nombre del archivo o carpeta tal como se muestra en la interfaz de usuario.
- `defaultOpen`: Esto muestra el estado inicial de una Folder. Si se establece en true, la Folder se expandirá cuando se renderice por primera vez.
- `open`: Si se pasa esta prop a un Folder, siempre estará abierto independientemente de la interacción del usuario. Esto es útil cuando deseas mantener ciertas secciones del árbol de archivos expuestas.
- `active`: Cuando se utiliza esta prop en un File, el nombre del archivo se resaltará. Esto es útil cuando deseas mostrar a los usuarios qué archivo está actualmente activo o seleccionado.

```tsx copy filename="file-tree.tsx"
<FileTree>
  <FileTree.Folder name="src" defaultOpen>
    <FileTree.File name="main.js" active />
    <FileTree.Folder name="components">
      <FileTree.File name="Header.js" />
      <FileTree.File name="Footer.js" />
    </FileTree.Folder>
  </FileTree.Folder>
  <FileTree.File name="package.json" />
</FileTree>
```

import { Feature, Features } from '@/components/features'

  </Feature>
