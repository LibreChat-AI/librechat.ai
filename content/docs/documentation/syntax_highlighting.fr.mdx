Cette documentation utilise [Shiki](https://shiki.matsu.io) pour la coloration syntaxique au moment de la compilation.

## Utilisation de base [#basic-usage]

Ajoutez des blocs de code en utilisant trois backticks avec un identifiant de langage :

````md
```js
console.log('hello, world')
```
````

S'affiche comme :

```js
console.log('hello, world')
```

## Langues prises en charge [#supported-languages]

Toutes les langues prises en charge par Shiki sont disponibles. Les plus courantes incluent :

- `js` / `javascript`
- `ts` / `typescript`
- `jsx` / `tsx`
- `bash` / `sh`
- `json`
- `yaml`
- `python`
- `go`
- `rust`
- `css`
- `html`
- `markdown` / `md`
- `sql`
- `dockerfile`

## Noms de fichiers [#filenames]

Vous pouvez ajouter un nom de fichier aux blocs de code :

````md
```js filename="example.js"
console.log('hello')
```
````
