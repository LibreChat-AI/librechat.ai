Esta documentación utiliza [Shiki](https://shiki.matsu.io) para el resaltado de sintaxis en tiempo de compilación.

## Uso básico [#basic-usage]

Agregue bloques de código usando tres comillas invertidas con un identificador de lenguaje:

````md
```js
console.log('hello, world')
```
````

Se renderiza como:

```js
console.log('hello, world')
```

## Idiomas compatibles [#supported-languages]

Todos los lenguajes soportados por Shiki están disponibles. Los más comunes incluyen:

- `js` / `javascript`
- `ts` / `typescript`
- `jsx` / `tsx`
- `bash` / `sh`
- `json`
- `yaml`
- `python`
- ir
- `rust`
- `css`
- `html`
- `markdown` / `md`
- `sql`
- `dockerfile`

## Nombres de archivo [#filenames]

Puedes añadir un nombre de archivo a los bloques de código:

````md
```js filename="example.js"
console.log('hello')
```
````

## Resaltado de líneas [#line-highlighting]

Resaltar líneas específicas:

````md
```js {1,3-4}
import React from 'react'

function Hello() {
  return <div>Hello</div>
}
```
````
