このドキュメントでは、ビルド時のシンタックスハイライトに [Shiki](https://shiki.matsu.io) を使用しています。

## 基本的な使用方法 [#basic-usage]

言語識別子を付けた3つのバッククォートを使用してコードブロックを追加します：

````md
```js
console.log('hello, world')
```
````

次のようにレンダリングされます:

```js
console.log('hello, world')
```

## サポートされている言語 [#supported-languages]

Shikiでサポートされているすべての言語が利用可能です。一般的な言語は以下の通りです：

- `js` / `javascript`
- `ts` / `typescript`
- `jsx` / `tsx`
- `bash` / `sh`
- `json`
- yaml
- `python`
- `go`
- `rust`
- `css`
- `html`
- `markdown` / `md`
- `sql`
- `dockerfile`

## ファイル名 [#filenames]

コードブロックにファイル名を追加できます：

````md
```js filename="example.js"
console.log('hello')
```
````

## 行のハイライト [#line-highlighting]

特定の行をハイライトする:

````md
```js {1,3-4}
import React from 'react'

function Hello() {
  return <div>Hello</div>
}
```
````
