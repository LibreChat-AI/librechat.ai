本指南使用 [Shiki](https://shiki.matsu.io) 在构建时进行语法高亮显示。

## 基本用法 [#basic-usage]

使用三个反引号并加上语言标识符来添加代码块：

````md
```js
console.log('hello, world')
```
````

渲染为：

```js
console.log('hello, world')
```

## 支持的语言 [#supported-languages]

支持 Shiki 的所有语言均可用。常见的包括：

- `js` / `javascript`
- `ts` / `typescript`
- `jsx` / `tsx`
- `bash` / `sh`
- `json`
- `yaml`
- `python`
- `go`
- `rust`
- `css`
- `html`
- `markdown` / `md`
- `sql`
- `dockerfile`

## 文件名 [#filenames]

您可以为代码块添加文件名：

````md
```js filename="example.js"
console.log('hello')
```
````

## 代码高亮 [#line-highlighting]

高亮特定行：

````md
```js {1,3-4}
import React from 'react'

function Hello() {
  return <div>Hello</div>
}
```
````
