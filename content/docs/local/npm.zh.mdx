对于大多数场景，由于 Docker Compose 简单、易用且可靠，它是推荐的安装方式。如果您更喜欢使用 npm，可以按照这些说明进行操作。

## 先决条件 [#prerequisites]

- Node.js `v24.16.0`: [https://nodejs.org/en/download](https://nodejs.org/en/download)
- npm `v11.16.0`
  - LibreChat 使用 CommonJS (CJS) 和 openid-client v6；Node 24 满足了所需的 CJS/ESM 互操作性、WebCrypto 和 Fetch API 运行时支持。
- Git: https://git-scm.com/download/
- MongoDB (Atlas 或 Community Server)
  - [MongoDB Atlas](/docs/configuration/mongodb/mongodb_atlas)
  - [MongoDB Community Server](/docs/configuration/mongodb/mongodb_community)

如果您使用 `nvm`，请安装并选择推荐的 Node.js 版本，然后更新 npm：

```bash filename="Use Node.js 24 and npm 11"
nvm install 24.16.0
nvm use 24.16.0
npm install -g npm@11.16.0
node -v
npm -v
```

在安装 LibreChat 依赖项之前，你应该看到 Node.js 的版本为 `v24.16.0`，npm 的版本为 `11.16.0`。

## 安装步骤 [#installation-steps]

### 准备工作 [#preparation]

在终端中运行以下命令：

```bash filename="Clone the Repository"
git clone https://github.com/danny-avila/LibreChat.git
```

```bash filename="Navigate to the LibreChat Directory"
cd LibreChat
```

```bash filename="Create a .env File from .env.example"
cp .env.example .env
```

> **注意：** **如果您使用的是 Windows 10，您可能需要使用 `copy` 而不是 `cp`。**

```bash filename="Update the MONGO_URI"
Important: Edit the newly created `.env` file to update the `MONGO_URI` with your own MongoDB instance URI.
```

<Callout title="更新 MONGO_URI" emoji="❗">
  编辑新创建的 `.env` 文件，以使用您自己的 `MONGO_URI` 进行更新
</Callout>

### 构建并启动 [#build-and-start]

完成准备步骤后，请运行以下命令：

```bash filename="Install dependencies"
npm run reinstall
```

`npm run reinstall` 执行干净的依赖安装并构建 LibreChat。在更改 Node.js 或 npm 版本后使用此命令，以便针对当前运行环境重新构建原生包。

```bash filename="Start LibreChat!"
npm run backend
```

<Callout type="success" title="访问 LibreChat！" emoji="🎉">
  **访问 [http://localhost:3080/](http://localhost:3080/)**
</Callout>

<Callout type="example" title="提示" emoji="🔥">
  - 下次您想启动 LibreChat 时，只需执行 `npm run backend`
</Callout>

## 更新 LibreChat [#update-librechat]

要将 LibreChat 更新至最新版本，请运行以下命令：

<Callout type="warning" emoji="">
  首先，停止 LibreChat（如果你还没有停止的话）。
</Callout>

```bash filename="Pull latest project changes"
git pull
```

```bash filename="Update dependencies"
npm run smart-reinstall
```

如果您在更新期间更改了 Node.js 或 npm 版本，请改为运行 `npm run reinstall`。

```bash filename="Start LibreChat!"
npm run backend
```
