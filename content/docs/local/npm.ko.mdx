대부분의 시나리오에서 Docker Compose는 단순성, 사용 편의성 및 신뢰성으로 인해 권장되는 설치 방법입니다. npm을 사용하는 것을 선호하신다면, 다음 지침을 따를 수 있습니다.

## 필수 조건 [#prerequisites]

- Node.js `v24.16.0`: [https://nodejs.org/en/download](https://nodejs.org/en/download)
- npm `v11.16.0`
  - LibreChat은 CommonJS(CJS)와 openid-client v6를 사용합니다. Node 24는 필요한 CJS/ESM 상호 운용성, WebCrypto 및 Fetch API 런타임 지원을 충족합니다.
- Git: https://git-scm.com/download/
- MongoDB (Atlas 또는 Community Server)
  - [MongoDB Atlas](/docs/configuration/mongodb/mongodb_atlas)
  - [MongoDB Community Server](/docs/configuration/mongodb/mongodb_community)

`nvm`을 사용하는 경우, 권장되는 Node.js 버전을 설치 및 선택한 다음 npm을 업데이트하세요:

```bash filename="Use Node.js 24 and npm 11"
nvm install 24.16.0
nvm use 24.16.0
npm install -g npm@11.16.0
node -v
npm -v
```

LibreChat 의존성을 설치하기 전에 Node.js는 `v24.16.0`, npm은 `11.16.0` 버전이어야 합니다.

## 설치 단계 [#installation-steps]

### 준비 사항 [#preparation]

터미널에서 다음 명령어를 실행하세요:

```bash filename="Clone the Repository"
git clone https://github.com/danny-avila/LibreChat.git
```

```bash filename="Navigate to the LibreChat Directory"
cd LibreChat
```

```bash filename="Create a .env File from .env.example"
cp .env.example .env
```

> **참고:** **Windows 10을 사용하는 경우 `cp` 대신 `copy`를 사용해야 할 수도 있습니다.**

```bash filename="Update the MONGO_URI"
Important: Edit the newly created `.env` file to update the `MONGO_URI` with your own MongoDB instance URI.
```

<Callout title="MONGO_URI 업데이트" emoji="❗">
  새로 생성된 `.env` 파일을 편집하여 `MONGO_URI`를 본인의 것으로 업데이트하세요.
</Callout>

### 빌드 및 시작 [#build-and-start]

준비 단계를 완료했다면, 다음 명령어를 실행하세요:

```bash filename="Install dependencies"
npm run reinstall
```

`npm run reinstall`은 의존성을 새로 설치하고 LibreChat을 빌드합니다. Node.js 또는 npm 버전을 변경한 후, 네이티브 패키지가 활성 런타임에 맞춰 다시 빌드되도록 이 명령어를 사용하세요.

```bash filename="Start LibreChat!"
npm run backend
```

<Callout type="success" title="LibreChat에 액세스하세요!" emoji="🎉">
  **[http://localhost:3080/](http://localhost:3080/)**를 방문하세요
</Callout>

<Callout type="example" title="팁" emoji="🔥">
  - 다음에 LibreChat을 시작하고 싶을 때는 `npm run backend`만 실행하면 됩니다.
</Callout>

## LibreChat 업데이트 [#update-librechat]

LibreChat을 최신 버전으로 업데이트하려면 다음 명령어를 실행하세요:

<Callout type="warning" emoji="">
  먼저, LibreChat을 중지하세요(아직 중지하지 않은 경우).
</Callout>

```bash filename="Pull latest project changes"
git pull
```

```bash filename="Update dependencies"
npm run smart-reinstall
```

업데이트 중에 Node.js 또는 npm 버전을 변경했다면, 대신 `npm run reinstall`을 실행하세요.

```bash filename="Start LibreChat!"
npm run backend
```
