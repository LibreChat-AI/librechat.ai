Ini adalah versi ringkas dari [Panduan Instalasi Lokal](/docs/local/) kami

## Langkah 1. Unduh Proyek [#step-1-download-the-project]

### Unduhan Manual [#manual-download]

1. **Buka Halaman Proyek**: Kunjungi [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat).

2. **Unduh File ZIP**: Klik tombol hijau "Code", lalu klik "Download ZIP."

3. **Ekstrak File ZIP**: Temukan file ZIP yang telah diunduh, klik kanan, lalu pilih "Extract All...".

### Menggunakan Git [#using-git]

Jalankan perintah [git](https://git-scm.com/) berikut di terminal Anda, dari direktori induk yang diinginkan:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Langkah 2. Install Docker [#step-2-install-docker]

1. **Download**: Buka [Halaman Unduh Docker Desktop](https://www.docker.com/products/docker-desktop) dan unduh Docker Desktop.
2. **Install**: Buka penginstal dan ikuti petunjuknya.
3. **Jalankan**: Buka Docker Desktop untuk memastikan aplikasi tersebut sedang berjalan.

**Catatan:**
- Docker Desktop direkomendasikan untuk sebagian besar pengguna. Jika Anda mencari pengaturan docker/container tingkat lanjut, terutama untuk instalasi server jarak jauh, lihat [Panduan Deployment Docker Ubuntu](/docs/remote/docker_linux) kami.
- Anda mungkin perlu memulai ulang komputer Anda setelah instalasi.

## Langkah 3. Jalankan Aplikasi [#step-3-run-the-app]

1. **Arahkan ke Direktori Proyek**

2. **Buat dan Konfigurasi File .env**:
   - Salin isi dari `.env.example` ke file baru bernama `.env`.
   - Isi nilai-nilai yang diperlukan.
      - Untuk konfigurasi lingkungan yang mendalam, lihat [.env File Configuration Guide](/docs/configuration/dotenv).

3. **Jalankan Aplikasi**:
   - Jalankan perintah berikut:

   ```bash
   docker compose up -d
   ```

## Kesimpulan [#conclusion]

**Selesai!** Anda sekarang seharusnya sudah memiliki **LibreChat** yang berjalan secara lokal di mesin Anda. Selamat menikmati!
