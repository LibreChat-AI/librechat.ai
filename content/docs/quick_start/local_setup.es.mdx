Esta es una versión condensada de nuestra [Guía de instalación local](/docs/local/)

## Paso 1. Descargar el proyecto [#step-1-download-the-project]

### Descarga manual [#manual-download]

1. **Ve a la página del proyecto**: Visita [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat).

2. **Descargar el archivo ZIP**: Haz clic en el botón verde "Code" y luego en "Download ZIP."

3. **Extraer el archivo ZIP**: Localice el archivo ZIP descargado, haga clic derecho y seleccione "Extraer todo...".

### Uso de Git [#using-git]

Ejecute el siguiente comando de [git](https://git-scm.com/) en su terminal, desde el directorio principal deseado:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Paso 2. Instalar Docker [#step-2-install-docker]

1. **Descarga**: Ve a la [Docker Desktop Download Page](https://www.docker.com/products/docker-desktop) y descarga Docker Desktop.
2. **Instalación**: Abra el instalador y siga las instrucciones.
3. **Ejecutar**: Abra Docker Desktop para asegurarse de que se esté ejecutando.

**Notas:**
- Docker Desktop es recomendado para la mayoría de los usuarios. Si buscas una configuración avanzada de docker/contenedores, especialmente para una instalación en servidor remoto, consulta nuestra [Guía de despliegue de Docker en Ubuntu](/docs/remote/docker_linux).
- Es posible que necesites reiniciar tu computadora después de la instalación.

## Paso 3. Ejecutar la aplicación [#step-3-run-the-app]

1. **Navega al directorio del proyecto**

2. **Crear y configurar el archivo .env**:
   - Copia el contenido de `.env.example` a un nuevo archivo llamado `.env`.
   - Complete cualquier valor necesario.
      - Para una configuración detallada del entorno, consulta la [.env File Configuration Guide](/docs/configuration/dotenv).

3. **Iniciar la aplicación**:
   - Ejecute el siguiente comando:

   ```bash
   docker compose up -d
   ```

## Conclusión [#conclusion]

**¡Eso es todo!** Ahora deberías tener **LibreChat** ejecutándose localmente en tu máquina. ¡Disfrútalo!
