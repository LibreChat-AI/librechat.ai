Esta é uma versão condensada do nosso [Guia de Instalação Local](/docs/local/)

## Passo 1. Baixe o Projeto [#step-1-download-the-project]

### Download Manual [#manual-download]

1. **Vá para a Página do Projeto**: Visite [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat).

2. **Baixe o arquivo ZIP**: Clique no botão verde "Code" e, em seguida, clique em "Download ZIP."

3. **Extraia o arquivo ZIP**: Localize o arquivo ZIP baixado, clique com o botão direito e selecione "Extrair tudo...".

### Usando o Git [#using-git]

Execute o seguinte comando [git](https://git-scm.com/) no seu terminal, a partir do diretório pai desejado:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Passo 2. Instalar o Docker [#step-2-install-docker]

1. **Download**: Vá para a [Docker Desktop Download Page](https://www.docker.com/products/docker-desktop) e baixe o Docker Desktop.
2. **Instalação**: Abra o instalador e siga as instruções.
3. **Executar**: Abra o Docker Desktop para garantir que ele esteja em execução.

**Notas:**
- O Docker Desktop é recomendado para a maioria dos usuários. Se você está procurando uma configuração avançada de docker/container, especialmente para uma instalação em servidor remoto, consulte nosso [Guia de Implantação Docker no Ubuntu](/docs/remote/docker_linux).
- Você pode precisar reiniciar seu computador após a instalação.

## Passo 3. Executar o App [#step-3-run-the-app]

1. **Navegue até o Diretório do Projeto**

2. **Criar e configurar o arquivo .env**:
   - Copie o conteúdo de `.env.example` para um novo arquivo chamado `.env`.
   - Preencha quaisquer valores necessários.
      - Para uma configuração de ambiente detalhada, consulte o [Guia de Configuração do Arquivo .env](/docs/configuration/dotenv).

3. **Iniciar a Aplicação**:
   - Execute o seguinte comando:

   ```bash
   docker compose up -d
   ```

## Conclusão [#conclusion]

**É isso!** Agora você deve ter o **LibreChat** rodando localmente em sua máquina. Aproveite!
