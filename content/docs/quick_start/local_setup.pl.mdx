To jest skrócona wersja naszego [Przewodnika instalacji lokalnej](/docs/local/)

## Krok 1. Pobierz projekt [#step-1-download-the-project]

### Ręczne pobieranie [#manual-download]

1. **Przejdź do strony projektu**: Odwiedź [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat).

2. **Pobierz plik ZIP**: Kliknij zielony przycisk "Code", a następnie kliknij "Download ZIP."

3. **Wypakuj plik ZIP**: Znajdź pobrany plik ZIP, kliknij go prawym przyciskiem myszy i wybierz „Wyodrębnij wszystko...”.

### Korzystanie z Git [#using-git]

Uruchom następujące polecenie [git](https://git-scm.com/) w swoim terminalu, z poziomu wybranego katalogu nadrzędnego:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Krok 2. Zainstaluj Docker [#step-2-install-docker]

1. **Pobieranie**: Przejdź do [strony pobierania Docker Desktop](https://www.docker.com/products/docker-desktop) i pobierz Docker Desktop.
2. **Instalacja**: Otwórz instalator i postępuj zgodnie z instrukcjami.
3. **Uruchomienie**: Otwórz Docker Desktop, aby upewnić się, że działa.

**Uwagi:**
- Docker Desktop jest zalecany dla większości użytkowników. Jeśli szukasz zaawansowanej konfiguracji docker/kontenerów, zwłaszcza w przypadku instalacji na zdalnym serwerze, zapoznaj się z naszym [Przewodnikiem wdrażania Docker na Ubuntu](/docs/remote/docker_linux).
- Po instalacji może być konieczne ponowne uruchomienie komputera.

## Krok 3. Uruchom aplikację [#step-3-run-the-app]

1. **Przejdź do katalogu projektu**

2. **Utwórz i skonfiguruj plik .env**:
   - Skopiuj zawartość `.env.example` do nowego pliku o nazwie `.env`.
   - Wypełnij wszelkie niezbędne wartości.
      - Aby uzyskać szczegółowe informacje na temat konfiguracji środowiska, zobacz [Przewodnik po konfiguracji pliku .env](/docs/configuration/dotenv).

3. **Uruchom aplikację**:
   - Uruchom następujące polecenie:

   ```bash
   docker compose up -d
   ```

## Podsumowanie [#conclusion]

**To wszystko!** Powinieneś teraz mieć **LibreChat** uruchomiony lokalnie na swoim komputerze. Miłego korzystania!
