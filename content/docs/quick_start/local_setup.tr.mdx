Bu, [Local Installation Guide](/docs/local/) kılavuzumuzun özetlenmiş bir versiyonudur.

## Adım 1. Projeyi İndirin [#step-1-download-the-project]

### Manuel İndirme [#manual-download]

1. **Proje Sayfasına Git**: [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat) adresini ziyaret edin.

2. **ZIP Dosyasını İndirin**: Yeşil "Code" butonuna tıklayın, ardından "Download ZIP" seçeneğine tıklayın.

3. **ZIP Dosyasını Çıkartın**: İndirilen ZIP dosyasını bulun, sağ tıklayın ve "Tümünü Ayıkla..." seçeneğini seçin.

### Git Kullanımı [#using-git]

Terminalinizde, istediğiniz üst dizinden aşağıdaki [git](https://git-scm.com/) komutunu çalıştırın:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Adım 2. Docker kurulumu [#step-2-install-docker]

1. **İndirme**: [Docker Desktop İndirme Sayfası](https://www.docker.com/products/docker-desktop) adresine gidin ve Docker Desktop'ı indirin.
2. **Kurulum**: Yükleyiciyi açın ve talimatları izleyin.
3. **Çalıştırma**: Çalıştığından emin olmak için Docker Desktop'ı açın.

**Notlar:**
- Çoğu kullanıcı için Docker Desktop önerilir. Özellikle uzak bir sunucu kurulumu için gelişmiş bir docker/konteyner kurulumu arıyorsanız, [Ubuntu Docker Deployment Guide](/docs/remote/docker_linux) bölümümüze göz atın.
- Kurulumdan sonra bilgisayarınızı yeniden başlatmanız gerekebilir.

## Adım 3. Uygulamayı Çalıştırın [#step-3-run-the-app]

1. **Proje Dizinine Gidin**

2. **.env Dosyası Oluşturma ve Yapılandırma**:
   - `.env.example` dosyasının içeriğini `.env` adında yeni bir dosyaya kopyalayın.
   - Gerekli değerleri doldurun.
      - Kapsamlı bir ortam yapılandırması için [.env File Configuration Guide](/docs/configuration/dotenv) bölümüne bakın.

3. **Uygulamayı Başlatın**:
   - Aşağıdaki komutu çalıştırın:

   ```bash
   docker compose up -d
   ```

## Sonuç [#conclusion]

**İşte bu kadar!** Artık **LibreChat** yerel makinenizde çalışıyor olmalı. Keyfini çıkarın!
