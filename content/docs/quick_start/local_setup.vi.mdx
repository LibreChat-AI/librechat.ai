Đây là phiên bản rút gọn của [Hướng dẫn cài đặt cục bộ](/docs/local/) của chúng tôi

## Bước 1. Tải xuống dự án [#step-1-download-the-project]

### Tải xuống thủ công [#manual-download]

1. **Đi tới Trang Dự án**: Truy cập [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat).

2. **Tải xuống tệp ZIP**: Nhấp vào nút "Code" màu xanh lá cây, sau đó nhấp vào "Download ZIP."

3. **Giải nén tệp ZIP**: Tìm tệp ZIP đã tải xuống, nhấp chuột phải và chọn "Extract All...".

### Sử dụng Git [#using-git]

Chạy lệnh [git](https://git-scm.com/) sau trong terminal của bạn, từ thư mục cha mong muốn:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Bước 2. Cài đặt Docker [#step-2-install-docker]

1. **Tải xuống**: Truy cập [Trang tải xuống Docker Desktop](https://www.docker.com/products/docker-desktop) và tải xuống Docker Desktop.
2. **Cài đặt**: Mở trình cài đặt và làm theo các hướng dẫn.
3. **Chạy**: Mở Docker Desktop để đảm bảo ứng dụng đang chạy.

**Ghi chú:**
- Docker Desktop được khuyến nghị cho hầu hết người dùng. Nếu bạn đang tìm kiếm một thiết lập docker/container nâng cao, đặc biệt là cho việc cài đặt trên máy chủ từ xa, hãy xem [Hướng dẫn triển khai Docker trên Ubuntu](/docs/remote/docker_linux) của chúng tôi.
- Bạn có thể cần khởi động lại máy tính sau khi cài đặt.

## Bước 3. Chạy ứng dụng [#step-3-run-the-app]

1. **Điều hướng đến Thư mục Dự án**

2. **Tạo và cấu hình tệp .env**:
   - Sao chép nội dung của `.env.example` sang một tệp mới có tên là `.env`.
   - Điền bất kỳ giá trị cần thiết nào.
      - Để biết cấu hình môi trường chuyên sâu, hãy xem [Hướng dẫn cấu hình tệp .env](/docs/configuration/dotenv).

3. **Khởi chạy ứng dụng**:
   - Chạy lệnh sau:

   ```bash
   docker compose up -d
   ```

## Kết luận [#conclusion]

**Vậy là xong!** Bây giờ bạn đã có **LibreChat** chạy cục bộ trên máy của mình. Chúc bạn trải nghiệm vui vẻ!
