Dies ist eine gekürzte Version unserer [Local Installation Guide](/docs/local/)

## Schritt 1. Das Projekt herunterladen [#step-1-download-the-project]

### Manueller Download [#manual-download]

1. **Zur Projektseite gehen**: Besuchen Sie [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat).

2. **ZIP-Datei herunterladen**: Klicken Sie auf die grüne Schaltfläche „Code“ und anschließend auf „Download ZIP“.

3. **Entpacken der ZIP-Datei**: Suchen Sie die heruntergeladene ZIP-Datei, klicken Sie mit der rechten Maustaste darauf und wählen Sie "Alle extrahieren...".

### Verwendung von Git [#using-git]

Führen Sie den folgenden [git](https://git-scm.com/) Befehl in Ihrem Terminal aus dem gewünschten übergeordneten Verzeichnis aus:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Schritt 2. Docker installieren [#step-2-install-docker]

1. **Download**: Gehen Sie zur [Docker Desktop Download Page](https://www.docker.com/products/docker-desktop) und laden Sie Docker Desktop herunter.
2. **Install**: Öffnen Sie das Installationsprogramm und folgen Sie den Anweisungen.
3. **Ausführen**: Öffnen Sie Docker Desktop, um sicherzustellen, dass es läuft.

**Hinweise:**
- Docker Desktop wird für die meisten Benutzer empfohlen. Wenn Sie nach einer fortgeschrittenen Docker-/Container-Einrichtung suchen, insbesondere für eine Installation auf einem Remote-Server, lesen Sie unseren [Ubuntu Docker Deployment Guide](/docs/remote/docker_linux).
- Möglicherweise müssen Sie Ihren Computer nach der Installation neu starten.

## Schritt 3. Die App ausführen [#step-3-run-the-app]

1. **Navigieren Sie zum Projektverzeichnis**

2. **Erstellen und Konfigurieren der .env Datei**:
   - Kopieren Sie den Inhalt von `.env.example` in eine neue Datei namens `.env`.
   - Tragen Sie alle erforderlichen Werte ein.
      - Für eine detaillierte Umgebungskonfiguration siehe den [.env File Configuration Guide](/docs/configuration/dotenv).

3. **Starten der Anwendung**:
   - Führen Sie den folgenden Befehl aus:

   ```bash
   docker compose up -d
   ```

## Fazit [#conclusion]

**Das war's!** Sie sollten nun **LibreChat** lokal auf Ihrem Rechner ausführen können. Viel Spaß!
