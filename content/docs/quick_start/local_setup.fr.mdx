Ceci est une version condensée de notre [Guide d'installation locale](/docs/local/)

## Étape 1. Télécharger le projet [#step-1-download-the-project]

### Téléchargement manuel [#manual-download]

1. **Allez sur la page du projet** : Visitez [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat).

2. **Télécharger le fichier ZIP** : Cliquez sur le bouton vert "Code", puis cliquez sur "Download ZIP."

3. **Extraire le fichier ZIP** : Localisez le fichier ZIP téléchargé, faites un clic droit et sélectionnez "Extraire tout...".

### Utilisation de Git [#using-git]

Exécutez la commande [git](https://git-scm.com/) suivante dans votre terminal, depuis le répertoire parent souhaité :

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Étape 2. Installer Docker [#step-2-install-docker]

1. **Téléchargement** : Allez sur la [page de téléchargement de Docker Desktop](https://www.docker.com/products/docker-desktop) et téléchargez Docker Desktop.
2. **Installation** : Ouvrez l'installateur et suivez les instructions.
3. **Exécution** : Ouvrez Docker Desktop pour vous assurer qu'il est en cours d'exécution.

**Notes :**
- Docker Desktop est recommandé pour la plupart des utilisateurs. Si vous recherchez une configuration docker/conteneur avancée, notamment pour une installation sur serveur distant, consultez notre [Guide de déploiement Docker sur Ubuntu](/docs/remote/docker_linux).
- Il peut être nécessaire de redémarrer votre ordinateur après l'installation.

## Étape 3. Exécuter l'application [#step-3-run-the-app]

1. **Accéder au répertoire du projet**

2. **Créer et configurer le fichier .env** :
   - Copiez le contenu de `.env.example` dans un nouveau fichier nommé `.env`.
   - Remplissez toutes les valeurs nécessaires.
      - Pour une configuration détaillée de l'environnement, consultez le [Guide de configuration du fichier .env](/docs/configuration/dotenv).

3. **Démarrer l'application** :
   - Exécutez la commande suivante :

   ```bash
   docker compose up -d
   ```

## Conclusion [#conclusion]

**C'est tout !** Vous devriez maintenant avoir **LibreChat** qui fonctionne localement sur votre machine. Profitez-en !
