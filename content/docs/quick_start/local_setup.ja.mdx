## ステップ 1. プロジェクトのダウンロード [#step-1-download-the-project]

### 手動ダウンロード [#manual-download]

1. **プロジェクトページへ移動**: [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat) にアクセスしてください。

2. **ZIPファイルをダウンロード**: 緑色の「Code」ボタンをクリックし、「Download ZIP」をクリックします。

3. **ZIPファイルを展開する**: ダウンロードしたZIPファイルを見つけ、右クリックして「すべて展開...」を選択します。

### Gitの使用 [#using-git]

ターミナルで、目的の親ディレクトリから以下の [git](https://git-scm.com/) コマンドを実行してください：

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## ステップ 2. Docker をインストールする [#step-2-install-docker]

1. **ダウンロード**: [Docker Desktop ダウンロードページ](https://www.docker.com/products/docker-desktop) にアクセスし、Docker Desktop をダウンロードしてください。
2. **インストール**: インストーラーを開き、指示に従ってください。
3. **実行**: Docker Desktop を開き、実行中であることを確認します。

**注記:**
- ほとんどのユーザーには Docker Desktop が推奨されます。高度な Docker/コンテナ設定、特にリモートサーバーへのインストールをお探しの場合は、[Ubuntu Docker Deployment Guide](/docs/remote/docker_linux) を参照してください。
- インストール後、コンピュータの再起動が必要になる場合があります。

## ステップ 3. アプリの実行 [#step-3-run-the-app]

1. **プロジェクトディレクトリに移動する**

2. **Create and Configure .env File**:
   - `.env.example` の内容をコピーして、`.env` という名前の新しいファイルを作成します。
   - 必要な値を入力してください。
      - 詳細な環境設定については、[.env File Configuration Guide](/docs/configuration/dotenv) を参照してください。

3. **アプリケーションの起動**:
   - 以下のコマンドを実行してください：

   ```bash
   docker compose up -d
   ```

## 結論 [#conclusion]

**以上です！** これでローカル環境で **LibreChat** が動作するはずです。お楽しみください！
