이 문서는 [Local Installation Guide](/docs/local/)의 요약 버전입니다.

## 1단계. 프로젝트 다운로드 [#step-1-download-the-project]

### 수동 다운로드 [#manual-download]

1. **프로젝트 페이지로 이동**: [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat)를 방문하세요.

2. **ZIP 파일 다운로드**: 녹색 "Code" 버튼을 클릭한 다음, "Download ZIP"을 클릭하세요.

3. **ZIP 파일 압축 풀기**: 다운로드한 ZIP 파일을 찾아 마우스 오른쪽 버튼으로 클릭한 다음 "모두 압축 풀기..."를 선택합니다.

### Git 사용하기 [#using-git]

원하는 상위 디렉토리에서 터미널에 다음 [git](https://git-scm.com/) 명령어를 실행하세요:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## 2단계. Docker 설치 [#step-2-install-docker]

1. **다운로드**: [Docker Desktop 다운로드 페이지](https://www.docker.com/products/docker-desktop)로 이동하여 Docker Desktop을 다운로드하세요.
2. **설치**: 설치 프로그램을 열고 지침을 따르세요.
3. **실행(Run)**: Docker Desktop을 열어 실행 중인지 확인하세요.

**참고:**
- 대부분의 사용자에게는 Docker Desktop을 권장합니다. 고급 docker/컨테이너 설정을 찾고 계시거나, 특히 원격 서버 설치를 원하시는 경우 [Ubuntu Docker Deployment Guide](/docs/remote/docker_linux)를 참조하세요.
- 설치 후 컴퓨터를 다시 시작해야 할 수도 있습니다.

## 3단계. 앱 실행하기 [#step-3-run-the-app]

1. **프로젝트 디렉토리로 이동**

2. **Create and Configure .env File**:
   - `.env.example`의 내용을 `.env`라는 이름의 새 파일로 복사하세요.
   - 필요한 값을 모두 입력하세요.
      - 심층적인 환경 구성을 보려면 [.env File Configuration Guide](/docs/configuration/dotenv)를 참조하세요.

3. **애플리케이션 시작**:
   - 다음 명령어를 실행하세요:

   ```bash
   docker compose up -d
   ```

## 결론 [#conclusion]

**이제 끝입니다!** 이제 로컬 머신에서 **LibreChat**이 실행되고 있을 것입니다. 즐겁게 사용하세요!
