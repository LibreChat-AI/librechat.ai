Dit is een verkorte versie van onze [Local Installation Guide](/docs/local/)

## Stap 1. Download het project [#step-1-download-the-project]

### Handmatige download [#manual-download]

1. **Ga naar de Projectpagina**: Bezoek [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat).

2. **Download het ZIP-bestand**: Klik op de groene "Code"-knop en klik vervolgens op "Download ZIP."

3. **Pak het ZIP-bestand uit**: Zoek het gedownloade ZIP-bestand, klik met de rechtermuisknop en selecteer "Alles uitpakken...".

### Git gebruiken [#using-git]

Voer het volgende [git](https://git-scm.com/) commando uit in je terminal, vanuit de gewenste bovenliggende map:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Stap 2. Installeer Docker [#step-2-install-docker]

1. **Download**: Ga naar de [Docker Desktop Download Page](https://www.docker.com/products/docker-desktop) en download Docker Desktop.
2. **Install**: Open het installatieprogramma en volg de instructies.
3. **Run**: Open Docker Desktop om er zeker van te zijn dat het draait.

**Opmerkingen:**
- Docker Desktop wordt voor de meeste gebruikers aanbevolen. Als je op zoek bent naar een geavanceerde docker/container-configuratie, vooral voor een installatie op een externe server, bekijk dan onze [Ubuntu Docker Deployment Guide](/docs/remote/docker_linux).
- Mogelijk moet je je computer opnieuw opstarten na de installatie.

## Stap 3. De app uitvoeren [#step-3-run-the-app]

1. **Navigeer naar de Project Directory**

2. **Maak en configureer het .env bestand**:
   - Kopieer de inhoud van `.env.example` naar een nieuw bestand genaamd `.env`.
   - Vul alle benodigde waarden in.
      - Voor een uitgebreide omgevingsconfiguratie, zie de [.env File Configuration Guide](/docs/configuration/dotenv).

3. **Start de Applicatie**:
   - Voer het volgende commando uit:

   ```bash
   docker compose up -d
   ```

## Conclusie [#conclusion]

**Dat is alles!** Je zou **LibreChat** nu lokaal op je machine moeten hebben draaien. Veel plezier!
