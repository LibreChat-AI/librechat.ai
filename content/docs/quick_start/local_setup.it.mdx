Questa è una versione condensata della nostra [Guida all'installazione locale](/docs/local/)

## Passaggio 1. Scarica il progetto [#step-1-download-the-project]

### Download manuale [#manual-download]

1. **Vai alla pagina del progetto**: Visita [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat).

2. **Scarica il file ZIP**: Fai clic sul pulsante verde "Code", quindi fai clic su "Download ZIP."

3. **Estrai il file ZIP**: Individua il file ZIP scaricato, fai clic con il tasto destro e seleziona "Estrai tutto...".

### Utilizzo di Git [#using-git]

Esegui il seguente comando [git](https://git-scm.com/) nel tuo terminale, dalla directory padre desiderata:

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## Passaggio 2. Installare Docker [#step-2-install-docker]

1. **Download**: Vai alla [Pagina di download di Docker Desktop](https://www.docker.com/products/docker-desktop) e scarica Docker Desktop.
2. **Install**: Apri il programma di installazione e segui le istruzioni.
3. **Esecuzione**: Apri Docker Desktop per assicurarti che sia in esecuzione.

**Note:**
- Docker Desktop è consigliato per la maggior parte degli utenti. Se stai cercando una configurazione avanzata di docker/container, specialmente per un'installazione su server remoto, consulta la nostra [Guida all'implementazione di Docker su Ubuntu](/docs/remote/docker_linux).
- Potrebbe essere necessario riavviare il computer dopo l'installazione.

## Passaggio 3. Esegui l'App [#step-3-run-the-app]

1. **Naviga nella directory del progetto**

2. **Crea e configura il file .env**:
   - Copia il contenuto di `.env.example` in un nuovo file chiamato `.env`.
   - Inserisci tutti i valori necessari.
      - Per una configurazione dell'ambiente approfondita, consulta la [Guida alla configurazione del file .env](/docs/configuration/dotenv).

3. **Avviare l'applicazione**:
   - Esegui il seguente comando:

   ```bash
   docker compose up -d
   ```

## Conclusione [#conclusion]

**Ecco fatto!** Ora dovresti avere **LibreChat** in esecuzione localmente sulla tua macchina. Buon divertimento!
