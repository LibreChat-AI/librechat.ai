这是我们 [本地安装指南](/docs/local/) 的精简版本。

## 第 1 步：下载项目 [#step-1-download-the-project]

### 手动下载 [#manual-download]

1. **前往项目页面**：访问 [https://github.com/danny-avila/LibreChat](https://github.com/danny-avila/LibreChat)。

2. **下载 ZIP 文件**：点击绿色的 "Code" 按钮，然后点击 "Download ZIP"。

3. **解压 ZIP 文件**：找到下载的 ZIP 文件，右键点击，然后选择“全部解压...”。

### 使用 Git [#using-git]

在您的终端中，从所需的父目录运行以下 [git](https://git-scm.com/) 命令：

```bash
git clone https://github.com/danny-avila/LibreChat.git
```

## 第 2 步：安装 Docker [#step-2-install-docker]

1. **下载**: 前往 [Docker Desktop 下载页面](https://www.docker.com/products/docker-desktop) 并下载 Docker Desktop。
2. **安装**: 打开安装程序并按照说明进行操作。
3. **运行**: 打开 Docker Desktop 以确保其正在运行。

**注意：**
- 对于大多数用户，推荐使用 Docker Desktop。如果您正在寻找高级 docker/容器设置，特别是针对远程服务器安装，请参阅我们的 [Ubuntu Docker Deployment Guide](/docs/remote/docker_linux)。
- 安装后您可能需要重启计算机。

## 第 3 步：运行应用 [#step-3-run-the-app]

1. **导航至项目目录**

2. **创建并配置 .env 文件**:
   - 将 `.env.example` 的内容复制到一个名为 `.env` 的新文件中。
   - 填写任何必要的值。
      - 有关深入的环境配置，请参阅 [.env File Configuration Guide](/docs/configuration/dotenv)。

3. **启动应用程序**：
   - 运行以下命令：

   ```bash
   docker compose up -d
   ```

## 结论 [#conclusion]

**就是这样！** 你现在应该已经在本地机器上运行了 **LibreChat**。尽情享受吧！
