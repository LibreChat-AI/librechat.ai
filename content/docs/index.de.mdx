<Callout type="info" title="Suchen Sie nach einem Desktop-Installer?">

LibreChat ist eine selbst gehostete Webanwendung, keine native Windows-App oder Linux-AppImage. Verwenden Sie unter Windows [Docker Desktop](/docs/local/docker) für die einfachste lokale Einrichtung. Für Linux-Desktops oder -Server verwenden Sie [Docker](/docs/local/docker), [npm](/docs/local/npm) oder eine [Anleitung für Remote-Hosting](/docs/remote).

</Callout>
