## Ikhtisar [#overview]

Objek `balance` memungkinkan administrator untuk mengonfigurasi bagaimana saldo kredit token dikelola bagi pengguna di dalam LibreChat. Pengaturan mencakup pengaktifan pelacakan saldo, inisialisasi saldo pengguna, dan konfigurasi perilaku pengisian ulang token otomatis.

**Bidang di bawah `balance`:**

- `enabled`
- `startBalance`
- `autoRefillEnabled`
- `refillIntervalValue`
- `refillIntervalUnit`
- `refillAmount`

**Catatan:**

- Konfigurasi `balance` berlaku secara global di seluruh aplikasi.
- Nilai default telah disediakan namun dapat disesuaikan berdasarkan kebutuhan.
- Logika kondisional dapat memodifikasi pengaturan ini secara dinamis berdasarkan konfigurasi lainnya.


## Contoh [#example]

```yaml filename="balance"
balance:
  enabled: false
  startBalance: 20000
  autoRefillEnabled: false
  refillIntervalValue: 30
  refillIntervalUnit: "days"
  refillAmount: 10000
```
