## Ejemplo [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **Tipo:** `String`
- **Descripción:** Tu clave de sitio única de Cloudflare Turnstile. Asegúrate de haber registrado tu dominio en Cloudflare y de haber obtenido esta clave desde la guía [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/).
- **Ejemplo:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **Tipo:** `Object`
- **Descripción:** Un objeto para configurar ajustes adicionales para el widget Turnstile.

**Subclaves:**
<OptionTable
  options={[
    ['language', 'String', 'Especifica el idioma para el widget de Turnstile. Use `auto` para detectar automáticamente el idioma del usuario, o proporcione un código de idioma ISO 639-1 (por ejemplo, `en`).', 'auto'],
    ['size', 'String', 'Determina el tamaño de visualización del widget. Las opciones válidas incluyen `normal`, `compact`, `flexible` o `invisible`.', 'normal'],

```yaml
turnstile:
  options:
    language: "auto"
    size: "normal"
```

### Notas [#notes]

- **Integración opcional:** El bloque de configuración `turnstile` es opcional. Si decide no utilizar Cloudflare Turnstile, puede omitir este bloque por completo.
- **Consistencia del Dashboard:** Asegúrese de que los valores que configure aquí coincidan con sus ajustes en el dashboard de Cloudflare.
- **Experiencia de usuario:** Personalice las subclaves de `options` según sea necesario para adaptar el comportamiento y la apariencia del widget a los requisitos de su aplicación.
