## Vue d'ensemble [#overview]

L'objet `balance` permet aux administrateurs de configurer la gestion des soldes de crédits de jetons pour les utilisateurs au sein de LibreChat. Les paramètres incluent l'activation du suivi du solde, l'initialisation des soldes des utilisateurs et la configuration du comportement de rechargement automatique des jetons.

**Champs sous `balance` :**

- `enabled`
- `startBalance`
- `autoRefillEnabled`
- `refillIntervalValue`
- `refillIntervalUnit`
- `refillAmount`

**Notes :**

- Les configurations `balance` s'appliquent globalement à toute l'application.
- Des valeurs par défaut sont fournies mais peuvent être personnalisées selon les besoins.
- La logique conditionnelle peut modifier dynamiquement ces paramètres en fonction d'autres configurations.


## Exemple [#example]

```yaml filename="balance"
balance:
  enabled: false
  startBalance: 20000
  autoRefillEnabled: false
  refillIntervalValue: 30
  refillIntervalUnit: "days"
  refillAmount: 10000
```
