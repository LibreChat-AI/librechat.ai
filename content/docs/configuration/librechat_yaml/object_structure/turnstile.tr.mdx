## Örnek [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **Tür:** `String`
- **Açıklama:** Benzersiz Cloudflare Turnstile site anahtarınız. Alan adınızı Cloudflare'e kaydettiğinizden ve bu anahtarı [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/) kılavuzundan aldığınızdan emin olun.
- **Örnek:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **Tür:** `Object`
- **Açıklama:** Turnstile widget'ı için ek ayarları yapılandırmaya yarayan bir nesne.

### Notlar [#notes]

- **İsteğe Bağlı Entegrasyon:** `turnstile` yapılandırma bloğu isteğe bağlıdır. Cloudflare Turnstile kullanmamayı tercih ederseniz, bu bloğu tamamen çıkarabilirsiniz.
- **Dashboard Tutarlılığı:** Burada yapılandırdığınız değerlerin Cloudflare dashboard'undaki ayarlarınızla eşleştiğinden emin olun.
- **Kullanıcı Deneyimi:** Widget'ın davranışını ve görünümünü uygulamanızın gereksinimlerine göre uyarlamak için `options` alt anahtarlarını gerektiği gibi özelleştirin.
