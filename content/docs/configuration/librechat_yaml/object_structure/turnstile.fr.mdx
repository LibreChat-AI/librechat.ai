## Exemple [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **Type :** `String`
- **Description :** Votre clé de site Cloudflare Turnstile unique. Assurez-vous d'avoir enregistré votre domaine auprès de Cloudflare et d'avoir obtenu cette clé via le guide [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/).
- **Exemple :**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **Type :** `Object`
- **Description :** Un objet pour configurer des paramètres supplémentaires pour le widget Turnstile.

**Sous-clés :**
<OptionTable
  options={[
    ['language', 'String', 'Spécifie la langue du widget Turnstile. Utilisez `auto` pour détecter automatiquement la langue de l\'utilisateur, ou fournissez un code de langue ISO 639-1 (par exemple, `en`).', 'auto'],
    ['size', 'String', 'Détermine la taille d\'affichage du widget. Les options valides incluent `normal`, `compact`, `flexible` ou `invisible`.', 'normal'],
    ]}
/>

```yaml
turnstile:
  options:
    language: "auto"
    size: "normal"
```

### Notes [#notes]

- **Intégration facultative :** Le bloc de configuration `turnstile` est facultatif. Si vous choisissez de ne pas utiliser Cloudflare Turnstile, vous pouvez omettre ce bloc entièrement.
- **Cohérence du tableau de bord :** Assurez-vous que les valeurs que vous configurez ici correspondent à vos paramètres dans le tableau de bord Cloudflare.
- **Expérience utilisateur :** Personnalisez les sous-clés `options` selon vos besoins pour adapter le comportement et l'apparence du widget aux exigences de votre application.
