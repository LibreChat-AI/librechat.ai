Elk item onder `models` maakt deel uit van een lijst met records, hetzij een booleaanse waarde of een Object:

**Bij het specificeren van een model als een object:**

Een object maakt gedetailleerde configuratie van het model mogelijk, inclusief de `deploymentName` en/of `version`. Deze modus wordt gebruikt voor meer gedetailleerde controle over de modellen, vooral bij het werken met meerdere versies of deployments onder één instantie of resourcegroep.

**Voorbeeld**:
```yaml filename="endpoints / azureOpenAI / groups / {group_item} / models / {model_item=Object}"
models:
  gpt-4-vision-preview:
    deploymentName: "gpt-4-vision-preview"
    version: "2024-02-15-preview"
```

**Opmerkingen:**
- **Deployment Names** en **Versions** zijn cruciaal om ervoor te zorgen dat het juiste model wordt gebruikt.
    - Controleer deze waarden dubbel op nauwkeurigheid om onverwacht gedrag te voorkomen.
