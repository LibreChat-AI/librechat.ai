## Panoramica [#overview]

L'oggetto `balance` consente agli amministratori di configurare come vengono gestiti i saldi dei crediti dei token per gli utenti all'interno di LibreChat. Le impostazioni includono l'abilitazione del monitoraggio del saldo, l'inizializzazione dei saldi utente e la configurazione del comportamento di ricarica automatica dei token.

**Campi sotto `balance`:**

- `enabled`
- `startBalance`
- `autoRefillEnabled`
- `refillIntervalValue`
- `refillIntervalUnit`
- `refillAmount`

**Note:**

- Le configurazioni `balance` si applicano globalmente a tutta l'applicazione.
- I valori predefiniti sono forniti ma possono essere personalizzati in base ai requisiti.
- La logica condizionale può modificare dinamicamente queste impostazioni in base ad altre configurazioni.


## Esempio [#example]

```yaml filename="balance"
balance:
  enabled: false
  startBalance: 20000
  autoRefillEnabled: false
  refillIntervalValue: 30
  refillIntervalUnit: "days"
  refillAmount: 10000
```
