## Esempio [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **Tipo:** `String`
- **Descrizione:** La tua chiave del sito Cloudflare Turnstile univoca. Assicurati di aver registrato il tuo dominio con Cloudflare e di aver ottenuto questa chiave dalla guida [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/).
- **Esempio:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **Tipo:** `Object`
- **Descrizione:** Un oggetto per configurare impostazioni aggiuntive per il widget Turnstile.

**Sottochiavi:**
<OptionTable
  options={[
    ['language', 'String', 'Specifica la lingua per il widget Turnstile. Usa `auto` per rilevare automaticamente la lingua dell\'utente, oppure fornisci un codice lingua ISO 639-1 (ad esempio `en`).', 'auto'],

```yaml
turnstile:
  options:
    language: "auto"
    size: "normal"
```

### Note [#notes]

- **Integrazione opzionale:** Il blocco di configurazione `turnstile` è opzionale. Se scegli di non utilizzare Cloudflare Turnstile, puoi omettere completamente questo blocco.
- **Coerenza della Dashboard:** Assicurati che i valori configurati qui corrispondano alle tue impostazioni nella dashboard di Cloudflare.
- **Esperienza utente:** Personalizza le sottochiavi `options` come necessario per adattare il comportamento e l'aspetto del widget ai requisiti della tua applicazione.
