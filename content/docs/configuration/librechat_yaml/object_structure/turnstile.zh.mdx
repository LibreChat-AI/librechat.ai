## 示例 [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **类型：** `String`
- **描述：** 您唯一的 Cloudflare Turnstile 站点密钥。请确保您已在 Cloudflare 注册了您的域名，并已从 [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/) 指南中获取此密钥。
- **示例：**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **类型：** `Object`
- **描述：** 用于配置 Turnstile 小部件附加设置的对象。

**子键 (Subkeys):**

```yaml
turnstile:
  options:
    language: "auto"
    size: "normal"
```

### 注意事项 [#notes]

- **可选集成：** `turnstile` 配置块是可选的。如果您选择不使用 Cloudflare Turnstile，则可以完全省略此块。
- **仪表板一致性：** 确保您在此处配置的值与您在 Cloudflare 仪表板中的设置相匹配。
- **用户体验：** 根据需要自定义 `options` 子键，以根据您应用程序的要求调整小部件的行为和外观。
