## Genel Bakış [#overview]

`balance` nesnesi, yöneticilerin LibreChat içindeki kullanıcılar için token kredi bakiyelerinin nasıl yönetileceğini yapılandırmasına olanak tanır. Ayarlar; bakiye takibini etkinleştirmeyi, kullanıcı bakiyelerini başlatmayı ve otomatik token yenileme davranışını yapılandırmayı içerir.

**`balance` altındaki alanlar:**

- `enabled`
- `startBalance`
- `autoRefillEnabled`
- `refillIntervalValue`
- `refillIntervalUnit`
- `refillAmount`

**Notlar:**

- `balance` yapılandırmaları uygulama genelinde geçerlidir.
- Varsayılan değerler sağlanmıştır ancak gereksinimlere göre özelleştirilebilir.
- Koşullu mantık, bu ayarları diğer yapılandırmalara dayalı olarak dinamik bir şekilde değiştirebilir.


## Örnek [#example]

```yaml filename="balance"
balance:
  enabled: false
  startBalance: 20000
  autoRefillEnabled: false
  refillIntervalValue: 30
  refillIntervalUnit: "days"
  refillAmount: 10000
```
