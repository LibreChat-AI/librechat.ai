## Beispiel [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **Typ:** `String`
- **Beschreibung:** Ihr eindeutiger Cloudflare Turnstile Site Key. Stellen Sie sicher, dass Sie Ihre Domain bei Cloudflare registriert und diesen Schlüssel über den [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/) Leitfaden erhalten haben.
- **Beispiel:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **Typ:** `Object`
- **Beschreibung:** Ein Objekt zur Konfiguration zusätzlicher Einstellungen für das Turnstile-Widget.

**Untergeordnete Schlüssel:**
<OptionTable

    ['size', 'String', 'Bestimmt die Anzeigegröße des Widgets. Gültige Optionen sind `normal`, `compact`, `flexible` oder `invisible`.', 'normal'],

```yaml
turnstile:
  options:
    language: "auto"
    size: "normal"
```

### Hinweise [#notes]

- **Optionale Integration:** Der `turnstile` Konfigurationsblock ist optional. Wenn Sie sich entscheiden, Cloudflare Turnstile nicht zu verwenden, können Sie diesen Block vollständig weglassen.
- **Dashboard-Konsistenz:** Stellen Sie sicher, dass die Werte, die Sie hier konfigurieren, mit Ihren Einstellungen im Cloudflare-Dashboard übereinstimmen.
- **Benutzererfahrung:** Passen Sie die `options`-Subkeys nach Bedarf an, um das Verhalten und das Erscheinungsbild des Widgets an die Anforderungen Ihrer Anwendung anzupassen.
