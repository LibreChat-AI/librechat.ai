## Genel Bakış [#overview]

`messageFilter` nesnesi, sunucu tarafındaki mesaj filtrelerini yapılandırır. v1.3.13 sürümünde desteklenen filtre türü `messageFilter.pii` şeklindedir.

`messageFilter.pii`, gönderilen mesaj metnini, metin denetlemeye, model isteğine veya mesaj kalıcılığına ulaşmadan önce kimlik bilgisi benzeri başlangıç kalıplarıyla veya operatör tarafından tanımlanan düzenli ifadelerle eşleşip eşleşmediğine göre reddeder.

## Örnek [#example]

```yaml filename="librechat.yaml"
messageFilter:
  pii:
    starterPatterns:
      - sk_prefix
      - bearer_header
      - api_key_header
    customPatterns:
      - id: anthropic_api_key
        label: Anthropic API key
        regex: 'sk-ant-[A-Za-z0-9_-]{20,}'
```

## Davranış [#behavior]

Gönderilen mesaj metni yapılandırılmış bir PII kalıbıyla eşleştiğinde, LibreChat `message_filter_pii_block` ile birlikte HTTP `400` hatası döndürür ve çağrıyı yapan taraftan eşleşen kimlik bilgisi türünü kaldırmasını isteyen bir mesaj iletir.

Filtre şunları kontrol eder:

- LibreChat arayüzü aracılığıyla gönderilen sohbet rotası kullanıcı metni.
- OpenAI uyumlu Chat Completions isteklerinde, arayan tarafından sağlanan tüm mesaj rolleri genelindeki metin içeriği.
- OpenAI uyumlu Yanıtlar isteklerindeki tüm arayan tarafından sağlanan mesaj rolleri genelindeki metin içeriği.

Filtre, istek gövdesinde sağlanan metni değerlendirir. Dosya içeriklerini taramaz.
