Setiap item di bawah `models` merupakan bagian dari daftar rekaman, baik berupa nilai boolean atau Objek:

**Saat menentukan model sebagai objek:**

Sebuah objek memungkinkan konfigurasi mendetail untuk model tersebut, termasuk `deploymentName` dan/atau `version`-nya. Mode ini digunakan untuk kontrol yang lebih terperinci atas model, terutama saat bekerja dengan beberapa versi atau deployment di bawah satu instance atau grup sumber daya.

**Contoh**:
```yaml filename="endpoints / azureOpenAI / groups / {group_item} / models / {model_item=Object}"
models:
  gpt-4-vision-preview:
    deploymentName: "gpt-4-vision-preview"
    version: "2024-02-15-preview"
```

**Catatan:**
- **Deployment Names** dan **Versions** sangat penting untuk memastikan bahwa model yang tepat digunakan.
    - Periksa kembali nilai-nilai ini untuk memastikan keakuratannya guna mencegah perilaku yang tidak diinginkan.

### deploymentName

**Kunci:**

**Wajib:** ya

**Contoh:**
```yaml filename="endpoints / azureOpenAI / groups / {group_item} / models / {model_item=Object} / deploymentName"

```
