## Visão geral [#overview]

O objeto `balance` permite que administradores configurem como os saldos de créditos de tokens são gerenciados para usuários dentro do LibreChat. As configurações incluem habilitar o rastreamento de saldo, inicializar saldos de usuários e configurar o comportamento de recarga automática de tokens.

**Campos sob `balance`:**

- `enabled`
- `startBalance`
- `autoRefillEnabled`
- `refillIntervalValue`
- `refillIntervalUnit`
- `refillAmount`

**Notas:**

- As configurações de `balance` aplicam-se globalmente em toda a aplicação.
- Os padrões são fornecidos, mas podem ser personalizados com base nos requisitos.
- A lógica condicional pode modificar dinamicamente essas configurações com base em outras configurações.


## Exemplo [#example]

```yaml filename="balance"
balance:
  enabled: false
  startBalance: 20000
  autoRefillEnabled: false
  refillIntervalValue: 30
  refillIntervalUnit: "days"
  refillAmount: 10000
```
