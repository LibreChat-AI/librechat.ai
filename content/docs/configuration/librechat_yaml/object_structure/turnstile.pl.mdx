## Przykład [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **Typ:** `String`
- **Opis:** Twój unikalny klucz witryny Cloudflare Turnstile. Upewnij się, że zarejestrowałeś swoją domenę w Cloudflare i uzyskałeś ten klucz z przewodnika [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/).
- **Przykład:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **Typ:** `Object`
- **Opis:** Obiekt służący do konfigurowania dodatkowych ustawień widżetu Turnstile.

**Podklucze:**

```yaml
turnstile:
  options:
    language: "auto"
    size: "normal"
```

### Uwagi [#notes]

- **Opcjonalna integracja:** Blok konfiguracyjny `turnstile` jest opcjonalny. Jeśli zdecydujesz się nie używać Cloudflare Turnstile, możesz całkowicie pominąć ten blok.
- **Spójność panelu sterowania:** Upewnij się, że wartości skonfigurowane w tym miejscu są zgodne z ustawieniami w panelu sterowania Cloudflare.
- **User Experience:** Dostosuj podklucze `options` zgodnie z potrzebami, aby dopasować zachowanie i wygląd widżetu do wymagań Twojej aplikacji.
