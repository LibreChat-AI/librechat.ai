## 例 [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **型:** `String`
- **説明:** あなた固有の Cloudflare Turnstile サイトキーです。ドメインを Cloudflare に登録し、[Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/) ガイドからこのキーを取得していることを確認してください。
- **例:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **型:** `Object`
- **説明:** Turnstileウィジェットの追加設定を構成するためのオブジェクトです。

**サブキー:**
<OptionTable
  options={[
    ['language', 'String', 'Turnstileウィジェットの言語を指定します。`auto`を使用するとユーザーの言語を自動検出し、ISO 639-1言語コード（例: `en`）を指定することもできます。', 'auto'],
    ['size', 'String', 'ウィジェットの表示サイズを決定します。有効なオプションは `normal`、`compact`、`flexible`、または `invisible` です。', 'normal'],
    ]}
/>

```yaml
turnstile:
  options:
    language: "auto"
    size: "normal"
```

### 注記 [#notes]

- **オプションの統合:** `turnstile` 設定ブロックはオプションです。Cloudflare Turnstileを使用しない場合は、このブロックを完全に省略できます。
- **ダッシュボードの整合性:** ここで設定する値が、Cloudflareダッシュボードの設定と一致していることを確認してください。
- **ユーザーエクスペリエンス:** アプリケーションの要件に合わせてウィジェットの動作や外観を調整するために、必要に応じて `options` サブキーをカスタマイズしてください。
