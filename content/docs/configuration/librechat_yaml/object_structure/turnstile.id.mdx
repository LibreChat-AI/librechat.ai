## Contoh [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **Tipe:** `String`
- **Deskripsi:** Kunci situs Cloudflare Turnstile unik Anda. Pastikan Anda telah mendaftarkan domain Anda di Cloudflare dan mendapatkan kunci ini dari panduan [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/).
- **Contoh:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **Tipe:** `Object`
- **Deskripsi:** Sebuah objek untuk mengonfigurasi pengaturan tambahan bagi widget Turnstile.

**Subkey:**


### Catatan [#notes]

- **Integrasi Opsional:** Blok konfigurasi `turnstile` bersifat opsional. Jika Anda memilih untuk tidak menggunakan Cloudflare Turnstile, Anda dapat menghilangkan blok ini sepenuhnya.
- **Konsistensi Dasbor:** Pastikan nilai yang Anda konfigurasi di sini sesuai dengan pengaturan Anda di dasbor Cloudflare.
- **Pengalaman Pengguna:** Sesuaikan subkunci `options` sesuai kebutuhan untuk menyesuaikan perilaku dan tampilan widget dengan persyaratan aplikasi Anda.
