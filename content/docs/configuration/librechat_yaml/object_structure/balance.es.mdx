## Descripción general [#overview]

El objeto `balance` permite a los administradores configurar cómo se gestionan los saldos de crédito de tokens para los usuarios dentro de LibreChat. Los ajustes incluyen habilitar el seguimiento de saldo, inicializar los saldos de los usuarios y configurar el comportamiento de recarga automática de tokens.

**Campos bajo `balance`:**

- `enabled`
- `startBalance`
- `autoRefillEnabled`
- `refillIntervalValue`
- `refillIntervalUnit`
- `refillAmount`

**Notas:**

- Las configuraciones de `balance` se aplican globalmente en toda la aplicación.
- Se proporcionan valores predeterminados, pero pueden personalizarse según los requisitos.
- La lógica condicional puede modificar dinámicamente estos ajustes basándose en otras configuraciones.


## Ejemplo [#example]

```yaml filename="balance"
balance:
  enabled: false
  startBalance: 20000
  autoRefillEnabled: false
  refillIntervalValue: 30
  refillIntervalUnit: "days"
  refillAmount: 10000
```
