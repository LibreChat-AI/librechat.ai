## Ví dụ [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **Loại:** `String`
- **Mô tả:** Khóa trang web Cloudflare Turnstile duy nhất của bạn. Hãy đảm bảo rằng bạn đã đăng ký tên miền của mình với Cloudflare và lấy khóa này từ hướng dẫn [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/).
- **Ví dụ:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **Kiểu:** `Object`
- **Mô tả:** Một đối tượng để cấu hình các cài đặt bổ sung cho tiện ích Turnstile.

```yaml
  options:

    size: "normal"
```

### Ghi chú [#notes]

- **Tích hợp tùy chọn:** Khối cấu hình `turnstile` là tùy chọn. Nếu bạn chọn không sử dụng Cloudflare Turnstile, bạn có thể bỏ qua hoàn toàn khối này.
- **Tính nhất quán của Dashboard:** Đảm bảo rằng các giá trị bạn cấu hình tại đây khớp với cài đặt của bạn trong dashboard Cloudflare.
- **Trải nghiệm người dùng:** Tùy chỉnh các khóa con `options` khi cần để điều chỉnh hành vi và giao diện của tiện ích theo yêu cầu ứng dụng của bạn.
