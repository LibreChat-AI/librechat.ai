Die Integration des Azure OpenAI Service in Ihre Anwendung ermﾃｶglicht es Ihnen, nahtlos mehrere Bereitstellungen und regionale Modelle zu nutzen, die von Azure OpenAI gehostet werden. Dieser Abschnitt beschreibt, wie Sie den Azure OpenAI endpoint fﾃｼr Ihre Anforderungen konfigurieren.

**[Fﾃｼr eine detaillierte Anleitung zur Einrichtung von Azure OpenAI-Konfigurationen, klicken Sie hier](/docs/configuration/librechat_yaml/ai_endpoints/azure)**

## Beispielkonfiguration [#example-configuration]

```yaml filename="Example Azure OpenAI Object Structure"
endpoints:
  azureOpenAI:
    titleModel: "gpt-4-turbo"
    groups:
      - group: "my-westus" # arbitrary name
        apiKey: "${WESTUS_API_KEY}"
        instanceName: "actual-instance-name" # name of the resource group or instance
        version: "2023-12-01-preview"
        # baseURL: https://prod.example.com
        # additionalHeaders:
        #   X-Custom-Header: value
        models:
          gpt-4-vision-preview:
            deploymentName: gpt-4-vision-preview
            version: "2024-02-15-preview"
          gpt-3.5-turbo:
            deploymentName: gpt-35-turbo
          gpt-3.5-turbo-1106:
            deploymentName: gpt-35-turbo-1106
          gpt-4:
            deploymentName: gpt-4
          gpt-4-1106-preview:
            deploymentName: gpt-4-1106-preview
      - group: "my-eastus"
        apiKey: "${EASTUS_API_KEY}"
        instanceName: "actual-eastus-instance-name"
        deploymentName: gpt-4-turbo
        version: "2024-02-15-preview"
        baseURL: "https://gateway.ai.cloudflare.com/v1/cloudflareId/azure/azure-openai/${INSTANCE_NAME}/${DEPLOYMENT_NAME}" # uses env variables
        additionalHeaders:
          X-Custom-Header: value
        models:
          gpt-4-turbo: true
```

> **Hinweis:** Der Azure OpenAI endpoint unterstﾃｼtzt alle [Shared Endpoint Settings](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings), einschlieﾃ殕ich `streamRate`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate` und `titleEndpoint`.
