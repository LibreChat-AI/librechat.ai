## Voorbeeld [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **Type:** `String`
- **Beschrijving:** Uw unieke Cloudflare Turnstile site key. Zorg ervoor dat u uw domein heeft geregistreerd bij Cloudflare en deze sleutel heeft verkregen via de [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/) handleiding.
- **Voorbeeld:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **Type:** `Object`
- **Beschrijving:** Een object om aanvullende instellingen voor de Turnstile-widget te configureren.

**Subkeys:**
<OptionTable

    ['size', 'String', 'Bepaalt de weergavegrootte van de widget. Geldige opties zijn `normal`, `compact`, `flexible` of `invisible`.', 'normal'],

```yaml
turnstile:
  options:
    language: "auto"
    size: "normal"
```

### Opmerkingen [#notes]

- **Optionele integratie:** Het `turnstile` configuratieblok is optioneel. Als u ervoor kiest om Cloudflare Turnstile niet te gebruiken, kunt u dit blok volledig weglaten.
- **Dashboard-consistentie:** Zorg ervoor dat de waarden die u hier configureert overeenkomen met uw instellingen in het Cloudflare-dashboard.
- **Gebruikerservaring:** Pas de `options` subkeys naar wens aan om het gedrag en het uiterlijk van de widget af te stemmen op de vereisten van uw applicatie.
