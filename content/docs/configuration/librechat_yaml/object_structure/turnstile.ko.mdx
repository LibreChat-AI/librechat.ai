## 예시 [#example]

```yaml filename="turnstile"
turnstile:
  siteKey: "your-site-key-here"
  options:
    language: "auto"    # "auto" or an ISO 639-1 language code (e.g., en)
    size: "normal"      # Options: "normal", "compact", "flexible", or "invisible"
```

#### siteKey

- **유형:** `String`
- **설명:** 고유한 Cloudflare Turnstile 사이트 키입니다. 도메인을 Cloudflare에 등록하고 [Cloudflare Turnstile Get Started](https://developers.cloudflare.com/turnstile/get-started/) 가이드에서 이 키를 획득했는지 확인하세요.
- **예시:**
```yaml
turnstile:
  siteKey: "your-site-key-here"
```

#### options

- **유형:** `Object`
- **설명:** Turnstile 위젯에 대한 추가 설정을 구성하기 위한 객체입니다.

```yaml
turnstile:
  options:
    language: "auto"

```

### 참고 사항 [#notes]

- **선택적 통합:** `turnstile` 구성 블록은 선택 사항입니다. Cloudflare Turnstile을 사용하지 않기로 선택한 경우, 이 블록을 완전히 생략할 수 있습니다.
- **대시보드 일관성:** 여기에 구성한 값이 Cloudflare 대시보드의 설정과 일치하는지 확인하세요.
- **사용자 경험:** 애플리케이션의 요구 사항에 맞춰 위젯의 동작과 외관을 조정하려면 필요에 따라 `options` 하위 키를 사용자 지정하세요.
