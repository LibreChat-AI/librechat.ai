## 주석이 포함된 예시 [#example-with-comments]

이 예제 설정 파일은 여러 주요 영역에 걸쳐 상세한 옵션으로 LibreChat을 설정합니다:

- **Caching**: 성능 향상을 위해 활성화됨.
- **파일 처리**:
  - **File Strategy**: 주석 처리되어 있지만 파일 저장을 위해 Firebase와의 통합 가능성을 암시합니다.
  - **파일 구성**: 다양한 endpoint에 대한 파일 업로드 제한 및 허용된 MIME 유형을 사용자 지정하며, 여기에는 전역 서버 파일 크기 제한과 사용자 아바타 이미지에 대한 특정 제한이 포함됩니다.
- **Rate Limiting**: 지정된 시간 내에 IP 및 사용자별로 허용되는 최대 파일 업로드 횟수에 대한 임계값을 정의하여 남용을 방지합니다.
- **등록**:
  - 지정된 소셜 로그인 제공업체 및 이메일 도메인으로부터의 가입을 허용하여 보안 및 사용자 관리를 강화합니다.
- **Endpoints**:
  - **Assistants**: assistants endpoint를 폴링 간격 및 작업 시간 초과 설정으로 구성하며, 빌더 인터페이스를 비활성화하는 옵션을 제공합니다.
  - **사용자 지정 엔드포인트(Custom Endpoints)**:
    - 두 개의 외부 AI 서비스 endpoint인 Mistral과 OpenRouter를 구성하며, 여기에는 API 키, 기본 URL, 모델 처리, 그리고 대화 제목, 요약, 매개변수 조정과 같은 특정 기능 토글이 포함됩니다.
    - Mistral의 경우, 동적 모델 가져오기를 활성화하고, 안전한 프롬프트를 위한 추가 매개변수를 적용하며, 지원되지 않는 매개변수를 명시적으로 삭제합니다.
    - OpenRouter의 경우, 동적 모델 가져오기 없이 기본 구성을 설정하며 대화 제목을 위한 모델을 지정합니다.
