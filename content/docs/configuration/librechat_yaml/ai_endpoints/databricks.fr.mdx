> **[Inscrivez-vous à Databricks]**(https://www.databricks.com/try-databricks#account)

**Notes :**

- Puisque Databricks fournit un endpoint de complétion complet, se terminant par "invocations" pour les "serving-endpoints", l'utilisation du paramètre [directEndpoint](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#directendpoint) est requise.
- [titleMessageRole](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#titlemessagerole) défini sur "user" est également requis pour la génération de titres, car les messages "system" isolés ne sont pas pris en charge.

```yaml filename="librechat.yaml"
    - name: 'Databricks'
      apiKey: '${DATABRICKS_API_KEY}'
      baseURL: 'https://your_databricks_serving_endpoint_url_here_ending_with/invocations'
      models:
        default: [
          "databricks-meta-llama-3-70b-instruct",
        ]
        fetch: false
      titleConvo: true
      titleModel: 'current_model'
      directEndpoint: true # required
      titleMessageRole: 'user' # required
```
