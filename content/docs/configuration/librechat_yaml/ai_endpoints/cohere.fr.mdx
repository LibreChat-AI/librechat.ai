> Clé API Cohere : [dashboard.cohere.com](https://dashboard.cohere.com/)

**Notes :**

- **Connu :** icône fournie.
- Expérimental : ne suit pas la spécification OpenAI, utilise une nouvelle méthode pour la compatibilité des endpoint, et partage certaines similitudes et paramètres.
- Pour une liste complète des paramètres spécifiques à Cohere, consultez la [documentation de l'API Cohere](https://docs.cohere.com/reference/chat).
- Remarque : Les paramètres suivants sont reconnus entre OpenAI et Cohere. La plupart sont supprimés dans l'exemple de configuration ci-dessous pour privilégier les paramètres par défaut de Cohere :
    - `stop` : mappé à `stopSequences`
    - `top_p` : mappé sur `p`, valeurs min/max différentes
    - `frequency_penalty` : mappé à `frequencyPenalty`, valeurs min/max différentes
    - `presence_penalty` : mappé à `presencePenalty`, valeurs min/max différentes
    - `model` : partagé, inclus par défaut.
    - `stream` : partagé, inclus par défaut.
    - `max_tokens` : partagé, mappé sur `maxTokens`, non inclus par défaut.


```yaml filename="librechat.yaml"
    - name: "cohere"
      apiKey: "${COHERE_API_KEY}"
      baseURL: "https://api.cohere.ai/v1"
      models:
        default: ["command-r","command-r-plus","command-light","command-light-nightly","command","command-nightly"]
        fetch: false
      modelDisplayLabel: "cohere"
      titleModel: "command"
      dropParams: ["stop", "user", "frequency_penalty", "presence_penalty", "temperature", "top_p"]
```
