> Clé API Moonshot : [platform.moonshot.cn](https://platform.moonshot.ai)

**Notes :**

- **Connu :** icône fournie.
- **Important :** Pour les modèles dotés de capacités de raisonnement/réflexion (par exemple, `kimi-k2.5`, `kimi-k2-thinking`), le `name` de l'endpoint **doit** être défini sur `"Moonshot"` (insensible à la casse) pour que le raisonnement entrelacé fonctionne correctement avec les appels d'outils. L'utilisation d'un nom différent entraînera des erreurs telles que `thinking is enabled but reasoning_content is missing in assistant tool call message`. Consultez la [documentation de Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) pour plus de détails.

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```
