Moonshot AI propose ses modèles Kimi via une API compatible avec OpenAI que vous pouvez ajouter à LibreChat en tant qu'endpoint personnalisé.

## Obtenir une clé API [#get-an-api-key]

Créez une clé depuis la [Moonshot platform](https://platform.moonshot.ai). Ajoutez-la à votre fichier `.env` :

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## Configuration [#configuration]

Ajoutez l'endpoint sous `endpoints.custom` dans votre `librechat.yaml` :

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## Notes [#notes]

- Pour les modèles dotés de capacités de raisonnement tels que `kimi-k2.5` et `kimi-k2-thinking`, le `name` de l'endpoint doit être défini sur `"Moonshot"` (insensible à la casse) pour que le raisonnement entrelacé fonctionne avec les appels d'outils. Un nom différent provoque des erreurs telles que `thinking is enabled but reasoning_content is missing in assistant tool call message`. Consultez la [documentation de Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) pour plus de détails.
