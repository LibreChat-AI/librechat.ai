> Deepseek API-Schlüssel: [platform.deepseek.com](https://platform.deepseek.com/usage)

**Hinweise:**

- **Bekannt:** Symbol bereitgestellt.
- `deepseek-chat` und `deepseek-coder` sind kompatibel mit [Agents + tools](/docs/features/agents).
- `deepseek-reasoner`, mit dem Codenamen „R1“, wird unterstützt und streamt seinen „Gedankengang“; bestimmte OpenAI API-Parameter werden jedoch von diesem spezifischen Modell möglicherweise nicht unterstützt.
- „R1“ ist möglicherweise auch mit Agents kompatibel, wenn keine Tools verwendet werden.
- `deepseek-chat` wird für die Titelgenerierung bevorzugt.

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```
