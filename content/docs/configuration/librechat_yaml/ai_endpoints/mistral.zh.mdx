> Mistral API key: [console.mistral.ai](https://console.mistral.ai/)

**注意：**

- **已知：** 提供图标，消息角色的特殊处理：系统消息仅允许出现在消息负载的顶部。

- API 对无法识别的参数要求严格，且错误信息描述不明确（通常为 "no body"）。

    - 必须使用 [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams) 来丢弃 "user"、"frequency_penalty" 和 "presence_penalty" 参数。
    - `stop` 不再作为默认参数包含在内，因此除非您希望完全禁止用户配置此字段，否则无需将其包含在 [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams) 中。
    
- 允许获取模型列表，但请注意不要将嵌入模型（embedding models）用于聊天。

```yaml filename="librechat.yaml"
    - name: "Mistral"
      apiKey: "${MISTRAL_API_KEY}"
      baseURL: "https://api.mistral.ai/v1"
      models:
        default: ["mistral-tiny", "mistral-small", "mistral-medium", "mistral -large-latest"]
        fetch: true
      titleConvo: true
      titleModel: "mistral-tiny"
      modelDisplayLabel: "Mistral"
      dropParams: ["stop", "user", "frequency_penalty", "presence_penalty"]
```
