APIpie ist ein Aggregator, der Modelle vieler Anbieter über einen einzigen OpenAI-kompatiblen endpoint bereitstellt.

## Einen API-Schlüssel erhalten [#get-an-api-key]

Erstellen Sie einen Schlüssel unter [apipie.ai/dashboard/profile/api-keys](https://apipie.ai/dashboard/profile/api-keys) und fügen Sie ihn Ihrer `.env` Datei hinzu:

```bash filename=".env"
APIPIE_API_KEY=your-api-key
```

## Konfiguration [#configuration]

Fügen Sie den endpoint unter `endpoints.custom` in Ihrer `librechat.yaml` hinzu:

```yaml filename="librechat.yaml"
    - name: "APIpie"
      apiKey: "${APIPIE_API_KEY}"
      baseURL: "https://apipie.ai/v1/"
      models:
        default: [
          "gpt-4",
          "gpt-4-turbo",
          "gpt-3.5-turbo",
          "claude-3-opus",
          "claude-3-sonnet",
          "claude-3-haiku",
          "llama-3-70b-instruct",
          "llama-3-8b-instruct",
          "gemini-pro-1.5",
          "gemini-pro",
          "mistral-large",
          "mistral-medium",
          "mistral-small",
          "mistral-tiny",
          "mixtral-8x22b",
          ]
        fetch: false
      titleConvo: true
      titleModel: "claude-3-haiku"
      summarize: false
      summaryModel: "claude-3-haiku"
      dropParams: ["stream"]
      modelDisplayLabel: "APIpie"
```

<Callout type="tip" title="Modelle abrufen und sortieren" collapsible>
Dieses Python-Skript kann die LLM-Modelle für dich abrufen und sortieren. Die Ausgabe wird in `models.txt` gespeichert und ist so formatiert, dass sie einfacher in die yaml-Konfiguration eingefügt werden kann.

```py filename="fetch.py"
import json
import requests

def fetch_and_order_models():
    # API endpoint
    url = "https://apipie.ai/models"

    # headers as per request example
    headers = {"Accept": "application/json"}

    # request parameters
    params = {"type": "llm"}

    # make request
    response = requests.get(url, headers=headers, params=params)

    # parse JSON response
    data = response.json()

    # extract an ordered list of unique model IDs
    model_ids = sorted(set([model["id"] for model in data]))

    # write result to a text file
    with open("models.txt", "w") as file:
        json.dump(model_ids, file, indent=2)

# execute the function
if __name__ == "__main__":
    fetch_and_order_models()
```
</Callout>

## Hinweise [#notes]

- Das automatische Abrufen von Modellen wird nicht unterstützt, daher ist `fetch` auf `false` gesetzt und die Modellliste wird inline definiert. Verwenden Sie das obige Skript, um die Liste neu zu generieren, wenn neue Modelle hinzugefügt werden.
- Die Ergebnisse der Konversationsbetitelung können inkonsistent sein.
- `dropParams: ["stream"]` entfernt den `stream`-Parameter, der derzeit nicht unterstützt wird.
