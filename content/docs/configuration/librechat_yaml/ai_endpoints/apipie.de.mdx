> **APIpie API-Schlüssel:** [apipie.ai/dashboard/profile/api-keys](https://apipie.ai/dashboard/profile/api-keys)

**Hinweise:**

- **Bekannt:** Ein Icon wird bereitgestellt; das Abrufen der Modellliste wird empfohlen, da API-Token-Raten und Preise für Token-Guthaben verwendet werden, wenn Modelle abgerufen werden.

- **Bekanntes Problem:**
  - Das Abrufen der Modellliste wird nicht unterstützt.
  - Ihr Erfolg bei der Benennung von Unterhaltungen kann variieren.
  - Streaming wird derzeit nicht unterstützt (ist jedoch seit dem 24. April 2024 geplant)

<Callout type="tip" title="Modelle abrufen und sortieren" collapsible>
Dieses Python-Skript kann die LLM-Modelle für dich abrufen und sortieren. Die Ausgabe wird in models.txt gespeichert und so formatiert, dass sie für dich einfacher in die yaml-Konfiguration zu übernehmen ist.

```py filename="fetch.py"
import json
import requests

def fetch_and_order_models():
    # API endpoint
    url = "https://apipie.ai/models"

    # headers as per request example
    headers = {"Accept": "application/json"}

    # request parameters
    params = {"type": "llm"}

    # make request
    response = requests.get(url, headers=headers, params=params)

    # parse JSON response
    data = response.json()

    # extract an ordered list of unique model IDs
    model_ids = sorted(set([model["id"] for model in data]))

    # write result to a text file
    with open("models.txt", "w") as file:
        json.dump(model_ids, file, indent=2)

# execute the function
if __name__ == "__main__":
    fetch_and_order_models()
```
</Callout>
