Helicone è un gateway AI che instrada le richieste verso modelli di OpenAI, Anthropic, Google, Meta, Mistral e altri provider attraverso un singolo endpoint compatibile con OpenAI, con registrazione delle richieste e analisi dell'utilizzo integrate.

## Ottieni una API key [#get-an-api-key]

Crea un account su [helicone.ai](https://helicone.ai/) e genera una chiave da [Settings → API Keys](https://us.helicone.ai/settings/api-keys). Aggiungila al tuo file `.env`:

```bash filename=".env"
HELICONE_KEY=your-api-key
```

Assicurati che il tuo account abbia dei crediti, altrimenti le richieste al modello falliranno.

## Configurazione [#configuration]

Aggiungi l'endpoint sotto `endpoints.custom` nel tuo `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Helicone"
      apiKey: "${HELICONE_KEY}"
      baseURL: "https://ai-gateway.helicone.ai"
      headers:
        x-librechat-body-parentmessageid: "{{LIBRECHAT_BODY_PARENTMESSAGEID}}"
      models:
        default: ["gpt-4o-mini", "claude-4.5-sonnet", "llama-3.1-8b-instruct", "gemini-2.5-flash-lite"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-4o-mini"
      modelDisplayLabel: "Helicone"
      iconURL: "https://marketing-assets-helicone.s3.us-west-2.amazonaws.com/helicone.png"
```

## Note [#notes]

- Con `fetch: true`, LibreChat carica l'elenco completo dei modelli da Helicone, quindi l'array `default` rappresenta solo la selezione iniziale. Esplora tutto ciò che è disponibile nella [libreria di modelli Helicone](https://helicone.ai/models).
- L'header `x-librechat-body-parentmessageid` trasmette l'ID del messaggio padre di ogni richiesta a Helicone, in modo che i log possano essere raggruppati per conversazione.
- Imposta i limiti di frequenza (rate limits), le politiche di caching e controlla l'utilizzo dalla [Helicone dashboard](https://us.helicone.ai/dashboard).
