> Deepseek API key: [platform.deepseek.com](https://platform.deepseek.com/usage)

**注記:**

- **既知:** アイコンが提供されています。
- `deepseek-chat` および `deepseek-coder` は、[Agents + tools](/docs/features/agents) と互換性があります。
- `deepseek-reasoner`（コードネーム "R1"）はサポートされており、その「思考プロセス」をストリーミングしますが、この特定のモデルでは一部の OpenAI API パラメータがサポートされていない場合があります。
- "R1" は、ツールを使用しない場合に Agents と互換性がある可能性があります。
- `deepseek-chat` はタイトル生成に推奨されます。

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```
