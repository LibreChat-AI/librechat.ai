> Moonshot API key: [platform.moonshot.cn](https://platform.moonshot.ai)

**注意：**

- **已知：** 已提供图标。
- **重要提示：** 对于具有推理/思考能力的模型（例如 `kimi-k2.5`、`kimi-k2-thinking`），endpoint 的 `name` **必须**设置为 `"Moonshot"`（不区分大小写），以便交错推理（interleaved reasoning）能够与工具调用正常配合。使用其他名称将导致类似 `thinking is enabled but reasoning_content is missing in assistant tool call message` 的错误。详情请参阅 [Moonshot 的文档](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions)。

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```
