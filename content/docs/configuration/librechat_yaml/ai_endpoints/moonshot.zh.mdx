Moonshot AI 通过 OpenAI 兼容的 API 提供其 Kimi 模型，你可以将其作为自定义 endpoint 添加到 LibreChat 中。

## 获取 API key [#get-an-api-key]

从 [Moonshot platform](https://platform.moonshot.ai) 创建一个 key。将其添加到你的 `.env` 文件中：

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## 配置 [#configuration]

在你的 `librechat.yaml` 中，将 endpoint 添加到 `endpoints.custom` 下：

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## 注意事项 [#notes]

- 对于具有推理能力的模型（如 `kimi-k2.5` 和 `kimi-k2-thinking`），endpoint 的 `name` 必须设置为 `"Moonshot"`（不区分大小写），以便交错推理（interleaved reasoning）能够与工具调用（tool calls）协同工作。使用其他名称会导致类似 `thinking is enabled but reasoning_content is missing in assistant tool call message` 的错误。详情请参阅 [Moonshot 的文档](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions)。
