TrueFoundry AI Gateway es la capa de proxy que se sitúa entre sus aplicaciones y los proveedores de LLM y servidores MCP. Es una plataforma de nivel empresarial que permite a los usuarios acceder a más de 1000 LLMs utilizando una interfaz unificada, al tiempo que se encarga de la observabilidad y la gobernanza.

## Detalles de configuración [#configuration-details]

Para utilizar [TrueFoundry's AI Gateway](https://www.truefoundry.com/ai-gateway), sigue la guía de inicio rápido [aquí](https://docs.truefoundry.com/gateway/quick-start).

### Obtención de la URL base y los nombres de los modelos [#getting-base-url-and-model-names]

- **Base URL y nombres de modelo**: Obtenga su URL de endpoint de TrueFoundry AI Gateway y el nombre del modelo desde el fragmento de código unificado (asegúrese de haber añadido el mismo nombre de modelo)
- **PAT**: Generar a partir de su Personal Access Token [PAT](https://docs.truefoundry.com/gateway/authentication#personal-access-token-pat) de TrueFoundry

## Uso de TrueFoundry AI Gateway con cualquier modelo LLM [#using-truefoundry-ai-gateway-with-any-llm-model]
```yaml filename="librechat.yaml"
      - name: "TrueFoundry"
        apiKey: "${TRUEFOUNDRY_API_KEY}"
        baseURL: "${TRUEFOUNDRY_GATEWAY_URL}"
        models:
            default: ["openai-main/gpt-4o-mini", "openai-main/gpt-4o"]
            fetch: true
        titleConvo: true
        titleModel: "current_model"
        summarize: false
        summaryModel: "current_model"
        modelDisplayLabel: "TrueFoundry:OpenAI"
```
Para más detalles, puedes consultar: [TrueFoundry Docs](https://docs.truefoundry.com/docs/introduction)
