NeurochainAI는 OpenAI 호환 추론 네트워크입니다. 아래의 `<=generated api key=>`를 [NeurochainAI REST API 문서](https://app.neurochain.ai/network-integrations/rest-api)에서 생성한 키로 교체하세요.

## 구성 [#configuration]

`librechat.yaml`의 `endpoints.custom` 아래에 해당 endpoint를 추가하세요:

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## 참고 사항 [#notes]

- 모델 목록은 시간이 지남에 따라 늘어나므로 예시가 최신 상태가 아닐 수 있습니다. 최신 모델은 [NeurochainAI](https://neurochain.ai)에서 확인하세요.
