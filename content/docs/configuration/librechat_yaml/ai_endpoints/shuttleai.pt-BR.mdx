O ShuttleAI fornece acesso aos seus modelos Shuttle por meio de uma API compatível com OpenAI, utilizada no LibreChat como um endpoint personalizado.

## Obtenha uma API key [#get-an-api-key]

Crie uma chave na [página de chaves do ShuttleAI](https://shuttleai.com/keys). Adicione-a ao seu arquivo `.env`:

```bash filename=".env"
SHUTTLEAI_API_KEY=your-api-key
```

## Configuração [#configuration]

Adicione o endpoint em `endpoints.custom` no seu `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
