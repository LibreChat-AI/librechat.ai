Deepseek menyajikan model chat, coder, dan reasoner-nya melalui API yang kompatibel dengan OpenAI yang dapat Anda tambahkan ke LibreChat sebagai custom endpoint.

## Dapatkan API key [#get-an-api-key]

Buat kunci dari [Deepseek platform](https://platform.deepseek.com/usage). Tambahkan ke file `.env` Anda:

```bash filename=".env"
DEEPSEEK_API_KEY=your-api-key
```

## Konfigurasi [#configuration]

Tambahkan endpoint di bawah `endpoints.custom` dalam `librechat.yaml` Anda:

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```

## Catatan [#notes]

- `deepseek-chat` dan `deepseek-coder` berfungsi dengan [Agents and tools](/docs/features/agents).
- `deepseek-reasoner`, dengan nama kode "R1," didukung dan melakukan streaming pada proses pemikirannya, namun beberapa parameter OpenAI API mungkin tidak berfungsi dengannya. R1 juga dapat bekerja dengan Agents saat tools tidak digunakan.
- `deepseek-chat` adalah model yang disarankan untuk pembuatan judul.
