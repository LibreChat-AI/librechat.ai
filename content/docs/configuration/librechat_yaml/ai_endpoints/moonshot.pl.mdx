Moonshot AI udostępnia swoje modele Kimi poprzez API zgodne z OpenAI, które możesz dodać do LibreChat jako niestandardowy endpoint.

## Uzyskaj klucz API [#get-an-api-key]

Utwórz klucz z [platformy Moonshot](https://platform.moonshot.ai). Dodaj go do swojego pliku `.env`:

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## Konfiguracja [#configuration]

Dodaj endpoint w sekcji `endpoints.custom` w swoim `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## Uwagi [#notes]

- W przypadku modeli z możliwościami wnioskowania, takich jak `kimi-k2.5` oraz `kimi-k2-thinking`, `name` endpointu musi być ustawione na `"Moonshot"` (wielkość liter nie ma znaczenia), aby przeplatane wnioskowanie działało z wywołaniami narzędzi. Inna nazwa spowoduje błędy typu `thinking is enabled but reasoning_content is missing in assistant tool call message`. Szczegółowe informacje znajdują się w [dokumentacji Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions).
