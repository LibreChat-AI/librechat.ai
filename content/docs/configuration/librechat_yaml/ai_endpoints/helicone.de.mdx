Helicone ist ein KI-Gateway, das Anfragen an Modelle von OpenAI, Anthropic, Google, Meta, Mistral und anderen Anbietern über einen einzigen OpenAI-kompatiblen endpoint leitet, mit integrierter Anfrageprotokollierung und Nutzungsanalysen.

## Einen API-Schlüssel erhalten [#get-an-api-key]

Erstellen Sie ein Konto bei [helicone.ai](https://helicone.ai/) und generieren Sie einen Schlüssel unter [Settings → API Keys](https://us.helicone.ai/settings/api-keys). Fügen Sie ihn Ihrer `.env` Datei hinzu:

```bash filename=".env"
HELICONE_KEY=your-api-key
```

Stellen Sie sicher, dass Ihr Konto über Guthaben verfügt, da Modellanfragen andernfalls fehlschlagen.

## Konfiguration [#configuration]

Fügen Sie den endpoint unter `endpoints.custom` in Ihrer `librechat.yaml` hinzu:

```yaml filename="librechat.yaml"
    - name: "Helicone"
      apiKey: "${HELICONE_KEY}"
      baseURL: "https://ai-gateway.helicone.ai"
      headers:
        x-librechat-body-parentmessageid: "{{LIBRECHAT_BODY_PARENTMESSAGEID}}"
      models:
        default: ["gpt-4o-mini", "claude-4.5-sonnet", "llama-3.1-8b-instruct", "gemini-2.5-flash-lite"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-4o-mini"
      modelDisplayLabel: "Helicone"
      iconURL: "https://marketing-assets-helicone.s3.us-west-2.amazonaws.com/helicone.png"
```

## Hinweise [#notes]

- Mit `fetch: true` lädt LibreChat die vollständige Modellliste von Helicone, sodass das `default`-Array nur die anfängliche Auswahl darstellt. Durchsuchen Sie alles, was in der [Helicone model library](https://helicone.ai/models) verfügbar ist.
- Der `x-librechat-body-parentmessageid` Header übergibt die übergeordnete Nachrichten-ID jeder Anfrage an Helicone, damit Protokolle nach Konversation gruppiert werden können.
- Legen Sie Ratenbegrenzungen, Caching-Richtlinien fest und überprüfen Sie die Nutzung über das [Helicone dashboard](https://us.helicone.ai/dashboard).
