Deepseek, chat, coder ve reasoner modellerini, LibreChat'e özel bir endpoint olarak ekleyebileceğiniz OpenAI uyumlu bir API aracılığıyla sunar.

## Bir API anahtarı alın [#get-an-api-key]

[Deepseek platform](https://platform.deepseek.com/usage) üzerinden bir anahtar oluşturun. Bunu `.env` dosyanıza ekleyin:

```bash filename=".env"
DEEPSEEK_API_KEY=your-api-key
```

## Yapılandırma [#configuration]

`librechat.yaml` dosyanızda `endpoints.custom` altına endpoint'i ekleyin:

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```

## Notlar [#notes]

- `deepseek-chat` ve `deepseek-coder`, [Agents and tools](/docs/features/agents) ile çalışır.
- `deepseek-reasoner` (kod adı "R1") desteklenmektedir ve düşünce sürecini akış olarak sunar, ancak bazı OpenAI API parametreleri bununla çalışmayabilir. R1, araçlar kullanılmadığında Ajanlarla (Agents) da çalışabilir.
- `deepseek-chat`, başlık oluşturma için tercih edilen modeldir.
