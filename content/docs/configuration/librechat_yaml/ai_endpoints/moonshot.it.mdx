Moonshot AI fornisce i suoi modelli Kimi tramite un'API compatibile con OpenAI che puoi aggiungere a LibreChat come endpoint personalizzato.

## Ottieni una API key [#get-an-api-key]

Crea una chiave dalla [piattaforma Moonshot](https://platform.moonshot.ai). Aggiungila al tuo file `.env`:

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## Configurazione [#configuration]

Aggiungi l'endpoint sotto `endpoints.custom` nel tuo `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## Note [#notes]

- Per i modelli con capacità di ragionamento come `kimi-k2.5` e `kimi-k2-thinking`, il `name` dell'endpoint deve essere impostato su `"Moonshot"` (senza distinzione tra maiuscole e minuscole) affinché il ragionamento interlacciato funzioni con le chiamate agli strumenti. Un nome diverso causa errori come `thinking is enabled but reasoning_content is missing in assistant tool call message`. Consulta la [documentazione di Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) per i dettagli.
