ShuttleAI는 LibreChat에서 커스텀 endpoint로 사용되는 OpenAI 호환 API를 통해 Shuttle 모델에 대한 액세스를 제공합니다.

## API 키 발급받기 [#get-an-api-key]

[ShuttleAI keys page](https://shuttleai.com/keys)에서 키를 생성하세요. 해당 키를 `.env` 파일에 추가하세요:

```bash filename=".env"
SHUTTLEAI_API_KEY=your-api-key
```

## 구성 [#configuration]

`librechat.yaml`의 `endpoints.custom` 아래에 해당 endpoint를 추가하세요:

```yaml filename="librechat.yaml"
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
