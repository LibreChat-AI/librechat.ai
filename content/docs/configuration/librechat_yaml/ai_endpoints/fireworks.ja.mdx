> Fireworks API key: [fireworks.ai/api-keys](https://fireworks.ai/api-keys)

**注記:**

- **既知の事項:** アイコンが提供されており、モデルリストの取得が推奨されます。
- - APIによっては一部のモデルに対して厳格であり、`user`のようなフィールドを許可しない場合があります。その場合は、[`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams)を使用してください。

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false 
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```
