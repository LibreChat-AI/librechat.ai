NEAR AI Cloud cung cấp một endpoint tương thích với OpenAI mà bạn có thể thêm vào LibreChat thông qua `endpoints.custom`.

## Nhận API key [#get-an-api-key]

Tạo một API key từ [NEAR AI Cloud](https://cloud.near.ai/). Thêm nó vào tệp `.env` của bạn:

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## Cấu hình [#configuration]

Thêm endpoint vào dưới `endpoints.custom` trong tệp `librechat.yaml` của bạn:

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

Để cho phép mỗi người dùng tự cung cấp khóa riêng thông qua giao diện người dùng LibreChat thay vì đọc từ `.env`, hãy đặt `apiKey: "user_provided"`.

## Ghi chú [#notes]

- Với `fetch: true`, LibreChat sẽ tải danh sách model từ endpoint `/v1/models` tương thích với OpenAI của NEAR AI Cloud. Mảng `default` chỉ là lựa chọn ban đầu.
- Sử dụng các model ID chính xác như cách chúng xuất hiện trong [NEAR AI Cloud model catalog](https://cloud-api.near.ai/v1/model/list).
- Tính khả dụng của các model có thể thay đổi theo thời gian. Nếu một model đã chọn không khả dụng, hãy chọn một model có khả năng trò chuyện khác từ danh mục.
