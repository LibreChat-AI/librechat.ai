Deepseek udostępnia swoje modele chat, coder oraz reasoner poprzez API zgodne z OpenAI, które możesz dodać do LibreChat jako niestandardowy endpoint.

## Uzyskaj klucz API [#get-an-api-key]

Utwórz klucz z [platformy Deepseek](https://platform.deepseek.com/usage). Dodaj go do swojego pliku `.env`:

```bash filename=".env"
DEEPSEEK_API_KEY=your-api-key
```

## Konfiguracja [#configuration]

Dodaj endpoint w sekcji `endpoints.custom` w swoim `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```

## Uwagi [#notes]

- `deepseek-chat` oraz `deepseek-coder` współpracują z [Agentami i narzędziami](/docs/features/agents).
- `deepseek-reasoner`, o kryptonimie "R1", jest obsługiwany i przesyła strumieniowo swój proces myślowy, jednak niektóre parametry OpenAI API mogą z nim nie działać. R1 może również współpracować z Agents, gdy narzędzia nie są używane.
- `deepseek-chat` jest preferowanym modelem do generowania tytułów.
