> Clave de API de Groq: [wow.groq.com](https://console.groq.com/keys)

**Notas:**

- **Conocido:** icono proporcionado.

- **Temperature:** Si estableces un valor de temperatura de 0, se convertirá a 1e-8. Si tienes algún problema, intenta establecer el valor en un float32 mayor que 0 y menor o igual a 2.

- Groq es actualmente gratuito pero tiene límites de tasa: 10 consultas/minuto, 100/hora.

```yaml filename="librechat.yaml"
    - name: "groq"
      apiKey: "${GROQ_API_KEY}"
      baseURL: "https://api.groq.com/openai/v1/"
      models:
        default: [
          "llama3-70b-8192",
          "llama3-8b-8192",
          "llama2-70b-4096",
          "mixtral-8x7b-32768",
          "gemma-7b-it",
          ]
        fetch: false
      titleConvo: true
      titleModel: "mixtral-8x7b-32768"
      modelDisplayLabel: "groq"
```
