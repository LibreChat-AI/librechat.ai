NEAR AI Cloud stellt einen OpenAI-kompatiblen endpoint bereit, den Sie in LibreChat über `endpoints.custom` hinzufügen können.

## Einen API-Schlüssel erhalten [#get-an-api-key]

Erstellen Sie einen API-Schlüssel von [NEAR AI Cloud](https://cloud.near.ai/). Fügen Sie ihn Ihrer `.env` Datei hinzu:

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## Konfiguration [#configuration]

Fügen Sie den endpoint unter `endpoints.custom` in Ihrer `librechat.yaml` hinzu:

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

Damit jeder Benutzer seinen eigenen Schlüssel über die LibreChat UI eingeben kann, anstatt ihn aus der `.env` zu lesen, setzen Sie `apiKey: "user_provided"`.

## Hinweise [#notes]

- Mit `fetch: true` lädt LibreChat die Modellliste vom OpenAI-kompatiblen `/v1/models` endpoint der NEAR AI Cloud. Das `default` Array ist lediglich die anfängliche Auswahl.
- Verwenden Sie die Modell-IDs exakt so, wie sie im [NEAR AI Cloud model catalog](https://cloud-api.near.ai/v1/model/list) aufgeführt sind.
- Die Verfügbarkeit von Modellen kann sich im Laufe der Zeit ändern. Wenn ein ausgewähltes Modell nicht verfügbar ist, wählen Sie ein anderes chatfähiges Modell aus dem Katalog.
