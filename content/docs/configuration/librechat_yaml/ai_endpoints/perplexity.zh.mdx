> Perplexity API key: [perplexity.ai/settings/api](https://www.perplexity.ai/settings/api)

**注意：**

- **已知：** 已提供图标。
- **已知问题：** 不支持获取模型列表。
- 对于某些模型，API 的要求可能非常严格，并且可能不允许使用 `stop` 等字段；当 `frequency_penalty` 设置为 0 时也可能会导致错误，在这种情况下，您应该使用 [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams)。
- 该示例包含一个模型列表，为了方便起见，该列表最后更新于 2024 年 7 月 3 日。

```yaml
    - name: "Perplexity"
      apiKey: "${PERPLEXITY_API_KEY}"
      baseURL: "https://api.perplexity.ai/"
      models:
        default: [
          "sonar-deep-research",
          "sonar-reasoning-pro",
          "sonar-reasoning",
          "sonar-pro",
          "sonar",
          "r1-1776"
          ]
        fetch: false # fetching list of models is not supported
      titleConvo: true
      titleModel: "llama-3-sonar-small-32k-chat"
      summarize: false
      summaryModel: "llama-3-sonar-small-32k-chat"
      dropParams: ["stop", "frequency_penalty"]
      modelDisplayLabel: "Perplexity"
```
