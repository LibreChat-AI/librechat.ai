> Portkey API-Schlüssel: [app.portkey.ai](https://app.portkey.ai/)


**Hinweise:**
- LibreChat erfordert, dass das Feld `API key` vorhanden ist. Wir benötigen es nicht für die Portkey-Integration, wir können einen `dummy` String dafür übergeben.
- Portkey lässt sich in LibreChat integrieren und bietet Observability, über 50 Guardrails, Caching sowie bedingtes Routing mit Fallbacks und Wiederholungsversuchen für zuverlässige, produktionsreife Deployments.
- Portkey unterstützt über 250 Modelle. Die Liste der Anbieter finden Sie in der [Portkey-Dokumentation](https://docs.portkey.ai/docs/integrations/llms)
- Sie können Portkey auf zwei Arten verwenden – über [Virtual Keys](https://docs.portkey.ai/docs/product/ai-gateway/virtual-keys) oder [Configs](https://docs.portkey.ai/docs/product/ai-gateway/configs).
- Sie können modellspezifische Parameter wie `top_p`, `max Tokens` usw. bei der Verwendung von Portkey Config nutzen. [Mehr erfahren](https://docs.portkey.ai/docs/product/ai-gateway/configs#configs)


## Verwendung von Portkey AI mit virtuellen Schlüsseln [#using-portkey-ai-with-virtual-keys]
```yaml filename="librechat.yaml"
      - name: "Portkey"
        apiKey: "dummy"  
        baseURL: ${PORTKEY_GATEWAY_URL}
        headers:
            x-portkey-api-key: "${PORTKEY_API_KEY}"
            x-portkey-virtual-key: "PORTKEY_OPENAI_VIRTUAL_KEY"
        models:
            default: ["gpt-4o-mini"]
            fetch: true
        titleConvo: true
        titleModel: "current_model"
        summarize: false
        summaryModel: "current_model"
        modelDisplayLabel: "Portkey:OpenAI"
        iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```


## Verwendung von Portkey AI mit Configs [#using-portkey-ai-with-configs]
```yaml filename="librechat.yaml"
    - name: "Portkey"
      apikey: "dummy"
      baseURL: ${PORTKEY_GATEWAY_URL}
      headers:
        x-portkey-api-key: "${PORTKEY_API_KEY}"
        x-portkey-config: "pc-libre-xxx"
      models:
        default: ["llama-3.2"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "Portkey:Llama"
      iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```
