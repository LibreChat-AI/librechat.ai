> Mistral API-Schlüssel: [console.mistral.ai](https://console.mistral.ai/)

**Hinweise:**

- **Bekannt:** Icon bereitgestellt, spezielle Handhabung von Nachrichten-Rollen: System-Nachrichten sind nur am Anfang des Nachrichten-Payloads erlaubt.

- Die API ist streng bei nicht erkannten Parametern und Fehlermeldungen sind nicht aussagekräftig (meistens "no body").

    - Die Verwendung von [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams), um die Parameter "user", "frequency_penalty" und "presence_penalty" zu entfernen, ist erforderlich.
    - `stop` ist nicht mehr als Standardparameter enthalten, daher ist es nicht mehr erforderlich, ihn in [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams) aufzunehmen, es sei denn, Sie möchten Benutzer vollständig daran hindern, dieses Feld zu konfigurieren.
    
- Ermöglicht das Abrufen der Modellliste, aber achten Sie darauf, keine Embedding-Modelle für den Chat zu verwenden.

```yaml filename="librechat.yaml"
    - name: "Mistral"
      apiKey: "${MISTRAL_API_KEY}"
      baseURL: "https://api.mistral.ai/v1"
      models:
        default: ["mistral-tiny", "mistral-small", "mistral-medium", "mistral -large-latest"]
        fetch: true
      titleConvo: true
      titleModel: "mistral-tiny"
      modelDisplayLabel: "Mistral"
      dropParams: ["stop", "user", "frequency_penalty", "presence_penalty"]
```
