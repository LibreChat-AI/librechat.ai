> groq API key: [wow.groq.com](https://console.groq.com/keys)

**注意：**

- **已知：** 已提供图标。

- **Temperature:** 如果您将 temperature 值设置为 0，它将被转换为 1e-8。如果您遇到任何问题，请尝试将该值设置为大于 0 且小于或等于 2 的 float32。

- Groq 目前免费，但有速率限制：每分钟 10 次查询，每小时 100 次。

```yaml filename="librechat.yaml"
    - name: "groq"
      apiKey: "${GROQ_API_KEY}"
      baseURL: "https://api.groq.com/openai/v1/"
      models:
        default: [
          "llama3-70b-8192",
          "llama3-8b-8192",
          "llama2-70b-4096",
          "mixtral-8x7b-32768",
          "gemma-7b-it",
          ]
        fetch: false
      titleConvo: true
      titleModel: "mixtral-8x7b-32768"
      modelDisplayLabel: "groq"
```
