Deepseek offre i suoi modelli chat, coder e reasoner tramite un'API compatibile con OpenAI che puoi aggiungere a LibreChat come endpoint personalizzato.

## Ottieni una API key [#get-an-api-key]

Crea una chiave dalla [piattaforma Deepseek](https://platform.deepseek.com/usage). Aggiungila al tuo file `.env`:

```bash filename=".env"
DEEPSEEK_API_KEY=your-api-key
```

## Configurazione [#configuration]

Aggiungi l'endpoint sotto `endpoints.custom` nel tuo `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```

## Note [#notes]

- `deepseek-chat` e `deepseek-coder` funzionano con [Agenti e strumenti](/docs/features/agents).
- `deepseek-reasoner`, nome in codice "R1", è supportato e trasmette in streaming il suo processo di pensiero, ma alcuni parametri dell'API OpenAI potrebbero non funzionare con esso. R1 può anche funzionare con gli Agents quando non vengono utilizzati strumenti.
- `deepseek-chat` è il modello preferito per la generazione dei titoli.
