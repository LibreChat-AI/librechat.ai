NeurochainAI es una red de inferencia compatible con OpenAI. Reemplaza `<=generated api key=>` a continuación con la clave que crees en la [documentación de la API REST de NeurochainAI](https://app.neurochain.ai/network-integrations/rest-api).

## Configuración [#configuration]

Agregue el endpoint bajo `endpoints.custom` en su `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## Notas [#notes]

- La lista de modelos crece con el tiempo, por lo que el ejemplo puede estar desactualizado. Consulta [NeurochainAI](https://neurochain.ai) para ver los modelos más recientes.
