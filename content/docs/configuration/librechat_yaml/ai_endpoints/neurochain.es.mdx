> Clave de API de NeurochainAI: Requerido - [Documentación de la API REST de NeurochainAI](https://app.neurochain.ai/network-integrations/rest-api)

**Notas:**

- La API se basa en la OpenAI API.
- La lista de modelos crece constantemente y puede no estar actualizada en este ejemplo. Mantente al tanto de [NeurochainAI](https://neurochain.ai) para conocer los modelos más recientes.
- El ejemplo incluye el modelo NeurochainAI: "Mistral-7B-OpenOrca-GPTQ".

```yaml filename="neurochainai.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```
