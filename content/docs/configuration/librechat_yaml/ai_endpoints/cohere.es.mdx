> Clave de API de Cohere: [dashboard.cohere.com](https://dashboard.cohere.com/)

**Notas:**

- **Conocido:** icono proporcionado.
- Experimental: no sigue la especificación de OpenAI, utiliza un nuevo método para la compatibilidad de endpoint y comparte algunas similitudes y parámetros.
- Para obtener una lista completa de los parámetros específicos de Cohere, consulte la [documentación de la API de Cohere](https://docs.cohere.com/reference/chat).
- Nota: Los siguientes parámetros son reconocidos entre OpenAI y Cohere. La mayoría se eliminan en la configuración de ejemplo a continuación para preferir los ajustes predeterminados de Cohere:
    - `stop`: mapeado a `stopSequences`
    - `top_p`: mapeado a `p`, diferentes valores mín/máx
    - `frequency_penalty`: asignado a `frequencyPenalty`, valores mínimos/máximos diferentes
    - `presence_penalty`: mapeado a `presencePenalty`, diferentes valores mín/máx
    - `model`: compartido, incluido de forma predeterminada.
    - `stream`: compartido, incluido de forma predeterminada.
    - `max_tokens`: compartido, mapeado a `maxTokens`, no incluido por defecto.


```yaml filename="librechat.yaml"
    - name: "cohere"
      apiKey: "${COHERE_API_KEY}"
      baseURL: "https://api.cohere.ai/v1"
      models:
        default: ["command-r","command-r-plus","command-light","command-light-nightly","command","command-nightly"]
        fetch: false
      modelDisplayLabel: "cohere"
      titleModel: "command"
      dropParams: ["stop", "user", "frequency_penalty", "presence_penalty", "temperature", "top_p"]
```
