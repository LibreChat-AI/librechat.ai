NEAR AI Cloud, LibreChat'e `endpoints.custom` aracılığıyla ekleyebileceğiniz OpenAI uyumlu bir endpoint sağlar.

## Bir API anahtarı alın [#get-an-api-key]

[NEAR AI Cloud](https://cloud.near.ai/) üzerinden bir API anahtarı oluşturun. Bunu `.env` dosyanıza ekleyin:

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## Yapılandırma [#configuration]

`librechat.yaml` dosyanızda `endpoints.custom` altına endpoint'i ekleyin:

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

Her kullanıcının `.env` dosyasından bir anahtar okumak yerine LibreChat arayüzü aracılığıyla kendi anahtarını sağlamasına izin vermek için `apiKey: "user_provided"` olarak ayarlayın.

## Notlar [#notes]

- `fetch: true` ile LibreChat, model listesini NEAR AI Cloud'un OpenAI uyumlu `/v1/models` endpoint'inden yükler. `default` dizisi yalnızca başlangıç seçimidir.
- Model kimliklerini [NEAR AI Cloud model kataloğunda](https://cloud-api.near.ai/v1/model/list) göründükleri şekliyle tam olarak kullanın.
- Model kullanılabilirliği zaman içinde değişebilir. Seçilen bir model kullanılamıyorsa, katalogdan sohbet destekli başka bir model seçin.
