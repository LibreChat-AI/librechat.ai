> Clave de API de Fireworks: [fireworks.ai/api-keys](https://fireworks.ai/api-keys)

**Notas:**

- **Conocido:** se proporciona el icono, se recomienda obtener la lista de modelos.
- - La API puede ser estricta con algunos modelos y podría no permitir campos como `user`, en cuyo caso, deberías usar [`dropParams`.](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams)

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false 
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```
