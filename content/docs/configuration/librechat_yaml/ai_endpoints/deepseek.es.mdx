Deepseek ofrece sus modelos de chat, coder y reasoner a través de una API compatible con OpenAI que puedes añadir a LibreChat como un endpoint personalizado.

## Obtener una API key [#get-an-api-key]

Cree una clave desde la [plataforma de Deepseek](https://platform.deepseek.com/usage). Añádala a su archivo `.env`:

```bash filename=".env"
DEEPSEEK_API_KEY=your-api-key
```

## Configuración [#configuration]

Agregue el endpoint bajo `endpoints.custom` en su `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```

## Notas [#notes]

- `deepseek-chat` y `deepseek-coder` funcionan con [Agents and tools](/docs/features/agents).
- `deepseek-reasoner`, cuyo nombre en clave es "R1", es compatible y transmite su proceso de razonamiento, pero es posible que algunos parámetros de la API de OpenAI no funcionen con él. R1 también puede funcionar con Agents cuando no se utilizan herramientas.
- `deepseek-chat` es el modelo preferido para la generación de títulos.
