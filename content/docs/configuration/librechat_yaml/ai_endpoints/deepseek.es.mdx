> Clave de API de Deepseek: [platform.deepseek.com](https://platform.deepseek.com/usage)

**Notas:**

- **Conocido:** icono proporcionado.
- `deepseek-chat` y `deepseek-coder` son compatibles con [Agents + tools](/docs/features/agents).
- `deepseek-reasoner`, cuyo nombre en clave es "R1", es compatible y transmitirá su "proceso de pensamiento"; sin embargo, es posible que ciertos parámetros de la API de OpenAI no sean compatibles con este modelo específico.
- "R1" también puede ser compatible con Agents cuando no se utilizan herramientas.
- `deepseek-chat` es preferido para la generación de títulos.

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```
