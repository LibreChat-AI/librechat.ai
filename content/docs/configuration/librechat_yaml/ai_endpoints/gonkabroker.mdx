[Gonka Broker](https://gonkabroker.com) is an OpenAI-compatible gateway to open-weight models such as Qwen3, Kimi, and MiniMax, served on the Gonka decentralized inference network. It exposes a public `/v1/models` endpoint, so LibreChat can auto-discover the available models.

## Get an API key

Create an account at [app.gonkabroker.com](https://app.gonkabroker.com/signup) and generate a key (it starts with `gnk-prx-`). Add it to your `.env` file:

```bash filename=".env"
GONKABROKER_API_KEY=gnk-prx-your-key-here
```

## Configuration

Add the endpoint under `endpoints.custom` in your `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Gonka Broker"
      apiKey: "${GONKABROKER_API_KEY}"
      baseURL: "https://proxy.gonkabroker.com/v1"
      models:
        default: ["Qwen/Qwen3-235B-A22B-Instruct-2507-FP8"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Gonka Broker"
      iconURL: "https://gonkabroker.com/img/gonka-broker-icon.svg"
```

To let each user supply their own key through the LibreChat UI instead of reading one from `.env`, set `apiKey: "user_provided"`. Users then see a key input field when they select the endpoint.

## Notes

- With `fetch: true`, LibreChat loads the current model list from the Gonka Broker `/v1/models` endpoint, so new models appear automatically. The `default` array is only the initial selection.
- API keys start with `gnk-prx-`. Create and manage them in the [Gonka Broker dashboard](https://app.gonkabroker.com).
- Prices are locked in at top-up. See [current pricing](https://gonkabroker.com/gonka-api-pricing/) for per-token rates.
