NeurochainAI adalah jaringan inferensi yang kompatibel dengan OpenAI. Ganti `<=generated api key=>` di bawah ini dengan kunci yang Anda buat di [dokumentasi REST API NeurochainAI](https://app.neurochain.ai/network-integrations/rest-api).

## Konfigurasi [#configuration]

Tambahkan endpoint di bawah `endpoints.custom` dalam `librechat.yaml` Anda:

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## Catatan [#notes]

- Daftar model terus bertambah seiring waktu, jadi contoh ini mungkin sudah usang. Periksa [NeurochainAI](https://neurochain.ai) untuk model terbaru.
