ShuttleAI fournit un accès à ses modèles Shuttle via une API compatible avec OpenAI, utilisée dans LibreChat en tant qu'endpoint personnalisé.

## Obtenir une clé API [#get-an-api-key]

Créez une clé depuis la [page des clés ShuttleAI](https://shuttleai.com/keys). Ajoutez-la à votre fichier `.env` :

```bash filename=".env"
SHUTTLEAI_API_KEY=your-api-key
```

## Configuration [#configuration]

Ajoutez l'endpoint sous `endpoints.custom` dans votre `librechat.yaml` :

```yaml filename="librechat.yaml"
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
