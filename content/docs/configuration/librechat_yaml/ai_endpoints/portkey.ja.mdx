> Portkey API key: [app.portkey.ai](https://app.portkey.ai/)


**注記:**
- LibreChatでは、`API key`フィールドが必須となっています。Portkey統合ではこのキーは不要なため、`dummy`という文字列を渡すことができます。
- PortkeyはLibreChatと統合されており、信頼性の高い本番環境向けのデプロイを実現するために、可観測性、50種類以上のガードレール、キャッシュ、およびフォールバックとリトライを備えた条件付きルーティングを提供します。
- Portkeyは250以上のモデルをサポートしています。プロバイダーのリストは[Portkey Docs](https://docs.portkey.ai/docs/integrations/llms)で確認できます。
- Portkeyは、[Virtual Keys](https://docs.portkey.ai/docs/product/ai-gateway/virtual-keys) または [Configs](https://docs.portkey.ai/docs/product/ai-gateway/configs) を通じて、2通りの方法で使用できます。
- Portkey Configを使用する際、`top_p`や`max Tokens`などのModel固有のパラメータを使用できます。[詳細はこちら](https://docs.portkey.ai/docs/product/ai-gateway/configs#configs)


## Virtual Keyを使用してPortkey AIを利用する [#using-portkey-ai-with-virtual-keys]
```yaml filename="librechat.yaml"
      - name: "Portkey"
        apiKey: "dummy"  
        baseURL: ${PORTKEY_GATEWAY_URL}
        headers:
            x-portkey-api-key: "${PORTKEY_API_KEY}"
            x-portkey-virtual-key: "PORTKEY_OPENAI_VIRTUAL_KEY"
        models:
            default: ["gpt-4o-mini"]
            fetch: true
        titleConvo: true
        titleModel: "current_model"
        summarize: false
        summaryModel: "current_model"
        modelDisplayLabel: "Portkey:OpenAI"
        iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```


## Configs を使用した Portkey AI の利用 [#using-portkey-ai-with-configs]
```yaml filename="librechat.yaml"
    - name: "Portkey"
      apikey: "dummy"
      baseURL: ${PORTKEY_GATEWAY_URL}
      headers:
        x-portkey-api-key: "${PORTKEY_API_KEY}"
        x-portkey-config: "pc-libre-xxx"
      models:
        default: ["llama-3.2"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "Portkey:Llama"
      iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```
