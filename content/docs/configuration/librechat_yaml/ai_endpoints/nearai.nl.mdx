NEAR AI Cloud biedt een OpenAI-compatibel endpoint dat je aan LibreChat kunt toevoegen via `endpoints.custom`.

## Een API-sleutel verkrijgen [#get-an-api-key]

Maak een API-sleutel aan via [NEAR AI Cloud](https://cloud.near.ai/). Voeg deze toe aan je `.env` bestand:

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## Configuratie [#configuration]

Voeg het endpoint toe onder `endpoints.custom` in je `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

Om elke gebruiker zijn eigen sleutel te laten opgeven via de LibreChat UI in plaats van er een te lezen uit `.env`, stel `apiKey: "user_provided"` in.

## Opmerkingen [#notes]

- Met `fetch: true` laadt LibreChat de modellijst van het OpenAI-compatibele `/v1/models` endpoint van NEAR AI Cloud. De `default` array is enkel de initiële selectie.
- Gebruik model-ID's exact zoals ze voorkomen in de [NEAR AI Cloud model catalog](https://cloud-api.near.ai/v1/model/list).
- De beschikbaarheid van modellen kan in de loop van de tijd veranderen. Als een geselecteerd model niet beschikbaar is, kies dan een ander chat-geschikt model uit de catalogus.
