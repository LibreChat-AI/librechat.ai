> NeurochainAI API key: 必須 - [NeurochainAI REST API Documentation](https://app.neurochain.ai/network-integrations/rest-api)

**注記:**

- ApiはOpenAI APIに基づいています。
- モデルリストは常に増え続けており、この例では最新の状態ではない可能性があります。最新のモデルについては [NeurochainAI](https://neurochain.ai) をご確認ください。
- この例には、NeurochainAIモデルである「Mistral-7B-OpenOrca-GPTQ」が含まれています。

```yaml filename="neurochainai.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```
