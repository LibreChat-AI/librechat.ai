NeurochainAIは、OpenAI互換の推論ネットワークです。以下の `<=generated api key=>` を、[NeurochainAI REST API documentation](https://app.neurochain.ai/network-integrations/rest-api) で作成したキーに置き換えてください。

## 設定 [#configuration]

`librechat.yaml` の `endpoints.custom` の下にエンドポイントを追加します：

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## 注記 [#notes]

- モデルリストは時間の経過とともに増加するため、この例は古くなっている可能性があります。最新のモデルについては [NeurochainAI](https://neurochain.ai) を確認してください。
