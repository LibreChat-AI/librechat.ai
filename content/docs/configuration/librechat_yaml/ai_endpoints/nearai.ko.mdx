NEAR AI Cloud는 LibreChat의 `endpoints.custom`을 통해 추가할 수 있는 OpenAI 호환 endpoint를 제공합니다.

## API 키 발급받기 [#get-an-api-key]

[NEAR AI Cloud](https://cloud.near.ai/)에서 API 키를 생성하세요. 이를 `.env` 파일에 추가하세요:

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## 구성 [#configuration]

`librechat.yaml`의 `endpoints.custom` 아래에 해당 endpoint를 추가하세요:

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

각 사용자가 `.env`에서 키를 읽어오는 대신 LibreChat UI를 통해 자신의 키를 직접 입력하게 하려면 `apiKey: "user_provided"`로 설정하세요.

## 참고 사항 [#notes]

- `fetch: true`를 설정하면, LibreChat은 NEAR AI Cloud의 OpenAI 호환 `/v1/models` endpoint에서 모델 목록을 불러옵니다. `default` 배열은 초기 선택 항목일 뿐입니다.
- [NEAR AI Cloud 모델 카탈로그](https://cloud-api.near.ai/v1/model/list)에 표시된 대로 정확하게 모델 ID를 사용하세요.
- 모델 가용성은 시간이 지남에 따라 변경될 수 있습니다. 선택한 모델을 사용할 수 없는 경우, 카탈로그에서 채팅이 가능한 다른 모델을 선택하세요.
