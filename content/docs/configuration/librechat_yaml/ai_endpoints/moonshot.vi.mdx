Moonshot AI cung cấp các mô hình Kimi của họ thông qua một API tương thích với OpenAI mà bạn có thể thêm vào LibreChat dưới dạng một endpoint tùy chỉnh.

## Nhận API key [#get-an-api-key]

Tạo một khóa từ [Moonshot platform](https://platform.moonshot.ai). Thêm nó vào tệp `.env` của bạn:

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## Cấu hình [#configuration]

Thêm endpoint vào dưới `endpoints.custom` trong tệp `librechat.yaml` của bạn:

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## Ghi chú [#notes]

- Đối với các model có khả năng suy luận như `kimi-k2.5` và `kimi-k2-thinking`, `name` của endpoint phải được đặt là `"Moonshot"` (không phân biệt chữ hoa chữ thường) để tính năng suy luận xen kẽ hoạt động với các lệnh gọi công cụ (tool calls). Một tên khác sẽ gây ra các lỗi như `thinking is enabled but reasoning_content is missing in assistant tool call message`. Xem [tài liệu của Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) để biết thêm chi tiết.
