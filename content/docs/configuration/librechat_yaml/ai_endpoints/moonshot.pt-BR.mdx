A Moonshot AI disponibiliza seus modelos Kimi por meio de uma API compatível com OpenAI que você pode adicionar ao LibreChat como um endpoint personalizado.

## Obtenha uma API key [#get-an-api-key]

Crie uma chave na [Moonshot platform](https://platform.moonshot.ai). Adicione-a ao seu arquivo `.env`:

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## Configuração [#configuration]

Adicione o endpoint em `endpoints.custom` no seu `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## Notas [#notes]

- Para modelos com capacidades de raciocínio, como `kimi-k2.5` e `kimi-k2-thinking`, o `name` do endpoint deve ser definido como `"Moonshot"` (insensível a maiúsculas/minúsculas) para que o raciocínio intercalado funcione com chamadas de ferramenta. Um nome diferente causa erros como `thinking is enabled but reasoning_content is missing in assistant tool call message`. Veja a [documentação da Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) para mais detalhes.
