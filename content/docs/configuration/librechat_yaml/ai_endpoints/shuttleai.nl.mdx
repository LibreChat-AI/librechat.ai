ShuttleAI biedt toegang tot zijn Shuttle-modellen via een OpenAI-compatibele API, die in LibreChat wordt gebruikt als een custom endpoint.

## Een API-sleutel verkrijgen [#get-an-api-key]

Maak een sleutel aan via de [ShuttleAI keys page](https://shuttleai.com/keys). Voeg deze toe aan je `.env` bestand:

```bash filename=".env"
SHUTTLEAI_API_KEY=your-api-key
```

## Configuratie [#configuration]

Voeg het endpoint toe onder `endpoints.custom` in je `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
