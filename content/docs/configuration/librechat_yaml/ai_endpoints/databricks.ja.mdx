> **[Databricksに登録する]**(https://www.databricks.com/try-databricks#account)

**注記:**

- Databricksは「serving-endpoints」に対して「invocations」で終わる完全な補完endpointを提供しているため、[directEndpoint](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#directendpoint)設定の使用が必須となります。
- [titleMessageRole](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#titlemessagerole) を "user" に設定することもタイトル生成に必要です。これは、単一の "system" メッセージはサポートされていないためです。

```yaml filename="librechat.yaml"
    - name: 'Databricks'
      apiKey: '${DATABRICKS_API_KEY}'
      baseURL: 'https://your_databricks_serving_endpoint_url_here_ending_with/invocations'
      models:
        default: [
          "databricks-meta-llama-3-70b-instruct",
        ]
        fetch: false
      titleConvo: true
      titleModel: 'current_model'
      directEndpoint: true # required
      titleMessageRole: 'user' # required
```
