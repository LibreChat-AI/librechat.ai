together.ai는 OpenAI 호환 API를 통해 방대한 오픈 모델 카탈로그를 제공하며, 이를 사용자 지정 endpoint로 LibreChat에 추가할 수 있습니다.

## API 키 발급받기 [#get-an-api-key]

[api.together.xyz/settings/api-keys](https://api.together.xyz/settings/api-keys)에서 키를 생성하세요. 해당 키를 `.env` 파일에 추가하세요:

```bash filename=".env"
TOGETHERAI_API_KEY=your-api-key
```

## 참고 사항 [#notes]

- 모델 목록 가져오기가 지원되지 않으므로 `fetch`는 `false`로 설정하고 `default` 배열은 직접 유지 관리해야 합니다.
- together.ai의 카탈로그는 자주 변경됩니다. 목록을 수동으로 편집하는 대신 아래 스크립트를 사용하여 현재 채팅 모델을 가져오세요.

<Callout type="tip" title="모델을 가져와 정렬합니다" collapsible>
이 Python 스크립트는 사용 가능한 채팅 모델을 가져와서 정렬한 뒤, yaml 설정에 붙여넣기 쉬운 형식으로 `models_togetherai.json`에 기록합니다. 먼저 API 키를 설정하세요.

```py filename="fetch_togetherai.py"
import json

import requests

# API key
api_key = ""

# API endpoint
url = "https://api.together.xyz/v1/models"

# headers
headers = {
    "accept": "application/json",
    "Authorization": f"Bearer {api_key}"
}

# make request
response = requests.get(url, headers=headers)

# parse JSON response
data = response.json()

# extract an ordered list of unique model IDs
model_ids = sorted(
    [
        model['id']
        for model in data
        if model['type'] == 'chat'
    ]
)

# write result to a text file
with open("models_togetherai.json", "w") as file:
    json.dump(model_ids, file, indent=2)
```
</Callout>
