APIpie est un agrégateur qui expose des modèles provenant de nombreux fournisseurs via un point de terminaison unique compatible avec OpenAI.

## Obtenir une clé API [#get-an-api-key]

Créez une clé depuis [apipie.ai/dashboard/profile/api-keys](https://apipie.ai/dashboard/profile/api-keys) et ajoutez-la à votre fichier `.env` :

```bash filename=".env"
APIPIE_API_KEY=your-api-key
```

## Configuration [#configuration]

Ajoutez l'endpoint sous `endpoints.custom` dans votre `librechat.yaml` :

```yaml filename="librechat.yaml"
    - name: "APIpie"
      apiKey: "${APIPIE_API_KEY}"
      baseURL: "https://apipie.ai/v1/"
      models:
        default: [
          "gpt-4",
          "gpt-4-turbo",
          "gpt-3.5-turbo",
          "claude-3-opus",
          "claude-3-sonnet",
          "claude-3-haiku",
          "llama-3-70b-instruct",
          "llama-3-8b-instruct",
          "gemini-pro-1.5",
          "gemini-pro",
          "mistral-large",
          "mistral-medium",
          "mistral-small",
          "mistral-tiny",
          "mixtral-8x22b",
          ]
        fetch: false
      titleConvo: true
      titleModel: "claude-3-haiku"
      summarize: false
      summaryModel: "claude-3-haiku"
      dropParams: ["stream"]
      modelDisplayLabel: "APIpie"
```

<Callout type="tip" title="Récupérer et ordonner les modèles" collapsible>
Ce script python peut récupérer et ordonner les modèles LLM pour vous. La sortie est enregistrée dans `models.txt`, formatée de manière à faciliter son intégration dans la configuration yaml.

```py filename="fetch.py"
import json
import requests

def fetch_and_order_models():
    # API endpoint
    url = "https://apipie.ai/models"

    # headers as per request example
    headers = {"Accept": "application/json"}

    # request parameters
    params = {"type": "llm"}

    # make request
    response = requests.get(url, headers=headers, params=params)

    # parse JSON response
    data = response.json()

    # extract an ordered list of unique model IDs
    model_ids = sorted(set([model["id"] for model in data]))

    # write result to a text file
    with open("models.txt", "w") as file:
        json.dump(model_ids, file, indent=2)

# execute the function
if __name__ == "__main__":
    fetch_and_order_models()
```
</Callout>

## Notes [#notes]

- La récupération automatique des modèles n'est pas prise en charge, donc `fetch` est défini sur `false` et la liste des modèles est définie en ligne. Utilisez le script ci-dessus pour régénérer la liste lorsque de nouveaux modèles sont ajoutés.
- Les résultats de l'attribution de titres aux conversations peuvent être incohérents.
- `dropParams: ["stream"]` supprime le paramètre `stream`, qui n'est pas pris en charge actuellement.
