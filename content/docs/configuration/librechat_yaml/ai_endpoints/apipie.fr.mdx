> **Clé API APIpie :** [apipie.ai/dashboard/profile/api-keys](https://apipie.ai/dashboard/profile/api-keys)

**Notes :**

- **Connu :** icône fournie, il est recommandé de récupérer la liste des modèles car les tarifs des jetons API et la tarification sont utilisés pour les soldes de crédits de jetons lorsque les modèles sont récupérés.

- **Problème connu :**
  - La récupération de la liste des modèles n'est pas prise en charge.
  - Votre succès peut varier avec le titrage des conversations
  - Le streaming n'est pas actuellement pris en charge (mais est prévu depuis le 24 avril 2024)

<Callout type="tip" title="Récupérer et ordonner les modèles" collapsible>
Ce script python peut récupérer et ordonner les modèles llm pour vous. Le résultat sera enregistré dans models.txt, formaté de manière à faciliter son inclusion dans la configuration yaml.

```py filename="fetch.py"
import json
import requests

def fetch_and_order_models():
    # API endpoint
    url = "https://apipie.ai/models"

    # headers as per request example
    headers = {"Accept": "application/json"}

    # request parameters
    params = {"type": "llm"}

    # make request
    response = requests.get(url, headers=headers, params=params)

    # parse JSON response
    data = response.json()

    # extract an ordered list of unique model IDs
    model_ids = sorted(set([model["id"] for model in data]))

    # write result to a text file
    with open("models.txt", "w") as file:
        json.dump(model_ids, file, indent=2)

# execute the function
if __name__ == "__main__":
    fetch_and_order_models()
```
