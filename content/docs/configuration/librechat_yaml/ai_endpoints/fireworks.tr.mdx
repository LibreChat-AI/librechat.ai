Fireworks AI, LibreChat'te özel bir endpoint olarak kullanılan, OpenAI uyumlu bir API aracılığıyla çeşitli açık modeller sunar.

## Bir API anahtarı alın [#get-an-api-key]

[Fireworks API keys page](https://fireworks.ai/api-keys) sayfasından bir anahtar oluşturun. Bunu `.env` dosyanıza ekleyin:

```bash filename=".env"
FIREWORKS_API_KEY=your-api-key
```

## Yapılandırma [#configuration]

`librechat.yaml` dosyanızda `endpoints.custom` altına endpoint'i ekleyin:

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```

## Notlar [#notes]

- API bazı modeller için katıdır ve `user` gibi alanları reddedebilir, bu yüzden bunları [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams) ile kaldırın.
