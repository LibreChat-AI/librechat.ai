Moonshot AI ofrece sus modelos Kimi a través de una API compatible con OpenAI que puedes añadir a LibreChat como un endpoint personalizado.

## Obtener una API key [#get-an-api-key]

Cree una clave desde la [plataforma Moonshot](https://platform.moonshot.ai). Añádala a su archivo `.env`:

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## Configuración [#configuration]

Agregue el endpoint bajo `endpoints.custom` en su `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## Notas [#notes]

- Para modelos con capacidades de razonamiento como `kimi-k2.5` y `kimi-k2-thinking`, el `name` del endpoint debe establecerse como `"Moonshot"` (sin distinguir entre mayúsculas y minúsculas) para que el razonamiento intercalado funcione con las llamadas a herramientas. Un nombre diferente provoca errores como `thinking is enabled but reasoning_content is missing in assistant tool call message`. Consulta la [documentación de Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) para obtener más detalles.
