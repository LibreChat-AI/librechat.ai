> Clave de API de Moonshot: [platform.moonshot.cn](https://platform.moonshot.ai)

**Notas:**

- **Conocido:** icono proporcionado.
- **Importante:** Para modelos con capacidades de razonamiento/pensamiento (por ejemplo, `kimi-k2.5`, `kimi-k2-thinking`), el `name` del endpoint **debe** establecerse como `"Moonshot"` (sin distinguir entre mayúsculas y minúsculas) para que el razonamiento intercalado funcione correctamente con las llamadas a herramientas. Usar un nombre diferente resultará en errores como `thinking is enabled but reasoning_content is missing in assistant tool call message`. Consulta la [documentación de Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) para obtener más detalles.

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```
