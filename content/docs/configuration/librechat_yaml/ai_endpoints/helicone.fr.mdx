Helicone est une passerelle IA qui achemine les requêtes vers des modèles d'OpenAI, Anthropic, Google, Meta, Mistral et d'autres fournisseurs via un endpoint unique compatible avec OpenAI, avec une journalisation des requêtes et des analyses d'utilisation intégrées.

## Obtenir une clé API [#get-an-api-key]

Créez un compte sur [helicone.ai](https://helicone.ai/) et générez une clé depuis [Settings → API Keys](https://us.helicone.ai/settings/api-keys). Ajoutez-la à votre fichier `.env` :

```bash filename=".env"
HELICONE_KEY=your-api-key
```

Assurez-vous que votre compte dispose de crédits, sinon les requêtes de modèle échoueront.

## Configuration [#configuration]

Ajoutez l'endpoint sous `endpoints.custom` dans votre `librechat.yaml` :

```yaml filename="librechat.yaml"
    - name: "Helicone"
      apiKey: "${HELICONE_KEY}"
      baseURL: "https://ai-gateway.helicone.ai"
      headers:
        x-librechat-body-parentmessageid: "{{LIBRECHAT_BODY_PARENTMESSAGEID}}"
      models:
        default: ["gpt-4o-mini", "claude-4.5-sonnet", "llama-3.1-8b-instruct", "gemini-2.5-flash-lite"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-4o-mini"
      modelDisplayLabel: "Helicone"
      iconURL: "https://marketing-assets-helicone.s3.us-west-2.amazonaws.com/helicone.png"
```

## Notes [#notes]

- Avec `fetch: true`, LibreChat charge la liste complète des modèles depuis Helicone, donc le tableau `default` n'est que la sélection initiale. Parcourez tout ce qui est disponible dans la [bibliothèque de modèles Helicone](https://helicone.ai/models).
- L'en-tête `x-librechat-body-parentmessageid` transmet l'ID du message parent de chaque requête à Helicone afin que les journaux puissent être regroupés par conversation.
- Définissez des limites de débit, des politiques de mise en cache et consultez l'utilisation depuis le [tableau de bord Helicone](https://us.helicone.ai/dashboard).
