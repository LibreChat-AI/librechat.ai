ShuttleAI proporciona acceso a sus modelos Shuttle a través de una API compatible con OpenAI, utilizada en LibreChat como un endpoint personalizado.

## Obtener una API key [#get-an-api-key]

Cree una clave desde la [página de claves de ShuttleAI](https://shuttleai.com/keys). Añádala a su archivo `.env`:

```bash filename=".env"
SHUTTLEAI_API_KEY=your-api-key
```

## Configuración [#configuration]

Agregue el endpoint bajo `endpoints.custom` en su `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
