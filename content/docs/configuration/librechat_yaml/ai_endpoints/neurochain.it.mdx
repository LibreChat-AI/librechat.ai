NeurochainAI è una rete di inferenza compatibile con OpenAI. Sostituisci `<=generated api key=>` qui sotto con la chiave che hai creato nella [documentazione della REST API di NeurochainAI](https://app.neurochain.ai/network-integrations/rest-api).

## Configurazione [#configuration]

Aggiungi l'endpoint sotto `endpoints.custom` nel tuo `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## Note [#notes]

- L'elenco dei modelli cresce nel tempo, quindi l'esempio potrebbe non essere aggiornato. Controlla [NeurochainAI](https://neurochain.ai) per i modelli più recenti.
