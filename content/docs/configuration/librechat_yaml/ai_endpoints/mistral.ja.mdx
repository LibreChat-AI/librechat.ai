> Mistral API key: [console.mistral.ai](https://console.mistral.ai/)

**注記:**

- **既知の事項:** アイコンが提供されており、メッセージロールの特別な処理が必要です。システムメッセージは、メッセージペイロードの先頭にのみ配置できます。

- APIは認識されないパラメータに対して厳格であり、エラーメッセージも詳細ではありません（通常は「no body」と表示されます）。

    - "user"、"frequency_penalty"、"presence_penalty" パラメータを除外するために、[`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams) を使用する必要があります。
    - `stop` はデフォルトのパラメータに含まれなくなったため、ユーザーがこのフィールドを設定することを完全に禁止したい場合を除き、[`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams) に含める必要はなくなりました。
    
- モデルリストの取得を許可しますが、チャット用に埋め込み（embedding）モデルを使用しないよう注意してください。

```yaml filename="librechat.yaml"
    - name: "Mistral"
      apiKey: "${MISTRAL_API_KEY}"
      baseURL: "https://api.mistral.ai/v1"
      models:
        default: ["mistral-tiny", "mistral-small", "mistral-medium", "mistral -large-latest"]
        fetch: true
      titleConvo: true
      titleModel: "mistral-tiny"
      modelDisplayLabel: "Mistral"
      dropParams: ["stop", "user", "frequency_penalty", "presence_penalty"]
```
