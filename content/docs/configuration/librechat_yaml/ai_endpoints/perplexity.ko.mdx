Perplexity는 LibreChat에서 커스텀 endpoint로 사용되는 검색 기반 Sonar 모델 제품군을 제공합니다.

## API 키 발급받기 [#get-an-api-key]

[Perplexity API settings](https://www.perplexity.ai/settings/api)에서 키를 생성하세요. 해당 키를 `.env` 파일에 추가하세요:

```bash filename=".env"
PERPLEXITY_API_KEY=your-api-key
```

## 구성 [#configuration]

`librechat.yaml`의 `endpoints.custom` 아래에 해당 endpoint를 추가하세요:

```yaml filename="librechat.yaml"
    - name: "Perplexity"
      apiKey: "${PERPLEXITY_API_KEY}"
      baseURL: "https://api.perplexity.ai/"
      models:
        default: [
          "sonar-deep-research",
          "sonar-reasoning-pro",
          "sonar-reasoning",
          "sonar-pro",
          "sonar",
          "r1-1776"
          ]
        fetch: false
      titleConvo: true
      titleModel: "llama-3-sonar-small-32k-chat"
      summarize: false
      summaryModel: "llama-3-sonar-small-32k-chat"
      dropParams: ["stop", "frequency_penalty"]
      modelDisplayLabel: "Perplexity"
```

## 참고 사항 [#notes]

- 모델 목록 가져오기는 지원되지 않으므로 `fetch: false`로 유지하고 `default` 목록을 수동으로 관리하세요.
- 일부 모델의 경우 API가 엄격합니다. `stop` 및 `frequency_penalty`와 같은 필드는 0으로 설정할 경우 오류를 일으킬 수 있으므로, [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams)를 사용하여 해당 필드를 제외하세요.
