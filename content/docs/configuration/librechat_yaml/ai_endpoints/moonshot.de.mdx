> Moonshot API-Schlüssel: [platform.moonshot.cn](https://platform.moonshot.ai)

**Hinweise:**

- **Bekannt:** Symbol bereitgestellt.
- **Wichtig:** Bei Modellen mit Reasoning-/Thinking-Funktionen (z. B. `kimi-k2.5`, `kimi-k2-thinking`) **muss** der `name` des endpoint auf `"Moonshot"` (Groß-/Kleinschreibung wird ignoriert) gesetzt werden, damit das verschachtelte Reasoning bei Tool-Aufrufen korrekt funktioniert. Die Verwendung eines anderen Namens führt zu Fehlern wie `thinking is enabled but reasoning_content is missing in assistant tool call message`. Weitere Details finden Sie in der [Dokumentation von Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions).

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```
