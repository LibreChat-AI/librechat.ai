ShuttleAI zapewnia dostęp do swoich modeli Shuttle poprzez API kompatybilne z OpenAI, używane w LibreChat jako niestandardowy endpoint.

## Uzyskaj klucz API [#get-an-api-key]

Utwórz klucz na [stronie kluczy ShuttleAI](https://shuttleai.com/keys). Dodaj go do swojego pliku `.env`:

```bash filename=".env"
SHUTTLEAI_API_KEY=your-api-key
```

## Konfiguracja [#configuration]

Dodaj endpoint w sekcji `endpoints.custom` w swoim `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
