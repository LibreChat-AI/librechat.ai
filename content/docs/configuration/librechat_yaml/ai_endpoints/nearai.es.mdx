NEAR AI Cloud proporciona un endpoint compatible con OpenAI que puedes añadir a LibreChat a través de `endpoints.custom`.

## Obtener una API key [#get-an-api-key]

Cree una clave de API desde [NEAR AI Cloud](https://cloud.near.ai/). Añádala a su archivo `.env`:

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## Configuración [#configuration]

Agregue el endpoint bajo `endpoints.custom` en su `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

Para permitir que cada usuario proporcione su propia clave a través de la interfaz de usuario de LibreChat en lugar de leer una desde `.env`, establezca `apiKey: "user_provided"`.

## Notas [#notes]

- Con `fetch: true`, LibreChat carga la lista de modelos desde el endpoint `/v1/models` compatible con OpenAI de NEAR AI Cloud. La matriz `default` es solo la selección inicial.
- Utilice los IDs de modelo exactamente como aparecen en el [catálogo de modelos de NEAR AI Cloud](https://cloud-api.near.ai/v1/model/list).
- La disponibilidad de los modelos puede cambiar con el tiempo. Si un modelo seleccionado no está disponible, elija otro modelo capaz de chatear del catálogo.
