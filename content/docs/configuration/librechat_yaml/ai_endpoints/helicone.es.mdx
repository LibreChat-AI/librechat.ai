> Clave de API de Helicone: [https://us.helicone.ai/settings/api-keys](https://us.helicone.ai/settings/api-keys)

**Notas:**

- **Conocido:** se proporciona el icono, se recomienda obtener la lista de modelos ya que Helicone proporciona acceso a más de 100 modelos de IA de múltiples proveedores.

- Helicone es un proveedor de AI Gateway que permite el acceso a modelos de OpenAI, Anthropic, Google, Meta, Mistral y otros proveedores con observabilidad y monitoreo integrados.

- **Características clave:**
  - 🚀 Acceso a más de 100 modelos de IA a través de una única puerta de enlace
  - 📊 Observabilidad y monitoreo integrados
  - 🔄 Soporte para múltiples proveedores
  - ⚡ Registro de solicitudes y análisis en el panel de control de Helicone

- **Consideraciones importantes:**
  - Asegúrate de que tu cuenta de Helicone tenga créditos para que puedas acceder a los modelos.
  - Puedes encontrar todos los modelos compatibles en la [Helicone Model Library](https://helicone.ai/models).
  - Puedes establecer tus propios límites de tasa y políticas de caché dentro del panel de control de Helicone.

```yaml
    - name: "Helicone"
      # For `apiKey` and `baseURL`, you can use environment variables that you define.
      # recommended environment variables:
      apiKey: "${HELICONE_KEY}"
      baseURL: "https://ai-gateway.helicone.ai"
      headers:
        x-librechat-body-parentmessageid: "{{LIBRECHAT_BODY_PARENTMESSAGEID}}"
      models:
        default: ["gpt-4o-mini", "claude-4.5-sonnet", "llama-3.1-8b-instruct", "gemini-2.5-flash-lite"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-4o-mini"
      modelDisplayLabel: "Helicone"
      iconURL: "https://marketing-assets-helicone.s3.us-west-2.amazonaws.com/helicone.png"
```

**Detalles de configuración:**

- **apiKey:** Utilice la variable de entorno `HELICONE_KEY` para almacenar su clave de API de Helicone.
- **baseURL:** El endpoint de Helicone AI Gateway: `https://ai-gateway.helicone.ai`
- **headers:** El encabezado `x-librechat-body-parentmessageid` es esencial para el seguimiento de mensajes y la continuidad de la conversación
- **models:** Establece los modelos predeterminados; sin embargo, al habilitar `fetch`, recuperarás automáticamente todos los modelos disponibles desde la API de Helicone.
- **fetch:** Configúrelo en `true` para recuperar automáticamente los modelos disponibles desde la API de Helicone

**Pasos de configuración:**
1. Regístrate para obtener una cuenta de Helicone en [helicone.ai](https://helicone.ai/)
2. Genera tu API key desde el [Helicone dashboard](https://us.helicone.ai/settings/api-keys)
3. Establece la variable de entorno `HELICONE_KEY` en tu archivo `.env`
4. Copie la configuración de ejemplo en su archivo `librechat.yaml`
5. Reconstruya sus contenedores de Docker si utiliza el despliegue con Docker
6. Reinicia LibreChat para cargar la nueva configuración
7. Prueba seleccionando Helicone del menú desplegable de proveedores
8. Dirígete al [Helicone dashboard](https://us.helicone.ai/dashboard) para revisar tu uso y configuración.

**Problemas potenciales:**

- **Acceso a modelos:** Verifique que tiene créditos en Helicone para poder acceder a los modelos.
- **Limitación de tasa (Rate Limiting):** Puedes establecer tus propios límites de tasa y políticas de caché dentro del panel de control de Helicone.
- **Variables de entorno:** Verifique dos veces que `HELICONE_KEY` esté configurada correctamente y sea accesible para su instancia de LibreChat

**Pruebas:**
1. Después de la configuración, seleccione Helicone en el menú desplegable del proveedor
2. Verifique que los modelos aparezcan en la selección de modelos
3. Envía un mensaje de prueba y confirma que aparece en tu panel de control de Helicone
4. Comprueba que el hilo de conversación funcione correctamente con el encabezado del ID del mensaje principal
