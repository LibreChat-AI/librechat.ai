Helicone es una puerta de enlace de IA que enruta solicitudes a modelos de OpenAI, Anthropic, Google, Meta, Mistral y otros proveedores a través de un único endpoint compatible con OpenAI, con registro de solicitudes y análisis de uso integrados.

## Obtener una API key [#get-an-api-key]

Cree una cuenta en [helicone.ai](https://helicone.ai/) y genere una clave desde [Settings → API Keys](https://us.helicone.ai/settings/api-keys). Añádala a su archivo `.env`:

```bash filename=".env"
HELICONE_KEY=your-api-key
```

Asegúrate de que tu cuenta tenga créditos, de lo contrario las solicitudes de modelo fallarán.

## Configuración [#configuration]

Agregue el endpoint bajo `endpoints.custom` en su `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Helicone"
      apiKey: "${HELICONE_KEY}"
      baseURL: "https://ai-gateway.helicone.ai"
      headers:
        x-librechat-body-parentmessageid: "{{LIBRECHAT_BODY_PARENTMESSAGEID}}"
      models:
        default: ["gpt-4o-mini", "claude-4.5-sonnet", "llama-3.1-8b-instruct", "gemini-2.5-flash-lite"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-4o-mini"
      modelDisplayLabel: "Helicone"
      iconURL: "https://marketing-assets-helicone.s3.us-west-2.amazonaws.com/helicone.png"
```

## Notas [#notes]

- Con `fetch: true`, LibreChat carga la lista completa de modelos desde Helicone, por lo que la matriz `default` es solo la selección inicial. Explora todo lo disponible en la [biblioteca de modelos de Helicone](https://helicone.ai/models).
- El encabezado `x-librechat-body-parentmessageid` pasa el ID del mensaje principal de cada solicitud a Helicone para que los registros puedan agruparse por conversación.
- Establezca límites de tasa, políticas de caché y revise el uso desde el [Helicone dashboard](https://us.helicone.ai/dashboard).
