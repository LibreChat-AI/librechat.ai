Moonshot AI, Kimi modellerini LibreChat'e özel bir endpoint olarak ekleyebileceğiniz OpenAI uyumlu bir API aracılığıyla sunar.

## Bir API anahtarı alın [#get-an-api-key]

[Moonshot platform](https://platform.moonshot.ai) üzerinden bir anahtar oluşturun. Bunu `.env` dosyanıza ekleyin:

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## Yapılandırma [#configuration]

`librechat.yaml` dosyanızda `endpoints.custom` altına endpoint'i ekleyin:

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## Notlar [#notes]

- `kimi-k2.5` ve `kimi-k2-thinking` gibi muhakeme yeteneklerine sahip modeller için, araç çağrılarıyla birlikte serpiştirilmiş muhakemenin (interleaved reasoning) çalışabilmesi adına endpoint `name` değeri `"Moonshot"` (büyük/küçük harf duyarsız) olarak ayarlanmalıdır. Farklı bir isim kullanılması `thinking is enabled but reasoning_content is missing in assistant tool call message` gibi hatalara yol açar. Ayrıntılar için [Moonshot's documentation](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) sayfasına bakın.
