Helicone adalah AI gateway yang merutekan permintaan ke model dari OpenAI, Anthropic, Google, Meta, Mistral, dan penyedia lainnya melalui satu endpoint yang kompatibel dengan OpenAI, dengan fitur pencatatan permintaan dan analitik penggunaan bawaan.

## Dapatkan API key [#get-an-api-key]

Buat akun di [helicone.ai](https://helicone.ai/) dan buat kunci dari [Settings → API Keys](https://us.helicone.ai/settings/api-keys). Tambahkan ke file `.env` Anda:

```bash filename=".env"
HELICONE_KEY=your-api-key
```

Pastikan akun Anda memiliki kredit, jika tidak, permintaan model akan gagal.

## Konfigurasi [#configuration]

Tambahkan endpoint di bawah `endpoints.custom` dalam `librechat.yaml` Anda:

```yaml filename="librechat.yaml"
    - name: "Helicone"
      apiKey: "${HELICONE_KEY}"
      baseURL: "https://ai-gateway.helicone.ai"
      headers:
        x-librechat-body-parentmessageid: "{{LIBRECHAT_BODY_PARENTMESSAGEID}}"
      models:
        default: ["gpt-4o-mini", "claude-4.5-sonnet", "llama-3.1-8b-instruct", "gemini-2.5-flash-lite"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-4o-mini"
      modelDisplayLabel: "Helicone"
      iconURL: "https://marketing-assets-helicone.s3.us-west-2.amazonaws.com/helicone.png"
```

## Catatan [#notes]

- Dengan `fetch: true`, LibreChat memuat daftar model lengkap dari Helicone, sehingga array `default` hanyalah pilihan awal. Telusuri semua yang tersedia di [Helicone model library](https://helicone.ai/models).
- Header `x-librechat-body-parentmessageid` meneruskan ID pesan induk setiap permintaan ke Helicone agar log dapat dikelompokkan berdasarkan percakapan.
- Atur batas laju (rate limits), kebijakan caching, dan tinjau penggunaan dari [dasbor Helicone](https://us.helicone.ai/dashboard).
