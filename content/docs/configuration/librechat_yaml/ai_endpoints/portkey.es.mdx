> Clave de API de Portkey: [app.portkey.ai](https://app.portkey.ai/)


**Notas:**
- LibreChat requiere que el campo `API key` esté presente. No lo necesitamos para la integración con Portkey, podemos pasar una cadena `dummy` para ello.
- Portkey se integra con LibreChat, ofreciendo observabilidad, más de 50 guardrails, almacenamiento en caché y enrutamiento condicional con alternativas (fallbacks) y reintentos para implementaciones confiables de nivel de producción.
- Portkey admite más de 250 modelos, puedes encontrar la lista de proveedores en [Portkey Docs](https://docs.portkey.ai/docs/integrations/llms)
- Puedes usar Portkey de dos maneras: a través de [Virtual Keys](https://docs.portkey.ai/docs/product/ai-gateway/virtual-keys) o [Configs](https://docs.portkey.ai/docs/product/ai-gateway/configs).
- Puede utilizar parámetros específicos de Model como `top_p`, `max Tokens`, etc., al usar Portkey Config. [Más información](https://docs.portkey.ai/docs/product/ai-gateway/configs#configs)


## Uso de Portkey AI con Virtual Keys [#using-portkey-ai-with-virtual-keys]
```yaml filename="librechat.yaml"
      - name: "Portkey"
        apiKey: "dummy"  
        baseURL: ${PORTKEY_GATEWAY_URL}
        headers:
            x-portkey-api-key: "${PORTKEY_API_KEY}"
            x-portkey-virtual-key: "PORTKEY_OPENAI_VIRTUAL_KEY"
        models:
            default: ["gpt-4o-mini"]
            fetch: true
        titleConvo: true
        titleModel: "current_model"
        summarize: false
        summaryModel: "current_model"
        modelDisplayLabel: "Portkey:OpenAI"
        iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```


## Uso de Portkey AI con Configs [#using-portkey-ai-with-configs]
```yaml filename="librechat.yaml"
    - name: "Portkey"
      apikey: "dummy"
      baseURL: ${PORTKEY_GATEWAY_URL}
      headers:
        x-portkey-api-key: "${PORTKEY_API_KEY}"
        x-portkey-config: "pc-libre-xxx"
      models:
        default: ["llama-3.2"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "Portkey:Llama"
      iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```
