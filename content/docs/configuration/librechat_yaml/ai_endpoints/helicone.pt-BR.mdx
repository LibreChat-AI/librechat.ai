Helicone é um gateway de IA que roteia solicitações para modelos da OpenAI, Anthropic, Google, Meta, Mistral e outros provedores através de um único endpoint compatível com OpenAI, com registro de solicitações e análise de uso integrados.

## Obtenha uma API key [#get-an-api-key]

Crie uma conta em [helicone.ai](https://helicone.ai/) e gere uma chave em [Settings → API Keys](https://us.helicone.ai/settings/api-keys). Adicione-a ao seu arquivo `.env`:

```bash filename=".env"
HELICONE_KEY=your-api-key
```

Certifique-se de que sua conta possua créditos, caso contrário, as solicitações de modelo falharão.

## Configuração [#configuration]

Adicione o endpoint em `endpoints.custom` no seu `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Helicone"
      apiKey: "${HELICONE_KEY}"
      baseURL: "https://ai-gateway.helicone.ai"
      headers:
        x-librechat-body-parentmessageid: "{{LIBRECHAT_BODY_PARENTMESSAGEID}}"
      models:
        default: ["gpt-4o-mini", "claude-4.5-sonnet", "llama-3.1-8b-instruct", "gemini-2.5-flash-lite"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-4o-mini"
      modelDisplayLabel: "Helicone"
      iconURL: "https://marketing-assets-helicone.s3.us-west-2.amazonaws.com/helicone.png"
```

## Notas [#notes]

- Com `fetch: true`, o LibreChat carrega a lista completa de modelos do Helicone, portanto o array `default` é apenas a seleção inicial. Navegue por tudo o que está disponível na [Helicone model library](https://helicone.ai/models).
- O cabeçalho `x-librechat-body-parentmessageid` passa o ID da mensagem pai de cada requisição para o Helicone, para que os logs possam ser agrupados por conversa.
- Defina limites de taxa, políticas de cache e revise o uso a partir do [Helicone dashboard](https://us.helicone.ai/dashboard).
