> Clé API Deepseek : [platform.deepseek.com](https://platform.deepseek.com/usage)

**Notes :**

- **Connu :** icône fournie.
- `deepseek-chat` et `deepseek-coder` sont compatibles avec les [Agents + outils](/docs/features/agents).
- `deepseek-reasoner`, dont le nom de code est « R1 », est pris en charge et diffusera son « processus de réflexion » ; cependant, certains paramètres de l'API OpenAI pourraient ne pas être pris en charge par ce modèle spécifique.
- « R1 » peut également être compatible avec les Agents lorsque vous n'utilisez pas d'outils.
- `deepseek-chat` est privilégié pour la génération de titres.

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```
