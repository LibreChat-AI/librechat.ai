APIpie은 단일 OpenAI 호환 endpoint를 통해 여러 제공업체의 모델을 노출하는 애그리게이터입니다.

## API 키 발급받기 [#get-an-api-key]

[apipie.ai/dashboard/profile/api-keys](https://apipie.ai/dashboard/profile/api-keys)에서 키를 생성하고 `.env` 파일에 추가하세요:

```bash filename=".env"
APIPIE_API_KEY=your-api-key
```

## 구성 [#configuration]

`librechat.yaml`의 `endpoints.custom` 아래에 해당 endpoint를 추가하세요:

```yaml filename="librechat.yaml"
    - name: "APIpie"
      apiKey: "${APIPIE_API_KEY}"
      baseURL: "https://apipie.ai/v1/"
      models:
        default: [
          "gpt-4",
          "gpt-4-turbo",
          "gpt-3.5-turbo",
          "claude-3-opus",
          "claude-3-sonnet",
          "claude-3-haiku",
          "llama-3-70b-instruct",
          "llama-3-8b-instruct",
          "gemini-pro-1.5",
          "gemini-pro",
          "mistral-large",
          "mistral-medium",
          "mistral-small",
          "mistral-tiny",
          "mixtral-8x22b",
          ]
        fetch: false
      titleConvo: true
      titleModel: "claude-3-haiku"
      summarize: false
      summaryModel: "claude-3-haiku"
      dropParams: ["stream"]
      modelDisplayLabel: "APIpie"
```

<Callout type="tip" title="모델을 가져와 정렬합니다" collapsible>
이 Python 스크립트는 LLM 모델을 가져와 정렬할 수 있습니다. 출력 결과는 `models.txt`에 저장되며, yaml 설정에 쉽게 넣을 수 있도록 형식이 지정되어 있습니다.

```py filename="fetch.py"
import json
import requests

def fetch_and_order_models():
    # API endpoint
    url = "https://apipie.ai/models"

    # headers as per request example
    headers = {"Accept": "application/json"}

    # request parameters
    params = {"type": "llm"}

    # make request
    response = requests.get(url, headers=headers, params=params)

    # parse JSON response
    data = response.json()

    # extract an ordered list of unique model IDs
    model_ids = sorted(set([model["id"] for model in data]))

    # write result to a text file
    with open("models.txt", "w") as file:
        json.dump(model_ids, file, indent=2)

# execute the function
if __name__ == "__main__":
    fetch_and_order_models()
```
</Callout>

## 참고 사항 [#notes]

- 자동 모델 가져오기는 지원되지 않으므로 `fetch`는 `false`로 설정되어 있으며 모델 목록은 인라인으로 정의됩니다. 새 모델이 추가될 때 목록을 다시 생성하려면 위의 스크립트를 사용하세요.
- 대화 제목 생성 결과가 일관되지 않을 수 있습니다.
- `dropParams: ["stream"]`은 현재 지원되지 않는 `stream` 매개변수를 제거합니다.
