A xAI disponibiliza seus modelos Grok por meio de uma API compatível com OpenAI, que você pode adicionar ao LibreChat como um endpoint personalizado.

## Obtenha uma API key [#get-an-api-key]

Crie uma chave no [xAI console](https://console.x.ai/). Adicione-a ao seu arquivo `.env`:

```bash filename=".env"
XAI_API_KEY=your-api-key
```

## Configuração [#configuration]

Adicione o endpoint em `endpoints.custom` no seu `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "xai"
      apiKey: "${XAI_API_KEY}"
      baseURL: "https://api.x.ai/v1"
      models:
        default: ["grok-beta"]
        fetch: false
      titleConvo: true
      titleMethod: "completion"
      titleModel: "grok-beta"
      summarize: false
      summaryModel: "grok-beta"
      modelDisplayLabel: "Grok"
```

## Notas [#notes]

- `titleMethod: "completion"` gera títulos de conversa através do endpoint de chat completions, que é o que o Grok suporta.
