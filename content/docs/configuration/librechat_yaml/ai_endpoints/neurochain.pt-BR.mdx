NeurochainAI é uma rede de inferência compatível com OpenAI. Substitua `<=generated api key=>` abaixo pela chave que você criar na [documentação da REST API do NeurochainAI](https://app.neurochain.ai/network-integrations/rest-api).

## Configuração [#configuration]

Adicione o endpoint em `endpoints.custom` no seu `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## Notas [#notes]

- A lista de modelos cresce com o tempo, portanto o exemplo pode estar desatualizado. Verifique a [NeurochainAI](https://neurochain.ai) para os modelos mais recentes.
