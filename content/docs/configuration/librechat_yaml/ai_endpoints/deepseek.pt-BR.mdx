O Deepseek disponibiliza seus modelos de chat, coder e reasoner por meio de uma API compatível com OpenAI, que você pode adicionar ao LibreChat como um endpoint personalizado.

## Obtenha uma API key [#get-an-api-key]

Crie uma chave na [plataforma Deepseek](https://platform.deepseek.com/usage). Adicione-a ao seu arquivo `.env`:

```bash filename=".env"
DEEPSEEK_API_KEY=your-api-key
```

## Configuração [#configuration]

Adicione o endpoint em `endpoints.custom` no seu `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Deepseek"
      apiKey: "${DEEPSEEK_API_KEY}"
      baseURL: "https://api.deepseek.com/v1"
      models:
        default: ["deepseek-chat", "deepseek-coder", "deepseek-reasoner"]
        fetch: false
      titleConvo: true
      titleModel: "deepseek-chat"
      modelDisplayLabel: "Deepseek"
```

## Notas [#notes]

- `deepseek-chat` e `deepseek-coder` funcionam com [Agents and tools](/docs/features/agents).
- `deepseek-reasoner`, codinome "R1", é suportado e transmite seu processo de raciocínio, mas alguns parâmetros da API OpenAI podem não funcionar com ele. O R1 também pode funcionar com Agents quando ferramentas não são utilizadas.
- `deepseek-chat` é o modelo preferido para a geração de títulos.
