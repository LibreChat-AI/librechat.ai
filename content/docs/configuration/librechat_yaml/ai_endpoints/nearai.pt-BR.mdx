O NEAR AI Cloud fornece um endpoint compatível com OpenAI que você pode adicionar ao LibreChat através de `endpoints.custom`.

## Obtenha uma API key [#get-an-api-key]

Crie uma chave de API no [NEAR AI Cloud](https://cloud.near.ai/). Adicione-a ao seu arquivo `.env`:

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## Configuração [#configuration]

Adicione o endpoint em `endpoints.custom` no seu `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

Para permitir que cada usuário forneça sua própria chave através da interface do LibreChat em vez de ler uma do `.env`, defina `apiKey: "user_provided"`.

## Notas [#notes]

- Com `fetch: true`, o LibreChat carrega a lista de modelos a partir do endpoint `/v1/models` compatível com OpenAI da NEAR AI Cloud. O array `default` é apenas a seleção inicial.
- Use os IDs de modelo exatamente como aparecem no [catálogo de modelos do NEAR AI Cloud](https://cloud-api.near.ai/v1/model/list).
- A disponibilidade de modelos pode mudar com o tempo. Se um modelo selecionado estiver indisponível, escolha outro modelo capaz de realizar chats no catálogo.
