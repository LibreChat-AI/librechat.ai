Helicone is een AI-gateway die verzoeken naar modellen van OpenAI, Anthropic, Google, Meta, Mistral en andere providers routeert via één enkel OpenAI-compatibel endpoint, met ingebouwde verzoeklogging en gebruiksanalyses.

## Een API-sleutel verkrijgen [#get-an-api-key]

Maak een account aan op [helicone.ai](https://helicone.ai/) en genereer een sleutel via [Settings → API Keys](https://us.helicone.ai/settings/api-keys). Voeg deze toe aan je `.env` bestand:

```bash filename=".env"
HELICONE_KEY=your-api-key
```

Zorg ervoor dat je account over tegoeden beschikt, anders zullen modelverzoeken mislukken.

## Configuratie [#configuration]

Voeg het endpoint toe onder `endpoints.custom` in je `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Helicone"
      apiKey: "${HELICONE_KEY}"
      baseURL: "https://ai-gateway.helicone.ai"
      headers:
        x-librechat-body-parentmessageid: "{{LIBRECHAT_BODY_PARENTMESSAGEID}}"
      models:
        default: ["gpt-4o-mini", "claude-4.5-sonnet", "llama-3.1-8b-instruct", "gemini-2.5-flash-lite"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-4o-mini"
      modelDisplayLabel: "Helicone"
      iconURL: "https://marketing-assets-helicone.s3.us-west-2.amazonaws.com/helicone.png"
```

## Opmerkingen [#notes]

- Met `fetch: true` laadt LibreChat de volledige modellijst van Helicone, dus de `default` array is slechts de initiële selectie. Blader door alles wat beschikbaar is in de [Helicone model library](https://helicone.ai/models).
- De `x-librechat-body-parentmessageid` header geeft het parent message ID van elk verzoek door aan Helicone, zodat logs per gesprek kunnen worden gegroepeerd.
- Stel snelheidslimieten, caching-beleid en gebruiksbeoordelingen in via het [Helicone dashboard](https://us.helicone.ai/dashboard).
