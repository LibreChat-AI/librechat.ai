Helicone, OpenAI, Anthropic, Google, Meta, Mistral ve diğer sağlayıcılardan gelen modellere yönelik istekleri, yerleşik istek günlüğü ve kullanım analitiği ile tek bir OpenAI uyumlu endpoint üzerinden yönlendiren bir yapay zeka ağ geçididir.

## Bir API anahtarı alın [#get-an-api-key]

[helicone.ai](https://helicone.ai/) adresinde bir hesap oluşturun ve [Settings → API Keys](https://us.helicone.ai/settings/api-keys) kısmından bir anahtar oluşturun. Bunu `.env` dosyanıza ekleyin:

```bash filename=".env"
HELICONE_KEY=your-api-key
```

Hesabınızda kredi olduğundan emin olun, aksi takdirde model istekleri başarısız olacaktır.

## Yapılandırma [#configuration]

`librechat.yaml` dosyanızda `endpoints.custom` altına endpoint'i ekleyin:

```yaml filename="librechat.yaml"
    - name: "Helicone"
      apiKey: "${HELICONE_KEY}"
      baseURL: "https://ai-gateway.helicone.ai"
      headers:
        x-librechat-body-parentmessageid: "{{LIBRECHAT_BODY_PARENTMESSAGEID}}"
      models:
        default: ["gpt-4o-mini", "claude-4.5-sonnet", "llama-3.1-8b-instruct", "gemini-2.5-flash-lite"]
        fetch: true
      titleConvo: true
      titleModel: "gpt-4o-mini"
      modelDisplayLabel: "Helicone"
      iconURL: "https://marketing-assets-helicone.s3.us-west-2.amazonaws.com/helicone.png"
```

## Notlar [#notes]

- `fetch: true` ile LibreChat, Helicone'dan tam model listesini yükler, bu nedenle `default` dizisi yalnızca ilk seçimdir. [Helicone model kütüphanesinde](https://helicone.ai/models) mevcut olan her şeye göz atın.
- `x-librechat-body-parentmessageid` başlığı, her isteğin üst mesaj kimliğini (parent message ID) Helicone'a iletir, böylece günlükler konuşmaya göre gruplandırılabilir.
- [Helicone panosundan](https://us.helicone.ai/dashboard) hız sınırlarını (rate limits), önbelleğe alma politikalarını ve kullanım incelemelerini ayarlayın.
