Fireworks AI menyediakan berbagai model terbuka melalui API yang kompatibel dengan OpenAI, yang digunakan di LibreChat sebagai endpoint kustom.

## Dapatkan API key [#get-an-api-key]

Buat kunci dari [halaman kunci API Fireworks](https://fireworks.ai/api-keys). Tambahkan ke file `.env` Anda:

```bash filename=".env"
FIREWORKS_API_KEY=your-api-key
```

## Konfigurasi [#configuration]

Tambahkan endpoint di bawah `endpoints.custom` dalam `librechat.yaml` Anda:

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```

## Catatan [#notes]

- API ini bersifat ketat untuk beberapa model dan mungkin menolak kolom seperti `user`, jadi hapus kolom tersebut dengan [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams).
