> Clé API Portkey : [app.portkey.ai](https://app.portkey.ai/)


**Notes :**
- LibreChat exige que le champ `API key` soit présent. Nous n'en avons pas besoin pour l'intégration Portkey, nous pouvons y transmettre une chaîne `dummy`.
- Portkey s'intègre avec LibreChat, offrant une observabilité, plus de 50 garde-fous, la mise en cache et un routage conditionnel avec des mécanismes de secours et de nouvelles tentatives pour des déploiements fiables et de qualité production.
- Portkey prend en charge plus de 250 modèles, vous pouvez trouver la liste des fournisseurs sur la [documentation de Portkey](https://docs.portkey.ai/docs/integrations/llms)
- Vous pouvez utiliser Portkey de deux manières : via des [Virtual Keys](https://docs.portkey.ai/docs/product/ai-gateway/virtual-keys) ou des [Configs](https://docs.portkey.ai/docs/product/ai-gateway/configs).
- Vous pouvez utiliser des paramètres spécifiques au modèle comme `top_p`, `max Tokens`, etc. en utilisant Portkey Config. [En savoir plus](https://docs.portkey.ai/docs/product/ai-gateway/configs#configs)


## Utilisation de Portkey AI avec des clés virtuelles [#using-portkey-ai-with-virtual-keys]
```yaml filename="librechat.yaml"
      - name: "Portkey"
        apiKey: "dummy"  
        baseURL: ${PORTKEY_GATEWAY_URL}
        headers:
            x-portkey-api-key: "${PORTKEY_API_KEY}"
            x-portkey-virtual-key: "PORTKEY_OPENAI_VIRTUAL_KEY"
        models:
            default: ["gpt-4o-mini"]
            fetch: true
        titleConvo: true
        titleModel: "current_model"
        summarize: false
        summaryModel: "current_model"
        modelDisplayLabel: "Portkey:OpenAI"
        iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```


## Utilisation de Portkey AI avec des Configs [#using-portkey-ai-with-configs]
```yaml filename="librechat.yaml"
    - name: "Portkey"
      apikey: "dummy"
      baseURL: ${PORTKEY_GATEWAY_URL}
      headers:
        x-portkey-api-key: "${PORTKEY_API_KEY}"
        x-portkey-config: "pc-libre-xxx"
      models:
        default: ["llama-3.2"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "Portkey:Llama"
      iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```
