> groq API key: [wow.groq.com](https://console.groq.com/keys)

**注記:**

- **既知:** アイコンが提供されています。

- **Temperature:** Temperatureの値を0に設定した場合、1e-8に変換されます。問題が発生した場合は、0より大きく2以下のfloat32の値を設定してみてください。

- Groqは現在無料ですが、レート制限があります：1分間に10クエリ、1時間に100クエリまでです。

```yaml filename="librechat.yaml"
    - name: "groq"
      apiKey: "${GROQ_API_KEY}"
      baseURL: "https://api.groq.com/openai/v1/"
      models:
        default: [
          "llama3-70b-8192",
          "llama3-8b-8192",
          "llama2-70b-4096",
          "mixtral-8x7b-32768",
          "gemma-7b-it",
          ]
        fetch: false
      titleConvo: true
      titleModel: "mixtral-8x7b-32768"
      modelDisplayLabel: "groq"
```
