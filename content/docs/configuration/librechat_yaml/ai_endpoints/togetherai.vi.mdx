together.ai cung cấp một danh mục lớn các mô hình mở thông qua API tương thích với OpenAI, mà bạn có thể thêm vào LibreChat dưới dạng một endpoint tùy chỉnh.

## Nhận API key [#get-an-api-key]

Tạo một khóa tại [api.together.xyz/settings/api-keys](https://api.together.xyz/settings/api-keys). Thêm nó vào tệp `.env` của bạn:

```bash filename=".env"
TOGETHERAI_API_KEY=your-api-key
```

## Ghi chú [#notes]

- Việc lấy danh sách model không được hỗ trợ, vì vậy `fetch` được đặt thành `false` và bạn tự duy trì mảng `default`.
- Danh mục của together.ai thay đổi thường xuyên. Hãy sử dụng tập lệnh bên dưới để lấy các mô hình chat hiện tại thay vì chỉnh sửa danh sách theo cách thủ công.

<Callout type="tip" title="Tìm nạp và sắp xếp các model" collapsible>
Tập lệnh Python này tìm nạp các mô hình trò chuyện khả dụng và ghi chúng, theo thứ tự đã sắp xếp, vào `models_togetherai.json` ở định dạng dễ dàng dán vào cấu hình yaml. Hãy thiết lập khóa API của bạn trước.

```py filename="fetch_togetherai.py"
import json

import requests

# API key
api_key = ""

# API endpoint
url = "https://api.together.xyz/v1/models"

# headers
headers = {
    "accept": "application/json",
    "Authorization": f"Bearer {api_key}"
}

# make request
response = requests.get(url, headers=headers)

# parse JSON response
data = response.json()

# extract an ordered list of unique model IDs
model_ids = sorted(
    [
        model['id']
        for model in data
        if model['type'] == 'chat'
    ]
)

# write result to a text file
with open("models_togetherai.json", "w") as file:
    json.dump(model_ids, file, indent=2)
```
</Callout>
