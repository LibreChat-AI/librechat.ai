> Moonshot API key: [platform.moonshot.cn](https://platform.moonshot.ai)

**注記:**

- **既知:** アイコンが提供されています。
- **重要:** 推論/思考機能を持つモデル（例: `kimi-k2.5`、`kimi-k2-thinking`）の場合、ツール呼び出しと連動した推論を正しく機能させるには、endpoint の `name` を `"Moonshot"`（大文字小文字を区別しません）に設定する**必要があります**。別の名前を使用すると、`thinking is enabled but reasoning_content is missing in assistant tool call message` のようなエラーが発生します。詳細については、[Moonshot のドキュメント](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions)を参照してください。

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```
