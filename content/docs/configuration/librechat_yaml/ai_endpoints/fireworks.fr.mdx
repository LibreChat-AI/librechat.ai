> Clé API Fireworks : [fireworks.ai/api-keys](https://fireworks.ai/api-keys)

**Notes :**

- **Connu :** icône fournie, il est recommandé de récupérer la liste des modèles.
- - L'API peut être stricte pour certains modèles et peut ne pas autoriser des champs comme `user`, auquel cas vous devriez utiliser [`dropParams`.](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams)

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false 
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```
