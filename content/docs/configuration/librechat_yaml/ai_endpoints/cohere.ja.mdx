> Cohere API key: [dashboard.cohere.com](https://dashboard.cohere.com/)

**注記:**

- **既知:** アイコンが提供されています。
- 実験的機能：OpenAI-specには準拠しておらず、エンドポイントの互換性のために新しい手法を使用していますが、いくつかの類似点やパラメータを共有しています。
- Cohere固有のパラメータの完全なリストについては、[Cohere API documentation](https://docs.cohere.com/reference/chat)を参照してください。
- 注: 以下のパラメータはOpenAIとCohereの間で共通して認識されます。以下の設定例では、Cohereのデフォルト設定を優先するため、ほとんどのパラメータは削除されています。
    - `stop`: `stopSequences` にマッピングされます
    - `top_p`: `p` にマッピングされ、最小値/最大値が異なります
    - `frequency_penalty`: `frequencyPenalty` にマッピングされます。最小値/最大値が異なります。
    - `presence_penalty`: `presencePenalty` にマッピングされ、最小値/最大値が異なります
    - `model`: 共有、デフォルトで含まれています。
    - `stream`: 共有、デフォルトで含まれます。
    - `max_tokens`: 共有され、`maxTokens` にマッピングされます。デフォルトでは含まれていません。


```yaml filename="librechat.yaml"
    - name: "cohere"
      apiKey: "${COHERE_API_KEY}"
      baseURL: "https://api.cohere.ai/v1"
      models:
        default: ["command-r","command-r-plus","command-light","command-light-nightly","command","command-nightly"]
        fetch: false
      modelDisplayLabel: "cohere"
      titleModel: "command"
      dropParams: ["stop", "user", "frequency_penalty", "presence_penalty", "temperature", "top_p"]
```
