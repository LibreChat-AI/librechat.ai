ShuttleAI, Shuttle modellerine erişimi, LibreChat'te özel bir endpoint olarak kullanılan OpenAI uyumlu bir API aracılığıyla sağlar.

## Bir API anahtarı alın [#get-an-api-key]

[ShuttleAI anahtarlar sayfasından](https://shuttleai.com/keys) bir anahtar oluşturun. Bunu `.env` dosyanıza ekleyin:

```bash filename=".env"
SHUTTLEAI_API_KEY=your-api-key
```

## Yapılandırma [#configuration]

`librechat.yaml` dosyanızda `endpoints.custom` altına endpoint'i ekleyin:

```yaml filename="librechat.yaml"
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
