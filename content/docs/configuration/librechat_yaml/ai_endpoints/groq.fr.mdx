> Clé API Groq : [wow.groq.com](https://console.groq.com/keys)

**Notes :**

- **Connu :** icône fournie.

- **Température :** Si vous définissez une valeur de température de 0, elle sera convertie en 1e-8. Si vous rencontrez des problèmes, veuillez essayer de définir la valeur sur un float32 supérieur à 0 et inférieur ou égal à 2.

- Groq est actuellement gratuit mais soumis à des limites de débit : 10 requêtes/minute, 100/heure.

```yaml filename="librechat.yaml"
    - name: "groq"
      apiKey: "${GROQ_API_KEY}"
      baseURL: "https://api.groq.com/openai/v1/"
      models:
        default: [
          "llama3-70b-8192",
          "llama3-8b-8192",
          "llama2-70b-4096",
          "mixtral-8x7b-32768",
          "gemma-7b-it",
          ]
        fetch: false
      titleConvo: true
      titleModel: "mixtral-8x7b-32768"
      modelDisplayLabel: "groq"
```
