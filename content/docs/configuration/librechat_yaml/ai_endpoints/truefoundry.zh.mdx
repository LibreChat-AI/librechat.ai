TrueFoundry AI Gateway 是位于您的应用程序与 LLM 提供商及 MCP Servers 之间的代理层。它是一个企业级平台，使用户能够通过统一的接口访问 1000 多种 LLM，同时兼顾可观测性和治理。

## 配置详情 [#configuration-details]

要使用 [TrueFoundry's AI Gateway](https://www.truefoundry.com/ai-gateway)，请按照[此处](https://docs.truefoundry.com/gateway/quick-start)的快速入门指南进行操作。

### 获取 Base URL 和模型名称 [#getting-base-url-and-model-names]

- **Base URL 和 Model Names**：从统一的代码片段中获取您的 TrueFoundry AI Gateway endpoint URL 和模型名称（确保您已添加相同的模型名称）
- **PAT**: 从您的 TrueFoundry 个人访问令牌 [PAT](https://docs.truefoundry.com/gateway/authentication#personal-access-token-pat) 生成

## 使用 TrueFoundry AI Gateway 连接任意 LLM 模型 [#using-truefoundry-ai-gateway-with-any-llm-model]
```yaml filename="librechat.yaml"
      - name: "TrueFoundry"
        apiKey: "${TRUEFOUNDRY_API_KEY}"
        baseURL: "${TRUEFOUNDRY_GATEWAY_URL}"
        models:
            default: ["openai-main/gpt-4o-mini", "openai-main/gpt-4o"]
            fetch: true
        titleConvo: true
        titleModel: "current_model"
        summarize: false
        summaryModel: "current_model"
        modelDisplayLabel: "TrueFoundry:OpenAI"
```
有关更多详细信息，您可以查看：[TrueFoundry Docs](https://docs.truefoundry.com/docs/introduction)
