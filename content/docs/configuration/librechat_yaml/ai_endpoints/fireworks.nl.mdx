Fireworks AI biedt een reeks open modellen aan via een OpenAI-compatibele API, die in LibreChat wordt gebruikt als een custom endpoint.

## Een API-sleutel verkrijgen [#get-an-api-key]

Maak een sleutel aan via de [Fireworks API keys page](https://fireworks.ai/api-keys). Voeg deze toe aan je `.env` bestand:

```bash filename=".env"
FIREWORKS_API_KEY=your-api-key
```

## Configuratie [#configuration]

Voeg het endpoint toe onder `endpoints.custom` in je `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```

## Opmerkingen [#notes]

- De API is strikt voor sommige modellen en kan velden zoals `user` afwijzen, dus verwijder deze met [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams).
