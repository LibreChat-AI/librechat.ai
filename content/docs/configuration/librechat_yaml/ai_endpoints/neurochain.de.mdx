NeurochainAI ist ein OpenAI-kompatibles Inferenz-Netzwerk. Ersetzen Sie `<=generated api key=>` unten durch den Schlüssel, den Sie in der [NeurochainAI REST API documentation](https://app.neurochain.ai/network-integrations/rest-api) erstellen.

## Konfiguration [#configuration]

Fügen Sie den endpoint unter `endpoints.custom` in Ihrer `librechat.yaml` hinzu:

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## Hinweise [#notes]

- Die Modellliste wächst mit der Zeit, daher könnte das Beispiel veraltet sein. Überprüfen Sie [NeurochainAI](https://neurochain.ai) auf die neuesten Modelle.
