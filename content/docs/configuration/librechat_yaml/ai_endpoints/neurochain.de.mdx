> NeurochainAI API-Schlüssel: Erforderlich - [NeurochainAI REST API Dokumentation](https://app.neurochain.ai/network-integrations/rest-api)

**Hinweise:**

- Die API basiert auf der OpenAI API.
- Die Modellliste wächst ständig und ist in diesem Beispiel möglicherweise nicht auf dem neuesten Stand. Behalten Sie [NeurochainAI](https://neurochain.ai) im Auge, um die neuesten Modelle zu sehen.
- Das Beispiel enthält das NeurochainAI-Modell: "Mistral-7B-OpenOrca-GPTQ".

```yaml filename="neurochainai.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```
