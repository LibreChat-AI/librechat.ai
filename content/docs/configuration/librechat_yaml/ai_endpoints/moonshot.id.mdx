Moonshot AI menyediakan model Kimi mereka melalui API yang kompatibel dengan OpenAI yang dapat Anda tambahkan ke LibreChat sebagai endpoint kustom.

## Dapatkan API key [#get-an-api-key]

Buat kunci dari [Moonshot platform](https://platform.moonshot.ai). Tambahkan ke file `.env` Anda:

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## Konfigurasi [#configuration]

Tambahkan endpoint di bawah `endpoints.custom` dalam `librechat.yaml` Anda:

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## Catatan [#notes]

- Untuk model dengan kemampuan penalaran seperti `kimi-k2.5` dan `kimi-k2-thinking`, `name` endpoint harus diatur ke `"Moonshot"` (tidak peka huruf besar/kecil) agar penalaran yang disisipkan (interleaved reasoning) dapat berfungsi dengan pemanggilan alat (tool calls). Nama yang berbeda akan menyebabkan kesalahan seperti `thinking is enabled but reasoning_content is missing in assistant tool call message`. Lihat [dokumentasi Moonshot](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) untuk detailnya.
