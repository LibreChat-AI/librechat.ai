Anyscale Endpoints ofrece modelos abiertos como Llama a través de una API compatible con OpenAI que puedes añadir a LibreChat como un endpoint personalizado.

## Obtener una API key [#get-an-api-key]

Cree una clave desde sus [Anyscale credentials](https://app.endpoints.anyscale.com/credentials). Añádala a su archivo `.env`:

```bash filename=".env"
ANYSCALE_API_KEY=your-api-key
```

## Configuración [#configuration]

Agregue el endpoint bajo `endpoints.custom` en su `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Anyscale"
      apiKey: "${ANYSCALE_API_KEY}"
      baseURL: "https://api.endpoints.anyscale.com/v1"
      models:
        default: [
          "meta-llama/Llama-2-7b-chat-hf",
          ]
        fetch: true
      titleConvo: true
      titleModel: "meta-llama/Llama-2-7b-chat-hf"
      summarize: false
      summaryModel: "meta-llama/Llama-2-7b-chat-hf"
      modelDisplayLabel: "Anyscale"
```

## Notas [#notes]

- Con `fetch: true`, LibreChat carga la lista de modelos disponibles desde Anyscale, por lo que la matriz `default` es solo la selección inicial.
