xAI ofrece sus modelos Grok a través de una API compatible con OpenAI que puedes añadir a LibreChat como un endpoint personalizado.

## Obtener una API key [#get-an-api-key]

Cree una clave desde la [xAI console](https://console.x.ai/). Añádala a su archivo `.env`:

```bash filename=".env"
XAI_API_KEY=your-api-key
```

## Configuración [#configuration]

Agregue el endpoint bajo `endpoints.custom` en su `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "xai"
      apiKey: "${XAI_API_KEY}"
      baseURL: "https://api.x.ai/v1"
      models:
        default: ["grok-beta"]
        fetch: false
      titleConvo: true
      titleMethod: "completion"
      titleModel: "grok-beta"
      summarize: false
      summaryModel: "grok-beta"
      modelDisplayLabel: "Grok"
```

## Notas [#notes]

- `titleMethod: "completion"` genera títulos de conversación a través del endpoint de chat completions, que es lo que admite Grok.
