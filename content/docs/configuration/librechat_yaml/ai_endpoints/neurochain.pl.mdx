NeurochainAI to sieć wnioskowania kompatybilna z OpenAI. Zastąp `<=generated api key=>` poniżej kluczem, który utworzysz w [dokumentacji NeurochainAI REST API](https://app.neurochain.ai/network-integrations/rest-api).

## Konfiguracja [#configuration]

Dodaj endpoint w sekcji `endpoints.custom` w swoim `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## Uwagi [#notes]

- Lista modeli z czasem się powiększa, więc przykład może być nieaktualny. Sprawdź [NeurochainAI](https://neurochain.ai), aby uzyskać najnowsze modele.
