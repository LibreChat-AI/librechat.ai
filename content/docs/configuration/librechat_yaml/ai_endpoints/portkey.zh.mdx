> Portkey API key: [app.portkey.ai](https://app.portkey.ai/)


**注意：**
- LibreChat 要求必须提供 `API key` 字段。对于 Portkey 集成，我们并不需要它，因此可以传入一个 `dummy` 字符串。
- Portkey 与 LibreChat 集成，提供可观测性、50 多种护栏（guardrails）、缓存以及带有回退和重试机制的条件路由，以实现可靠的生产级部署。
- Portkey 支持 250 多种模型，你可以在 [Portkey Docs](https://docs.portkey.ai/docs/integrations/llms) 上找到提供商列表。
- 您可以通过两种方式使用 Portkey——通过 [Virtual Keys](https://docs.portkey.ai/docs/product/ai-gateway/virtual-keys) 或 [Configs](https://docs.portkey.ai/docs/product/ai-gateway/configs)。
- 您可以在使用 Portkey Config 时使用 `top_p`、`max Tokens` 等 Model 特定参数。[了解更多](https://docs.portkey.ai/docs/product/ai-gateway/configs#configs)


## 使用 Portkey AI 和虚拟密钥 (Virtual Keys) [#using-portkey-ai-with-virtual-keys]
```yaml filename="librechat.yaml"
      - name: "Portkey"
        apiKey: "dummy"  
        baseURL: ${PORTKEY_GATEWAY_URL}
        headers:
            x-portkey-api-key: "${PORTKEY_API_KEY}"
            x-portkey-virtual-key: "PORTKEY_OPENAI_VIRTUAL_KEY"
        models:
            default: ["gpt-4o-mini"]
            fetch: true
        titleConvo: true
        titleModel: "current_model"
        summarize: false
        summaryModel: "current_model"
        modelDisplayLabel: "Portkey:OpenAI"
        iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```


## 使用 Portkey AI 和 Configs [#using-portkey-ai-with-configs]
```yaml filename="librechat.yaml"
    - name: "Portkey"
      apikey: "dummy"
      baseURL: ${PORTKEY_GATEWAY_URL}
      headers:
        x-portkey-api-key: "${PORTKEY_API_KEY}"
        x-portkey-config: "pc-libre-xxx"
      models:
        default: ["llama-3.2"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "Portkey:Llama"
      iconURL: https://images.crunchbase.com/image/upload/c_pad,f_auto,q_auto:eco,dpr_1/rjqy7ghvjoiu4cd1xjbf
```
