ShuttleAIは、OpenAI互換APIを通じてShuttleモデルへのアクセスを提供しており、LibreChatではカスタムendpointとして使用されます。

## APIキーを取得する [#get-an-api-key]

[ShuttleAI keys page](https://shuttleai.com/keys) からキーを作成します。それを `.env` ファイルに追加してください：

```bash filename=".env"
SHUTTLEAI_API_KEY=your-api-key
```

## 設定 [#configuration]

`librechat.yaml` の `endpoints.custom` の下にエンドポイントを追加します：

```yaml filename="librechat.yaml"
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
