NeurochainAI is een OpenAI-compatibel inferentienetwerk. Vervang `<=generated api key=>` hieronder door de sleutel die je aanmaakt in de [NeurochainAI REST API documentation](https://app.neurochain.ai/network-integrations/rest-api).

## Configuratie [#configuration]

Voeg het endpoint toe onder `endpoints.custom` in je `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## Opmerkingen [#notes]

- De modellijst groeit na verloop van tijd, dus het voorbeeld kan verouderd zijn. Kijk op [NeurochainAI](https://neurochain.ai) voor de nieuwste modellen.
