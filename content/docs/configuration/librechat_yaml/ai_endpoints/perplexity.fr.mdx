> Clé API Perplexity : [perplexity.ai/settings/api](https://www.perplexity.ai/settings/api)

**Notes :**

- **Connu :** icône fournie.
- **Problème connu :** la récupération de la liste des modèles n'est pas prise en charge.
- L'API peut être stricte pour certains modèles et peut ne pas autoriser des champs comme `stop` ; `frequency_penalty` peut provoquer une erreur lorsqu'il est défini sur 0, auquel cas vous devriez utiliser [`dropParams`.](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams)
- L'exemple inclut une liste de modèles, mise à jour pour la dernière fois le 3 juillet 2024, pour votre commodité.

```yaml
    - name: "Perplexity"
      apiKey: "${PERPLEXITY_API_KEY}"
      baseURL: "https://api.perplexity.ai/"
      models:
        default: [
          "sonar-deep-research",
          "sonar-reasoning-pro",
          "sonar-reasoning",
          "sonar-pro",
          "sonar",
          "r1-1776"
          ]
        fetch: false # fetching list of models is not supported
      titleConvo: true
      titleModel: "llama-3-sonar-small-32k-chat"
      summarize: false
      summaryModel: "llama-3-sonar-small-32k-chat"
      dropParams: ["stop", "frequency_penalty"]
      modelDisplayLabel: "Perplexity"
```
