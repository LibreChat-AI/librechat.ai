Moonshot AI biedt zijn Kimi-modellen aan via een OpenAI-compatibele API die je als custom endpoint aan LibreChat kunt toevoegen.

## Een API-sleutel verkrijgen [#get-an-api-key]

Maak een sleutel aan via het [Moonshot platform](https://platform.moonshot.ai). Voeg deze toe aan je `.env` bestand:

```bash filename=".env"
MOONSHOT_API_KEY=your-api-key
```

## Configuratie [#configuration]

Voeg het endpoint toe onder `endpoints.custom` in je `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Moonshot"
      apiKey: "${MOONSHOT_API_KEY}"
      baseURL: "https://api.moonshot.ai/v1"
      models:
        default: ["kimi-k2.5"]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      modelDisplayLabel: "Moonshot"
```

## Opmerkingen [#notes]

- Voor modellen met redeneervermogen zoals `kimi-k2.5` en `kimi-k2-thinking`, moet de endpoint `name` worden ingesteld op `"Moonshot"` (niet hoofdlettergevoelig) om interleaved redeneren te laten werken met tool calls. Een andere naam veroorzaakt fouten zoals `thinking is enabled but reasoning_content is missing in assistant tool call message`. Zie [Moonshot's documentation](https://platform.moonshot.ai/docs/guide/use-kimi-k2-thinking-model#frequently-asked-questions) voor details.
