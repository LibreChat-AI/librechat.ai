> Clave de API de Mistral: [console.mistral.ai](https://console.mistral.ai/)

**Notas:**

- **Conocido:** icono proporcionado, manejo especial de los roles de mensaje: el mensaje del sistema solo se permite al principio del payload de mensajes.

- La API es estricta con los parámetros no reconocidos y los errores no son descriptivos (generalmente "no body")

    - El uso de [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams) para eliminar los parámetros "user", "frequency_penalty", "presence_penalty" es obligatorio.
    - `stop` ya no se incluye como parámetro predeterminado, por lo que ya no es necesario incluirlo en [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams), a menos que desee evitar por completo que los usuarios configuren este campo.
    
- Permite obtener la lista de modelos, pero tenga cuidado de no utilizar modelos de embedding para el chat.

```yaml filename="librechat.yaml"
    - name: "Mistral"
      apiKey: "${MISTRAL_API_KEY}"
      baseURL: "https://api.mistral.ai/v1"
      models:
        default: ["mistral-tiny", "mistral-small", "mistral-medium", "mistral -large-latest"]
        fetch: true
      titleConvo: true
      titleModel: "mistral-tiny"
      modelDisplayLabel: "Mistral"
      dropParams: ["stop", "user", "frequency_penalty", "presence_penalty"]
```
