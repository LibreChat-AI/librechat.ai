Anyscale Endpoints offre modelli aperti come Llama tramite un'API compatibile con OpenAI che puoi aggiungere a LibreChat come endpoint personalizzato.

## Ottieni una API key [#get-an-api-key]

Crea una chiave dalle tue [Anyscale credentials](https://app.endpoints.anyscale.com/credentials). Aggiungila al tuo file `.env`:

```bash filename=".env"
ANYSCALE_API_KEY=your-api-key
```

## Configurazione [#configuration]

Aggiungi l'endpoint sotto `endpoints.custom` nel tuo `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Anyscale"
      apiKey: "${ANYSCALE_API_KEY}"
      baseURL: "https://api.endpoints.anyscale.com/v1"
      models:
        default: [
          "meta-llama/Llama-2-7b-chat-hf",
          ]
        fetch: true
      titleConvo: true
      titleModel: "meta-llama/Llama-2-7b-chat-hf"
      summarize: false
      summaryModel: "meta-llama/Llama-2-7b-chat-hf"
      modelDisplayLabel: "Anyscale"
```

## Note [#notes]

- Con `fetch: true`, LibreChat carica l'elenco dei modelli disponibili da Anyscale, quindi l'array `default` rappresenta solo la selezione iniziale.
