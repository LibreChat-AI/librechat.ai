NEAR AI Cloud udostępnia endpoint zgodny z OpenAI, który możesz dodać do LibreChat poprzez `endpoints.custom`.

## Uzyskaj klucz API [#get-an-api-key]

Utwórz klucz API z [NEAR AI Cloud](https://cloud.near.ai/). Dodaj go do swojego pliku `.env`:

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## Konfiguracja [#configuration]

Dodaj endpoint w sekcji `endpoints.custom` w swoim `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

Aby pozwolić każdemu użytkownikowi na podanie własnego klucza przez interfejs LibreChat zamiast odczytywania go z pliku `.env`, ustaw `apiKey: "user_provided"`.

## Uwagi [#notes]

- Dzięki `fetch: true`, LibreChat wczytuje listę modeli z punktu końcowego `/v1/models` zgodnego z OpenAI w chmurze NEAR AI. Tablica `default` stanowi jedynie początkowy wybór.
- Używaj identyfikatorów modeli dokładnie tak, jak pojawiają się one w [katalogu modeli NEAR AI Cloud](https://cloud-api.near.ai/v1/model/list).
- Dostępność modeli może zmieniać się w czasie. Jeśli wybrany model jest niedostępny, wybierz inny model obsługujący czat z katalogu.
