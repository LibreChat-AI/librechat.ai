Fireworks AI udostępnia szereg otwartych modeli za pośrednictwem API zgodnego z OpenAI, używanego w LibreChat jako niestandardowy endpoint.

## Uzyskaj klucz API [#get-an-api-key]

Utwórz klucz na [stronie kluczy API Fireworks](https://fireworks.ai/api-keys). Dodaj go do swojego pliku `.env`:

```bash filename=".env"
FIREWORKS_API_KEY=your-api-key
```

## Konfiguracja [#configuration]

Dodaj endpoint w sekcji `endpoints.custom` w swoim `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```

## Uwagi [#notes]

- API jest rygorystyczne w przypadku niektórych modeli i może odrzucać pola takie jak `user`, więc usuń je za pomocą [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams).
