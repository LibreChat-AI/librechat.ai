> groq API-Schlüssel: [wow.groq.com](https://console.groq.com/keys)

**Hinweise:**

- **Bekannt:** Icon bereitgestellt.

- **Temperatur:** Wenn Sie einen Temperaturwert von 0 festlegen, wird dieser in 1e-8 umgewandelt. Falls Probleme auftreten, versuchen Sie bitte, den Wert auf einen float32-Wert größer als 0 und kleiner oder gleich 2 zu setzen.

- Groq ist derzeit kostenlos, unterliegt jedoch Ratenbegrenzungen: 10 Anfragen/Minute, 100/Stunde.

```yaml filename="librechat.yaml"
    - name: "groq"
      apiKey: "${GROQ_API_KEY}"
      baseURL: "https://api.groq.com/openai/v1/"
      models:
        default: [
          "llama3-70b-8192",
          "llama3-8b-8192",
          "llama2-70b-4096",
          "mixtral-8x7b-32768",
          "gemma-7b-it",
          ]
        fetch: false
      titleConvo: true
      titleModel: "mixtral-8x7b-32768"
      modelDisplayLabel: "groq"
```
