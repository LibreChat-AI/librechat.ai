together.ai menyediakan katalog besar model terbuka melalui API yang kompatibel dengan OpenAI, yang dapat Anda tambahkan ke LibreChat sebagai endpoint kustom.

## Dapatkan API key [#get-an-api-key]

Buat kunci di [api.together.xyz/settings/api-keys](https://api.together.xyz/settings/api-keys). Tambahkan ke file `.env` Anda:

```bash filename=".env"
TOGETHERAI_API_KEY=your-api-key
```

## Catatan [#notes]

- Pengambilan daftar model tidak didukung, jadi `fetch` diatur ke `false` dan Anda harus mengelola array `default` sendiri.
- Katalog together.ai sering berubah. Gunakan skrip di bawah ini untuk menarik model chat terkini alih-alih menyunting daftar tersebut secara manual.

<Callout type="tip" title="Ambil dan urutkan model" collapsible>
Skrip Python ini mengambil model obrolan yang tersedia dan menulisnya, dalam urutan yang telah diurutkan, ke `models_togetherai.json` dalam format yang mudah ditempelkan ke dalam konfigurasi yaml. Atur kunci API Anda terlebih dahulu.

```py filename="fetch_togetherai.py"
import json

import requests

# API key
api_key = ""

# API endpoint
url = "https://api.together.xyz/v1/models"

# headers
headers = {
    "accept": "application/json",
    "Authorization": f"Bearer {api_key}"
}

# make request
response = requests.get(url, headers=headers)

# parse JSON response
data = response.json()

# extract an ordered list of unique model IDs
model_ids = sorted(
    [
        model['id']
        for model in data
        if model['type'] == 'chat'
    ]
)

# write result to a text file
with open("models_togetherai.json", "w") as file:
    json.dump(model_ids, file, indent=2)
```
</Callout>
