> Fireworks API-Schlüssel: [fireworks.ai/api-keys](https://fireworks.ai/api-keys)

**Hinweise:**

- **Bekannt:** Icon bereitgestellt, das Abrufen der Modellliste wird empfohlen.
- - Die API kann für einige Modelle streng sein und Felder wie `user` möglicherweise nicht zulassen. In diesem Fall sollten Sie [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams) verwenden.

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false 
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```
