Fireworks AI stellt eine Reihe offener Modelle über eine OpenAI-kompatible API bereit, die in LibreChat als benutzerdefinierter endpoint verwendet werden.

## Einen API-Schlüssel erhalten [#get-an-api-key]

Erstellen Sie einen Schlüssel auf der [Fireworks API keys page](https://fireworks.ai/api-keys). Fügen Sie ihn Ihrer `.env` Datei hinzu:

```bash filename=".env"
FIREWORKS_API_KEY=your-api-key
```

## Konfiguration [#configuration]

Fügen Sie den endpoint unter `endpoints.custom` in Ihrer `librechat.yaml` hinzu:

```yaml filename="librechat.yaml"
    - name: "Fireworks"
      apiKey: "${FIREWORKS_API_KEY}"
      baseURL: "https://api.fireworks.ai/inference/v1"
      models:
        default: [
          "accounts/fireworks/models/mixtral-8x7b-instruct",
          ]
        fetch: true
      titleConvo: true
      titleModel: "accounts/fireworks/models/llama-v2-7b-chat"
      summarize: false
      summaryModel: "accounts/fireworks/models/llama-v2-7b-chat"
      modelDisplayLabel: "Fireworks"
      dropParams: ["user"]
```

## Hinweise [#notes]

- Die API ist bei einigen Modellen streng und lehnt möglicherweise Felder wie `user` ab. Entfernen Sie diese daher mit [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams).
