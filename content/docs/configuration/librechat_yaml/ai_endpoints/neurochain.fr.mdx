> Clé API NeurochainAI : Requise - [Documentation de l'API REST NeurochainAI](https://app.neurochain.ai/network-integrations/rest-api)

**Notes :**

- L'API est basée sur l'API OpenAI.
- La liste des modèles est en constante évolution et peut ne pas être à jour dans cet exemple. Gardez un œil sur [NeurochainAI](https://neurochain.ai) pour connaître les derniers modèles.
- L'exemple inclut le modèle NeurochainAI : "Mistral-7B-OpenOrca-GPTQ".

```yaml filename="neurochainai.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```
