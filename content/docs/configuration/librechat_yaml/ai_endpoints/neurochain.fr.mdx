NeurochainAI est un réseau d'inférence compatible avec OpenAI. Remplacez `<=generated api key=>` ci-dessous par la clé que vous créez dans la [documentation de l'API REST de NeurochainAI](https://app.neurochain.ai/network-integrations/rest-api).

## Configuration [#configuration]

Ajoutez l'endpoint sous `endpoints.custom` dans votre `librechat.yaml` :

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## Notes [#notes]

- La liste des modèles s'agrandit avec le temps, l'exemple peut donc être obsolète. Consultez [NeurochainAI](https://neurochain.ai) pour connaître les derniers modèles.
