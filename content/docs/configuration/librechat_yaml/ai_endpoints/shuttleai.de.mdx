> ShuttleAI API-Schlüssel: [shuttleai.com/keys](https://shuttleai.com/keys)

**Hinweise:**

- **Bekannt:** Icon bereitgestellt, das Abrufen der Modellliste wird empfohlen.

```yaml
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
