ShuttleAI bietet Zugriff auf seine Shuttle-Modelle über eine OpenAI-kompatible API, die in LibreChat als benutzerdefinierter endpoint verwendet wird.

## Einen API-Schlüssel erhalten [#get-an-api-key]

Erstellen Sie einen Schlüssel auf der [ShuttleAI keys page](https://shuttleai.com/keys). Fügen Sie ihn Ihrer `.env` Datei hinzu:

```bash filename=".env"
SHUTTLEAI_API_KEY=your-api-key
```

## Konfiguration [#configuration]

Fügen Sie den endpoint unter `endpoints.custom` in Ihrer `librechat.yaml` hinzu:

```yaml filename="librechat.yaml"
    - name: "ShuttleAI"
      apiKey: "${SHUTTLEAI_API_KEY}"
      baseURL: "https://api.shuttleai.com/v1"
      models:
        default: [
          "shuttle-2.5", "shuttle-2.5-mini"
          ]
        fetch: true
      titleConvo: true
      titleModel: "shuttle-2.5-mini"
      summarize: false
      summaryModel: "shuttle-2.5-mini"
      modelDisplayLabel: "ShuttleAI"
      dropParams: ["user", "stop"]
```
