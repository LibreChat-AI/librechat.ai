> Clé API Mistral : [console.mistral.ai](https://console.mistral.ai/)

**Notes :**

- **Connu :** icône fournie, traitement spécial des rôles de message : le message système est uniquement autorisé au début de la charge utile (payload) des messages.

- L'API est stricte concernant les paramètres non reconnus et les erreurs ne sont pas descriptives (généralement « no body »)

    - L'utilisation de [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams) pour supprimer les paramètres "user", "frequency_penalty" et "presence_penalty" est requise.
    - `stop` n'est plus inclus en tant que paramètre par défaut, il n'est donc plus nécessaire de l'inclure dans [`dropParams`](/docs/configuration/librechat_yaml/object_structure/custom_endpoint#dropparams), à moins que vous ne souhaitiez empêcher complètement les utilisateurs de configurer ce champ.
    
- Permet de récupérer la liste des modèles, mais veillez à ne pas utiliser de modèles d'embedding pour le chat.

```yaml filename="librechat.yaml"
    - name: "Mistral"
      apiKey: "${MISTRAL_API_KEY}"
      baseURL: "https://api.mistral.ai/v1"
      models:
        default: ["mistral-tiny", "mistral-small", "mistral-medium", "mistral -large-latest"]
        fetch: true
      titleConvo: true
      titleModel: "mistral-tiny"
      modelDisplayLabel: "Mistral"
      dropParams: ["stop", "user", "frequency_penalty", "presence_penalty"]
```
