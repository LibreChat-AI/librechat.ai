NEAR AI Cloud fournit un endpoint compatible avec OpenAI que vous pouvez ajouter à LibreChat via `endpoints.custom`.

## Obtenir une clé API [#get-an-api-key]

Créez une clé API depuis [NEAR AI Cloud](https://cloud.near.ai/). Ajoutez-la à votre fichier `.env` :

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## Configuration [#configuration]

Ajoutez l'endpoint sous `endpoints.custom` dans votre `librechat.yaml` :

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

Pour permettre à chaque utilisateur de fournir sa propre clé via l'interface utilisateur de LibreChat au lieu d'en lire une depuis `.env`, définissez `apiKey: "user_provided"`.

## Notes [#notes]

- Avec `fetch: true`, LibreChat charge la liste des modèles depuis le endpoint `/v1/models` compatible OpenAI de NEAR AI Cloud. Le tableau `default` ne constitue que la sélection initiale.
- Utilisez les IDs de modèle exactement tels qu'ils apparaissent dans le [catalogue de modèles NEAR AI Cloud](https://cloud-api.near.ai/v1/model/list).
- La disponibilité des modèles peut changer au fil du temps. Si un modèle sélectionné n'est pas disponible, choisissez un autre modèle capable de discuter dans le catalogue.
