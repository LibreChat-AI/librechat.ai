> Cohere API key: [dashboard.cohere.com](https://dashboard.cohere.com/)

**注意：**

- **已知：** 已提供图标。
- 实验性功能：不遵循 OpenAI-spec，使用了一种新的端点兼容性方法，并共享了部分相似之处和参数。
- 有关 Cohere 特定参数的完整列表，请参阅 [Cohere API documentation](https://docs.cohere.com/reference/chat)。
- 注意：以下参数在 OpenAI 和 Cohere 之间通用。下方的示例配置中移除了大部分参数，以优先使用 Cohere 的默认设置：
    - `stop`：映射到 `stopSequences`
    - `top_p`：映射为 `p`，具有不同的最小值/最大值
    - `frequency_penalty`：映射至 `frequencyPenalty`，具有不同的最小值/最大值
    - `presence_penalty`：映射到 `presencePenalty`，具有不同的最小值/最大值
    - `model`: 共享，默认包含。
    - `stream`：共享，默认包含。
    - `max_tokens`：共享，映射为 `maxTokens`，默认不包含。


```yaml filename="librechat.yaml"
    - name: "cohere"
      apiKey: "${COHERE_API_KEY}"
      baseURL: "https://api.cohere.ai/v1"
      models:
        default: ["command-r","command-r-plus","command-light","command-light-nightly","command","command-nightly"]
        fetch: false
      modelDisplayLabel: "cohere"
      titleModel: "command"
      dropParams: ["stop", "user", "frequency_penalty", "presence_penalty", "temperature", "top_p"]
```
