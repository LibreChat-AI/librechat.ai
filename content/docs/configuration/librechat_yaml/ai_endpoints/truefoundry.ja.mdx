TrueFoundry AI Gatewayは、アプリケーションとLLMプロバイダーおよびMCP Serversの間に位置するプロキシ層です。これは、可観測性とガバナンスを確保しながら、統一されたインターフェースを使用して1000以上のLLMへのアクセスを可能にするエンタープライズグレードのプラットフォームです。

## 設定の詳細 [#configuration-details]

[TrueFoundry's AI Gateway](https://www.truefoundry.com/ai-gateway) を使用するには、[こちら](https://docs.truefoundry.com/gateway/quick-start)のクイックスタートガイドに従ってください。

### ベースURLとモデル名の取得 [#getting-base-url-and-model-names]

- **Base URL and Model Names**: TrueFoundry AI Gatewayのendpoint URLとモデル名を統合コードスニペットから取得します（同じモデル名を追加していることを確認してください）。
- **PAT**: TrueFoundry Personal Access Token [PAT](https://docs.truefoundry.com/gateway/authentication#personal-access-token-pat) から生成します。

## TrueFoundry AI Gateway を任意の LLM モデルで使用する [#using-truefoundry-ai-gateway-with-any-llm-model]
```yaml filename="librechat.yaml"
      - name: "TrueFoundry"
        apiKey: "${TRUEFOUNDRY_API_KEY}"
        baseURL: "${TRUEFOUNDRY_GATEWAY_URL}"
        models:
            default: ["openai-main/gpt-4o-mini", "openai-main/gpt-4o"]
            fetch: true
        titleConvo: true
        titleModel: "current_model"
        summarize: false
        summaryModel: "current_model"
        modelDisplayLabel: "TrueFoundry:OpenAI"
```
詳細については、[TrueFoundry Docs](https://docs.truefoundry.com/docs/introduction) をご確認ください。
