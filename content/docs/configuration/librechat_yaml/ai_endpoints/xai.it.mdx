xAI fornisce i suoi modelli Grok tramite un'API compatibile con OpenAI che puoi aggiungere a LibreChat come endpoint personalizzato.

## Ottieni una API key [#get-an-api-key]

Crea una chiave dalla [xAI console](https://console.x.ai/). Aggiungila al tuo file `.env`:

```bash filename=".env"
XAI_API_KEY=your-api-key
```

## Configurazione [#configuration]

Aggiungi l'endpoint sotto `endpoints.custom` nel tuo `librechat.yaml`:

```yaml filename="librechat.yaml"
    - name: "xai"
      apiKey: "${XAI_API_KEY}"
      baseURL: "https://api.x.ai/v1"
      models:
        default: ["grok-beta"]
        fetch: false
      titleConvo: true
      titleMethod: "completion"
      titleModel: "grok-beta"
      summarize: false
      summaryModel: "grok-beta"
      modelDisplayLabel: "Grok"
```

## Note [#notes]

- `titleMethod: "completion"` genera i titoli delle conversazioni tramite l'endpoint chat completions, che è ciò che Grok supporta.
