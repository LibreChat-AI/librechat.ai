NeurochainAI, OpenAI ile uyumlu bir çıkarım (inference) ağıdır. Aşağıdaki `<=generated api key=>` kısmını, [NeurochainAI REST API belgelerinde](https://app.neurochain.ai/network-integrations/rest-api) oluşturduğunuz anahtar ile değiştirin.

## Yapılandırma [#configuration]

`librechat.yaml` dosyanızda `endpoints.custom` altına endpoint'i ekleyin:

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## Notlar [#notes]

- Model listesi zamanla büyüdüğü için örnek güncelliğini yitirmiş olabilir. En güncel modeller için [NeurochainAI](https://neurochain.ai) adresini kontrol edin.
