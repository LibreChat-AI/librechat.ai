NEAR AI Cloud menyediakan endpoint yang kompatibel dengan OpenAI yang dapat Anda tambahkan ke LibreChat melalui `endpoints.custom`.

## Dapatkan API key [#get-an-api-key]

Buat kunci API dari [NEAR AI Cloud](https://cloud.near.ai/). Tambahkan ke file `.env` Anda:

```bash filename=".env"
NEARAI_API_KEY=your-api-key
```

## Konfigurasi [#configuration]

Tambahkan endpoint di bawah `endpoints.custom` dalam `librechat.yaml` Anda:

```yaml filename="librechat.yaml"
    - name: "nearai"
      apiKey: "${NEARAI_API_KEY}"
      baseURL: "https://cloud-api.near.ai/v1"
      models:
        default:
          - "z-ai/glm-5.2"
          - "deepseek-ai/DeepSeek-V4-Flash"
        fetch: true
      titleConvo: true
      titleModel: "z-ai/glm-5.2"
      modelDisplayLabel: "NEAR AI Cloud"
```

Untuk membiarkan setiap pengguna memasukkan kunci mereka sendiri melalui UI LibreChat alih-alih membacanya dari `.env`, atur `apiKey: "user_provided"`.

## Catatan [#notes]

- Dengan `fetch: true`, LibreChat memuat daftar model dari endpoint `/v1/models` yang kompatibel dengan OpenAI milik NEAR AI Cloud. Array `default` hanyalah pilihan awal.
- Gunakan ID model persis seperti yang muncul di [katalog model NEAR AI Cloud](https://cloud-api.near.ai/v1/model/list).
- Ketersediaan model dapat berubah sewaktu-waktu. Jika model yang dipilih tidak tersedia, pilih model lain yang mendukung obrolan dari katalog.
