Anyscale Endpoints, Llama gibi açık kaynaklı modelleri, LibreChat'e özel bir endpoint olarak ekleyebileceğiniz OpenAI uyumlu bir API aracılığıyla sunar.

## Bir API anahtarı alın [#get-an-api-key]

[Anyscale kimlik bilgilerinizden](https://app.endpoints.anyscale.com/credentials) bir anahtar oluşturun. Bunu `.env` dosyanıza ekleyin:

```bash filename=".env"
ANYSCALE_API_KEY=your-api-key
```

## Yapılandırma [#configuration]

`librechat.yaml` dosyanızda `endpoints.custom` altına endpoint'i ekleyin:

```yaml filename="librechat.yaml"
    - name: "Anyscale"
      apiKey: "${ANYSCALE_API_KEY}"
      baseURL: "https://api.endpoints.anyscale.com/v1"
      models:
        default: [
          "meta-llama/Llama-2-7b-chat-hf",
          ]
        fetch: true
      titleConvo: true
      titleModel: "meta-llama/Llama-2-7b-chat-hf"
      summarize: false
      summaryModel: "meta-llama/Llama-2-7b-chat-hf"
      modelDisplayLabel: "Anyscale"
```

## Notlar [#notes]

- `fetch: true` ile LibreChat, mevcut model listesini Anyscale'den yükler, bu nedenle `default` dizisi yalnızca başlangıç seçimidir.
