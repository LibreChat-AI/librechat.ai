NeurochainAI 是一个兼容 OpenAI 的推理网络。将下方的 `<=generated api key=>` 替换为您在 [NeurochainAI REST API documentation](https://app.neurochain.ai/network-integrations/rest-api) 中创建的密钥。

## 配置 [#configuration]

在你的 `librechat.yaml` 中，将 endpoint 添加到 `endpoints.custom` 下：

```yaml filename="librechat.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```

## 注意事项 [#notes]

- 模型列表会随时间增加，因此示例可能已过时。请查看 [NeurochainAI](https://neurochain.ai) 以获取最新模型。
