> NeurochainAI API key: 必需 - [NeurochainAI REST API Documentation](https://app.neurochain.ai/network-integrations/rest-api)

**注意：**

- Api 基于 OpenAI API。
- 模型列表在不断增长，此示例中的内容可能不是最新的。请关注 [NeurochainAI](https://neurochain.ai) 以获取最新模型。
- 该示例包含 NeurochainAI 模型：“Mistral-7B-OpenOrca-GPTQ”。

```yaml filename="neurochainai.yaml"
    - name: "NeurochainAI"
      apiKey: "<=generated api key=>"
      baseURL: "https://ncmb.neurochain.io/v1/"
      models:
        default: [
          "Mistral-7B-OpenOrca-GPTQ"
        ]
        fetch: true
      titleConvo: true
      titleModel: "current_model"
      summarize: false
      summaryModel: "current_model"
      modelDisplayLabel: "NeurochainAI"
      iconURL: "https://raw.githubusercontent.com/LibreChat-AI/librechat-config-yaml/refs/heads/main/icons/NeurochainAI.png"
```
