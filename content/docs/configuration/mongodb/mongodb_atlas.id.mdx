  Atlas
---

<Steps>
<Step>

### Membuat Akun MongoDB Atlas [#create-a-mongodb-atlas-account]

1. Buka tab baru di peramban web Anda dan kunjungi [account.mongodb.com/account/register](https://account.mongodb.com/account/register).
2. Isi informasi yang diperlukan dan buat akun Anda.

</Step>
<Step>

### Membuat Proyek Baru [#create-a-new-project]

Setelah menyiapkan akun Anda, klik tombol "New Project" dan beri nama (contoh: "LibreChat").

</Step>
<Step>

### Membangun Database [#build-a-database]

Klik tombol "Build a Database".

</Step>
<Step>

### Pilih Tingkat Gratis [#choose-the-free-tier]

Pilih opsi "Shared Clusters", yang merupakan tingkat gratis.

</Step>
<Step>

### Beri Nama Klaster Anda [#name-your-cluster]

Berikan nama pada klaster Anda (contoh: "LibreChat-Cluster") dan klik "Create Cluster".

</Step>
<Step>

### Mengatur Kredensial Basis Data [#set-up-database-credentials]

1. Klik opsi "Database & Network Access" di bilah sisi.
2. Klik tombol "Add New Database User".
3. Masukkan nama pengguna dan kata sandi yang aman, lalu klik "Add User".

</Step>
<Step>

### Konfigurasi Akses Jaringan [#configure-network-access]

1. Klik opsi "IP Access List" di bilah sisi.
2. Klik tombol "Add IP Address".
3. Masukkan "0.0.0.0/0" lalu klik "Confirm".

</Step>
<Step>

### Dapatkan Connection String Anda [#get-your-connection-string]

1. Klik opsi "Project Overview" di bilah sisi.
2. Di "Application Development" klik "Get connection string".
3. Klik tombol "Connect".
4. Pilih "Connect Your Application".
5. Salin string koneksi yang disediakan.
6. Ganti `<db_username>` dan `<db_password>` pada string koneksi dengan nama pengguna dan kata sandi yang Anda tetapkan pada langkah kredensial. Hapus karakter `<>` di sekitar kata sandi.

String koneksi akhir Anda seharusnya terlihat seperti ini:

```sh filename="Connection String"
mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

</Step>
<Step>

### Perbarui File .env [#update-the-env-file]

1. Di proyek LibreChat Anda, buka file `.env`.
2. Cari variabel `MONGO_URI` dan tempelkan string koneksi Anda:

```sh filename=".env"
MONGO_URI=mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

</Step>
</Steps>

Selesai! Anda kini telah menyiapkan basis data MongoDB daring untuk LibreChat menggunakan MongoDB Atlas, dan Anda telah memperbarui aplikasi LibreChat Anda untuk menggunakan koneksi basis data ini. Aplikasi Anda sekarang seharusnya sudah dapat terhubung ke basis data MongoDB daring tersebut.

## Catatan tentang Docker [#note-about-docker]

<Callout type="note" title="Docker">
**Catatan:** Jika Anda menggunakan LibreChat dengan Docker, Anda perlu menggunakan file `docker-compose.override.yml`. File pengganti (override) ini memungkinkan Anda untuk mencegah penginstalan instans MongoDB yang disertakan. Sebagai gantinya, kontainer Docker LibreChat Anda akan menggunakan database MongoDB Atlas daring yang baru saja Anda siapkan. Untuk informasi lebih lanjut mengenai penggunaan file override, silakan merujuk ke [Panduan Override Docker](/docs/configuration/docker_override) kami.
</Callout>
