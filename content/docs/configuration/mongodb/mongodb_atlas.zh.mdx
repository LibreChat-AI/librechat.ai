---

## 1. 创建 MongoDB Atlas 账户 [#1-create-a-mongodb-atlas-account]
1. 在您的网络浏览器中打开一个新标签页，并访问 [account.mongodb.com/account/register](https://account.mongodb.com/account/register)。
2. 填写所需信息并创建您的账户。

## 2. 创建新项目 [#2-create-a-new-project]
1. 设置好您的账户后，点击“New Project”按钮并为其命名（例如：“LibreChat”）。

## 3. 构建数据库 [#3-build-a-database]
1. 点击“Build a Database”按钮。

## 4. 选择免费层级 [#4-choose-the-free-tier]
1. 选择“Shared Clusters”选项，这是免费层级。

## 5. 为您的集群命名 [#5-name-your-cluster]
1. 为您的集群命名（例如 "LibreChat-Cluster"），然后点击 "Create Cluster"。

## 6. 设置数据库凭据 [#6-set-up-database-credentials]
1. 点击侧边栏中的 “Database & Network Access” 选项。
2. 点击“Add New Database User”按钮。
3. 输入用户名和安全密码，然后点击“Add User”。

## 7. 配置网络访问 [#7-configure-network-access]
1. 点击侧边栏中的“IP Access List”选项。
2. 点击“Add IP Address”按钮。
3. 输入 "0.0.0.0/0" 并点击 "Confirm"。

## 8. 获取您的连接字符串 [#8-get-your-connection-string]
1. 点击侧边栏中的“Project Overview”选项。
2. 在 "Application Development" 中点击 "Get connection string"。
2. 点击“Connect”按钮。
3. 选择 "Connect Your Application"。
4. 复制提供的连接字符串。
5. 将连接字符串中的 `<db_username>` 和 `<db_password>` 替换为您在第 6 步中设置的用户名和密码。请移除密码周围的 `<>` 字符。

您最终的连接字符串应该类似于这样：

```sh filename="Connection String"
mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

## 9. 更新 .env 文件 [#9-update-the-env-file]
1. 在你的 LibreChat 项目中，打开 `.env` 文件。
2. 找到 `MONGO_URI` 变量并粘贴您的连接字符串：

```sh filename=".env"
MONGO_URI=mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

就是这样！你现在已经使用 MongoDB Atlas 为 LibreChat 设置了一个在线 MongoDB 数据库，并且已经更新了你的 LibreChat 应用程序以使用此数据库连接。你的应用程序现在应该能够连接到在线 MongoDB 数据库了。

## 关于 Docker 的说明 [#note-about-docker]

<Callout type="note" title="Docker">
**注意：** 如果你正在使用 Docker 运行 LibreChat，则需要利用 `docker-compose.override.yml` 文件。此覆盖文件允许你阻止安装随附的 MongoDB 实例。相反，你的 LibreChat Docker 容器将使用你刚刚设置的在线 MongoDB Atlas 数据库。有关使用覆盖文件的更多信息，请参阅我们的 [Docker Override Guide](/docs/configuration/docker_override)。
</Callout>
