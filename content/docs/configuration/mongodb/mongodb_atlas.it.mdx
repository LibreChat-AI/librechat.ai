  Come configurare un database MongoDB online per LibreChat utilizzando MongoDB
  Atlas

<Steps>
<Step>

### Crea un account MongoDB Atlas [#create-a-mongodb-atlas-account]

1. Apri una nuova scheda nel tuo browser web e vai su [account.mongodb.com/account/register](https://account.mongodb.com/account/register).
2. Inserisci le informazioni richieste e crea il tuo account.

</Step>
<Step>

### Crea un nuovo progetto [#create-a-new-project]

Dopo aver configurato il tuo account, clicca sul pulsante "New Project" e assegnagli un nome (ad esempio, "LibreChat").

</Step>
<Step>

### Crea un database [#build-a-database]

Fai clic sul pulsante "Build a Database".

</Step>
<Step>

### Scegli il piano gratuito [#choose-the-free-tier]

Seleziona l'opzione "Shared Clusters", che è il piano gratuito.

</Step>
<Step>

### Dai un nome al tuo cluster [#name-your-cluster]

Dai un nome al tuo cluster (ad esempio, "LibreChat-Cluster") e clicca su "Create Cluster".

</Step>
<Step>

### Configura le credenziali del database [#set-up-database-credentials]

1. Fai clic sull'opzione "Database & Network Access" nella barra laterale.
2. Fai clic sul pulsante "Add New Database User".
3. Inserisci un nome utente e una password sicura, quindi fai clic su "Add User".

</Step>
<Step>

### Configurare l'accesso alla rete [#configure-network-access]

1. Fai clic sull'opzione "IP Access List" nella barra laterale.
2. Fai clic sul pulsante "Add IP Address".
3. Inserisci "0.0.0.0/0" e clicca su "Confirm".

</Step>
<Step>

### Ottieni la tua stringa di connessione [#get-your-connection-string]

1. Fai clic sull'opzione "Project Overview" nella barra laterale.
2. In "Application Development" fai clic su "Get connection string".
3. Fai clic sul pulsante "Connect".
4. Seleziona "Connect Your Application".
5. Copia la stringa di connessione fornita.
6. Sostituisci `<db_username>` e `<db_password>` nella stringa di connessione con il nome utente e la password impostati nel passaggio delle credenziali. Rimuovi i caratteri `<>` attorno alla password.

La tua stringa di connessione finale dovrebbe apparire simile a questa:

```sh filename="Connection String"
mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

</Step>
<Step>

### Aggiorna il file .env [#update-the-env-file]

1. Nel tuo progetto LibreChat, apri il file `.env`.
2. Trova la variabile `MONGO_URI` e incolla la tua stringa di connessione:

```sh filename=".env"
MONGO_URI=mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

</Step>
</Steps>

Ecco fatto! Hai configurato un database MongoDB online per LibreChat utilizzando MongoDB Atlas e hai aggiornato la tua applicazione LibreChat per utilizzare questa connessione al database. La tua applicazione dovrebbe ora essere in grado di connettersi al database MongoDB online.

## Nota su Docker [#note-about-docker]

<Callout type="note" title="Docker">
**Nota:** Se stai utilizzando LibreChat con Docker, dovrai utilizzare il file `docker-compose.override.yml`. Questo file di override ti consente di impedire l'installazione dell'istanza MongoDB inclusa. Invece, il tuo container Docker di LibreChat utilizzerà il database online MongoDB Atlas che hai appena configurato. Per ulteriori informazioni sull'utilizzo del file di override, consulta la nostra [Guida all'override di Docker](/docs/configuration/docker_override).
</Callout>
