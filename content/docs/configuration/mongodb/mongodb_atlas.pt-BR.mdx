description: >-
  Como configurar um banco de dados MongoDB online para o LibreChat usando o
  MongoDB Atlas
---

<Steps>
<Step>

### Criar uma conta no MongoDB Atlas [#create-a-mongodb-atlas-account]

1. Abra uma nova aba no seu navegador e acesse [account.mongodb.com/account/register](https://account.mongodb.com/account/register).
2. Preencha as informações necessárias e crie sua conta.

</Step>
<Step>

### Criar um Novo Projeto [#create-a-new-project]

Após configurar sua conta, clique no botão "New Project" e dê um nome a ele (por exemplo, "LibreChat").

</Step>
<Step>

### Criar um Banco de Dados [#build-a-database]

Clique no botão "Build a Database".

</Step>
<Step>

### Escolha o Plano Gratuito [#choose-the-free-tier]

Selecione a opção "Shared Clusters", que é o nível gratuito.

</Step>
<Step>

### Nomeie seu Cluster [#name-your-cluster]

Dê um nome ao seu cluster (por exemplo, "LibreChat-Cluster") e clique em "Create Cluster".

</Step>
<Step>

### Configurar Credenciais do Banco de Dados [#set-up-database-credentials]

1. Clique na opção "Database & Network Access" na barra lateral.
2. Clique no botão "Add New Database User".
3. Insira um nome de usuário e uma senha segura, depois clique em "Add User".

</Step>
<Step>

### Configurar Acesso à Rede [#configure-network-access]

1. Clique na opção "IP Access List" na barra lateral.
2. Clique no botão "Add IP Address".
3. Insira "0.0.0.0/0" e clique em "Confirm".

</Step>
<Step>

### Obtenha sua Connection String [#get-your-connection-string]

1. Clique na opção "Project Overview" na barra lateral.
2. Em "Application Development", clique em "Get connection string".
3. Clique no botão "Connect".
4. Selecione "Connect Your Application".
5. Copie a connection string fornecida.
6. Substitua `<db_username>` e `<db_password>` na string de conexão pelo nome de usuário e senha que você definiu na etapa de credenciais. Remova os caracteres `<>` ao redor da senha.

Sua string de conexão final deve ser parecida com isto:

```sh filename="Connection String"
mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

</Step>
<Step>

### Atualize o arquivo .env [#update-the-env-file]

1. No seu projeto LibreChat, abra o arquivo `.env`.
2. Encontre a variável `MONGO_URI` e cole sua string de conexão:

```sh filename=".env"
MONGO_URI=mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

</Step>
</Steps>

É isso! Você configurou agora um banco de dados MongoDB online para o LibreChat usando o MongoDB Atlas e atualizou sua aplicação LibreChat para usar essa conexão de banco de dados. Sua aplicação agora deve ser capaz de se conectar ao banco de dados MongoDB online.

## Nota sobre o Docker [#note-about-docker]

<Callout type="note" title="Docker">
**Nota:** Se você estiver usando o LibreChat com Docker, precisará utilizar o arquivo `docker-compose.override.yml`. Este arquivo de substituição permite que você impeça a instalação da instância do MongoDB incluída. Em vez disso, seu container Docker do LibreChat usará o banco de dados online MongoDB Atlas que você acabou de configurar. Para mais informações sobre como usar o arquivo de substituição, consulte nosso [Guia de Substituição do Docker](/docs/configuration/docker_override).
</Callout>
