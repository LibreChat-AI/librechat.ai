---
title: MongoDB Atlas
icon: MongoDB
description: >-
  Comment configurer une base de données MongoDB en ligne pour LibreChat en
  utilisant MongoDB Atlas
---

<Steps>
<Step>

### Créer un compte MongoDB Atlas [#create-a-mongodb-atlas-account]

1. Ouvrez un nouvel onglet dans votre navigateur web et accédez à [account.mongodb.com/account/register](https://account.mongodb.com/account/register).
2. Remplissez les informations requises et créez votre compte.

</Step>
<Step>

### Créer un nouveau projet [#create-a-new-project]

Après avoir configuré votre compte, cliquez sur le bouton "New Project" et donnez-lui un nom (par exemple, "LibreChat").

</Step>
<Step>

### Créer une base de données [#build-a-database]

Cliquez sur le bouton "Build a Database".

</Step>
<Step>

### Choisir l'offre gratuite [#choose-the-free-tier]

Sélectionnez l'option "Shared Clusters", qui correspond à l'offre gratuite.

</Step>
<Step>

### Nommez votre cluster [#name-your-cluster]

Donnez un nom à votre cluster (par exemple, "LibreChat-Cluster") et cliquez sur "Create Cluster".

</Step>
<Step>

### Configurer les identifiants de la base de données [#set-up-database-credentials]

1. Cliquez sur l'option "Database & Network Access" dans la barre latérale.
2. Cliquez sur le bouton "Add New Database User".
3. Saisissez un nom d'utilisateur et un mot de passe sécurisé, puis cliquez sur "Add User".

</Step>
<Step>

### Configurer l'accès réseau [#configure-network-access]

1. Cliquez sur l'option "IP Access List" dans la barre latérale.
2. Cliquez sur le bouton "Add IP Address".
3. Saisissez "0.0.0.0/0" et cliquez sur "Confirm".

</Step>
<Step>

### Obtenez votre chaîne de connexion [#get-your-connection-string]

1. Cliquez sur l'option "Project Overview" dans la barre latérale.
2. Dans "Application Development", cliquez sur "Get connection string".
3. Cliquez sur le bouton "Connect".
4. Sélectionnez "Connect Your Application".
5. Copiez la chaîne de connexion fournie.
6. Remplacez `<db_username>` et `<db_password>` dans la chaîne de connexion par le nom d'utilisateur et le mot de passe que vous avez définis lors de l'étape des identifiants. Supprimez les caractères `<>` autour du mot de passe.

Votre chaîne de connexion finale devrait ressembler à ceci :

```sh filename="Connection String"
mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

</Step>
<Step>

### Mettre à jour le fichier .env [#update-the-env-file]

1. Dans votre projet LibreChat, ouvrez le fichier `.env`.
2. Trouvez la variable `MONGO_URI` et collez votre chaîne de connexion :

```sh filename=".env"
MONGO_URI=mongodb+srv://username:password@cluster-url.mongodb.net/LibreChat?retryWrites=true
```

</Step>
</Steps>

C'est tout ! Vous avez maintenant configuré une base de données MongoDB en ligne pour LibreChat en utilisant MongoDB Atlas, et vous avez mis à jour votre application LibreChat pour utiliser cette connexion à la base de données. Votre application devrait désormais être en mesure de se connecter à la base de données MongoDB en ligne.

## Note sur Docker [#note-about-docker]

<Callout type="note" title="Docker">
**Note :** Si vous utilisez LibreChat avec Docker, vous devrez utiliser le fichier `docker-compose.override.yml`. Ce fichier de remplacement vous permet d'empêcher l'installation de l'instance MongoDB incluse. Au lieu de cela, votre conteneur Docker LibreChat utilisera la base de données MongoDB Atlas en ligne que vous venez de configurer. Pour plus d'informations sur l'utilisation du fichier de remplacement, veuillez consulter notre [Docker Override Guide](/docs/configuration/docker_override).
</Callout>
