La herramienta Wolfram|Alpha brinda a los agentes acceso a cálculos, matemáticas, conocimiento curado, conversión de unidades, datos científicos y datos en tiempo real. Se debe proporcionar un AppID en todas las llamadas a la API de Wolfram|Alpha.

- Nota: Las llamadas a la API de Wolfram están limitadas a 100 llamadas/día y 2000/mes para usuarios regulares.

### Crear una cuenta [#make-an-account]
- Visita: **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** para crear tu cuenta

### Obtén tu AppID [#get-your-appid]
- Visita el **[Developer Portal](https://developer.wolframalpha.com/portal/myapps/)** y haz clic en `Get an AppID`
- Seleccione `LLM API` como la `API` y copie la clave

### Configurar LibreChat [#configure-librechat]

Agregue su AppID a `.env`:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

Reinicie LibreChat después de cambiar `.env`.

| Despliegue | Comando |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Detenga el servidor, luego ejecute `npm run backend` de nuevo |

#### Agregar la herramienta a un agente [#add-the-tool-to-an-agent]

En LibreChat, selecciona **Agents**, crea o edita un agente, abre la lista de **Tools** del agente, selecciona **Wolfram** y guarda el agente. Consulta la sección [Agents](/docs/features/agents#tools) para obtener más información.
