<Callout type="info" title="Cerchi la ricerca web?">

Questa pagina copre lo strumento agente **Google Custom Search**. Per la funzionalità di **Web Search** integrata di LibreChat (Serper/SearXNG + Firecrawl + Jina), consulta [Web Search](/docs/features/web_search).

</Callout>

Lo strumento Google Search consente ai tuoi agenti di interrogare Google utilizzando la Custom Search JSON API. Avrai bisogno di un ID del motore di ricerca personalizzato (Custom Search Engine ID) e di una chiave API di Google.

## Setup [#setup]

<Steps>
  <Step>

### Creare un Programmable Search Engine [#create-a-programmable-search-engine]

Vai al [pannello di controllo di Programmable Search Engine](https://programmablesearchengine.google.com/controlpanel/all) e accedi con il tuo account Google.

Fai clic su **Add** per creare un nuovo motore di ricerca. Inserisci un nome, seleziona **Search the entire web** e fai clic su **Create**.

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)

  </Step>
  <Step>

### Copia il tuo Search Engine ID [#copy-your-search-engine-id]

Dopo aver creato il motore, vedrai il tuo **Search engine ID**. Copialo: lo aggiungerai al tuo file `.env` come `GOOGLE_CSE_ID`.

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)

  </Step>
  <Step>

### Ottieni una Google Search API Key [#get-a-google-search-api-key]

Vai alla [pagina di introduzione della Custom Search JSON API](https://developers.google.com/custom-search/v1/introduction) e clicca su **Get a Key**.

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)

Dai un nome al tuo progetto, accetta i Termini di Servizio e copia la chiave API.

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)

  </Step>
  <Step>

### Aggiungere variabili d'ambiente [#add-environment-variables]

Aggiungi entrambi i valori al tuo file `.env`:

```bash filename=".env"
GOOGLE_SEARCH_API_KEY=your-api-key-here
GOOGLE_CSE_ID=your-search-engine-id-here
```

  </Step>
  <Step>

### Aggiungi lo strumento a un Agent [#add-the-tool-to-an-agent]

In LibreChat, vai al pannello **Agents** e crea o modifica un agente. Nell'elenco **Tools** dell'agente, seleziona **Google Search**.

  </Step>
  <Step>

### Riavvia e testa [#restart-and-test]

| Distribuzione | Comando |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Locale | Interrompi (Ctrl+C) poi `npm run backend` |

Invia un messaggio come "Cerca le ultime notizie sull'IA" al tuo agente. L'agente utilizzerà Google Custom Search per trovare e restituire risultati pertinenti.

  </Step>
</Steps>

<Callout type="warn" title="Problemi comuni">

Se la ricerca non restituisce risultati, verifica che il tuo Programmable Search Engine sia impostato su **Cerca nell'intero web** (non limitato a siti specifici). Conferma inoltre che sia `GOOGLE_SEARCH_API_KEY` che `GOOGLE_CSE_ID` siano impostati nel tuo file `.env` e di aver riavviato LibreChat dopo aver apportato le modifiche.

</Callout>
