Calculator adalah alat agen bawaan untuk perhitungan aritmatika dan simbolik. Alat ini tidak memerlukan kunci API atau variabel lingkungan.

## Pengaturan [#setup]

<Steps>
  <Step>

### Menambahkan Kalkulator ke Agen [#add-calculator-to-an-agent]

Di LibreChat, pilih **Agents**, buat atau edit agen, buka daftar **Tools** agen tersebut, pilih **Calculator**, lalu simpan agen tersebut.

  </Step>
  <Step>

### Uji Coba [#test-it]

Minta agen untuk menghitung sesuatu yang memerlukan pemanggilan tool:

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
