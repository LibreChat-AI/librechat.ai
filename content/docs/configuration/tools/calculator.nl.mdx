Calculator is een ingebouwde agent-tool voor rekenkundige en symbolische berekeningen. Het vereist geen API-sleutel of omgevingsvariabele.

## Setup [#setup]

<Steps>
  <Step>

### Rekenmachine toevoegen aan een Agent [#add-calculator-to-an-agent]

Selecteer in LibreChat **Agents**, maak een agent aan of bewerk er een, open de **Tools**-lijst van de agent, selecteer **Calculator** en sla de agent op.

  </Step>
  <Step>

### Test het [#test-it]

Vraag de agent om iets te berekenen dat baat heeft bij een tool call:

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
