Traversaal es una herramienta de búsqueda de agentes integrada que envía una consulta en lenguaje natural a Traversaal Ares y devuelve una respuesta con las URLs de origen cuando están disponibles.

## Configuración [#setup]

<Steps>
  <Step>

### Obtener una clave de API de Traversaal [#get-a-traversaal-api-key]

Crea una cuenta y obtén una API key en [api.traversaal.ai](https://api.traversaal.ai/).

  </Step>
  <Step>

### Agregar la variable de entorno [#add-the-environment-variable]

Agregue la clave a su archivo `.env`:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### Reiniciar LibreChat [#restart-librechat]

| Despliegue | Comando |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Detenga el servidor, luego ejecute `npm run backend` nuevamente |

  </Step>
  <Step>

### Agregar Traversaal a un Agente [#add-traversaal-to-an-agent]

En LibreChat, selecciona **Agents**, crea o edita un agente, abre la lista de **Tools** del agente, selecciona **Traversaal** y guarda el agente.

  </Step>
</Steps>

## Parámetros [#parameters]

| Parámetro | Descripción |
|-----------|-------------|
| `query` | Una oración completa que describe lo que el agente debe buscar. Obligatorio. |

## Ejemplo de Prompts [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Solución de problemas [#troubleshooting]

- Si Traversaal devuelve un error de autenticación, confirme que `TRAVERSAAL_API_KEY` esté configurado y reinicie LibreChat.
- Si la herramienta no es visible en el Agent Builder, verifica [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) y la capacidad `tools` del agente.
