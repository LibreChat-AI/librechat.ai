Alat Wolfram|Alpha memberikan agen akses ke komputasi, matematika, pengetahuan terkurasi, konversi satuan, data ilmiah, dan data waktu nyata. AppID harus disertakan dalam semua panggilan ke API Wolfram|Alpha.

- Catatan: Panggilan Wolfram API dibatasi hingga 100 panggilan/hari dan 2000/bulan untuk pengguna reguler.

### Buat akun [#make-an-account]
- Kunjungi: **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** untuk membuat akun Anda

### Dapatkan AppID Anda [#get-your-appid]
- Kunjungi **[Developer Portal](https://developer.wolframalpha.com/access)** dan klik `Get an AppID`
- Pilih `LLM API` sebagai `API` dan salin kuncinya

### Konfigurasi LibreChat [#configure-librechat]

Tambahkan AppID Anda ke `.env`:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

Mulai ulang LibreChat setelah mengubah `.env`.

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Hentikan server, lalu jalankan `npm run backend` kembali |

#### Menambahkan Tool ke Agent [#add-the-tool-to-an-agent]

Di LibreChat, pilih **Agents**, buat atau edit agen, buka daftar **Tools** agen tersebut, pilih **Wolfram**, dan simpan agennya. Lihat bagian [Agents](/docs/features/agents#tools) untuk informasi lebih lanjut.
