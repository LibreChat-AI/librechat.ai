Tavily Search ist ein integriertes Agenten-Tool für aktuelle Web-Recherchen. Es liefert strukturierte Suchergebnisse und kann optional Antworten, Bilder, rohe Seiteninhalte, Domain-Filter und Aktualitätsfilter einbeziehen.

## Einrichtung [#setup]

<Steps>
  <Step>

### Einen Tavily API-Schlüssel erhalten [#get-a-tavily-api-key]

Erstellen Sie ein Tavily-Konto und kopieren Sie Ihren API-Schlüssel von [app.tavily.com](https://app.tavily.com/).

  </Step>
  <Step>

### Fügen Sie die Umgebungsvariable hinzu [#add-the-environment-variable]

Fügen Sie den Schlüssel zu Ihrer `.env` Datei hinzu:

```bash filename=".env"
TAVILY_API_KEY=tvly-your-api-key
```

  </Step>
  <Step>

### LibreChat neu starten [#restart-librechat]

| Deployment | Befehl |
| ---------- | ------------------------------------------------- |
| Docker     | `docker compose down && docker compose up -d`     |
| Lokal      | Stoppen Sie den Server und führen Sie dann erneut `npm run backend` aus |

  </Step>
  <Step>

### Tavily zu einem Agenten hinzufügen [#add-tavily-to-an-agent]

Wählen Sie in LibreChat **Agents** aus, erstellen oder bearbeiten Sie einen Agenten, öffnen Sie die **Tools**-Liste des Agenten, wählen Sie **Tavily Search** aus und speichern Sie den Agenten.

  </Step>
</Steps>

## Parameter [#parameters]

| Parameter | Beschreibung |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `query` | Suchanfrage. Erforderlich. |
| `max_results` | Anzahl der zurückzugebenden Ergebnisse, von 1 bis 10. Standardwert ist 5. |
| `search_depth` | `basic` für schnellere Ergebnisse oder `advanced` für qualitativ hochwertigere Ergebnisse. Erweiterte Suchen zählen als 2 Anfragen. |
| `include_answer` | Tavily's generierte Antwort in die Antwort einbeziehen. |
| `include_images` | Bildergebnisse einbeziehen. |
| `include_image_descriptions` | Beschreibungen für zurückgegebene Bilder einbeziehen, wenn Bilder aktiviert sind. |
| `include_raw_content` | Rohen Seiteninhalt in das Ergebnis einbeziehen. |
| `include_domains` | Ergebnisse auf bestimmte Domains beschränken. |
| `exclude_domains` | Bestimmte Domains ausschließen. |
| `topic` | `general`, `news` oder `finance`. |
| `time_range` | Ergebnisse auf einen aktuellen Zeitraum beschränken, wie z. B. `day`, `week`, `month` oder `year`. |
| `days` | Anzahl der Tage, die für Nachrichtensuchen zurückverfolgt werden sollen. |

## Hinweise [#notes]

- Tavily Search ist ein Agent-Tool. LibreChat kann Tavily auch als [Web Search](/docs/features/web_search)-Anbieter oder Scraper verwenden, was separat konfiguriert wird.
- Tavily-Anfragen verwenden die globale Umgebungsvariable `PROXY`, wenn diese konfiguriert ist. Wenn `PROXY` nicht gesetzt ist, berücksichtigen unterstützte serverseitige Clients `HTTP_PROXY`, `HTTPS_PROXY` und `NO_PROXY`/`no_proxy`.
- Wenn das Tool nicht im Agent Builder erscheint, bestätigen Sie, dass `TAVILY_API_KEY` gesetzt ist, und überprüfen Sie [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools) oder [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools).
