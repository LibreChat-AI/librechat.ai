title: 计算器
icon: Calculator
description: 在 LibreChat 智能体中使用内置的 Calculator 工具
---

Calculator 是一个用于算术和符号计算的内置智能体工具。它不需要 API 密钥或环境变量。

## 设置 [#setup]

<Steps>
  <Step>

### 为 Agent 添加计算器 [#add-calculator-to-an-agent]

在 LibreChat 中，选择 **Agents**，创建或编辑一个 agent，打开该 agent 的 **Tools** 列表，选择 **Calculator**，然后保存该 agent。

  </Step>
  <Step>

### 测试它 [#test-it]

要求智能体计算一些需要调用工具才能完成的内容：

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
