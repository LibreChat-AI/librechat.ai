Traversaal é uma ferramenta de busca de agente integrada que envia uma consulta em linguagem natural para o Traversaal Ares e retorna uma resposta com URLs de origem quando disponíveis.

## Configuração [#setup]

<Steps>
  <Step>

### Obtenha uma chave de API da Traversaal [#get-a-traversaal-api-key]

Crie uma conta e obtenha uma API key em [api.traversaal.ai](https://api.traversaal.ai/).

  </Step>
  <Step>

### Adicione a Variável de Ambiente [#add-the-environment-variable]

Adicione a chave ao seu arquivo `.env`:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### Reiniciar o LibreChat [#restart-librechat]

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Pare o servidor e, em seguida, execute `npm run backend` novamente |

  </Step>
  <Step>

### Adicionar Traversaal a um Agent [#add-traversaal-to-an-agent]

No LibreChat, selecione **Agents**, crie ou edite um agente, abra a lista de **Tools** do agente, selecione **Traversaal** e salve o agente.

  </Step>
</Steps>

## Parâmetros [#parameters]

| Parâmetro | Descrição |
|-----------|-------------|
| `query` | Uma frase completa descrevendo o que o agente deve pesquisar. Obrigatório. |

## Example Prompts [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Solução de problemas [#troubleshooting]

- Se o Traversaal retornar um erro de autenticação, confirme se a `TRAVERSAAL_API_KEY` está definida e reinicie o LibreChat.
- Se a ferramenta não estiver visível no Agent Builder, verifique [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) e a capacidade `tools` do agente.
