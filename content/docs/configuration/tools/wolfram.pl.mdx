Narzędzie Wolfram|Alpha zapewnia agentom dostęp do obliczeń, matematyki, wyselekcjonowanej wiedzy, konwersji jednostek, danych naukowych oraz danych w czasie rzeczywistym. Identyfikator AppID musi być podany we wszystkich wywołaniach API Wolfram|Alpha.

- Uwaga: Wywołania Wolfram API są ograniczone do 100 wywołań dziennie i 2000 miesięcznie dla zwykłych użytkowników.

### Utwórz konto [#make-an-account]
- Odwiedź: **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)**, aby utworzyć swoje konto

### Uzyskaj swój AppID [#get-your-appid]
- Odwiedź **[Developer Portal](https://developer.wolframalpha.com/access)** i kliknij `Get an AppID`
- Wybierz `LLM API` jako `API` i skopiuj klucz

### Konfiguracja LibreChat [#configure-librechat]

Dodaj swój AppID do `.env`:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

Zrestartuj LibreChat po zmianie `.env`.

| Wdrożenie | Polecenie |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokalne | Zatrzymaj serwer, a następnie ponownie uruchom `npm run backend` |

#### Dodawanie narzędzia do Agenta [#add-the-tool-to-an-agent]

W LibreChat wybierz **Agents**, utwórz lub edytuj agenta, otwórz listę **Tools** agenta, wybierz **Wolfram** i zapisz agenta. Więcej informacji znajdziesz w sekcji [Agents](/docs/features/agents#tools).
