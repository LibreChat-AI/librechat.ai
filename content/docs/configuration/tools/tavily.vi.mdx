Tavily Search là một công cụ tác nhân (agent tool) tích hợp sẵn dành cho việc nghiên cứu web hiện tại. Nó trả về các kết quả tìm kiếm có cấu trúc và có thể tùy chọn bao gồm câu trả lời, hình ảnh, nội dung trang thô, bộ lọc tên miền và bộ lọc độ mới.

## Thiết lập [#setup]

<Steps>
  <Step>

### Nhận API Key Tavily [#get-a-tavily-api-key]

Tạo tài khoản Tavily và sao chép API key của bạn từ [app.tavily.com](https://app.tavily.com/).

  </Step>
  <Step>

### Thêm biến môi trường [#add-the-environment-variable]

Thêm khóa vào tệp `.env` của bạn:

```bash filename=".env"
TAVILY_API_KEY=tvly-your-api-key
```

  </Step>
  <Step>

### Khởi động lại LibreChat [#restart-librechat]

| Triển khai | Lệnh |
| ---------- | ------------------------------------------------- |
| Docker     | `docker compose down && docker compose up -d`     |
| Local      | Dừng máy chủ, sau đó chạy lại `npm run backend` |

  </Step>
  <Step>

### Thêm Tavily vào một Agent [#add-tavily-to-an-agent]

Trong LibreChat, hãy chọn **Agents**, tạo hoặc chỉnh sửa một agent, mở danh sách **Tools** của agent đó, chọn **Tavily Search**, và lưu agent lại.

  </Step>
</Steps>

## Các tham số [#parameters]

| Tham số | Mô tả |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `query` | Truy vấn tìm kiếm. Bắt buộc. |
| `max_results` | Số lượng kết quả trả về, từ 1 đến 10. Mặc định là 5. |
| `search_depth` | `basic` cho kết quả nhanh hơn hoặc `advanced` cho kết quả chất lượng cao hơn. Tìm kiếm nâng cao được tính là 2 yêu cầu. |
| `include_answer` | Bao gồm câu trả lời do Tavily tạo trong phản hồi. |
| `include_images` | Bao gồm kết quả hình ảnh. |
| `include_image_descriptions` | Bao gồm mô tả cho các hình ảnh được trả về khi tính năng hình ảnh được bật. |
| `include_raw_content` | Bao gồm nội dung trang thô trong kết quả. |
| `include_domains` | Giới hạn kết quả trong các tên miền cụ thể. |
| `exclude_domains` | Loại trừ các tên miền cụ thể. |
| `topic` | `general`, `news`, hoặc `finance`. |
| `time_range` | Giới hạn kết quả trong một khoảng thời gian gần đây như `day`, `week`, `month`, hoặc `year`. |
| `days` | Số ngày quay ngược lại để bao gồm trong các tìm kiếm tin tức. |

## Ghi chú [#notes]

- Tavily Search là một công cụ tác nhân (agent tool). LibreChat cũng có thể sử dụng Tavily làm nhà cung cấp hoặc trình thu thập [Web Search](/docs/features/web_search), được cấu hình riêng biệt.
- Các yêu cầu Tavily sử dụng biến môi trường `PROXY` toàn cục khi nó được cấu hình. Khi `PROXY` không được thiết lập, các máy khách phía máy chủ được hỗ trợ sẽ tuân theo `HTTP_PROXY`, `HTTPS_PROXY` và `NO_PROXY`/`no_proxy`.
- Nếu công cụ không xuất hiện trong Agent Builder, hãy xác nhận `TAVILY_API_KEY` đã được thiết lập và kiểm tra [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools) hoặc [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools).
