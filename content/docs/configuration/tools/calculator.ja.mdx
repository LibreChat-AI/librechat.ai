Calculatorは、算術計算および記号計算のための組み込みエージェントツールです。APIキーや環境変数は必要ありません。

## セットアップ [#setup]

<Steps>
  <Step>

### エージェントに計算機を追加する [#add-calculator-to-an-agent]

LibreChatで、**Agents**を選択し、エージェントを作成または編集して、エージェントの**Tools**リストを開き、**Calculator**を選択して、エージェントを保存します。

  </Step>
  <Step>

### テストする [#test-it]

ツール呼び出しを活用して計算を行うようエージェントに依頼してください：

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>

## トラブルシューティング [#troubleshooting]

Agent BuilderでCalculatorが表示されない場合は、`librechat.yaml`で [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools) または [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) が使用されていないか確認してください。
