Traversaal is een ingebouwde agent-zoektool die een query in natuurlijke taal naar Traversaal Ares stuurt en een antwoord met bron-URL's retourneert wanneer deze beschikbaar zijn.

## Setup [#setup]

<Steps>
  <Step>

### Verkrijg een Traversaal API-sleutel [#get-a-traversaal-api-key]

Maak een account aan en verkrijg een API-sleutel via [api.traversaal.ai](https://api.traversaal.ai/).

  </Step>
  <Step>

### Voeg de omgevingsvariabele toe [#add-the-environment-variable]

Voeg de sleutel toe aan je `.env` bestand:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### LibreChat herstarten [#restart-librechat]

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokaal | Stop de server en voer daarna opnieuw `npm run backend` uit |

  </Step>
  <Step>

### Traversaal toevoegen aan een Agent [#add-traversaal-to-an-agent]

Selecteer in LibreChat **Agents**, maak een agent aan of bewerk er een, open de **Tools**-lijst van de agent, selecteer **Traversaal** en sla de agent op.

  </Step>
</Steps>

## Parameters [#parameters]

| Parameter | Description |
|-----------|-------------|
| `query` | Een volledige zin die beschrijft waar de agent naar moet zoeken. Vereist. |

## Voorbeeldprompts [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Probleemoplossing [#troubleshooting]

- Als Traversaal een authenticatiefout retourneert, bevestig dan dat `TRAVERSAAL_API_KEY` is ingesteld en herstart LibreChat.
- Als de tool niet zichtbaar is in de Agent Builder, controleer dan [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) en de `tools` capability van de agent.
