Công cụ Wolfram|Alpha cung cấp cho các agent quyền truy cập vào tính toán, toán học, kiến thức được chọn lọc, chuyển đổi đơn vị, dữ liệu khoa học và dữ liệu thời gian thực. Một AppID phải được cung cấp trong tất cả các lệnh gọi đến API Wolfram|Alpha.

- Lưu ý: Các lệnh gọi Wolfram API bị giới hạn ở mức 100 lệnh/ngày và 2000 lệnh/tháng đối với người dùng thông thường.

### Tạo tài khoản [#make-an-account]
- Truy cập: **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** để tạo tài khoản của bạn

### Lấy AppID của bạn [#get-your-appid]
- Truy cập **[Developer Portal](https://developer.wolframalpha.com/access)** và nhấp vào `Get an AppID`
- Chọn `LLM API` làm `API` và sao chép khóa

### Cấu hình LibreChat [#configure-librechat]

Thêm AppID của bạn vào `.env`:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

Khởi động lại LibreChat sau khi thay đổi `.env`.

| Triển khai | Lệnh |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Dừng server, sau đó chạy lại `npm run backend` |

#### Thêm Công cụ vào một Agent [#add-the-tool-to-an-agent]

Trong LibreChat, hãy chọn **Agents**, tạo hoặc chỉnh sửa một agent, mở danh sách **Tools** của agent đó, chọn **Wolfram**, và lưu agent lại. Xem phần [Agents](/docs/features/agents#tools) để biết thêm thông tin.
