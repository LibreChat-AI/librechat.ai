A ferramenta Wolfram|Alpha fornece aos agentes acesso a computação, matemática, conhecimento curado, conversão de unidades, dados científicos e dados em tempo real. Um AppID deve ser fornecido em todas as chamadas para a API do Wolfram|Alpha.

- Nota: As chamadas da API Wolfram são limitadas a 100 chamadas/dia e 2000/mês para usuários regulares.

### Crie uma conta [#make-an-account]
- Visite: **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** para criar sua conta

### Obtenha seu AppID [#get-your-appid]
- Visite o **[Developer Portal](https://developer.wolframalpha.com/access)** e clique em `Get an AppID`
- Selecione `LLM API` como a `API` e copie a chave

### Configurar o LibreChat [#configure-librechat]

Adicione seu AppID ao `.env`:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

Reinicie o LibreChat após alterar o `.env`.

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Pare o servidor e, em seguida, execute `npm run backend` novamente |

#### Adicionar a Ferramenta a um Agente [#add-the-tool-to-an-agent]

No LibreChat, selecione **Agents**, crie ou edite um agent, abra a lista de **Tools** do agent, selecione **Wolfram** e salve o agent. Veja a seção [Agents](/docs/features/agents#tools) para mais informações.
