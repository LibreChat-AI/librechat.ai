Traversaal là một công cụ tìm kiếm tác nhân (agent) tích hợp sẵn, gửi truy vấn ngôn ngữ tự nhiên đến Traversaal Ares và trả về phản hồi kèm theo các URL nguồn khi có sẵn.

## Thiết lập [#setup]

<Steps>
  <Step>

### Nhận API Key Traversaal [#get-a-traversaal-api-key]

Tạo một tài khoản và lấy API key từ [api.traversaal.ai](https://api.traversaal.ai/).

  </Step>
  <Step>

### Thêm biến môi trường [#add-the-environment-variable]

Thêm khóa vào tệp `.env` của bạn:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### Khởi động lại LibreChat [#restart-librechat]

| Triển khai | Lệnh |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Dừng máy chủ, sau đó chạy lại `npm run backend` |

  </Step>
  <Step>

### Thêm Traversaal vào một Agent [#add-traversaal-to-an-agent]

Trong LibreChat, hãy chọn **Agents**, tạo hoặc chỉnh sửa một agent, mở danh sách **Tools** của agent đó, chọn **Traversaal**, và lưu agent lại.

  </Step>
</Steps>

## Các tham số [#parameters]

| Tham số | Mô tả |
|-----------|-------------|
| `query` | Một câu hoàn chỉnh mô tả những gì agent cần tìm kiếm. Bắt buộc. |

## Các câu lệnh mẫu [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Khắc phục sự cố [#troubleshooting]

- Nếu Traversaal trả về lỗi xác thực, hãy xác nhận `TRAVERSAAL_API_KEY` đã được thiết lập và khởi động lại LibreChat.
- Nếu công cụ không hiển thị trong Agent Builder, hãy kiểm tra [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) và khả năng `tools` của agent.
