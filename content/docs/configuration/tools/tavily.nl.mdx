Tavily Search is een ingebouwde agent-tool voor actueel webonderzoek. Het retourneert gestructureerde zoekresultaten en kan optioneel antwoorden, afbeeldingen, onbewerkte pagina-inhoud, domeinfilters en recentheidsfilters bevatten.

## Setup [#setup]

<Steps>
  <Step>

### Verkrijg een Tavily API-sleutel [#get-a-tavily-api-key]

Maak een Tavily-account aan en kopieer je API-sleutel van [app.tavily.com](https://app.tavily.com/).

  </Step>
  <Step>

### Voeg de omgevingsvariabele toe [#add-the-environment-variable]

Voeg de sleutel toe aan je `.env` bestand:

```bash filename=".env"
TAVILY_API_KEY=tvly-your-api-key
```

  </Step>
  <Step>

### LibreChat herstarten [#restart-librechat]

| Deployment | Command |
| ---------- | ------------------------------------------------- |
| Docker     | `docker compose down && docker compose up -d`     |
| Lokaal     | Stop de server en voer daarna opnieuw `npm run backend` uit |

  </Step>
  <Step>

### Tavily toevoegen aan een Agent [#add-tavily-to-an-agent]

Selecteer in LibreChat **Agents**, maak een agent aan of bewerk er een, open de **Tools**-lijst van de agent, selecteer **Tavily Search** en sla de agent op.

  </Step>
</Steps>

## Parameters [#parameters]

| Parameter | Description |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `query` | Zoekopdracht. Verplicht. |
| `max_results` | Aantal resultaten om terug te geven, van 1 tot 10. Standaard is 5. |
| `search_depth` | `basic` voor snellere resultaten of `advanced` voor resultaten van hogere kwaliteit. Geavanceerde zoekopdrachten tellen als 2 verzoeken. |
| `include_answer` | Voeg het door Tavily gegenereerde antwoord toe aan het antwoord. |
| `include_images` | Voeg afbeeldingsresultaten toe. |
| `include_image_descriptions` | Voeg beschrijvingen toe voor geretourneerde afbeeldingen wanneer afbeeldingen zijn ingeschakeld. |
| `include_raw_content` | Voeg de onbewerkte pagina-inhoud toe aan het resultaat. |
| `include_domains` | Beperk resultaten tot specifieke domeinen. |
| `exclude_domains` | Sluit specifieke domeinen uit. |
| `topic` | `general`, `news`, of `finance`. |
| `time_range` | Beperk resultaten tot een recente periode zoals `day`, `week`, `month`, of `year`. |
| `days` | Aantal dagen terug om op te nemen voor nieuwszoekopdrachten. |

## Opmerkingen [#notes]

- Tavily Search is een agent-tool. LibreChat kan Tavily ook gebruiken als [Web Search](/docs/features/web_search)-provider of scraper, wat afzonderlijk wordt geconfigureerd.
- Tavily-verzoeken gebruiken de globale `PROXY` omgevingsvariabele wanneer deze is geconfigureerd. Wanneer `PROXY` niet is ingesteld, respecteren ondersteunde server-side clients `HTTP_PROXY`, `HTTPS_PROXY` en `NO_PROXY`/`no_proxy`.
- Als de tool niet verschijnt in de Agent Builder, bevestig dan dat `TAVILY_API_KEY` is ingesteld en controleer [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools) of [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools).
