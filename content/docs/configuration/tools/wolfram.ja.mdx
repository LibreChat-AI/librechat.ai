Wolfram|Alphaツールを使用すると、エージェントは計算、数学、精選された知識、単位換算、科学データ、およびリアルタイムデータにアクセスできるようになります。Wolfram|Alpha APIへのすべての呼び出しにおいて、AppIDを指定する必要があります。

- 注: Wolfram APIの呼び出しは、通常ユーザーの場合、1日あたり100回、1ヶ月あたり2000回までに制限されています。

### アカウントを作成する [#make-an-account]
- **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** にアクセスしてアカウントを作成してください。

### AppIDを取得する [#get-your-appid]
- **[Developer Portal](https://developer.wolframalpha.com/portal/myapps/)** にアクセスし、`Get an AppID` をクリックします。
- `API` として `LLM API` を選択し、キーをコピーします

### LibreChat を設定する [#configure-librechat]

`.env` に AppID を追加します:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

`.env` を変更した後は、LibreChat を再起動してください。

| デプロイ | コマンド |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| ローカル | サーバーを停止し、再度 `npm run backend` を実行します |

#### エージェントにツールを追加する [#add-the-tool-to-an-agent]

LibreChatで**Agents**を選択し、エージェントを作成または編集して、そのエージェントの**Tools**リストを開き、**Wolfram**を選択してエージェントを保存します。詳細については、[Agents](/docs/features/agents#tools)セクションを参照してください。
