Das Wolfram|Alpha-Tool bietet Agents Zugriff auf Berechnungen, Mathematik, kuratiertes Wissen, Einheitenumrechnungen, wissenschaftliche Daten und Echtzeitdaten. Bei allen Aufrufen der Wolfram|Alpha-API muss eine AppID angegeben werden.

- Hinweis: Wolfram API-Aufrufe sind für reguläre Benutzer auf 100 Aufrufe/Tag und 2000/Monat begrenzt.

### Konto erstellen [#make-an-account]
- Besuchen Sie: **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)**, um Ihr Konto zu erstellen

### Holen Sie sich Ihre AppID [#get-your-appid]
- Besuchen Sie das **[Developer Portal](https://developer.wolframalpha.com/access)** und klicken Sie auf `Get an AppID`
- Wählen Sie `LLM API` als `API` aus und kopieren Sie den Schlüssel

### LibreChat konfigurieren [#configure-librechat]

Fügen Sie Ihre AppID zur `.env` hinzu:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

Starten Sie LibreChat nach dem Ändern der `.env` neu.

| Bereitstellung | Befehl |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokal | Stoppen Sie den Server und führen Sie dann erneut `npm run backend` aus |

#### Das Tool zu einem Agenten hinzufügen [#add-the-tool-to-an-agent]

Wählen Sie in LibreChat **Agents** aus, erstellen oder bearbeiten Sie einen Agenten, öffnen Sie die **Tools**-Liste des Agenten, wählen Sie **Wolfram** aus und speichern Sie den Agenten. Weitere Informationen finden Sie im Abschnitt [Agents](/docs/features/agents#tools).
