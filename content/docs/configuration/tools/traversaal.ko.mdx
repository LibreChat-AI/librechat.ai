Traversaal은 자연어 쿼리를 Traversaal Ares로 전송하고, 사용 가능한 경우 소스 URL이 포함된 응답을 반환하는 내장 에이전트 검색 도구입니다.

## 설정 [#setup]

<Steps>
  <Step>

### Traversaal API 키 발급받기 [#get-a-traversaal-api-key]

[api.traversaal.ai](https://api.traversaal.ai/)에서 계정을 생성하고 API 키를 받으세요.

  </Step>
  <Step>

### 환경 변수 추가 [#add-the-environment-variable]

`.env` 파일에 키를 추가하세요:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### LibreChat 재시작 [#restart-librechat]

| 배포 | 명령어 |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | 서버를 중지한 후, `npm run backend`를 다시 실행하세요 |

  </Step>
  <Step>

### 에이전트에 Traversaal 추가하기 [#add-traversaal-to-an-agent]

LibreChat에서 **Agents**를 선택하고, 에이전트를 생성하거나 편집한 다음, 에이전트의 **Tools** 목록을 열어 **Traversaal**을 선택하고 에이전트를 저장하세요.

  </Step>
</Steps>

## 매개변수 [#parameters]

| 매개변수 | 설명 |
|-----------|-------------|
| `query` | 에이전트가 검색해야 할 내용을 설명하는 완전한 문장입니다. 필수 항목입니다. |

## 예시 프롬프트 [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## 문제 해결 [#troubleshooting]

- Traversaal에서 인증 오류가 반환되면 `TRAVERSAAL_API_KEY`가 설정되어 있는지 확인하고 LibreChat을 재시작하세요.
- Agent Builder에서 도구가 보이지 않는다면, [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) 및 해당 에이전트의 `tools` 기능을 확인하세요.
