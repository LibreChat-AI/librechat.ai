---

Wolfram|Alpha 工具使智能体能够访问计算、数学、精选知识、单位换算、科学数据以及实时数据。在所有调用 Wolfram|Alpha API 的请求中，必须提供 AppID。

- 注意：Wolfram API 调用对于普通用户限制为每天 100 次，每月 2000 次。

### 创建账户 [#make-an-account]
- 访问：**[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** 以创建您的账户

### 获取您的 AppID [#get-your-appid]
- 访问 **[Developer Portal](https://developer.wolframalpha.com/portal/myapps/)** 并点击 `Get an AppID`
- 选择 `LLM API` 作为 `API` 并复制密钥

### 配置 LibreChat [#configure-librechat]

将您的 AppID 添加到 `.env`：

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

修改 `.env` 后请重启 LibreChat。

| 部署方式 | 命令 |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| 本地 | 停止服务器，然后再次运行 `npm run backend` |

#### 将工具添加到 Agent [#add-the-tool-to-an-agent]

在 LibreChat 中，选择 **Agents**，创建或编辑一个 agent，打开该 agent 的 **Tools** 列表，选择 **Wolfram**，然后保存该 agent。更多信息请参阅 [Agents](/docs/features/agents#tools) 部分。
