<Callout type="info" title="正在寻找网页搜索功能？">

本页面涵盖 **Google Custom Search** 代理工具。关于 LibreChat 内置的 **Web Search** 功能（Serper/SearXNG + Firecrawl + Jina），请参阅 [Web Search](/docs/features/web_search)。

</Callout>

Google Search 工具允许您的智能体使用 Custom Search JSON API 查询 Google。您将需要一个 Google Custom Search Engine ID 和一个 API 密钥。

## 设置 [#setup]

<Steps>
  <Step>

### 创建一个可编程搜索引擎 [#create-a-programmable-search-engine]

前往 [Programmable Search Engine control panel](https://programmablesearchengine.google.com/controlpanel/all) 并使用您的 Google 账号登录。

点击 **Add** 以创建一个新的搜索引擎。填写名称，选择 **Search the entire web**，然后点击 **Create**。

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)

  </Step>
  <Step>

### 复制您的搜索引擎 ID [#copy-your-search-engine-id]

创建引擎后，您将看到您的 **Search engine ID**。请将其复制——您需要将其添加到您的 `.env` 文件中，作为 `GOOGLE_CSE_ID`。

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)

  </Step>
  <Step>

### 获取 Google Search API Key [#get-a-google-search-api-key]

前往 [Custom Search JSON API 介绍页面](https://developers.google.com/custom-search/v1/introduction) 并点击 **Get a Key**。

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)

为您的项目命名，同意服务条款，并复制 API key。

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)

  </Step>
  <Step>

### 添加环境变量 [#add-environment-variables]

将这两个值添加到您的 `.env` 文件中：

```bash filename=".env"
GOOGLE_SEARCH_API_KEY=your-api-key-here
GOOGLE_CSE_ID=your-search-engine-id-here
```

  </Step>
  <Step>

### 将工具添加到 Agent [#add-the-tool-to-an-agent]

在 LibreChat 中，前往 **Agents** 面板并创建或编辑一个 agent。在 agent 的 **Tools** 列表中，选择 **Google Search**。

  </Step>
  <Step>

### 重启并测试 [#restart-and-test]

| 部署 | 命令 |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| 本地 | 停止 (Ctrl+C) 然后运行 `npm run backend` |

向您的 agent 发送类似“搜索关于 AI 的最新新闻”的消息。该 agent 将使用 Google Custom Search 来查找并返回相关结果。

  </Step>
</Steps>

<Callout type="warn" title="常见问题">

如果搜索没有返回结果，请确认您的 Programmable Search Engine 已设置为 **搜索整个网络**（而非仅限于特定网站）。同时，请确认 `.env` 文件中已正确设置 `GOOGLE_SEARCH_API_KEY` 和 `GOOGLE_CSE_ID`，并在进行更改后重启了 LibreChat。

</Callout>
