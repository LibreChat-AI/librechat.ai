<Callout type="info" title="Op zoek naar Web Search?">

Deze pagina behandelt de **Google Custom Search** agent tool. Voor de ingebouwde **Web Search** functie van LibreChat (Serper/SearXNG + Firecrawl + Jina), zie [Web Search](/docs/features/web_search).

</Callout>

De Google Search tool stelt je agents in staat om Google te doorzoeken met behulp van de Custom Search JSON API. Je hebt een Google Custom Search Engine ID en een API-sleutel nodig.

## Setup [#setup]

<Steps>
  <Step>

### Een Programmable Search Engine maken [#create-a-programmable-search-engine]

Ga naar het [Programmable Search Engine control panel](https://programmablesearchengine.google.com/controlpanel/all) en log in met je Google-account.

Klik op **Add** om een nieuwe zoekmachine aan te maken. Vul een naam in, selecteer **Search the entire web** en klik op **Create**.

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)

  </Step>
  <Step>

### Kopieer je Search Engine ID [#copy-your-search-engine-id]

Nadat je de engine hebt aangemaakt, zie je jouw **Search engine ID**. Kopieer deze -- je zult deze toevoegen aan je `.env` bestand als `GOOGLE_CSE_ID`.

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)

  </Step>
  <Step>

### Verkrijg een Google Search API-sleutel [#get-a-google-search-api-key]

Ga naar de [Custom Search JSON API introduction page](https://developers.google.com/custom-search/v1/introduction) en klik op **Get a Key**.

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)

Geef je project een naam, ga akkoord met de Terms of Service en kopieer de API-sleutel.

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)

  </Step>
  <Step>

### Omgevingsvariabelen toevoegen [#add-environment-variables]

Voeg beide waarden toe aan je `.env` bestand:

```bash filename=".env"
GOOGLE_SEARCH_API_KEY=your-api-key-here
GOOGLE_CSE_ID=your-search-engine-id-here
```

  </Step>
  <Step>

### De Tool toevoegen aan een Agent [#add-the-tool-to-an-agent]

Ga in LibreChat naar het **Agents**-paneel en maak een agent aan of bewerk er een. Selecteer in de **Tools**-lijst van de agent **Google Search**.

  </Step>
  <Step>

### Herstarten en testen [#restart-and-test]

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokaal | Stop (Ctrl+C) en daarna `npm run backend` |

Stuur een bericht zoals "Zoek naar het laatste nieuws over AI" naar je agent. De agent zal Google Custom Search gebruiken om relevante resultaten te vinden en terug te sturen.

  </Step>
</Steps>

<Callout type="warn" title="Veelvoorkomende problemen">

Als de zoekopdracht geen resultaten oplevert, controleer dan of je Programmable Search Engine is ingesteld op **Het hele web doorzoeken** (niet beperkt tot specifieke sites). Bevestig ook dat zowel `GOOGLE_SEARCH_API_KEY` als `GOOGLE_CSE_ID` zijn ingesteld in je `.env` bestand en dat je LibreChat opnieuw hebt opgestart nadat je wijzigingen hebt aangebracht.

</Callout>
