Wolfram|Alpha 도구는 에이전트에게 계산, 수학, 엄선된 지식, 단위 변환, 과학 데이터 및 실시간 데이터에 대한 액세스 권한을 제공합니다. Wolfram|Alpha API를 호출할 때는 항상 AppID를 제공해야 합니다.

- 참고: Wolfram API 호출은 일반 사용자의 경우 하루 100회, 월 2000회로 제한됩니다.

### 계정 만들기 [#make-an-account]
- 다음 사이트를 방문하여 계정을 생성하세요: **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)**

### AppID 가져오기 [#get-your-appid]
- **[Developer Portal](https://developer.wolframalpha.com/access)**을 방문하여 `Get an AppID`를 클릭하세요.
- `API`로 `LLM API`를 선택하고 키를 복사하세요

### LibreChat 구성하기 [#configure-librechat]

`.env` 파일에 AppID를 추가하세요:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

`.env`를 변경한 후 LibreChat을 재시작하세요.

| 배포 | 명령어 |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | 서버를 중지한 후, `npm run backend`를 다시 실행하세요 |

#### 에이전트에 도구 추가하기 [#add-the-tool-to-an-agent]

LibreChat에서 **Agents**를 선택하고, 에이전트를 생성하거나 편집한 다음, 에이전트의 **Tools** 목록을 열어 **Wolfram**을 선택하고 에이전트를 저장하세요. 자세한 내용은 [Agents](/docs/features/agents#tools) 섹션을 참조하세요.
