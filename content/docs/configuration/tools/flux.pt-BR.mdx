Flux é uma ferramenta poderosa de geração de imagens que pode criar imagens de alta qualidade a partir de descrições de texto. Ele suporta vários estilos artísticos e oferece amplas opções de personalização.

## Instruções de Configuração [#setup-instructions]

1. Obtenha sua chave de API em [bfl.ml](https://bfl.ml)
2. Defina a variável de ambiente `FLUX_API_KEY` no seu arquivo `.env`:

```bash
FLUX_API_KEY=your_api_key_here
```

3. Reinicie o LibreChat e adicione **Flux** à lista de **Tools** de um agente.

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Pare o servidor e, em seguida, execute `npm run backend` novamente |

## Recursos [#features]

### Recursos Principais [#core-capabilities]
- Gere imagens de alta qualidade a partir de descrições de texto detalhadas
- Suporte para múltiplos estilos artísticos
- Dimensões de imagem personalizáveis
- Parâmetros de geração ajustáveis
- Múltiplas opções de endpoint para diferentes casos de uso
- Suporte para geração em lote (até 24 imagens)

### Endpoints Disponíveis [#available-endpoints]
- `/v1/flux-pro` - endpoint padrão (default)
- `/v1/flux-pro-1.1` - Versão aprimorada
- `/v1/flux-dev` - Versão de desenvolvimento
- `/v1/flux-pro-1.1-ultra` - endpoint de qualidade premium

### Parâmetros [#parameters]

A ferramenta Flux suporta três ações principais:

1. **generate** - Crie uma nova imagem a partir de um prompt de texto
2. **generate_finetuned** - Crie uma imagem usando um modelo ajustado (fine-tuned)
3. **list_finetunes** - Lista os modelos personalizados disponíveis para o usuário


Para a ação `generate`:

• **prompt** – Descrição em texto para a imagem (obrigatório)
• **width** – Largura em pixels (múltiplo de 32)
• **height** – Altura em pixels (múltiplo de 32)
• **prompt_upsampling** – Se deve realizar upsampling no prompt (padrão: false)
• **steps** – Número de passos de difusão (1-50, padrão: 40)
• **seed** – Seed opcional para reprodutibilidade
• **safety_tolerance** – Nível de tolerância para moderação (0-6, padrão: 6)
• **endpoint** – endpoint do modelo a ser utilizado:
  - `/v1/flux-pro-1.1` (padrão)
  - `/v1/flux-pro`
  - `/v1/flux-dev`
  - `/v1/flux-pro-1.1-ultra`
• **raw** – Gera imagens menos processadas (apenas para o endpoint ultra, padrão: false)

Para a ação `generate_finetuned`:

• Todos os parâmetros de `generate` mais:  
• **finetune_id** – ID do modelo ajustado (obrigatório)  
• **finetune_strength** – Intensidade do efeito de ajuste fino (0.1-1.2, padrão: 1.1)  
• **guidance** – Escala de orientação (padrão: 2.5)  
• **aspect_ratio** – Proporção de aspecto para modelos ultra (padrão: "16:9")  
• **endpoint** – Deve ser um dos seguintes:
  - `/v1/flux-pro-finetuned` (padrão)
  - `/v1/flux-pro-1.1-ultra-finetuned`


## Melhores Práticas [#best-practices]

### Escrita de Prompts [#prompt-writing]
1. Seja específico e detalhado nas descrições
2. Inclua elementos-chave:
   - Assunto
   - Estilo e abordagem artística
   - Detalhes da composição
   - Iluminação e atmosfera
   - Preferências de cor
   - Especificações técnicas

### Dicas para Melhores Resultados [#tips-for-best-results]
- Escreva prompts em inglês
- Equilibre a especificidade com a liberdade criativa
- Evite conceitos conflitantes
- Foque em descrições visuais
- Considere camadas de composição (primeiro plano, plano intermediário, plano de fundo)

## Detalhes Técnicos [#technical-details]

### Processamento de Imagem [#image-processing]
- As imagens são salvas e gerenciadas automaticamente
- Suporta formato de saída PNG
- Inclui recursos integrados de moderação e segurança
- Geração assíncrona com rastreamento de status

### Recursos de Integração [#integration-features]
- Integração perfeita com interfaces de chat
- Saída formatada em Markdown
- Tratamento de erros e registro (logging) integrados
- Suporte para processamento em lote

## Exemplos de uso [#usage-examples]

Aqui estão alguns exemplos de prompts que funcionam bem com o Flux:

> Uma paisagem montanhosa serena ao pôr do sol, com picos cobertos de neve refletidos em um lago alpino cristalino. Uma luz dourada quente ilumina nuvens tênues, criando uma atmosfera dramática. Estilo fotorrealista com cores ricas e detalhes nítidos.

> Uma paisagem urbana futurista à noite, apresentando arranha-céus iluminados por neon e veículos voadores. Estilo cyberpunk com azuis e roxos profundos, acentuados por cores neon brilhantes. Ruas molhadas pela chuva refletem as luzes da cidade, criando uma atmosfera melancólica.

## Tratamento de Erros [#error-handling]

Mensagens de erro comuns e soluções:
- Problemas com a API key: Verifique se sua API key está definida corretamente nas variáveis de ambiente
- Falhas de geração: Verifique o tamanho do prompt e as diretrizes de conteúdo
- Erros de timeout: Podem ocorrer durante alta carga no servidor, tente novamente após uma breve espera

## Limites de Taxa e Uso [#rate-limits-and-usage]

- O nível gratuito inclui limites de uso generosos
- Múltiplas imagens podem ser geradas em uma única solicitação
- Considere usar contagens de passos menores para gerações mais rápidas durante os testes
