Flux는 텍스트 설명으로부터 고품질 이미지를 생성할 수 있는 강력한 이미지 생성 도구입니다. 다양한 예술적 스타일을 지원하며 광범위한 사용자 지정 옵션을 제공합니다.

## 설정 지침 [#setup-instructions]

1. [bfl.ml](https://bfl.ml)에서 API 키를 받으세요.
2. `.env` 파일에 `FLUX_API_KEY` 환경 변수를 설정하세요:

```bash
FLUX_API_KEY=your_api_key_here
```

3. LibreChat을 재시작하고 에이전트의 **Tools** 목록에 **Flux**를 추가하세요.

| 배포 | 명령어 |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | 서버를 중지한 후, `npm run backend`를 다시 실행하세요 |

## 기능 [#features]

### 핵심 기능 [#core-capabilities]
- 상세한 텍스트 설명으로부터 고품질 이미지를 생성합니다
- 다양한 예술적 스타일 지원
- 사용자 지정 이미지 크기
- 조정 가능한 생성 매개변수
- 다양한 사용 사례를 위한 다중 endpoint 옵션
- 배치 생성 지원 (최대 24개 이미지)

### 사용 가능한 endpoint [#available-endpoints]
- `/v1/flux-pro` - 표준 endpoint (기본값)
- `/v1/flux-pro-1.1` - 향상된 버전
- `/v1/flux-dev` - 개발 버전
- `/v1/flux-pro-1.1-ultra` - 프리미엄 품질 endpoint

### 매개변수 [#parameters]

Flux 도구는 세 가지 주요 작업을 지원합니다:

1. **generate** - 텍스트 프롬프트로부터 새로운 이미지를 생성합니다
2. **generate_finetuned** - 파인튜닝된 모델을 사용하여 이미지 생성
3. **list_finetunes** - 사용자가 사용할 수 있는 커스텀 모델 목록을 표시합니다


`generate` 작업의 경우:

• **prompt** – 이미지에 대한 텍스트 설명 (필수)
• **width** – 픽셀 단위의 너비 (32의 배수)
• **height** – 픽셀 단위의 높이 (32의 배수)
• **prompt_upsampling** – 프롬프트에 업샘플링을 수행할지 여부 (기본값: false)
• **steps** – 확산 단계 수 (1-50, 기본값: 40)
• **seed** – 재현성을 위한 선택적 시드
• **safety_tolerance** – 조정 허용 수준 (0-6, 기본값: 6)
• **endpoint** – 사용할 모델 endpoint:
  - `/v1/flux-pro-1.1` (기본값)
  - `/v1/flux-pro`
  - `/v1/flux-dev`
  - `/v1/flux-pro-1.1-ultra`
• **raw** – 덜 처리된 이미지를 생성합니다 (ultra endpoint 전용, 기본값: false)

`generate_finetuned` 액션의 경우:

• `generate`의 모든 매개변수와 다음 항목:  
• **finetune_id** – 파인튜닝된 모델의 ID (필수)  
• **finetune_strength** – 파인튜닝 효과의 강도 (0.1-1.2, 기본값: 1.1)  
• **guidance** – 가이던스 스케일 (기본값: 2.5)  
• **aspect_ratio** – ultra 모델의 가로세로 비율 (기본값: "16:9")  
• **endpoint** – 다음 중 하나여야 함:
  - `/v1/flux-pro-finetuned` (기본값)
  - `/v1/flux-pro-1.1-ultra-finetuned`


## 모범 사례 [#best-practices]

### 프롬프트 작성 [#prompt-writing]
1. 설명은 구체적이고 상세하게 작성하세요.
2. 주요 요소 포함:
   - 주제
   - 스타일 및 예술적 접근 방식
   - 구성 세부 정보
   - 조명 및 분위기
   - 색상 기본 설정
   - 기술 사양

### 최상의 결과를 위한 팁 [#tips-for-best-results]
- 영어로 프롬프트 작성
- 구체성과 창의적 자유 사이의 균형을 맞추세요
- 상충되는 개념을 피하세요
- 시각적 묘사에 집중하세요
- 구성 레이어(전경, 중경, 배경)를 고려하세요

## 기술적 세부 사항 [#technical-details]

### 이미지 처리 [#image-processing]
- 이미지는 자동으로 저장 및 관리됩니다
- PNG 출력 형식을 지원합니다
- 내장된 조정 및 안전 기능을 포함합니다
- 상태 추적을 포함한 비동기 생성

### 통합 기능 [#integration-features]
- 채팅 인터페이스와의 원활한 통합
- Markdown 형식의 출력
- 내장된 오류 처리 및 로깅
- 일괄 처리(batch processing) 지원

## 사용 예시 [#usage-examples]

Flux에서 잘 작동하는 몇 가지 프롬프트 예시는 다음과 같습니다:

> 일몰 무렵의 평온한 산 풍경으로, 눈 덮인 봉우리가 수정처럼 맑은 고산 호수에 비치고 있습니다. 따뜻한 황금빛이 옅은 구름을 비추며 극적인 분위기를 자아냅니다. 풍부한 색감과 선명한 디테일이 돋보이는 사실적인 사진 스타일입니다.

> 네온 조명이 켜진 고층 빌딩과 비행체가 특징인 미래 지향적인 야경 도시. 짙은 파란색과 보라색을 바탕으로 밝은 네온 색상이 강조된 사이버펑크 스타일. 비에 젖은 거리에 도시의 불빛이 반사되어 분위기 있는 느낌을 자아냅니다.

## 오류 처리 [#error-handling]

일반적인 오류 메시지 및 해결 방법:
- API 키 문제: API 키가 환경 변수에 올바르게 설정되었는지 확인하세요.
- 생성 실패: 프롬프트 길이 및 콘텐츠 가이드라인을 확인하세요
- 타임아웃 오류: 서버 부하가 높을 때 발생할 수 있으며, 잠시 기다린 후 다시 시도하세요.

## 속도 제한 및 사용량 [#rate-limits-and-usage]

- 무료 티어에는 넉넉한 사용 제한이 포함되어 있습니다
- 단일 요청으로 여러 이미지를 생성할 수 있습니다.
- 테스트 중 더 빠른 생성을 위해 더 낮은 단계(step) 수를 사용하는 것을 고려하세요.
