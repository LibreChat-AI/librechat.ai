Traversaalは、自然言語のクエリをTraversaal Aresに送信し、利用可能な場合はソースURLを含む応答を返す組み込みのエージェント検索ツールです。

## セットアップ [#setup]

<Steps>
  <Step>

### Traversaal APIキーを取得する [#get-a-traversaal-api-key]

[api.traversaal.ai](https://api.traversaal.ai/) でアカウントを作成し、APIキーを取得してください。

  </Step>
  <Step>

### 環境変数の追加 [#add-the-environment-variable]

`.env` ファイルにキーを追加します：

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### LibreChatを再起動する [#restart-librechat]

| デプロイメント | コマンド |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| ローカル | サーバーを停止し、再度 `npm run backend` を実行します |

  </Step>
  <Step>

### エージェントにTraversaalを追加する [#add-traversaal-to-an-agent]

LibreChatで、**Agents**を選択し、エージェントを作成または編集して、エージェントの**Tools**リストを開き、**Traversaal**を選択して、エージェントを保存します。

  </Step>
</Steps>

## パラメータ [#parameters]

| パラメータ | 説明 |
|-----------|-------------|
| `query` | エージェントが何を検索すべきかを説明する完全な文章。必須。 |

## Example Prompts [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## トラブルシューティング [#troubleshooting]

- Traversaalが認証エラーを返す場合は、`TRAVERSAAL_API_KEY`が設定されていることを確認し、LibreChatを再起動してください。
- Agent Builderでツールが表示されない場合は、[`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools)、[`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools)、およびエージェントの`tools`機能を確認してください。
