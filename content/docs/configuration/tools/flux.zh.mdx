Flux 是一款功能强大的图像生成工具，能够根据文本描述创建高质量图像。它支持多种艺术风格，并提供丰富的自定义选项。

## 设置说明 [#setup-instructions]

1. 从 [bfl.ml](https://bfl.ml) 获取您的 API key。
2. 在您的 `.env` 文件中设置 `FLUX_API_KEY` 环境变量：

```bash
FLUX_API_KEY=your_api_key_here
```

3. 重启 LibreChat 并将 **Flux** 添加到代理的 **Tools** 列表中。

| 部署方式 | 命令 |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| 本地 | 停止服务器，然后再次运行 `npm run backend` |

## 功能特性 [#features]

### 核心功能 [#core-capabilities]
- 根据详细的文本描述生成高质量图像
- 支持多种艺术风格
- 可自定义的图像尺寸
- 可调节的生成参数
- 针对不同使用场景的多种 endpoint 选项
- 支持批量生成（最多 24 张图像）

### 可用 endpoint [#available-endpoints]
- `/v1/flux-pro` - 标准 endpoint（默认）
- `/v1/flux-pro-1.1` - 增强版
- `/v1/flux-dev` - 开发版本
- `/v1/flux-pro-1.1-ultra` - 高级质量 endpoint

### 参数 [#parameters]

Flux 工具支持三个主要操作：

1. **generate** - 根据文本提示词创建新图像
2. **generate_finetuned** - 使用微调模型生成图像
3. **list_finetunes** - 列出用户可用的自定义模型


对于 `generate` 操作：

• **prompt** – 图像的文本描述（必填）
• **width** – 宽度（像素，需为 32 的倍数）
• **height** – 高度（像素，需为 32 的倍数）
• **prompt_upsampling** – 是否对提示词进行上采样（默认：false）
• **steps** – 扩散步数（1-50，默认：40）
• **seed** – 用于可复现性的可选种子值
• **safety_tolerance** – 内容审核的容忍度级别（0-6，默认：6）
• **endpoint** – 要使用的模型 endpoint：
  - `/v1/flux-pro-1.1` (默认)
  - `/v1/flux-pro`
  - `/v1/flux-dev`
  - `/v1/flux-pro-1.1-ultra`
• **raw** – 生成处理较少的图像（仅适用于 ultra endpoint，默认值：false）

对于 `generate_finetuned` 操作：

• `generate` 的所有参数，外加：
• **finetune_id** – 微调模型的 ID（必填）
• **finetune_strength** – 微调效果的强度（0.1-1.2，默认值：1.1）
• **guidance** – 指导比例（default: 2.5）
• **aspect_ratio** – ultra 模型的宽高比（默认值："16:9"）
• **endpoint** – 必须是以下之一：
  - `/v1/flux-pro-finetuned`（默认）
  - `/v1/flux-pro-1.1-ultra-finetuned`


## 最佳实践 [#best-practices]

### 提示词编写 [#prompt-writing]
1. 在描述中保持具体和详细
2. 包含关键要素：
   - 主题内容
   - 风格与艺术手法
   - 组成详情
   - 灯光与氛围
   - 颜色偏好
   - 技术规格

### 获得最佳效果的提示 [#tips-for-best-results]
- 用英文编写提示词
- 在具体性与创造性自由之间取得平衡
- 避免概念冲突
- 专注于视觉描述
- 考虑构图层次（前景、中景、背景）

## 技术细节 [#technical-details]

### 图像处理 [#image-processing]
- 图像会自动保存并管理
- 支持 PNG 输出格式
- 包含内置的审核和安全功能
- 具有状态跟踪的异步生成

### 集成功能 [#integration-features]
- 与聊天界面无缝集成
- Markdown 格式的输出
- 内置错误处理和日志记录
- 支持批量处理

## 使用示例 [#usage-examples]

以下是一些在 Flux 上表现良好的提示词示例：

> 日落时分宁静的山景，白雪皑皑的山峰倒映在清澈见底的高山湖泊中。温暖的金色光芒照亮了缕缕云彩，营造出富有戏剧性的氛围。照片级真实风格，色彩丰富，细节清晰。

> 一座充满未来感的夜间城市景观，拥有霓虹灯闪烁的摩天大楼和飞行载具。赛博朋克风格，以深蓝色和紫色为主调，点缀着明亮的霓虹色彩。雨水打湿的街道反射着城市灯光，营造出一种忧郁的氛围。

## 错误处理 [#error-handling]

常见错误消息及解决方案：
- API key 问题：请验证您的 API key 是否已在环境变量中正确设置
- 生成失败：请检查提示词长度和内容准则
- 超时错误：可能在服务器高负载期间发生，请稍作等待后重试

## 速率限制与使用情况 [#rate-limits-and-usage]

- 免费层级包含慷慨的使用限额
- 在单个请求中可以生成多张图像
- 在测试期间，考虑使用较低的步数以实现更快的生成速度。
