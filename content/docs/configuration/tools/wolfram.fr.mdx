---

L'outil Wolfram|Alpha donne aux agents accès au calcul, aux mathématiques, aux connaissances organisées, à la conversion d'unités, aux données scientifiques et aux données en temps réel. Un AppID doit être fourni dans tous les appels à l'API Wolfram|Alpha.

- Remarque : Les appels à l'API Wolfram sont limités à 100 appels/jour et 2000/mois pour les utilisateurs réguliers.

### Créer un compte [#make-an-account]
- Visitez : **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** pour créer votre compte

### Obtenez votre AppID [#get-your-appid]
- Visitez le **[Developer Portal](https://developer.wolframalpha.com/access)** et cliquez sur `Get an AppID`
- Sélectionnez `LLM API` comme `API` et copiez la clé

### Configurer LibreChat [#configure-librechat]

Ajoutez votre AppID à `.env` :

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

Redémarrez LibreChat après avoir modifié `.env`.

| Déploiement | Commande |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Arrêtez le serveur, puis exécutez à nouveau `npm run backend` |

#### Ajouter l'outil à un Agent [#add-the-tool-to-an-agent]

Dans LibreChat, sélectionnez **Agents**, créez ou modifiez un agent, ouvrez la liste des **Tools** de l'agent, sélectionnez **Wolfram**, puis enregistrez l'agent. Consultez la section [Agents](/docs/features/agents#tools) pour plus d'informations.
