Lo strumento Wolfram|Alpha fornisce agli agenti accesso a calcoli, matematica, conoscenze curate, conversione di unità, dati scientifici e dati in tempo reale. Un AppID deve essere fornito in tutte le chiamate all'API di Wolfram|Alpha.

- Nota: le chiamate alle API di Wolfram sono limitate a 100 chiamate al giorno e 2000 al mese per gli utenti regolari.

### Crea un account [#make-an-account]
- Visita: **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** per creare il tuo account

### Ottieni il tuo AppID [#get-your-appid]
- Visita il **[Developer Portal](https://developer.wolframalpha.com/access)** e clicca su `Get an AppID`
- Seleziona `LLM API` come `API` e copia la chiave

### Configurare LibreChat [#configure-librechat]

Aggiungi il tuo AppID al file `.env`:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

Riavvia LibreChat dopo aver modificato `.env`.

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Locale | Arresta il server, quindi esegui nuovamente `npm run backend` |

#### Aggiungere lo strumento a un Agent [#add-the-tool-to-an-agent]

In LibreChat, seleziona **Agents**, crea o modifica un agente, apri l'elenco **Tools** dell'agente, seleziona **Wolfram** e salva l'agente. Consulta la sezione [Agents](/docs/features/agents#tools) per ulteriori informazioni.
