De Wolfram|Alpha tool geeft agents toegang tot berekeningen, wiskunde, samengestelde kennis, eenheidsconversie, wetenschappelijke gegevens en real-time gegevens. Een AppID moet worden opgegeven bij alle aanroepen naar de Wolfram|Alpha API.

- Let op: Wolfram API-aanroepen zijn beperkt tot 100 aanroepen per dag en 2000 per maand voor reguliere gebruikers.

### Maak een account aan [#make-an-account]
- Bezoek: **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** om je account aan te maken

### Verkrijg je AppID [#get-your-appid]
- Bezoek de **[Developer Portal](https://developer.wolframalpha.com/access)** en klik op `Get an AppID`
- Selecteer `LLM API` als de `API` en kopieer de sleutel

### LibreChat configureren [#configure-librechat]

Voeg je AppID toe aan `.env`:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

Herstart LibreChat na het wijzigen van `.env`.

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokaal | Stop de server en voer daarna opnieuw `npm run backend` uit |

#### Voeg de Tool toe aan een Agent [#add-the-tool-to-an-agent]

Selecteer in LibreChat **Agents**, maak een agent aan of bewerk er een, open de **Tools**-lijst van de agent, selecteer **Wolfram** en sla de agent op. Zie de sectie [Agents](/docs/features/agents#tools) voor meer informatie.
