Flux adalah alat pembuat gambar yang canggih yang dapat menciptakan gambar berkualitas tinggi dari deskripsi teks. Alat ini mendukung berbagai gaya artistik dan menawarkan opsi kustomisasi yang ekstensif.

## Petunjuk Pengaturan [#setup-instructions]

1. Dapatkan kunci API Anda dari [bfl.ml](https://bfl.ml)
2. Atur variabel lingkungan `FLUX_API_KEY` di dalam file `.env` Anda:

```bash
FLUX_API_KEY=your_api_key_here
```

3. Mulai ulang LibreChat dan tambahkan **Flux** ke daftar **Tools** agen.

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Hentikan server, lalu jalankan `npm run backend` kembali |

## Fitur [#features]

### Kemampuan Inti [#core-capabilities]
- Hasilkan gambar berkualitas tinggi dari deskripsi teks yang mendetail
- Dukungan untuk berbagai gaya artistik
- Dimensi gambar yang dapat disesuaikan
- Parameter generasi yang dapat disesuaikan
- Beberapa opsi endpoint untuk berbagai kasus penggunaan
- Dukungan pembuatan batch (hingga 24 gambar)

### Endpoint yang Tersedia [#available-endpoints]
- `/v1/flux-pro` - endpoint standar (default)
- `/v1/flux-pro-1.1` - Versi yang ditingkatkan
- `/v1/flux-dev` - Versi pengembangan
- `/v1/flux-pro-1.1-ultra` - endpoint kualitas premium

### Parameter [#parameters]

Alat Flux mendukung tiga tindakan utama:

1. **generate** - Membuat gambar baru dari prompt teks
2. **generate_finetuned** - Membuat gambar menggunakan model yang telah di-fine-tune
3. **list_finetunes** - Mencantumkan model kustom yang tersedia untuk pengguna


Untuk tindakan `generate`:

• **prompt** – Deskripsi teks untuk gambar (wajib)  
• **width** – Lebar dalam piksel (kelipatan 32)  
• **height** – Tinggi dalam piksel (kelipatan 32)  
• **prompt_upsampling** – Apakah akan melakukan upsampling pada prompt (default: false)  
• **steps** – Jumlah langkah difusi (1-50, default: 40)  
• **seed** – Seed opsional untuk reproduktifitas  
• **safety_tolerance** – Tingkat toleransi untuk moderasi (0-6, default: 6)  
• **endpoint** – endpoint model yang akan digunakan:
  - `/v1/flux-pro-1.1` (bawaan)
  - `/v1/flux-pro`
  - `/v1/flux-dev`
  - `/v1/flux-pro-1.1-ultra`
• **raw** – Menghasilkan gambar yang kurang diproses (hanya untuk endpoint ultra, default: false)

Untuk action `generate_finetuned`:

• Semua parameter dari `generate` ditambah:
• **finetune_id** – ID dari model yang telah di-fine-tune (wajib)
• **finetune_strength** – Kekuatan efek fine-tuning (0.1-1.2, default: 1.1)
• **guidance** – Skala panduan (default: 2.5)
• **aspect_ratio** – Rasio aspek untuk model ultra (default: "16:9")
• **endpoint** – Harus salah satu dari:
  - `/v1/flux-pro-finetuned` (default)
  - `/v1/flux-pro-1.1-ultra-finetuned`


## Praktik Terbaik [#best-practices]

### Penulisan Prompt [#prompt-writing]
1. Jadilah spesifik dan mendetail dalam deskripsi
2. Sertakan elemen-elemen kunci:
   - Subjek materi
   - Gaya dan pendekatan artistik
   - Detail komposisi
   - Pencahayaan dan suasana
   - Preferensi warna
   - Spesifikasi teknis

### Tips untuk Hasil Terbaik [#tips-for-best-results]
- Tulis prompt dalam bahasa Inggris
- Seimbangkan spesifisitas dengan kebebasan kreatif
- Hindari konsep yang bertentangan
- Fokus pada deskripsi visual
- Pertimbangkan lapisan komposisi (latar depan, latar tengah, latar belakang)

## Detail Teknis [#technical-details]

### Pemrosesan Gambar [#image-processing]
- Gambar disimpan dan dikelola secara otomatis
- Mendukung format output PNG
- Termasuk fitur moderasi dan keamanan bawaan
- Pembuatan asinkron dengan pelacakan status

### Fitur Integrasi [#integration-features]
- Integrasi mulus dengan antarmuka chat
- Output berformat Markdown
- Penanganan kesalahan dan pencatatan log bawaan
- Dukungan untuk pemrosesan batch

## Contoh Penggunaan [#usage-examples]

Berikut adalah beberapa contoh prompt yang berfungsi dengan baik pada Flux:

> Pemandangan pegunungan yang tenang saat matahari terbenam, dengan puncak bersalju yang terpantul di danau pegunungan yang jernih. Cahaya keemasan yang hangat menyinari awan tipis, menciptakan suasana yang dramatis. Gaya fotorealistik dengan warna-warna yang kaya dan detail yang tajam.

> Pemandangan kota futuristik di malam hari, menampilkan gedung pencakar langit dengan lampu neon dan kendaraan terbang. Gaya cyberpunk dengan warna biru dan ungu tua, diberi aksen warna neon yang cerah. Jalanan yang basah karena hujan memantulkan lampu kota, menciptakan suasana yang murung.

## Penanganan Error [#error-handling]

Pesan kesalahan umum dan solusinya:
- Masalah API key: Pastikan API key Anda telah diatur dengan benar di environment variables
- Kegagalan pembuatan: Periksa panjang prompt dan pedoman konten
- Galat waktu habis (Timeout errors): Dapat terjadi selama beban server yang tinggi, coba lagi setelah menunggu sebentar

## Batas Kecepatan dan Penggunaan [#rate-limits-and-usage]

- Tingkat gratis mencakup batas penggunaan yang besar
- Beberapa gambar dapat dibuat dalam satu permintaan
- Pertimbangkan untuk menggunakan jumlah langkah (step counts) yang lebih rendah untuk pembuatan yang lebih cepat selama pengujian
