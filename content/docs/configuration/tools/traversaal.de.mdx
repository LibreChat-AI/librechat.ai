Traversaal ist ein integriertes Agenten-Suchwerkzeug, das eine natürlichsprachliche Anfrage an Traversaal Ares sendet und eine Antwort mit Quell-URLs zurückgibt, sofern diese verfügbar sind.

## Einrichtung [#setup]

<Steps>
  <Step>

### Einen Traversaal API-Schlüssel erhalten [#get-a-traversaal-api-key]

Erstellen Sie ein Konto und erhalten Sie einen API-Schlüssel von [api.traversaal.ai](https://api.traversaal.ai/).

  </Step>
  <Step>

### Fügen Sie die Umgebungsvariable hinzu [#add-the-environment-variable]

Fügen Sie den Schlüssel zu Ihrer `.env` Datei hinzu:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### LibreChat neu starten [#restart-librechat]

| Bereitstellung | Befehl |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokal | Stoppen Sie den Server und führen Sie dann erneut `npm run backend` aus |

  </Step>
  <Step>

### Traversaal zu einem Agenten hinzufügen [#add-traversaal-to-an-agent]

Wählen Sie in LibreChat **Agents** aus, erstellen oder bearbeiten Sie einen Agenten, öffnen Sie die **Tools**-Liste des Agenten, wählen Sie **Traversaal** aus und speichern Sie den Agenten.

  </Step>
</Steps>

## Parameter [#parameters]

| Parameter | Beschreibung |
|-----------|--------------|
| `query` | Ein vollständiger Satz, der beschreibt, wonach der Agent suchen soll. Erforderlich. |

## Beispiel-Prompts [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Fehlerbehebung [#troubleshooting]

- Falls Traversaal einen Authentifizierungsfehler zurückgibt, bestätigen Sie, dass `TRAVERSAAL_API_KEY` gesetzt ist, und starten Sie LibreChat neu.
- Wenn das Tool im Agent Builder nicht sichtbar ist, überprüfen Sie [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) und die `tools`-Fähigkeit des Agenten.
