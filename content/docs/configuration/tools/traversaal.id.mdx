Traversaal adalah alat pencarian agen bawaan yang mengirimkan kueri bahasa alami ke Traversaal Ares dan mengembalikan respons beserta URL sumber jika tersedia.

## Pengaturan [#setup]

<Steps>
  <Step>

### Dapatkan Kunci API Traversaal [#get-a-traversaal-api-key]

Buat akun dan dapatkan API key dari [api.traversaal.ai](https://api.traversaal.ai/).

  </Step>
  <Step>

### Tambahkan Variabel Lingkungan [#add-the-environment-variable]

Tambahkan kunci tersebut ke file `.env` Anda:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### Restart LibreChat [#restart-librechat]

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Hentikan server, lalu jalankan `npm run backend` kembali |

  </Step>
  <Step>

### Menambahkan Traversaal ke Agen [#add-traversaal-to-an-agent]

Di LibreChat, pilih **Agents**, buat atau edit agen, buka daftar **Tools** agen tersebut, pilih **Traversaal**, lalu simpan agen tersebut.

  </Step>
</Steps>

## Parameter [#parameters]

| Parameter | Deskripsi |
|-----------|-------------|
| `query` | Kalimat lengkap yang mendeskripsikan apa yang harus dicari oleh agen. Wajib diisi. |

## Contoh Prompt [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Pemecahan Masalah [#troubleshooting]

- Jika Traversaal mengembalikan kesalahan autentikasi, pastikan `TRAVERSAAL_API_KEY` telah diatur dan mulai ulang LibreChat.
- Jika alat tersebut tidak terlihat di Agent Builder, periksa [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools), dan kapabilitas `tools` pada agen tersebut.
