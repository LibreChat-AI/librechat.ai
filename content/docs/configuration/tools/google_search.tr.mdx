<Callout type="info" title="Web Araması mı arıyorsunuz?">

Bu sayfa **Google Custom Search** aracı eklentisini kapsamaktadır. LibreChat'in yerleşik **Web Search** özelliği (Serper/SearXNG + Firecrawl + Jina) için [Web Search](/docs/features/web_search) bölümüne bakınız.

</Callout>

Google Search aracı, temsilcilerinizin Custom Search JSON API kullanarak Google'da sorgu yapmasını sağlar. Bir Google Custom Search Engine ID ve bir API anahtarına ihtiyacınız olacaktır.

## Kurulum [#setup]

<Steps>
  <Step>

### Programlanabilir Arama Motoru Oluşturun [#create-a-programmable-search-engine]

[Programmable Search Engine kontrol paneline](https://programmablesearchengine.google.com/controlpanel/all) gidin ve Google hesabınızla giriş yapın.

Yeni bir arama motoru oluşturmak için **Add** düğmesine tıklayın. Bir isim girin, **Search the entire web** seçeneğini belirleyin ve **Create** düğmesine tıklayın.

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)

  </Step>
  <Step>

### Arama Motoru Kimliğinizi (Search Engine ID) Kopyalayın [#copy-your-search-engine-id]

Motoru oluşturduktan sonra **Arama motoru kimliğinizi (Search engine ID)** göreceksiniz. Bunu kopyalayın; `.env` dosyanıza `GOOGLE_CSE_ID` olarak ekleyeceksiniz.

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)

  </Step>
  <Step>

### Google Search API Anahtarı Alın [#get-a-google-search-api-key]

[Custom Search JSON API giriş sayfasına](https://developers.google.com/custom-search/v1/introduction) gidin ve **Get a Key** düğmesine tıklayın.

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)

Projenizi adlandırın, Hizmet Şartlarını kabul edin ve API anahtarını kopyalayın.

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)

  </Step>
  <Step>

### Ortam Değişkenlerini Ekle [#add-environment-variables]

Her iki değeri de `.env` dosyanıza ekleyin:

```bash filename=".env"
GOOGLE_SEARCH_API_KEY=your-api-key-here
GOOGLE_CSE_ID=your-search-engine-id-here
```

  </Step>
  <Step>

### Aracı bir Ajan'a Ekleme [#add-the-tool-to-an-agent]

LibreChat'te **Agents** paneline gidin ve bir ajan oluşturun veya düzenleyin. Ajanın **Tools** listesinden **Google Search** seçeneğini seçin.

  </Step>
  <Step>

### Yeniden Başlatma ve Test Etme [#restart-and-test]

| Dağıtım | Komut |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Yerel | Durdur (Ctrl+C) ardından `npm run backend` |

Temsilcinize "Yapay zeka hakkındaki en son haberleri ara" gibi bir mesaj gönderin. Temsilci, ilgili sonuçları bulmak ve döndürmek için Google Custom Search kullanacaktır.

  </Step>
</Steps>

<Callout type="warn" title="Yaygın Sorunlar">

Arama sonuç vermiyorsa, Programmable Search Engine ayarınızın **Tüm web'de ara** (belirli sitelerle kısıtlanmamış) olarak ayarlandığından emin olun. Ayrıca `.env` dosyanızda hem `GOOGLE_SEARCH_API_KEY` hem de `GOOGLE_CSE_ID` değerlerinin ayarlandığını ve değişiklikleri yaptıktan sonra LibreChat'i yeniden başlattığınızı doğrulayın.

</Callout>
