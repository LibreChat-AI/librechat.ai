Flux ist ein leistungsstarkes Tool zur Bilderzeugung, das hochwertige Bilder aus Textbeschreibungen erstellen kann. Es unterstützt verschiedene künstlerische Stile und bietet umfangreiche Anpassungsmöglichkeiten.

## Einrichtungsanweisungen [#setup-instructions]

1. Holen Sie sich Ihren API-Schlüssel von [bfl.ml](https://bfl.ml)
2. Setzen Sie die Umgebungsvariable `FLUX_API_KEY` in Ihrer `.env` Datei:

```bash
FLUX_API_KEY=your_api_key_here
```

3. Starten Sie LibreChat neu und fügen Sie **Flux** zur **Tools**-Liste eines Agenten hinzu.

| Bereitstellung | Befehl |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokal | Stoppen Sie den Server und führen Sie dann erneut `npm run backend` aus |

## Funktionen [#features]

### Kernfunktionen [#core-capabilities]
- Erstellen Sie hochwertige Bilder aus detaillierten Textbeschreibungen
- Unterstützung für mehrere künstlerische Stile
- Anpassbare Bildabmessungen
- Anpassbare Generierungsparameter
- Mehrere endpoint-Optionen für verschiedene Anwendungsfälle
- Unterstützung für Batch-Generierung (bis zu 24 Bilder)

### Verfügbare Endpoints [#available-endpoints]
- `/v1/flux-pro` - Standard-endpoint (Standard)
- `/v1/flux-pro-1.1` - Verbesserte Version
- `/v1/flux-dev` - Entwicklungsversion
- `/v1/flux-pro-1.1-ultra` - Premium-Qualitäts-endpoint

### Parameter [#parameters]

Das Flux-Tool unterstützt drei Hauptaktionen:

1. **generate** - Erstelle ein neues Bild aus einem Text-Prompt
2. **generate_finetuned** - Erstellen Sie ein Bild mithilfe eines feinabgestimmten Modells
3. **list_finetunes** - Verfügbare benutzerdefinierte Modelle für den Benutzer auflisten


Für die `generate` Aktion:

• **prompt** – Textbeschreibung für das Bild (erforderlich)
• **width** – Breite in Pixeln (Vielfaches von 32)
• **height** – Höhe in Pixeln (Vielfaches von 32)
• **prompt_upsampling** – Ob ein Upsampling des Prompts durchgeführt werden soll (Standard: false)
• **steps** – Anzahl der Diffusionsschritte (1-50, Standard: 40)
• **seed** – Optionaler Seed für die Reproduzierbarkeit
• **safety_tolerance** – Toleranzstufe für die Moderation (0-6, Standard: 6)
• **endpoint** – Zu verwendender endpoint:
  - `/v1/flux-pro-1.1` (Standard)
  - `/v1/flux-pro`
  - `/v1/flux-dev`
  - `/v1/flux-pro-1.1-ultra`
• **raw** – Generiert weniger bearbeitete Bilder (nur für ultra endpoint, Standard: false)

Für die `generate_finetuned` Aktion:

• Alle Parameter von `generate` plus:
• **finetune_id** – ID des feinabgestimmten Modells (erforderlich)
• **finetune_strength** – Stärke des Fine-Tuning-Effekts (0,1–1,2, Standard: 1,1)
• **guidance** – Guidance-Skala (Standard: 2,5)
• **aspect_ratio** – Seitenverhältnis für Ultra-Modelle (Standard: "16:9")
• **endpoint** – Muss einer der folgenden sein:
  - `/v1/flux-pro-finetuned` (Standard)
  - `/v1/flux-pro-1.1-ultra-finetuned`


## Best Practices [#best-practices]

### Prompt-Erstellung [#prompt-writing]
1. Seien Sie präzise und detailliert in Ihren Beschreibungen
2. Enthaltene Schlüsselelemente:
   - Gegenstand
   - Stil und künstlerischer Ansatz
   - Zusammensetzungsdetails
   - Beleuchtung und Atmosphäre
   - Farbeinstellungen
   - Technische Spezifikationen

### Tipps für beste Ergebnisse [#tips-for-best-results]
- Schreibe Prompts auf Englisch
- Bringen Sie Spezifität und kreative Freiheit in Einklang
- Vermeiden Sie widersprüchliche Konzepte
- Konzentrieren Sie sich auf visuelle Beschreibungen
- Berücksichtigen Sie Kompositionsebenen (Vordergrund, Mittelgrund, Hintergrund)

## Technische Details [#technical-details]

### Bildverarbeitung [#image-processing]
- Bilder werden automatisch gespeichert und verwaltet
- Unterstützt das PNG-Ausgabeformat
- Enthält integrierte Moderations- und Sicherheitsfunktionen
- Asynchrone Generierung mit Statusverfolgung

### Integrationsfunktionen [#integration-features]
- Nahtlose Integration mit Chat-Oberflächen
- Markdown-formatierte Ausgabe
- Integrierte Fehlerbehandlung und Protokollierung
- Unterstützung für Stapelverarbeitung

## Anwendungsbeispiele [#usage-examples]

Hier sind einige Beispiel-Prompts, die gut mit Flux funktionieren:

> Eine ruhige Berglandschaft bei Sonnenuntergang, mit schneebedeckten Gipfeln, die sich in einem kristallklaren Alpensee spiegeln. Warmes goldenes Licht beleuchtet zarte Wolken und erzeugt eine dramatische Atmosphäre. Fotorealistischer Stil mit satten Farben und scharfen Details.

> Eine futuristische Stadtlandschaft bei Nacht mit neonbeleuchteten Wolkenkratzern und fliegenden Fahrzeugen. Cyberpunk-Stil in tiefen Blau- und Violetttönen, akzentuiert durch leuchtende Neonfarben. Regenfeuchte Straßen spiegeln die Lichter der Stadt wider und erzeugen eine stimmungsvolle Atmosphäre.

## Fehlerbehandlung [#error-handling]

Häufige Fehlermeldungen und Lösungen:
- Probleme mit dem API-Schlüssel: Überprüfen Sie, ob Ihr API-Schlüssel korrekt in den Umgebungsvariablen gesetzt ist
- Generierungsfehler: Überprüfen Sie die Prompt-Länge und die Inhaltsrichtlinien
- Timeout-Fehler: Können bei hoher Serverlast auftreten; versuchen Sie es nach einer kurzen Wartezeit erneut.

## Ratenbegrenzungen und Nutzung [#rate-limits-and-usage]

- Der kostenlose Tarif beinhaltet großzügige Nutzungslimits
- Mehrere Bilder können in einer einzigen Anfrage generiert werden
- Erwägen Sie die Verwendung niedrigerer Schrittzahlen für schnellere Generierungen während des Testens.
