Stable Diffusion adalah alat agen bawaan yang menghubungkan LibreChat ke API **[AUTOMATIC1111 Stable Diffusion WebUI](https://github.com/AUTOMATIC1111/stable-diffusion-webui)**. Untuk penyebaran Stable Diffusion berbasis Docker, Anda juga dapat menggunakan **[stable-diffusion-webui-docker](https://github.com/AbdBarho/stable-diffusion-webui-docker)**.

Dengan deployment Docker, Anda dapat melewati langkah 2 dan langkah 3, gunakan instruksi penyiapan dari repositori mereka sebagai gantinya.

- Catatan: Anda memerlukan GPU yang kompatibel ("hanya CPU" dimungkinkan tetapi sangat lambat). Nvidia direkomendasikan, namun tidak ada sumber daya yang jelas mengenai GPU yang tidak kompatibel. GPU apa pun yang layak seharusnya dapat berfungsi.

### 1. Ikuti petunjuk pengunduhan dan instalasi [#1-follow-download-and-installation-instructions]

Ikuti langkah-langkah penyiapan dari **[stable-diffusion-webui readme](https://github.com/AUTOMATIC1111/stable-diffusion-webui)**.

### 2. Edit pengaturan skrip jalankan Anda [#2-edit-your-run-script-settings]

#### Windows [#windows]

 - Edit file **webui-user.bat** Anda dengan menambahkan baris berikut sebelum perintah call:
- `set COMMANDLINE_ARGS=--api`

    - File .bat Anda harus terlihat seperti ini dengan semua pengaturan lainnya dibiarkan default
    ```shell 
    @echo off

    set PYTHON=
    set GIT=
    set VENV_DIR=
    set COMMANDLINE_ARGS=--api

    call webui.bat
    ```
#### Lainnya (belum diuji tetapi seharusnya berfungsi) [#others-not-tested-but-should-work]

 - Edit file **webui-user.sh** Anda dengan menambahkan baris berikut:
 - `export COMMANDLINE_ARGS="--api"`

     - File .sh Anda harus terlihat seperti ini dengan semua pengaturan lainnya dibiarkan default
    ```bash 

    export COMMANDLINE_ARGS="--api"

    #!/bin/bash
    #########################################################
    # Uncomment and change the variables below to your need:#
    #########################################################

    # ...rest
    ```

### 3. Jalankan Stable Diffusion (gunakan file .sh atau .bat sesuai dengan sistem operasi Anda) [#3-run-stable-diffusion-either-sh-or-bat-file-according-to-your-operating-system]

### 4. Atur URL Stable Diffusion di LibreChat [#4-set-the-stable-diffusion-url-in-librechat]

> **Catatan: Port default untuk Gradio adalah `7860`. Jika Anda telah mengubahnya, harap perbarui nilainya sebagaimana mestinya.**

#### Instalasi Docker [#docker-install]
- Gunakan `SD_WEBUI_URL=http://host.docker.internal:7860` di dalam file `.env`
- Atau `http://host.docker.internal:7860` dari webui

#### Instalasi Lokal [#local-install]
- Gunakan `SD_WEBUI_URL=http://127.0.0.1:7860` di dalam file `.env`
- Atau `http://127.0.0.1:7860` dari webui

Mulai ulang LibreChat setelah mengubah `.env`.

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Hentikan server, lalu jalankan `npm run backend` kembali |

#### Menambahkan Tool ke Agent [#add-the-tool-to-an-agent]

Di LibreChat, pilih **Agents**, buat atau edit agen, buka daftar **Tools** agen tersebut, pilih **Stable Diffusion**, lalu simpan agennya. Lihat bagian [Agents](/docs/features/agents#tools) untuk informasi lebih lanjut.
