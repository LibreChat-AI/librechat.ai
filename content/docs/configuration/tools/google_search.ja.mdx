<Callout type="info" title="Web検索をお探しですか？">

このページでは、**Google Custom Search** エージェントツールについて説明します。LibreChat に組み込まれている **Web Search** 機能（Serper/SearXNG + Firecrawl + Jina）については、[Web Search](/docs/features/web_search) を参照してください。

</Callout>

Google Searchツールを使用すると、エージェントはCustom Search JSON APIを介してGoogleで検索クエリを実行できるようになります。これには、Googleカスタム検索エンジンIDとAPIキーが必要です。

## セットアップ [#setup]

<Steps>
  <Step>

### Programmable Search Engineを作成する [#create-a-programmable-search-engine]

[Programmable Search Engine control panel](https://programmablesearchengine.google.com/controlpanel/all) にアクセスし、Googleアカウントでサインインします。

**Add** をクリックして新しい検索エンジンを作成します。名前を入力し、**Search the entire web** を選択して、**Create** をクリックします。

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)

  </Step>
  <Step>

### 検索エンジンIDをコピーする [#copy-your-search-engine-id]

エンジンを作成した後、**Search engine ID**が表示されます。これをコピーして、`.env`ファイルの`GOOGLE_CSE_ID`に追加してください。

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)

  </Step>
  <Step>

### Google Search APIキーを取得する [#get-a-google-search-api-key]

[Custom Search JSON API introduction page](https://developers.google.com/custom-search/v1/introduction) にアクセスし、**Get a Key** をクリックします。

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)

プロジェクトに名前を付け、利用規約に同意し、APIキーをコピーします。

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)

  </Step>
  <Step>

### 環境変数の追加 [#add-environment-variables]

両方の値を `.env` ファイルに追加します：

```bash filename=".env"
GOOGLE_SEARCH_API_KEY=your-api-key-here
GOOGLE_CSE_ID=your-search-engine-id-here
```

  </Step>
  <Step>

### エージェントにツールを追加する [#add-the-tool-to-an-agent]

LibreChatで、**Agents**パネルに移動し、エージェントを作成または編集します。エージェントの**Tools**リストで、**Google Search**を選択します。

  </Step>
  <Step>

### 再起動とテスト [#restart-and-test]

| デプロイ | コマンド |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| ローカル | 停止 (Ctrl+C) 後に `npm run backend` |

「AIに関する最新ニュースを検索して」のようなメッセージをエージェントに送信します。エージェントはGoogle Custom Searchを使用して関連する結果を検索し、返信します。

  </Step>
</Steps>

<Callout type="warn" title="よくある質問">

検索結果が何も返されない場合は、Programmable Search Engineが**ウェブ全体を検索**するように設定されていること（特定のサイトに制限されていないこと）を確認してください。また、`.env`ファイルに`GOOGLE_SEARCH_API_KEY`と`GOOGLE_CSE_ID`の両方が設定されていること、および変更後にLibreChatを再起動したことを確認してください。

</Callout>
