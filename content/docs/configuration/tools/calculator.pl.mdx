Calculator to wbudowane narzędzie agenta służące do obliczeń arytmetycznych i symbolicznych. Nie wymaga ono klucza API ani zmiennej środowiskowej.

## Konfiguracja [#setup]

<Steps>
  <Step>

### Dodawanie kalkulatora do Agenta [#add-calculator-to-an-agent]

W LibreChat wybierz **Agents**, utwórz lub edytuj agenta, otwórz listę **Tools** agenta, wybierz **Calculator** i zapisz agenta.

  </Step>
  <Step>

### Przetestuj to [#test-it]

Poproś agenta o obliczenie czegoś, co wymaga użycia narzędzia:

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
