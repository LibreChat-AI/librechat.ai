Calculator es una herramienta de agente integrada para cálculos aritméticos y simbólicos. No requiere una clave de API ni una variable de entorno.

## Configuración [#setup]

<Steps>
  <Step>

### Agregar una calculadora a un Agent [#add-calculator-to-an-agent]

En LibreChat, selecciona **Agents**, crea o edita un agente, abre la lista de **Tools** del agente, selecciona **Calculator** y guarda el agente.

  </Step>
  <Step>

### Pruébalo [#test-it]

Pídele al agente que calcule algo que se beneficie de una llamada a una herramienta:

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
