Calculator è uno strumento agente integrato per calcoli aritmetici e simbolici. Non richiede una chiave API o una variabile d'ambiente.

## Setup [#setup]

<Steps>
  <Step>

### Aggiungere una calcolatrice a un Agent [#add-calculator-to-an-agent]

In LibreChat, seleziona **Agents**, crea o modifica un agente, apri l'elenco **Tools** dell'agente, seleziona **Calculator** e salva l'agente.

  </Step>
  <Step>

### Testalo [#test-it]

Chiedi all'agente di calcolare qualcosa che tragga vantaggio da una chiamata a uno strumento:

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
