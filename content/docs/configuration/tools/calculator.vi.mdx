Calculator là một công cụ tác nhân (agent tool) tích hợp sẵn cho các phép tính số học và ký hiệu. Nó không yêu cầu khóa API hoặc biến môi trường.

## Thiết lập [#setup]

<Steps>
  <Step>

### Thêm Máy tính vào một Agent [#add-calculator-to-an-agent]

Trong LibreChat, hãy chọn **Agents**, tạo hoặc chỉnh sửa một agent, mở danh sách **Tools** của agent đó, chọn **Calculator**, và lưu agent lại.

  </Step>
  <Step>

### Kiểm tra thử [#test-it]

Yêu cầu tác nhân tính toán một thứ gì đó cần sử dụng đến tool call:

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
