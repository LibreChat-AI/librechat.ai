Calculator é uma ferramenta de agente integrada para cálculos aritméticos e simbólicos. Ela não requer uma chave de API ou variável de ambiente.

## Configuração [#setup]

<Steps>
  <Step>

### Adicionar Calculadora a um Agente [#add-calculator-to-an-agent]

No LibreChat, selecione **Agents**, crie ou edite um agente, abra a lista de **Tools** do agente, selecione **Calculator** e salve o agente.

  </Step>
  <Step>

### Teste-o [#test-it]

Peça ao agente para calcular algo que se beneficie de uma chamada de ferramenta:

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
