Tavily Search é uma ferramenta de agente integrada para pesquisa web atual. Ela retorna resultados de pesquisa estruturados e pode, opcionalmente, incluir respostas, imagens, conteúdo bruto de páginas, filtros de domínio e filtros de atualidade.

## Configuração [#setup]

<Steps>
  <Step>

### Obtenha uma chave de API da Tavily [#get-a-tavily-api-key]

Crie uma conta no Tavily e copie sua chave de API em [app.tavily.com](https://app.tavily.com/).

  </Step>
  <Step>

### Adicione a Variável de Ambiente [#add-the-environment-variable]

Adicione a chave ao seu arquivo `.env`:

```bash filename=".env"
TAVILY_API_KEY=tvly-your-api-key
```

  </Step>
  <Step>

### Reiniciar o LibreChat [#restart-librechat]

| Deployment | Command |
| ---------- | ------------------------------------------------- |
| Docker     | `docker compose down && docker compose up -d`     |
| Local      | Pare o servidor e execute `npm run backend` novamente |

  </Step>
  <Step>

### Adicionar Tavily a um Agent [#add-tavily-to-an-agent]

No LibreChat, selecione **Agents**, crie ou edite um agente, abra a lista de **Tools** do agente, selecione **Tavily Search** e salve o agente.

  </Step>
</Steps>

## Parâmetros [#parameters]

| Parâmetro                    | Descrição                                                                                                   |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `query`                      | Consulta de pesquisa. Obrigatório.                                                                          |
| `max_results`                | Número de resultados a retornar, de 1 a 10. O padrão é 5.                                                   |
| `search_depth`               | `basic` para resultados mais rápidos ou `advanced` para resultados de maior qualidade. Pesquisas avançadas contam como 2 requisições. |
| `include_answer`             | Incluir a resposta gerada pelo Tavily na resposta.                                                          |
| `include_images`             | Incluir resultados de imagens.                                                                              |
| `include_image_descriptions` | Incluir descrições para as imagens retornadas quando as imagens estiverem habilitadas.                      |
| `include_raw_content`        | Incluir o conteúdo bruto da página no resultado.                                                            |
| `include_domains`            | Limitar os resultados a domínios específicos.                                                               |
| `exclude_domains`            | Excluir domínios específicos.                                                                               |
| `topic`                      | `general`, `news` ou `finance`.                                                                             |
| `time_range`                 | Limitar os resultados a um período recente, como `day`, `week`, `month` ou `year`.                          |
| `days`                       | Número de dias retroativos a incluir para pesquisas de notícias.                                            |

## Notas [#notes]

- Tavily Search é uma ferramenta de agente. O LibreChat também pode usar o Tavily como um provedor de [Web Search](/docs/features/web_search) ou scraper, que é configurado separadamente.
- As solicitações do Tavily usam a variável de ambiente global `PROXY` quando ela está configurada. Quando `PROXY` não está definida, os clientes do lado do servidor suportados respeitam `HTTP_PROXY`, `HTTPS_PROXY` e `NO_PROXY`/`no_proxy`.
- Se a ferramenta não aparecer no Agent Builder, confirme se `TAVILY_API_KEY` está definida e verifique [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools) ou [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools).
