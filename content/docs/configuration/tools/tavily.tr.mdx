Tavily Search, güncel web araştırmaları için yerleşik bir agent aracıdır. Yapılandırılmış arama sonuçları döndürür ve isteğe bağlı olarak yanıtlar, görseller, ham sayfa içeriği, alan adı filtreleri ve güncellik filtreleri içerebilir.

## Kurulum [#setup]

<Steps>
  <Step>

### Tavily API Anahtarı Alın [#get-a-tavily-api-key]

Bir Tavily hesabı oluşturun ve [app.tavily.com](https://app.tavily.com/) adresinden API anahtarınızı kopyalayın.

  </Step>
  <Step>

### Ortam Değişkenini Ekle [#add-the-environment-variable]

Anahtarı `.env` dosyanıza ekleyin:

```bash filename=".env"
TAVILY_API_KEY=tvly-your-api-key
```

  </Step>
  <Step>

### LibreChat'i Yeniden Başlatın [#restart-librechat]

| Dağıtım | Komut |
| ---------- | ------------------------------------------------- |
| Docker     | `docker compose down && docker compose up -d`     |
| Yerel      | Sunucuyu durdurun, ardından tekrar `npm run backend` komutunu çalıştırın |

  </Step>
  <Step>

### Bir Ajan'a Tavily Ekleme [#add-tavily-to-an-agent]

LibreChat içinde **Agents** kısmını seçin, bir agent oluşturun veya düzenleyin, agent'ın **Tools** listesini açın, **Tavily Search** seçeneğini belirleyin ve agent'ı kaydedin.

  </Step>
</Steps>

## Parametreler [#parameters]

| Parametre                    | Açıklama                                                                                                    |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `query`                      | Arama sorgusu. Zorunludur.                                                                                  |
| `max_results`                | Döndürülecek sonuç sayısı, 1 ile 10 arası. Varsayılan değer 5'tir.                                         |
| `search_depth`               | Daha hızlı sonuçlar için `basic` veya daha yüksek kaliteli sonuçlar için `advanced`. Gelişmiş aramalar 2 istek sayılır. |
| `include_answer`             | Tavily tarafından oluşturulan yanıtı yanıta dahil et.                                                       |
| `include_images`             | Görsel sonuçlarını dahil et.                                                                                |
| `include_image_descriptions` | Görseller etkinleştirildiğinde döndürülen görseller için açıklamaları dahil et.                             |
| `include_raw_content`        | Sonuca ham sayfa içeriğini dahil et.                                                                        |
| `include_domains`            | Sonuçları belirli alan adlarıyla sınırla.                                                                   |
| `exclude_domains`            | Belirli alan adlarını hariç tut.                                                                            |
| `topic`                      | `general`, `news` veya `finance`.                                                                           |
| `time_range`                 | Sonuçları `day`, `week`, `month` veya `year` gibi yakın bir dönemle sınırla.                                |
| `days`                       | Haber aramaları için dahil edilecek geriye dönük gün sayısı.                                                |

## Notlar [#notes]

- Tavily Search bir ajan aracıdır. LibreChat, Tavily'yi ayrıca yapılandırılan bir [Web Search](/docs/features/web_search) sağlayıcısı veya kazıyıcı olarak da kullanabilir.
- Tavily istekleri, yapılandırıldığında global `PROXY` ortam değişkenini kullanır. `PROXY` ayarlanmadığında, desteklenen sunucu tarafı istemcileri `HTTP_PROXY`, `HTTPS_PROXY` ve `NO_PROXY`/`no_proxy` değişkenlerini dikkate alır.
- Eğer araç Agent Builder içinde görünmüyorsa, `TAVILY_API_KEY` değerinin ayarlandığından emin olun ve [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools) veya [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) ayarlarını kontrol edin.
