<Callout type="info" title="Auf der Suche nach der Websuche?">

Diese Seite behandelt das **Google Custom Search** Agenten-Tool. Für die integrierte **Web Search**-Funktion von LibreChat (Serper/SearXNG + Firecrawl + Jina), siehe [Web Search](/docs/features/web_search).

</Callout>

Das Google Search-Tool ermöglicht es Ihren Agents, Google mithilfe der Custom Search JSON API abzufragen. Sie benötigen eine Google Custom Search Engine ID und einen API-Schlüssel.

## Einrichtung [#setup]

<Steps>
  <Step>

### Erstellen einer Programmable Search Engine [#create-a-programmable-search-engine]

Gehen Sie zum [Programmable Search Engine control panel](https://programmablesearchengine.google.com/controlpanel/all) und melden Sie sich mit Ihrem Google-Konto an.

Klicken Sie auf **Add**, um eine neue Suchmaschine zu erstellen. Geben Sie einen Namen ein, wählen Sie **Search the entire web** aus und klicken Sie auf **Create**.

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)

  </Step>
  <Step>

### Kopieren Sie Ihre Suchmaschinen-ID [#copy-your-search-engine-id]

Nachdem Sie die Engine erstellt haben, sehen Sie Ihre **Search engine ID**. Kopieren Sie diese – Sie werden sie Ihrer `.env` Datei als `GOOGLE_CSE_ID` hinzufügen.

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)

  </Step>
  <Step>

### Einen Google Search API-Schlüssel erhalten [#get-a-google-search-api-key]

Gehen Sie zur [Einführungsseite der Custom Search JSON API](https://developers.google.com/custom-search/v1/introduction) und klicken Sie auf **Get a Key**.

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)

Geben Sie Ihrem Projekt einen Namen, stimmen Sie den Nutzungsbedingungen zu und kopieren Sie den API-Schlüssel.

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)

  </Step>
  <Step>

### Umgebungsvariablen hinzufügen [#add-environment-variables]

Fügen Sie beide Werte zu Ihrer `.env` Datei hinzu:

```bash filename=".env"
GOOGLE_SEARCH_API_KEY=your-api-key-here
GOOGLE_CSE_ID=your-search-engine-id-here
```

  </Step>
  <Step>

### Das Tool zu einem Agenten hinzufügen [#add-the-tool-to-an-agent]

Gehen Sie in LibreChat zum Bereich **Agents** und erstellen oder bearbeiten Sie einen Agenten. Wählen Sie in der **Tools**-Liste des Agenten **Google Search** aus.

  </Step>
  <Step>

### Neustart und Test [#restart-and-test]

| Bereitstellung | Befehl |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokal | Stoppen (Strg+C) und dann `npm run backend` |

Sende eine Nachricht wie „Suche nach den neuesten Nachrichten über KI“ an deinen Agenten. Der Agent wird die Google Custom Search verwenden, um relevante Ergebnisse zu finden und zurückzugeben.

  </Step>
</Steps>

<Callout type="warn" title="Häufige Probleme">

Wenn die Suche keine Ergebnisse liefert, stellen Sie sicher, dass Ihre Programmable Search Engine auf **Das gesamte Web durchsuchen** eingestellt ist (nicht auf bestimmte Websites beschränkt). Bestätigen Sie außerdem, dass sowohl `GOOGLE_SEARCH_API_KEY` als auch `GOOGLE_CSE_ID` in Ihrer `.env` Datei gesetzt sind und dass Sie LibreChat nach den Änderungen neu gestartet haben.

</Callout>
