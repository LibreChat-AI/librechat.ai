Stable Diffusion is een ingebouwde agent-tool die LibreChat verbindt met een **[AUTOMATIC1111 Stable Diffusion WebUI](https://github.com/AUTOMATIC1111/stable-diffusion-webui)** API. Voor een gecontaineriseerde Stable Diffusion-implementatie via Docker kun je ook **[stable-diffusion-webui-docker](https://github.com/AbdBarho/stable-diffusion-webui-docker)** gebruiken.

Bij de Docker-implementatie kun je stap 2 en stap 3 overslaan; gebruik in plaats daarvan de installatie-instructies uit hun repository.

- Let op: je hebt een compatibele GPU nodig ("CPU-only" is mogelijk, maar erg traag). Nvidia wordt aanbevolen, maar er is geen duidelijke bron over incompatibele GPU's. Elke degelijke GPU zou moeten werken.

### 1. Volg de download- en installatie-instructies [#1-follow-download-and-installation-instructions]

Volg de installatiestappen uit de **[stable-diffusion-webui readme](https://github.com/AUTOMATIC1111/stable-diffusion-webui)**.

### 2. Bewerk je instellingen voor het uitvoeringsscript [#2-edit-your-run-script-settings]

#### Windows [#windows]

 - Bewerk je **webui-user.bat** bestand door de volgende regel toe te voegen vóór het call-commando:
- `set COMMANDLINE_ARGS=--api`

    - Je .bat bestand zou er zo uit moeten zien met alle andere instellingen op standaard
    ```shell 
    @echo off

    set PYTHON=
    set GIT=
    set VENV_DIR=
    set COMMANDLINE_ARGS=--api

    call webui.bat
    ```
#### Overige (niet getest, maar zou moeten werken) [#others-not-tested-but-should-work]

 - Bewerk je **webui-user.sh** bestand door de volgende regel toe te voegen:
 - `export COMMANDLINE_ARGS="--api"`

     - Je .sh bestand zou er zo uit moeten zien met alle andere instellingen op standaard
    ```bash 

    export COMMANDLINE_ARGS="--api"

    #!/bin/bash
    #########################################################
    # Uncomment and change the variables below to your need:#
    #########################################################

    # ...rest
    ```

### 3. Voer Stable Diffusion uit (gebruik het .sh of .bat bestand, afhankelijk van je besturingssysteem) [#3-run-stable-diffusion-either-sh-or-bat-file-according-to-your-operating-system]

### 4. Stel de Stable Diffusion URL in LibreChat in [#4-set-the-stable-diffusion-url-in-librechat]

> **Let op: De standaardpoort voor Gradio is `7860`. Als je deze hebt gewijzigd, werk de waarde dan dienovereenkomstig bij.**

#### Docker Installatie [#docker-install]
- Gebruik `SD_WEBUI_URL=http://host.docker.internal:7860` in het `.env` bestand
- Of `http://host.docker.internal:7860` vanuit de webui

#### Lokale installatie [#local-install]
- Gebruik `SD_WEBUI_URL=http://127.0.0.1:7860` in het `.env` bestand
- Of `http://127.0.0.1:7860` vanuit de webui

Herstart LibreChat na het wijzigen van `.env`.

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokaal | Stop de server en voer daarna opnieuw `npm run backend` uit |

#### Voeg de Tool toe aan een Agent [#add-the-tool-to-an-agent]

Selecteer in LibreChat **Agents**, maak een agent aan of bewerk er een, open de **Tools**-lijst van de agent, selecteer **Stable Diffusion** en sla de agent op. Zie de sectie [Agents](/docs/features/agents#tools) voor meer informatie.
