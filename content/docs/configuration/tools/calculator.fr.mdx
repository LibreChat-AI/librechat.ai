Calculator est un outil d'agent intégré pour les calculs arithmétiques et symboliques. Il ne nécessite aucune clé API ni variable d'environnement.

## Configuration [#setup]

<Steps>
  <Step>

### Ajouter une calculatrice à un Agent [#add-calculator-to-an-agent]

Dans LibreChat, sélectionnez **Agents**, créez ou modifiez un agent, ouvrez la liste **Tools** de l'agent, sélectionnez **Calculator**, puis enregistrez l'agent.

  </Step>
  <Step>

### Testez-le [#test-it]

Demandez à l'agent de calculer quelque chose qui nécessite un appel d'outil :

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
