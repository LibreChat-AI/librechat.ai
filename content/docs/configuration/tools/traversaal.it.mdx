Traversaal è uno strumento di ricerca agente integrato che invia una query in linguaggio naturale a Traversaal Ares e restituisce una risposta con gli URL delle fonti quando disponibili.

## Setup [#setup]

<Steps>
  <Step>

### Ottieni una API Key di Traversaal [#get-a-traversaal-api-key]

Crea un account e ottieni una API key da [api.traversaal.ai](https://api.traversaal.ai/).

  </Step>
  <Step>

### Aggiungi la variabile d'ambiente [#add-the-environment-variable]

Aggiungi la chiave al tuo file `.env`:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### Riavviare LibreChat [#restart-librechat]

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Locale | Arresta il server, quindi esegui nuovamente `npm run backend` |

  </Step>
  <Step>

### Aggiungere Traversaal a un Agent [#add-traversaal-to-an-agent]

In LibreChat, seleziona **Agents**, crea o modifica un agente, apri l'elenco **Tools** dell'agente, seleziona **Traversaal** e salva l'agente.

  </Step>
</Steps>

## Parametri [#parameters]

| Parametro | Descrizione |
|-----------|-------------|
| `query` | Una frase completa che descrive ciò che l'agente deve cercare. Obbligatorio. |

## Esempio di Prompt [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Risoluzione dei problemi [#troubleshooting]

- Se Traversaal restituisce un errore di autenticazione, conferma che `TRAVERSAAL_API_KEY` sia impostata e riavvia LibreChat.
- Se lo strumento non è visibile nell'Agent Builder, controlla [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) e la funzionalità `tools` dell'agente.
