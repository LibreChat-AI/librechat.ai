Wolfram|Alpha aracı, ajanlara hesaplama, matematik, derlenmiş bilgi, birim dönüştürme, bilimsel veriler ve gerçek zamanlı verilere erişim sağlar. Wolfram|Alpha API'sine yapılan tüm çağrılarda bir AppID sağlanmalıdır.

- Not: Wolfram API çağrıları, normal kullanıcılar için günlük 100 ve aylık 2000 çağrı ile sınırlıdır.

### Bir hesap oluşturun [#make-an-account]
- Hesabınızı oluşturmak için **[products.wolframalpha.com/api/](https://products.wolframalpha.com/api/)** adresini ziyaret edin

### AppID'nizi alın [#get-your-appid]
- **[Geliştirici Portalı](https://developer.wolframalpha.com/access)** adresini ziyaret edin ve `Get an AppID` seçeneğine tıklayın
- `API` olarak `LLM API` seçeneğini belirleyin ve anahtarı kopyalayın

### LibreChat'i Yapılandırın [#configure-librechat]

AppID değerinizi `.env` dosyasına ekleyin:

```bash filename=".env"
WOLFRAM_APP_ID=your-app-id
```

`.env` dosyasını değiştirdikten sonra LibreChat'i yeniden başlatın.

| Dağıtım | Komut |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Yerel | Sunucuyu durdurun, ardından tekrar `npm run backend` komutunu çalıştırın |

#### Aracı bir Ajan'a Ekleme [#add-the-tool-to-an-agent]

LibreChat içinde **Agents** kısmını seçin, bir ajan oluşturun veya düzenleyin, ajanın **Tools** listesini açın, **Wolfram** seçeneğini belirleyin ve ajanı kaydedin. Daha fazla bilgi için [Agents](/docs/features/agents#tools) bölümüne bakın.
