Tavily Search adalah alat agen bawaan untuk riset web terkini. Alat ini mengembalikan hasil pencarian terstruktur dan secara opsional dapat menyertakan jawaban, gambar, konten halaman mentah, filter domain, dan filter keterkinian.

## Pengaturan [#setup]

<Steps>
  <Step>

### Dapatkan API Key Tavily [#get-a-tavily-api-key]

Buat akun Tavily dan salin kunci API Anda dari [app.tavily.com](https://app.tavily.com/).

  </Step>
  <Step>

### Tambahkan Variabel Lingkungan [#add-the-environment-variable]

Tambahkan kunci tersebut ke file `.env` Anda:

```bash filename=".env"
TAVILY_API_KEY=tvly-your-api-key
```

  </Step>
  <Step>

### Restart LibreChat [#restart-librechat]

| Deployment | Command                                           |
| ---------- | ------------------------------------------------- |
| Docker     | `docker compose down && docker compose up -d`     |
| Local      | Hentikan server, lalu jalankan kembali `npm run backend` |

  </Step>
  <Step>

### Menambahkan Tavily ke Agent [#add-tavily-to-an-agent]

Di LibreChat, pilih **Agents**, buat atau edit agen, buka daftar **Tools** agen tersebut, pilih **Tavily Search**, lalu simpan agen tersebut.

  </Step>
</Steps>

## Parameter [#parameters]

| Parameter | Deskripsi |
| --------- | --------- |
| `query` | Kueri pencarian. Wajib diisi. |
| `max_results` | Jumlah hasil yang akan dikembalikan, dari 1 hingga 10. Defaultnya adalah 5. |
| `search_depth` | `basic` untuk hasil yang lebih cepat atau `advanced` untuk hasil berkualitas lebih tinggi. Pencarian tingkat lanjut dihitung sebagai 2 permintaan. |
| `include_answer` | Sertakan jawaban yang dihasilkan Tavily dalam respons. |
| `include_images` | Sertakan hasil gambar. |
| `include_image_descriptions` | Sertakan deskripsi untuk gambar yang dikembalikan saat gambar diaktifkan. |
| `include_raw_content` | Sertakan konten halaman mentah dalam hasil. |
| `include_domains` | Batasi hasil ke domain tertentu. |
| `exclude_domains` | Kecualikan domain tertentu. |
| `topic` | `general`, `news`, atau `finance`. |
| `time_range` | Batasi hasil ke periode terkini seperti `day`, `week`, `month`, atau `year`. |
| `days` | Jumlah hari ke belakang yang disertakan untuk pencarian berita. |

## Catatan [#notes]

- Tavily Search adalah alat agen. LibreChat juga dapat menggunakan Tavily sebagai penyedia atau scraper [Web Search](/docs/features/web_search), yang dikonfigurasi secara terpisah.
- Permintaan Tavily menggunakan variabel lingkungan `PROXY` global saat dikonfigurasi. Ketika `PROXY` tidak diatur, klien sisi server yang didukung akan menghormati `HTTP_PROXY`, `HTTPS_PROXY`, dan `NO_PROXY`/`no_proxy`.
- Jika alat tersebut tidak muncul di Agent Builder, pastikan `TAVILY_API_KEY` telah diatur dan periksa [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools) atau [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools).
