Traversaal to wbudowane narzędzie wyszukiwania agenta, które wysyła zapytanie w języku naturalnym do Traversaal Ares i zwraca odpowiedź wraz z adresami URL źródeł, jeśli są dostępne.

## Konfiguracja [#setup]

<Steps>
  <Step>

### Uzyskaj klucz API Traversaal [#get-a-traversaal-api-key]

Utwórz konto i uzyskaj klucz API z [api.traversaal.ai](https://api.traversaal.ai/).

  </Step>
  <Step>

### Dodaj zmienną środowiskową [#add-the-environment-variable]

Dodaj klucz do swojego pliku `.env`:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### Restart LibreChat [#restart-librechat]

| Wdrożenie | Polecenie |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Lokalne | Zatrzymaj serwer, a następnie ponownie uruchom `npm run backend` |

  </Step>
  <Step>

### Dodaj Traversaal do Agenta [#add-traversaal-to-an-agent]

W LibreChat wybierz **Agents**, utwórz lub edytuj agenta, otwórz listę **Tools** agenta, wybierz **Traversaal** i zapisz agenta.

  </Step>
</Steps>

## Parametry [#parameters]

| Parametr | Opis |
|-----------|-------------|
| `query` | Pełne zdanie opisujące, czego agent ma szukać. Wymagane. |

## Przykładowe prompty [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Rozwiązywanie problemów [#troubleshooting]

- Jeśli Traversaal zwraca błąd uwierzytelniania, upewnij się, że `TRAVERSAAL_API_KEY` jest ustawiony i zrestartuj LibreChat.
- Jeśli narzędzie nie jest widoczne w Agent Builder, sprawdź [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) oraz funkcję `tools` agenta.
