Traversaal est un outil de recherche par agent intégré qui envoie une requête en langage naturel à Traversaal Ares et renvoie une réponse avec les URL des sources lorsqu'elles sont disponibles.

## Configuration [#setup]

<Steps>
  <Step>

### Obtenir une clé API Traversaal [#get-a-traversaal-api-key]

Créez un compte et obtenez une clé API sur [api.traversaal.ai](https://api.traversaal.ai/).

  </Step>
  <Step>

### Ajouter la variable d'environnement [#add-the-environment-variable]

Ajoutez la clé à votre fichier `.env` :

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### Redémarrer LibreChat [#restart-librechat]

| Déploiement | Commande |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Arrêtez le serveur, puis exécutez à nouveau `npm run backend` |

  </Step>
  <Step>

### Ajouter Traversaal à un Agent [#add-traversaal-to-an-agent]

Dans LibreChat, sélectionnez **Agents**, créez ou modifiez un agent, ouvrez la liste **Tools** de l'agent, sélectionnez **Traversaal**, puis enregistrez l'agent.

  </Step>
</Steps>

## Paramètres [#parameters]

| Paramètre | Description |
|-----------|-------------|
| `query` | Une phrase complète décrivant ce que l'agent doit rechercher. Requis. |

## Exemples de prompts [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Dépannage [#troubleshooting]

- Si Traversaal renvoie une erreur d'authentification, confirmez que `TRAVERSAAL_API_KEY` est défini et redémarrez LibreChat.
- Si l'outil n'est pas visible dans l'Agent Builder, vérifiez [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools), et la capacité `tools` de l'agent.
