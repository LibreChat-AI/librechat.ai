<Callout type="info" title="Procurando pela Pesquisa na Web?">

Esta página aborda a ferramenta de agente **Google Custom Search**. Para o recurso de **Web Search** integrado do LibreChat (Serper/SearXNG + Firecrawl + Jina), consulte [Web Search](/docs/features/web_search).

</Callout>

A ferramenta Google Search permite que seus agentes consultem o Google usando a Custom Search JSON API. Você precisará de um ID de Google Custom Search Engine e uma chave de API.

## Configuração [#setup]

<Steps>
  <Step>

### Criar um Programmable Search Engine [#create-a-programmable-search-engine]

Vá para o [painel de controle do Programmable Search Engine](https://programmablesearchengine.google.com/controlpanel/all) e faça login com sua conta Google.

Clique em **Add** para criar um novo mecanismo de busca. Preencha um nome, selecione **Search the entire web** e clique em **Create**.

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)

  </Step>
  <Step>

### Copie seu ID do Mecanismo de Pesquisa [#copy-your-search-engine-id]

Após criar o mecanismo, você verá seu **Search engine ID**. Copie-o -- você o adicionará ao seu `.env` file como `GOOGLE_CSE_ID`.

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)

  </Step>
  <Step>

### Obtenha uma chave de API do Google Search [#get-a-google-search-api-key]

Vá para a [página de introdução da Custom Search JSON API](https://developers.google.com/custom-search/v1/introduction) e clique em **Get a Key**.

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)

Dê um nome ao seu projeto, concorde com os Termos de Serviço e copie a chave de API.

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)

  </Step>
  <Step>

### Adicionar Variáveis de Ambiente [#add-environment-variables]

Adicione ambos os valores ao seu arquivo `.env`:

```bash filename=".env"
GOOGLE_SEARCH_API_KEY=your-api-key-here
GOOGLE_CSE_ID=your-search-engine-id-here
```

  </Step>
  <Step>

### Adicionar a Ferramenta a um Agent [#add-the-tool-to-an-agent]

No LibreChat, vá para o painel **Agents** e crie ou edite um agente. Na lista de **Tools** do agente, selecione **Google Search**.

  </Step>
  <Step>

### Reiniciar e Testar [#restart-and-test]

| Implantação | Comando |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Parar (Ctrl+C) e então `npm run backend` |

Envie uma mensagem como "Search for the latest news about AI" para o seu agente. O agente usará o Google Custom Search para encontrar e retornar resultados relevantes.

  </Step>
</Steps>

<Callout type="warn" title="Problemas Comuns">

Se a pesquisa não retornar resultados, verifique se o seu Programmable Search Engine está configurado para **Pesquisar em toda a web** (não restrito a sites específicos). Confirme também se `GOOGLE_SEARCH_API_KEY` e `GOOGLE_CSE_ID` estão definidos no seu arquivo `.env` e que você reiniciou o LibreChat após fazer as alterações.

</Callout>
