---

Traversaal 是一个内置的智能体搜索工具，它将自然语言查询发送给 Traversaal Ares，并在可用时返回包含源 URL 的响应。

## 设置 [#setup]

<Steps>
  <Step>

### 获取 Traversaal API Key [#get-a-traversaal-api-key]

创建一个账户并从 [api.traversaal.ai](https://api.traversaal.ai/) 获取 API key。

  </Step>
  <Step>

### 添加环境变量 [#add-the-environment-variable]

将该密钥添加到您的 `.env` 文件中：

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### 重启 LibreChat [#restart-librechat]

| 部署方式 | 命令 |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| 本地 | 停止服务器，然后再次运行 `npm run backend` |

  </Step>
  <Step>

### 将 Traversaal 添加到 Agent [#add-traversaal-to-an-agent]

在 LibreChat 中，选择 **Agents**，创建或编辑一个 agent，打开该 agent 的 **Tools** 列表，选择 **Traversaal**，然后保存该 agent。

  </Step>
</Steps>

## 参数 [#parameters]

| 参数 | 描述 |
|-----------|-------------|
| `query` | 描述代理应搜索内容的完整句子。必填。 |

## 示例提示词 [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## 故障排除 [#troubleshooting]

- 如果 Traversaal 返回身份验证错误，请确认已设置 `TRAVERSAAL_API_KEY` 并重启 LibreChat。
- 如果该工具在 Agent Builder 中不可见，请检查 [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools)、[`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) 以及该 agent 的 `tools` 功能。
