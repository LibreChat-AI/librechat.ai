Tavily Search는 최신 웹 조사를 위한 내장 에이전트 도구입니다. 구조화된 검색 결과를 반환하며, 선택적으로 답변, 이미지, 원본 페이지 콘텐츠, 도메인 필터 및 최신성 필터를 포함할 수 있습니다.

## 설정 [#setup]

<Steps>
  <Step>

### Tavily API 키 받기 [#get-a-tavily-api-key]

Tavily 계정을 만들고 [app.tavily.com](https://app.tavily.com/)에서 API 키를 복사하세요.

  </Step>
  <Step>

### 환경 변수 추가 [#add-the-environment-variable]

`.env` 파일에 키를 추가하세요:

```bash filename=".env"
TAVILY_API_KEY=tvly-your-api-key
```

  </Step>
  <Step>

### LibreChat 재시작 [#restart-librechat]

| 배포 | 명령어 |
| ---------- | ------------------------------------------------- |
| Docker     | `docker compose down && docker compose up -d`     |
| Local      | 서버를 중지한 후, `npm run backend`를 다시 실행하세요 |

  </Step>
  <Step>

### 에이전트에 Tavily 추가하기 [#add-tavily-to-an-agent]

LibreChat에서 **Agents**를 선택하고, 에이전트를 생성하거나 편집한 다음, 해당 에이전트의 **Tools** 목록을 열어 **Tavily Search**를 선택하고 에이전트를 저장하세요.

  </Step>
</Steps>

## 매개변수 [#parameters]

| 매개변수 | 설명 |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `query`                      | 검색 쿼리. 필수 항목입니다. |
| `max_results`                | 반환할 결과 수(1에서 10 사이). 기본값은 5입니다. |
| `search_depth`               | 더 빠른 결과를 원하면 `basic`, 더 높은 품질의 결과를 원하면 `advanced`를 사용하세요. Advanced 검색은 2번의 요청으로 계산됩니다. |
| `include_answer`             | 응답에 Tavily가 생성한 답변을 포함합니다. |
| `include_images`             | 이미지 결과를 포함합니다. |
| `include_image_descriptions` | 이미지가 활성화된 경우 반환된 이미지에 대한 설명을 포함합니다. |
| `include_raw_content`        | 결과에 원본 페이지 콘텐츠를 포함합니다. |
| `include_domains`            | 특정 도메인으로 결과를 제한합니다. |
| `exclude_domains`            | 특정 도메인을 제외합니다. |
| `topic`                      | `general`, `news`, 또는 `finance`. |
| `time_range`                 | 결과를 `day`, `week`, `month`, 또는 `year`와 같은 최근 기간으로 제한합니다. |
| `days`                       | 뉴스 검색 시 포함할 과거 일수입니다. |

## 참고 사항 [#notes]

- Tavily Search는 에이전트 도구입니다. LibreChat은 Tavily를 별도로 구성되는 [Web Search](/docs/features/web_search) 제공자 또는 스크레이퍼로 사용할 수도 있습니다.
- Tavily 요청은 구성된 경우 전역 `PROXY` 환경 변수를 사용합니다. `PROXY`가 설정되지 않은 경우, 지원되는 서버 측 클라이언트는 `HTTP_PROXY`, `HTTPS_PROXY` 및 `NO_PROXY`/`no_proxy`를 따릅니다.
- Agent Builder에 도구가 나타나지 않으면 `TAVILY_API_KEY`가 설정되어 있는지 확인하고 [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools) 또는 [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools)를 확인하세요.
