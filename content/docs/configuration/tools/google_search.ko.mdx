<Callout type="info" title="웹 검색을 찾으시나요?">

이 페이지는 **Google Custom Search** 에이전트 도구에 대해 다룹니다. LibreChat의 내장 **Web Search** 기능(Serper/SearXNG + Firecrawl + Jina)에 대해서는 [Web Search](/docs/features/web_search)를 참조하세요.

</Callout>

Google Search 도구는 에이전트가 Custom Search JSON API를 사용하여 Google에 쿼리할 수 있도록 합니다. Google Custom Search Engine ID와 API 키가 필요합니다.

## 설정 [#setup]

<Steps>
  <Step>

### 프로그래밍 가능한 검색 엔진(Programmable Search Engine) 만들기 [#create-a-programmable-search-engine]

[Programmable Search Engine 제어판](https://programmablesearchengine.google.com/controlpanel/all)으로 이동하여 Google 계정으로 로그인하세요.

**Add**를 클릭하여 새로운 검색 엔진을 생성합니다. 이름을 입력하고 **Search the entire web**을 선택한 다음, **Create**를 클릭합니다.

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)

  </Step>
  <Step>

### 검색 엔진 ID 복사하기 [#copy-your-search-engine-id]

엔진을 생성한 후, **Search engine ID**가 표시됩니다. 이를 복사하여 `.env` 파일에 `GOOGLE_CSE_ID`로 추가하세요.

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)

  </Step>
  <Step>

### Google Search API 키 가져오기 [#get-a-google-search-api-key]

[Custom Search JSON API 소개 페이지](https://developers.google.com/custom-search/v1/introduction)로 이동하여 **Get a Key**를 클릭하세요.

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)

프로젝트 이름을 지정하고, 서비스 약관에 동의한 뒤, API 키를 복사하세요.

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)

  </Step>
  <Step>

### 환경 변수 추가 [#add-environment-variables]

두 값을 `.env` 파일에 추가하세요:

```bash filename=".env"
GOOGLE_SEARCH_API_KEY=your-api-key-here
GOOGLE_CSE_ID=your-search-engine-id-here
```

  </Step>
  <Step>

### 에이전트에 도구 추가하기 [#add-the-tool-to-an-agent]

LibreChat에서 **Agents** 패널로 이동하여 에이전트를 생성하거나 편집하세요. 에이전트의 **Tools** 목록에서 **Google Search**를 선택하세요.

  </Step>
  <Step>

### 재시작 및 테스트 [#restart-and-test]

| 배포 | 명령어 |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | 중지 (Ctrl+C) 후 `npm run backend` |

"AI에 대한 최신 뉴스 검색해 줘"와 같은 메시지를 에이전트에게 보내세요. 에이전트는 Google Custom Search를 사용하여 관련 결과를 찾아 반환합니다.

  </Step>
</Steps>

<Callout type="warn" title="일반적인 문제">

검색 결과가 나오지 않는다면, Programmable Search Engine이 **전체 웹 검색(Search the entire web)**으로 설정되어 있는지(특정 사이트로 제한되지 않았는지) 확인하세요. 또한 `.env` 파일에 `GOOGLE_SEARCH_API_KEY`와 `GOOGLE_CSE_ID`가 모두 설정되어 있는지, 그리고 변경 사항을 적용한 후 LibreChat을 재시작했는지 확인하시기 바랍니다.

</Callout>
