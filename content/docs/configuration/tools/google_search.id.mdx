<Callout type="info" title="Mencari Pencarian Web?">

Halaman ini membahas alat agen **Google Custom Search**. Untuk fitur **Web Search** bawaan LibreChat (Serper/SearXNG + Firecrawl + Jina), lihat [Web Search](/docs/features/web_search).

</Callout>

Alat Google Search memungkinkan agen Anda melakukan kueri ke Google menggunakan Custom Search JSON API. Anda akan memerlukan Google Custom Search Engine ID dan API key.

## Pengaturan [#setup]

<Steps>
  <Step>

### Membuat Programmable Search Engine [#create-a-programmable-search-engine]

Buka [Programmable Search Engine control panel](https://programmablesearchengine.google.com/controlpanel/all) dan masuk dengan akun Google Anda.

Klik **Add** untuk membuat mesin pencari baru. Isi nama, pilih **Search the entire web**, lalu klik **Create**.

![google_search-2](https://github.com/danny-avila/LibreChat/assets/32828263/152cfe7c-4796-49c6-9160-92cddf38f1c8)

  </Step>
  <Step>

### Salin ID Mesin Pencari Anda [#copy-your-search-engine-id]

Setelah membuat mesin pencari, Anda akan melihat **Search engine ID** Anda. Salin ID tersebut -- Anda akan menambahkannya ke file `.env` Anda sebagai `GOOGLE_CSE_ID`.

![google_search-4](https://github.com/danny-avila/LibreChat/assets/32828263/e03b5c79-87e5-4a68-b83e-61faf4f2f718)

  </Step>
  <Step>

### Dapatkan Kunci API Google Search [#get-a-google-search-api-key]

Buka [halaman pengantar Custom Search JSON API](https://developers.google.com/custom-search/v1/introduction) dan klik **Get a Key**.

![google_search-5](https://github.com/danny-avila/LibreChat/assets/32828263/2b93a2f9-5ed2-4794-96a8-a114e346a602)

Beri nama proyek Anda, setujui Ketentuan Layanan, dan salin kunci API tersebut.

![google_search-6](https://github.com/danny-avila/LibreChat/assets/32828263/82c9c3ef-7363-40cd-a89e-fc45088e4c86)

  </Step>
  <Step>

### Menambahkan Variabel Lingkungan [#add-environment-variables]

Tambahkan kedua nilai tersebut ke dalam file `.env` Anda:

```bash filename=".env"
GOOGLE_SEARCH_API_KEY=your-api-key-here
GOOGLE_CSE_ID=your-search-engine-id-here
```

  </Step>
  <Step>

### Menambahkan Tool ke Agent [#add-the-tool-to-an-agent]

Di LibreChat, buka panel **Agents** dan buat atau edit agen. Dalam daftar **Tools** agen tersebut, pilih **Google Search**.

  </Step>
  <Step>

### Mulai Ulang dan Uji [#restart-and-test]

| Deployment | Command |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Local | Hentikan (Ctrl+C) lalu `npm run backend` |

Kirim pesan seperti "Cari berita terbaru tentang AI" ke agen Anda. Agen tersebut akan menggunakan Google Custom Search untuk menemukan dan mengembalikan hasil yang relevan.

  </Step>
</Steps>

<Callout type="warn" title="Masalah Umum">

Jika pencarian tidak memberikan hasil, pastikan Programmable Search Engine Anda diatur ke **Search the entire web** (tidak dibatasi pada situs tertentu). Pastikan juga `GOOGLE_SEARCH_API_KEY` dan `GOOGLE_CSE_ID` telah diatur di dalam file `.env` Anda dan Anda telah memulai ulang LibreChat setelah melakukan perubahan.

</Callout>
