---
title: 계산기

---

Calculator는 산술 및 기호 계산을 위한 내장 에이전트 도구입니다. API 키나 환경 변수가 필요하지 않습니다.

## 설정 [#setup]

<Steps>
  <Step>

### 에이전트에 계산기 추가하기 [#add-calculator-to-an-agent]

LibreChat에서 **Agents**를 선택하고, 에이전트를 생성하거나 편집한 다음, 에이전트의 **Tools** 목록을 열어 **Calculator**를 선택하고 에이전트를 저장하세요.

  </Step>
  <Step>

### 테스트하기 [#test-it]

도구 호출을 통해 이점을 얻을 수 있는 계산을 에이전트에게 요청하세요:

```text
Calculate 12345 * 6789 and show the result.
```

  </Step>
</Steps>
