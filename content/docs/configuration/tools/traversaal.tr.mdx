Traversaal, doğal dildeki bir sorguyu Traversaal Ares'e gönderen ve mevcut olduğunda kaynak URL'leri ile birlikte bir yanıt döndüren yerleşik bir aracı arama aracıdır.

## Kurulum [#setup]

<Steps>
  <Step>

### Traversaal API Anahtarı Alın [#get-a-traversaal-api-key]

Bir hesap oluşturun ve [api.traversaal.ai](https://api.traversaal.ai/) adresinden bir API anahtarı alın.

  </Step>
  <Step>

### Ortam Değişkenini Ekle [#add-the-environment-variable]

Anahtarı `.env` dosyanıza ekleyin:

```bash filename=".env"
TRAVERSAAL_API_KEY=your-api-key
```

  </Step>
  <Step>

### LibreChat'i Yeniden Başlatın [#restart-librechat]

| Dağıtım | Komut |
|------------|---------|
| Docker | `docker compose down && docker compose up -d` |
| Yerel | Sunucuyu durdurun, ardından tekrar `npm run backend` komutunu çalıştırın |

  </Step>
  <Step>

### Bir Ajan'a Traversaal Ekleme [#add-traversaal-to-an-agent]

LibreChat içinde **Agents** kısmını seçin, bir agent oluşturun veya düzenleyin, agent'ın **Tools** listesini açın, **Traversaal** seçeneğini belirleyin ve agent'ı kaydedin.

  </Step>
</Steps>

## Parametreler [#parameters]

| Parametre | Açıklama |
|-----------|-------------|
| `query` | Ajanın neyi araması gerektiğini tanımlayan tam bir cümle. Zorunludur. |

## Örnek İstemi (Prompt) [#example-prompts]

```text
Find recent reporting about open source AI coding agents.
Search for sources comparing renewable energy adoption in Europe and North America.
```

## Sorun Giderme [#troubleshooting]

- Eğer Traversaal bir kimlik doğrulama hatası döndürürse, `TRAVERSAAL_API_KEY` değerinin ayarlandığından emin olun ve LibreChat'i yeniden başlatın.
- Eğer araç Agent Builder içinde görünmüyorsa, [`includedTools`](/docs/configuration/librechat_yaml/object_structure/config#includedtools), [`filteredTools`](/docs/configuration/librechat_yaml/object_structure/config#filteredtools) ve ajanın `tools` yeteneğini kontrol edin.
