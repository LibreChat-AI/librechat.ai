Tampilkan pesan penting kepada semua pengguna Anda di bagian atas aplikasi. Sangat cocok untuk pengumuman, pemberitahuan pemeliharaan, atau pembaruan.

<Callout type="info" title="Tinjauan Singkat">
  - Hanya **satu banner** yang dapat aktif dalam satu waktu
  - Jadwalkan banner agar muncul dan hilang secara otomatis
  - Pilih apakah pengguna dapat menutup banner atau tidak
  - Tampilkan banner kepada semua orang, atau hanya kepada pengguna yang telah masuk
</Callout>

---

## Membuat Banner [#creating-a-banner]

Jalankan perintah ini untuk membuat atau memperbarui banner:

```bash
npm run update-banner
```

Anda akan dipandu melalui beberapa perintah sederhana:

```ansi
--------------------------
Update the banner!
--------------------------
Display From (Format: yyyy-mm-ddTHH:MM:SSZ, Default: now):
> 2025-12-02T09:00:00Z

Display To (Format: yyyy-mm-ddTHH:MM:SSZ, Default: not specified):
> 2025-12-31T23:59:59Z

Enter your message (Enter a single dot "." on a new line to finish):
> 🎉 Welcome to LibreChat! Check out our new features.
> .

Is public (y/N):
> n

Is persistable (cannot be dismissed) (y/N):
> n
```

<Callout type="tip" title="Apa arti opsi-opsi ini?">
  - **Tampilkan Dari/Hingga**: Kapan banner harus muncul dan menghilang. Biarkan kosong untuk "sekarang" dan "selamanya"
  - **Is public**: Tampilkan kepada pengunjung yang belum masuk (seperti pada halaman masuk)
  - **Is persistable**: Jika ya, pengguna tidak dapat menutup banner — gunakan untuk pemberitahuan penting
</Callout>

---

## Menghapus Banner [#deleting-a-banner]

```bash
npm run delete-banner
```

Anda akan melihat banner saat ini dan diminta untuk melakukan konfirmasi sebelum menghapus.

## Format Tanggal [#date-format]

Gunakan format ini untuk tanggal: `yyyy-mm-ddTHH:MM:SSZ`

**Contoh:**
- `2025-12-25T09:00:00Z` → 25 Desember 2025 pukul 09.00 (UTC)
- `2025-01-01T00:00:00Z` → 1 Januari 2025 tengah malam (UTC)

Biarkan tanggal kosong untuk menggunakan waktu saat ini atau tanpa tanggal berakhir.
