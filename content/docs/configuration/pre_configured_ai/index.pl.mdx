Ta sekcja zawiera szczegółowe przewodniki konfiguracyjne, które pomogą Ci skonfigurować różnych dostawców AI oraz ich odpowiednie API i dane uwierzytelniające w LibreChat.

## Konfiguracja endpointów [#endpoints-configuration]

Termin "Endpoints" odnosi się do dostawcy AI, konfiguracji lub API, które należy skonfigurować i zintegrować z LibreChat. Każdy endpoint posiada własny proces konfiguracji, który może obejmować uzyskanie kluczy API, danych uwierzytelniających lub postępowanie zgodnie ze szczegółowymi instrukcjami konfiguracji.

Poniższe przewodniki pomogą Ci skonfigurować różne endpoint:

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock)**
    - Konfiguracja integracji AWS Bedrock
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic)**
    - Zintegruj modele Anthropic AI
- **[OpenAI](/docs/configuration/pre_configured_ai/openai)**
    - Skonfiguruj integrację z OpenAI API
- **[Google](/docs/configuration/pre_configured_ai/google)**
    - Skonfiguruj usługi Google AI
- **[Asystenci](/docs/configuration/pre_configured_ai/assistants)**
    - Włącz i skonfiguruj asystentów OpenAI

## Konfiguracja niestandardowego endpoint [#custom-endpoint-configuration]

**[Przewodniki konfiguracji librechat.yaml](/docs/configuration/librechat_yaml)** zawierają szczegółowe instrukcje dotyczące konfigurowania niestandardowych endpoint wewnątrz LibreChat.

Oprócz wstępnie skonfigurowanych endpoint, plik konfiguracyjny librechat pozwala na dodawanie i konfigurowanie własnych endpoint. Obejmuje to integrację z dostawcami AI, takimi jak Ollama, Mistral AI, OpenRouter oraz wieloma innymi usługami zewnętrznymi.

Postępując zgodnie z tymi przewodnikami konfiguracji, możesz bezproblemowo zintegrować różnych dostawców AI, odblokować ich możliwości i wzbogacić swoje doświadczenie z LibreChat dzięki mocy wielu modeli i usług AI.
