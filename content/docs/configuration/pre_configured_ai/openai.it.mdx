Per ottenere la tua API key di OpenAI, devi:

- Vai su **[https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)**
- Crea un account o accedi con quello esistente
- Aggiungi un metodo di pagamento al tuo account
- Vai su https://platform.openai.com/api-keys per ottenere una chiave.
- Dovrai impostare la seguente variabile d'ambiente con la tua chiave, oppure puoi impostarla su `user_provided` per consentire agli utenti di fornire la propria.

```bash filename=".env"
OPENAI_API_KEY=user_provided
```

- Puoi determinare quali modelli desideri avere a disposizione con `OPENAI_MODELS`
  - Quando `OPENAI_API_KEY` è impostato su `user_provided` → saranno disponibili solo i modelli inseriti in questo elenco
    - ⚠️I nuovi modelli non verranno visualizzati automaticamente; dovrai prima aggiungerli a questo elenco
  - Quando `OPENAI_API_KEY` è impostata sul valore effettivo della chiave API → finché `OPENAI_MODELS` viene lasciato commentato, verrà effettuata una chiamata API per scoprire quali modelli sono disponibili, il che dovrebbe includere tutti quelli nuovi.

```bash filename=".env"
OPENAI_MODELS=gpt-5,gpt-5-codex,gpt-5-mini,gpt-5-nano,o3-pro,o3,o4-mini,gpt-4.1,gpt-4.1-mini,gpt-4.1-nano,o3-mini,o1-pro,o1,gpt-4o,gpt-4o-mini
```

**Note:**

- I valori predefiniti interni di OpenAI includono l'attuale catalogo di frontiera, tra cui `gpt-5.5`, `gpt-5.5-pro`, `chat-latest`, `gpt-5.4`, `gpt-5.4-pro`, `gpt-5.4-mini`, `gpt-5.4-nano`, `gpt-5.3-codex` e `gpt-5.2`. I valori predefiniti legacy deprecati non sono più inclusi a meno che non vengano aggiunti esplicitamente con `OPENAI_MODELS`.
- Non è necessario selezionare un modello vision per i messaggi con allegati, poiché verrà sostituito automaticamente per te. Se non hai selezionato esplicitamente un modello vision, questo verrà utilizzato solo per la richiesta vision e dovresti comunque vedere il modello non-vision che avevi selezionato dopo che la richiesta è stata completata con successo.
- I modelli OpenAI Vision consentono messaggi senza allegati
- L'endpoint OpenAI supporta tutte le [Shared Endpoint Settings](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings) tramite il file di configurazione `librechat.yaml`, inclusi `streamRate`, `headers`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate` e `titleEndpoint`
