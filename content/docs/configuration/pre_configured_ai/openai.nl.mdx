Om je OpenAI API key te verkrijgen, moet je:

- Ga naar **[https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)**
- Maak een account aan of log in met je bestaande account
- Voeg een betaalmethode toe aan je account
- Ga naar https://platform.openai.com/api-keys om een sleutel te verkrijgen.
- Je zult de volgende omgevingsvariabele moeten instellen op jouw sleutel, of je kunt deze instellen op `user_provided` zodat gebruikers hun eigen sleutel kunnen opgeven.

```bash filename=".env"
OPENAI_API_KEY=user_provided
```

- Je kunt bepalen welke modellen je beschikbaar wilt hebben met `OPENAI_MODELS`
  - Wanneer `OPENAI_API_KEY` is ingesteld op `user_provided` → zullen alleen de modellen in deze lijst beschikbaar zijn
    - ⚠️Nieuwe modellen verschijnen niet automatisch; je moet ze eerst aan deze lijst toevoegen
  - Wanneer `OPENAI_API_KEY` is ingesteld op de daadwerkelijke API-sleutelwaarde → zolang `OPENAI_MODELS` uitgeschakeld (gecommentarieerd) blijft, zal het een API-aanroep doen om te achterhalen welke modellen beschikbaar zijn, wat ook alle nieuwe modellen zou moeten bevatten.

```bash filename=".env"
OPENAI_MODELS=gpt-5,gpt-5-codex,gpt-5-mini,gpt-5-nano,o3-pro,o3,o4-mini,gpt-4.1,gpt-4.1-mini,gpt-4.1-nano,o3-mini,o1-pro,o1,gpt-4o,gpt-4o-mini
```

**Opmerkingen:**

- Interne OpenAI-standaarden bevatten de huidige frontier-catalogus, inclusief `gpt-5.5`, `gpt-5.5-pro`, `chat-latest`, `gpt-5.4`, `gpt-5.4-pro`, `gpt-5.4-mini`, `gpt-5.4-nano`, `gpt-5.3-codex` en `gpt-5.2`. Verouderde legacy-standaarden zijn niet langer inbegrepen, tenzij je ze expliciet toevoegt met `OPENAI_MODELS`.
- Het selecteren van een vision-model voor berichten met bijlagen is niet nodig, aangezien dit achter de schermen voor je wordt gewisseld. Als je niet expliciet een vision-model hebt geselecteerd, wordt dit alleen gebruikt voor het vision-verzoek en zou je na een succesvol verzoek nog steeds het niet-vision-model moeten zien dat je had geselecteerd.
- OpenAI Vision-modellen staan berichten zonder bijlagen toe
- Het OpenAI endpoint ondersteunt alle [Shared Endpoint Settings](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings) via het `librechat.yaml` configuratiebestand, inclusief `streamRate`, `headers`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate` en `titleEndpoint`
