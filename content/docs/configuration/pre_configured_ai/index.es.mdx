Esta sección proporciona guías de configuración detalladas para ayudarle a configurar varios proveedores de IA y sus respectivas APIs y credenciales en LibreChat.

## Configuración de Endpoints [#endpoints-configuration]

El término "Endpoints" se refiere al proveedor de IA, configuración o API que necesitas configurar e integrar con LibreChat. Cada endpoint tiene su propio proceso de configuración, el cual puede implicar la obtención de claves de API, credenciales o seguir instrucciones de configuración específicas.

Las siguientes guías están disponibles para ayudarle a configurar diferentes endpoint:

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock)**
    - Configurar la integración de AWS Bedrock
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic)**
    - Integrar modelos de Anthropic AI
- **[OpenAI](/docs/configuration/pre_configured_ai/openai)**
    - Configurar la integración de la API de OpenAI
- **[Google](/docs/configuration/pre_configured_ai/google)**
    - Configurar los servicios de Google AI
- **[Assistants](/docs/configuration/pre_configured_ai/assistants)**
    - Habilitar y configurar los Assistants de OpenAI

## Configuración de endpoint personalizado [#custom-endpoint-configuration]

Las **[Guías de configuración de librechat.yaml](/docs/configuration/librechat_yaml)** proporcionan instrucciones detalladas sobre cómo configurar endpoints personalizados dentro de LibreChat.

Además de los endpoints preconfigurados, el archivo de configuración librechat te permite añadir y configurar endpoints personalizados. Esto incluye la integración con proveedores de IA como Ollama, Mistral AI, OpenRouter y una multitud de otros servicios de terceros.

Al seguir estas guías de configuración, puede integrar sin problemas varios proveedores de IA, desbloquear sus capacidades y mejorar su experiencia en LibreChat con el poder de múltiples modelos y servicios de IA.
