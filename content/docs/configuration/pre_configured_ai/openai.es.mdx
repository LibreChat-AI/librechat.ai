Para obtener tu API key de OpenAI, necesitas:

- Ve a **[https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)**
- Crea una cuenta o inicia sesión con la que ya tienes
- Agregue un método de pago a su cuenta
- Ve a https://platform.openai.com/api-keys para obtener una clave.
- Deberá configurar la siguiente variable de entorno con su clave, o puede establecerla en `user_provided` para que los usuarios proporcionen la suya propia.

```bash filename=".env"
OPENAI_API_KEY=user_provided
```

- Puedes determinar qué modelos te gustaría tener disponibles con `OPENAI_MODELS`
  - Cuando `OPENAI_API_KEY` se establece en `user_provided` → solo estarán disponibles los modelos incluidos en esta lista
    - ⚠️Los nuevos modelos no aparecerán automáticamente; primero deberás agregarlos a esta lista
  - Cuando `OPENAI_API_KEY` se establece con el valor real de la clave API → siempre que `OPENAI_MODELS` se deje comentado, realizará una llamada a la API para averiguar qué modelos están disponibles, lo cual debería incluir cualquiera nuevo.

```bash filename=".env"
OPENAI_MODELS=gpt-5,gpt-5-codex,gpt-5-mini,gpt-5-nano,o3-pro,o3,o4-mini,gpt-4.1,gpt-4.1-mini,gpt-4.1-nano,o3-mini,o1-pro,o1,gpt-4o,gpt-4o-mini
```

**Notas:**

- Los valores predeterminados internos de OpenAI incluyen el catálogo actual de vanguardia, incluyendo `gpt-5.5`, `gpt-5.5-pro`, `chat-latest`, `gpt-5.4`, `gpt-5.4-pro`, `gpt-5.4-mini`, `gpt-5.4-nano`, `gpt-5.3-codex` y `gpt-5.2`. Los valores predeterminados heredados y obsoletos ya no se incluyen a menos que los añada explícitamente con `OPENAI_MODELS`.
- No es necesario seleccionar un modelo de visión para los mensajes con archivos adjuntos, ya que se cambiará automáticamente en segundo plano. Si no seleccionaste explícitamente un modelo de visión, este solo se utilizará para la solicitud de visión y deberías seguir viendo el modelo sin visión que habías seleccionado una vez que la solicitud se complete con éxito.
- Los modelos Vision de OpenAI permiten mensajes sin archivos adjuntos
- El endpoint de OpenAI admite todas las [Shared Endpoint Settings](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings) a través del archivo de configuración `librechat.yaml`, incluyendo `streamRate`, `headers`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate` y `titleEndpoint`.
