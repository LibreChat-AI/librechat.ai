Questa sezione fornisce guide di configurazione dettagliate per aiutarti a impostare vari provider di IA e le loro rispettive API e credenziali in LibreChat.

## Configurazione degli endpoint [#endpoints-configuration]

Il termine "Endpoints" si riferisce al provider di IA, alla configurazione o all'API che è necessario impostare e integrare con LibreChat. Ogni endpoint ha il proprio processo di configurazione, che può comportare l'ottenimento di chiavi API, credenziali o il seguire specifiche istruzioni di configurazione.

Le seguenti guide sono disponibili per aiutarti a configurare diversi endpoint:

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock)**
    - Configurazione dell'integrazione AWS Bedrock
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic)**
    - Integra i modelli AI di Anthropic
- **[OpenAI](/docs/configuration/pre_configured_ai/openai)**
    - Configura l'integrazione dell'API OpenAI
- **[Google](/docs/configuration/pre_configured_ai/google)**
    - Configura i servizi Google AI
- **[Assistants](/docs/configuration/pre_configured_ai/assistants)**
    - Abilita e configura gli Assistants di OpenAI

## Configurazione endpoint personalizzato [#custom-endpoint-configuration]

Le **[Guide di configurazione di librechat.yaml](/docs/configuration/librechat_yaml)** forniscono istruzioni dettagliate su come configurare endpoint personalizzati all'interno di LibreChat.

Oltre agli endpoint preconfigurati, il file di configurazione librechat ti consente di aggiungere e configurare endpoint personalizzati. Ciò include l'integrazione con provider di IA come Ollama, Mistral AI, OpenRouter e una moltitudine di altri servizi di terze parti.

Seguendo queste guide di configurazione, puoi integrare perfettamente vari provider di IA, sbloccare le loro funzionalità e migliorare la tua esperienza con LibreChat grazie alla potenza di molteplici modelli e servizi di IA.
