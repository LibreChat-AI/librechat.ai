이 섹션에서는 LibreChat에서 다양한 AI 제공업체와 각 API 및 자격 증명을 설정하는 데 도움이 되는 상세한 구성 가이드를 제공합니다.

## Endpoints Configuration [#endpoints-configuration]

"Endpoints"라는 용어는 LibreChat과 통합하기 위해 설정해야 하는 AI 제공업체, 구성 또는 API를 의미합니다. 각 endpoint는 API 키나 자격 증명을 획득하거나 특정 설정 지침을 따르는 등의 고유한 구성 과정을 거칩니다.

다음 가이드는 다양한 endpoint를 구성하는 데 도움이 되도록 제공됩니다:

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock)**
    - AWS Bedrock 통합 설정
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic)**
    - Anthropic AI 모델 통합
- **[OpenAI](/docs/configuration/pre_configured_ai/openai)**
    - OpenAI API 통합 설정
- **[Google](/docs/configuration/pre_configured_ai/google)**
    - Google AI 서비스 구성
- **[Assistants](/docs/configuration/pre_configured_ai/assistants)**
    - OpenAI의 Assistants 활성화 및 구성

## 사용자 지정 endpoint 구성 [#custom-endpoint-configuration]

**[librechat.yaml 구성 가이드](/docs/configuration/librechat_yaml)**는 LibreChat 내에서 사용자 지정 endpoint를 구성하는 방법에 대한 자세한 지침을 제공합니다.

사전 구성된 endpoint 외에도, librechat.yaml 설정 파일을 통해 사용자 지정 endpoint를 추가하고 구성할 수 있습니다. 여기에는 Ollama, Mistral AI, OpenRouter와 같은 AI 제공업체 및 기타 수많은 타사 서비스와의 통합이 포함됩니다.

이 구성 가이드를 따라 다양한 AI 제공업체를 원활하게 통합하고, 해당 기능을 잠금 해제하며, 여러 AI 모델 및 서비스의 강력한 성능으로 LibreChat 경험을 향상할 수 있습니다.
