本节提供了详细的配置指南，旨在帮助您在 LibreChat 中设置各种 AI 提供商及其各自的 API 和凭据。

## Endpoints Configuration [#endpoints-configuration]

“Endpoints”一词指的是您需要设置并与 LibreChat 集成的 AI 提供商、配置或 API。每个 endpoint 都有其各自的配置流程，其中可能涉及获取 API 密钥、凭据或遵循特定的设置说明。

以下指南可帮助您配置不同的 endpoint：

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock)**
    - 设置 AWS Bedrock 集成
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic)**
    - 集成 Anthropic AI 模型
- **[OpenAI](/docs/configuration/pre_configured_ai/openai)**
    - 设置 OpenAI API 集成
- **[Google](/docs/configuration/pre_configured_ai/google)**
    - 配置 Google AI 服务
- **[Assistants](/docs/configuration/pre_configured_ai/assistants)**
    - 启用并配置 OpenAI 的 Assistants

## 自定义 endpoint 配置 [#custom-endpoint-configuration]

**[librechat.yaml 配置指南](/docs/configuration/librechat_yaml)** 提供了关于如何在 LibreChat 中配置自定义 endpoint 的详细说明。

除了预配置的 endpoint 外，librechat.yaml 配置文件还允许您添加和配置自定义 endpoint。这包括与 Ollama、Mistral AI、OpenRouter 以及众多其他第三方服务等 AI 提供商进行集成。

通过遵循这些配置指南，您可以无缝集成各种 AI 提供商，解锁其功能，并利用多种 AI 模型和服务来增强您的 LibreChat 体验。
