Um Ihren OpenAI API-Schlüssel zu erhalten, müssen Sie:

- Gehen Sie zu **[https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)**
- Erstellen Sie ein Konto oder melden Sie sich mit Ihrem bestehenden Konto an
- Fügen Sie Ihrem Konto eine Zahlungsmethode hinzu
- Gehe zu https://platform.openai.com/api-keys, um einen Schlüssel zu erhalten.
- Sie müssen die folgende Umgebungsvariable auf Ihren Schlüssel setzen, oder Sie können sie auf `user_provided` setzen, damit Benutzer ihre eigenen Schlüssel angeben können.

```bash filename=".env"
OPENAI_API_KEY=user_provided
```

- Sie können mit `OPENAI_MODELS` festlegen, welche Modelle verfügbar sein sollen.
  - Wenn `OPENAI_API_KEY` auf `user_provided` gesetzt ist → sind nur die in dieser Liste aufgeführten Modelle verfügbar
    - ⚠️Neue Modelle erscheinen nicht automatisch; du musst sie zuerst zu dieser Liste hinzufügen
  - Wenn `OPENAI_API_KEY` auf den tatsächlichen API-Schlüsselwert gesetzt ist → solange `OPENAI_MODELS` auskommentiert bleibt, wird ein API-Aufruf getätigt, um herauszufinden, welche Modelle verfügbar sind, was auch alle neuen Modelle einschließen sollte.

```bash filename=".env"
OPENAI_MODELS=gpt-5,gpt-5-codex,gpt-5-mini,gpt-5-nano,o3-pro,o3,o4-mini,gpt-4.1,gpt-4.1-mini,gpt-4.1-nano,o3-mini,o1-pro,o1,gpt-4o,gpt-4o-mini
```

**Hinweise:**

- Interne OpenAI-Standardwerte umfassen den aktuellen Frontier-Katalog, einschließlich `gpt-5.5`, `gpt-5.5-pro`, `chat-latest`, `gpt-5.4`, `gpt-5.4-pro`, `gpt-5.4-mini`, `gpt-5.4-nano`, `gpt-5.3-codex` und `gpt-5.2`. Veraltete Legacy-Standardwerte sind nicht mehr enthalten, es sei denn, Sie fügen sie explizit mit `OPENAI_MODELS` hinzu.
- Die Auswahl eines Vision-Modells für Nachrichten mit Anhängen ist nicht erforderlich, da dies im Hintergrund automatisch für Sie erledigt wird. Wenn Sie nicht explizit ein Vision-Modell ausgewählt haben, wird es nur für die Vision-Anfrage verwendet, und Sie sollten nach erfolgreichem Abschluss der Anfrage weiterhin das von Ihnen gewählte Nicht-Vision-Modell sehen.
- OpenAI Vision-Modelle erlauben Nachrichten ohne Anhänge
- Der OpenAI endpoint unterstützt alle [Shared Endpoint Settings](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings) über die Konfigurationsdatei `librechat.yaml`, einschließlich `streamRate`, `headers`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate` und `titleEndpoint`.
