Bagian ini menyediakan panduan konfigurasi terperinci untuk membantu Anda menyiapkan berbagai penyedia AI beserta API dan kredensial masing-masing di LibreChat.

## Konfigurasi Endpoints [#endpoints-configuration]

Istilah "Endpoints" merujuk pada penyedia AI, konfigurasi, atau API yang perlu Anda siapkan dan integrasikan dengan LibreChat. Setiap endpoint memiliki proses konfigurasinya sendiri, yang mungkin melibatkan perolehan kunci API, kredensial, atau mengikuti instruksi pengaturan tertentu.

Panduan berikut tersedia untuk membantu Anda mengonfigurasi berbagai endpoint:

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock)**
    - Menyiapkan integrasi AWS Bedrock
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic)**
    - Integrasikan model Anthropic AI
- **[OpenAI](/docs/configuration/pre_configured_ai/openai)**
    - Siapkan integrasi OpenAI API
- **[Google](/docs/configuration/pre_configured_ai/google)**
    - Konfigurasikan layanan Google AI
- **[Assistants](/docs/configuration/pre_configured_ai/assistants)**
    - Aktifkan dan konfigurasikan Assistants OpenAI

## Konfigurasi Endpoint Kustom [#custom-endpoint-configuration]

**[Panduan Konfigurasi librechat.yaml](/docs/configuration/librechat_yaml)** menyediakan instruksi mendetail tentang cara mengonfigurasi endpoint kustom di dalam LibreChat.

Selain endpoint yang telah dikonfigurasi sebelumnya, file konfigurasi librechat memungkinkan Anda untuk menambahkan dan mengonfigurasi endpoint kustom. Ini termasuk integrasi dengan penyedia AI seperti Ollama, Mistral AI, OpenRouter, dan berbagai layanan pihak ketiga lainnya.

Dengan mengikuti panduan konfigurasi ini, Anda dapat mengintegrasikan berbagai penyedia AI dengan lancar, membuka kapabilitas mereka, serta meningkatkan pengalaman LibreChat Anda dengan kekuatan dari berbagai model dan layanan AI.
