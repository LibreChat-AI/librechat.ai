Phần này cung cấp các hướng dẫn cấu hình chi tiết để giúp bạn thiết lập nhiều nhà cung cấp AI khác nhau cùng các API và thông tin xác thực tương ứng của họ trong LibreChat.

## Cấu hình Endpoints [#endpoints-configuration]

Thuật ngữ "Endpoints" đề cập đến nhà cung cấp AI, cấu hình hoặc API mà bạn cần thiết lập và tích hợp với LibreChat. Mỗi endpoint có quy trình cấu hình riêng, có thể bao gồm việc lấy khóa API, thông tin xác thực hoặc tuân theo các hướng dẫn thiết lập cụ thể.

Các hướng dẫn sau đây có sẵn để giúp bạn cấu hình các endpoint khác nhau:

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock)**
    - Thiết lập tích hợp AWS Bedrock
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic)**
    - Tích hợp các mô hình Anthropic AI
- **[OpenAI](/docs/configuration/pre_configured_ai/openai)**
    - Thiết lập tích hợp OpenAI API
- **[Google](/docs/configuration/pre_configured_ai/google)**
    - Cấu hình các dịch vụ Google AI
- **[Assistants](/docs/configuration/pre_configured_ai/assistants)**
    - Bật và cấu hình Assistants của OpenAI

## Cấu hình endpoint tùy chỉnh [#custom-endpoint-configuration]

**[Hướng dẫn cấu hình librechat.yaml](/docs/configuration/librechat_yaml)** cung cấp các chỉ dẫn chi tiết về cách cấu hình các endpoint tùy chỉnh trong LibreChat.

Ngoài các endpoint được cấu hình sẵn, tệp cấu hình librechat cho phép bạn thêm và cấu hình các endpoint tùy chỉnh. Điều này bao gồm việc tích hợp với các nhà cung cấp AI như Ollama, Mistral AI, OpenRouter và vô số dịch vụ bên thứ ba khác.

Bằng cách làm theo các hướng dẫn cấu hình này, bạn có thể tích hợp liền mạch nhiều nhà cung cấp AI khác nhau, mở khóa các khả năng của họ và nâng cao trải nghiệm LibreChat của bạn với sức mạnh của nhiều mô hình và dịch vụ AI.
