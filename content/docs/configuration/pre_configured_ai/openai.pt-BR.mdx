Para obter sua chave de API da OpenAI, você precisa:

- Vá para **[https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)**
- Crie uma conta ou faça login com a sua já existente
- Adicione uma forma de pagamento à sua conta
- Vá para https://platform.openai.com/api-keys para obter uma chave.
- Você precisará definir a seguinte variável de ambiente com sua chave, ou você pode defini-la como `user_provided` para que os usuários forneçam as suas próprias.

```bash filename=".env"
OPENAI_API_KEY=user_provided
```

- Você pode determinar quais modelos deseja disponibilizar com `OPENAI_MODELS`
  - Quando `OPENAI_API_KEY` estiver definido como `user_provided` → apenas os modelos colocados nesta lista estarão disponíveis
    - ⚠️Novos modelos não aparecerão automaticamente; você precisará adicioná-los a esta lista primeiro
  - Quando `OPENAI_API_KEY` é definida com o valor real da chave de API → desde que `OPENAI_MODELS` seja deixada comentada, ela fará uma chamada de API para descobrir quais modelos estão disponíveis, o que deve incluir quaisquer modelos novos.

```bash filename=".env"
OPENAI_MODELS=gpt-5,gpt-5-codex,gpt-5-mini,gpt-5-nano,o3-pro,o3,o4-mini,gpt-4.1,gpt-4.1-mini,gpt-4.1-nano,o3-mini,o1-pro,o1,gpt-4o,gpt-4o-mini
```

**Notas:**

- Os padrões internos da OpenAI incluem o catálogo de fronteira atual, incluindo `gpt-5.5`, `gpt-5.5-pro`, `chat-latest`, `gpt-5.4`, `gpt-5.4-pro`, `gpt-5.4-mini`, `gpt-5.4-nano`, `gpt-5.3-codex` e `gpt-5.2`. Padrões legados obsoletos não estão mais incluídos, a menos que você os adicione explicitamente com `OPENAI_MODELS`.
- Não é necessário selecionar um modelo de visão para mensagens com anexos, pois ele será alternado automaticamente nos bastidores para você. Se você não selecionou explicitamente um modelo de visão, ele será usado apenas para a solicitação de visão e você ainda deverá ver o modelo sem visão que havia selecionado após a conclusão bem-sucedida da solicitação.
- Os modelos Vision da OpenAI permitem mensagens sem anexos
- O endpoint OpenAI suporta todas as [Configurações de Endpoint Compartilhadas](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings) via o arquivo de configuração `librechat.yaml`, incluindo `streamRate`, `headers`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate` e `titleEndpoint`
