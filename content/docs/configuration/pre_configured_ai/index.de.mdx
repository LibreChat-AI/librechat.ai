Dieser Abschnitt enthält detaillierte Konfigurationsanleitungen, die Ihnen bei der Einrichtung verschiedener KI-Anbieter sowie deren APIs und Zugangsdaten in LibreChat helfen.

## Endpoints-Konfiguration [#endpoints-configuration]

Der Begriff „Endpoints“ bezieht sich auf den KI-Anbieter, die Konfiguration oder die API, die Sie einrichten und in LibreChat integrieren müssen. Jeder Endpoint hat seinen eigenen Konfigurationsprozess, der das Einholen von API-Schlüsseln, Anmeldeinformationen oder das Befolgen spezifischer Einrichtungsanweisungen beinhalten kann.

Die folgenden Anleitungen stehen zur Verfügung, um Ihnen bei der Konfiguration verschiedener endpoints zu helfen:

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock)**
    - AWS Bedrock Integration einrichten
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic)**
    - Integrieren Sie Anthropic AI-Modelle
- **[OpenAI](/docs/configuration/pre_configured_ai/openai)**
    - OpenAI API-Integration einrichten
- **[Google](/docs/configuration/pre_configured_ai/google)**
    - Google AI-Dienste konfigurieren
- **[Assistants](/docs/configuration/pre_configured_ai/assistants)**
    - OpenAI Assistants aktivieren und konfigurieren

## Benutzerdefinierte Endpoint-Konfiguration [#custom-endpoint-configuration]

Die **[librechat.yaml Konfigurationsanleitungen](/docs/configuration/librechat_yaml)** bieten detaillierte Anweisungen zur Konfiguration benutzerdefinierter endpoints innerhalb von LibreChat.

Zusätzlich zu den vorkonfigurierten endpoints ermöglicht Ihnen die librechat Konfigurationsdatei das Hinzufügen und Konfigurieren benutzerdefinierter endpoints. Dies umfasst die Integration von KI-Anbietern wie Ollama, Mistral AI, OpenRouter und einer Vielzahl anderer Drittanbieterdienste.

Indem Sie diesen Konfigurationsanleitungen folgen, können Sie verschiedene KI-Anbieter nahtlos integrieren, deren Funktionen freischalten und Ihre LibreChat-Erfahrung mit der Leistungsfähigkeit mehrerer KI-Modelle und -Dienste erweitern.
