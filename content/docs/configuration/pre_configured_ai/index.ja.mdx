このセクションでは、LibreChatでさまざまなAIプロバイダーとそのAPIおよび認証情報を設定するための詳細な構成ガイドを提供します。

## エンドポイントの設定 [#endpoints-configuration]

「Endpoints」という用語は、LibreChatと統合するために設定が必要なAIプロバイダー、構成、またはAPIを指します。各endpointには独自の構成プロセスがあり、APIキーや認証情報の取得、あるいは特定のセットアップ手順に従う必要があります。

以下のガイドは、さまざまな endpoint を設定するのに役立ちます。

- **[AWS Bedrock](/docs/configuration/pre_configured_ai/bedrock)**
    - AWS Bedrock 統合のセットアップ
- **[Anthropic](/docs/configuration/pre_configured_ai/anthropic)**
    - Anthropic AIモデルを統合する
- **[OpenAI](/docs/configuration/pre_configured_ai/openai)**
    - OpenAI API統合のセットアップ
- **[Google](/docs/configuration/pre_configured_ai/google)**
    - Google AI サービスを設定する
- **[Assistants](/docs/configuration/pre_configured_ai/assistants)**
    - OpenAIのAssistantsを有効化および設定する

## カスタム endpoint 設定 [#custom-endpoint-configuration]

**[librechat.yaml 設定ガイド](/docs/configuration/librechat_yaml)**では、LibreChat内でカスタムendpointを設定する方法について詳細な手順を説明しています。

事前設定済みのendpointに加え、librechat.yaml設定ファイルを使用することで、カスタムendpointを追加および設定できます。これには、Ollama、Mistral AI、OpenRouterなどのAIプロバイダーや、その他多数のサードパーティサービスとの統合が含まれます。

これらの設定ガイドに従うことで、さまざまなAIプロバイダーをシームレスに統合し、その機能を最大限に活用して、複数のAIモデルとサービスの力でLibreChatの体験を向上させることができます。
