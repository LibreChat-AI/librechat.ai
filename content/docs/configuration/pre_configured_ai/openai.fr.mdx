Pour obtenir votre clé API OpenAI, vous devez :

- Allez sur **[https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)**
- Créez un compte ou connectez-vous avec votre compte existant
- Ajouter un moyen de paiement à votre compte
- Allez sur https://platform.openai.com/api-keys pour obtenir une clé.
- Vous devrez définir la variable d'environnement suivante avec votre clé, ou vous pouvez la définir sur `user_provided` pour permettre aux utilisateurs de fournir la leur.

```bash filename=".env"
OPENAI_API_KEY=user_provided
```

- Vous pouvez déterminer quels modèles vous souhaitez rendre disponibles avec `OPENAI_MODELS`
  - Lorsque `OPENAI_API_KEY` est défini sur `user_provided` → seuls les modèles inclus dans cette liste seront disponibles
    - ⚠️Les nouveaux modèles n'apparaîtront pas automatiquement ; vous devrez d'abord les ajouter à cette liste
  - Lorsque `OPENAI_API_KEY` est défini avec la valeur réelle de la clé API → tant que `OPENAI_MODELS` reste commenté, un appel API sera effectué pour découvrir quels modèles sont disponibles, ce qui devrait inclure tous les nouveaux modèles.

```bash filename=".env"
OPENAI_MODELS=gpt-5,gpt-5-codex,gpt-5-mini,gpt-5-nano,o3-pro,o3,o4-mini,gpt-4.1,gpt-4.1-mini,gpt-4.1-nano,o3-mini,o1-pro,o1,gpt-4o,gpt-4o-mini
```

**Notes :**

- Les valeurs par défaut internes d'OpenAI incluent le catalogue de pointe actuel, notamment `gpt-5.5`, `gpt-5.5-pro`, `chat-latest`, `gpt-5.4`, `gpt-5.4-pro`, `gpt-5.4-mini`, `gpt-5.4-nano`, `gpt-5.3-codex` et `gpt-5.2`. Les valeurs par défaut héritées et obsolètes ne sont plus incluses, sauf si vous les ajoutez explicitement avec `OPENAI_MODELS`.
- Il n'est pas nécessaire de sélectionner un modèle de vision pour les messages avec pièces jointes, car le changement s'effectuera automatiquement en arrière-plan. Si vous n'avez pas explicitement sélectionné un modèle de vision, celui-ci sera utilisé uniquement pour la requête de vision et vous devriez toujours voir le modèle sans vision que vous aviez sélectionné une fois la requête terminée avec succès.
- Les modèles OpenAI Vision permettent des messages sans pièces jointes
- L'endpoint OpenAI prend en charge tous les [Shared Endpoint Settings](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings) via le fichier de configuration `librechat.yaml`, y compris `streamRate`, `headers`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate` et `titleEndpoint`.
