Untuk mendapatkan API key OpenAI Anda, Anda perlu:

- Buka **[https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)**
- Buat akun atau masuk dengan akun yang sudah Anda miliki
- Tambahkan metode pembayaran ke akun Anda
- Buka https://platform.openai.com/api-keys untuk mendapatkan kunci.
- Anda perlu mengatur variabel lingkungan berikut ke kunci Anda, atau Anda dapat mengaturnya ke `user_provided` agar pengguna dapat menyediakan kunci mereka sendiri.

```bash filename=".env"
OPENAI_API_KEY=user_provided
```

- Anda dapat menentukan model mana yang ingin Anda sediakan dengan `OPENAI_MODELS`
  - Ketika `OPENAI_API_KEY` diatur ke `user_provided` → hanya model yang dimasukkan ke dalam daftar ini yang akan tersedia
    - ⚠️Model baru tidak akan muncul secara otomatis; Anda harus menambahkannya ke daftar ini terlebih dahulu
  - Ketika `OPENAI_API_KEY` diatur ke nilai kunci API yang sebenarnya → selama `OPENAI_MODELS` dibiarkan dalam keadaan dikomentari (commented-out), sistem akan melakukan panggilan API untuk mengetahui model apa saja yang tersedia, yang seharusnya mencakup model baru apa pun.

```bash filename=".env"
OPENAI_MODELS=gpt-5,gpt-5-codex,gpt-5-mini,gpt-5-nano,o3-pro,o3,o4-mini,gpt-4.1,gpt-4.1-mini,gpt-4.1-nano,o3-mini,o1-pro,o1,gpt-4o,gpt-4o-mini
```

**Catatan:**

- Default OpenAI internal mencakup katalog frontier saat ini, termasuk `gpt-5.5`, `gpt-5.5-pro`, `chat-latest`, `gpt-5.4`, `gpt-5.4-pro`, `gpt-5.4-mini`, `gpt-5.4-nano`, `gpt-5.3-codex`, dan `gpt-5.2`. Default lama yang sudah tidak digunakan lagi tidak lagi disertakan kecuali Anda menambahkannya secara eksplisit dengan `OPENAI_MODELS`.
- Memilih model vision untuk pesan dengan lampiran tidak diperlukan karena model tersebut akan dialihkan secara otomatis untuk Anda. Jika Anda tidak memilih model vision secara langsung, model tersebut hanya akan digunakan untuk permintaan vision dan Anda tetap akan melihat model non-vision yang telah Anda pilih setelah permintaan berhasil.
- Model OpenAI Vision memungkinkan pesan tanpa lampiran
- endpoint OpenAI mendukung semua [Shared Endpoint Settings](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings) melalui file konfigurasi `librechat.yaml`, termasuk `streamRate`, `headers`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate`, dan `titleEndpoint`
