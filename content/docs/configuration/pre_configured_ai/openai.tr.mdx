OpenAI API anahtarınızı almak için şunları yapmanız gerekir:

- **[https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)** adresine gidin
- Bir hesap oluşturun veya mevcut hesabınızla giriş yapın
- Hesabınıza bir ödeme yöntemi ekleyin
- Bir anahtar almak için https://platform.openai.com/api-keys adresine gidin.
- Aşağıdaki ortam değişkenini anahtarınızla ayarlamanız gerekecek veya kullanıcıların kendi anahtarlarını sağlamalarına izin vermek için bunu `user_provided` olarak ayarlayabilirsiniz.

```bash filename=".env"
OPENAI_API_KEY=user_provided
```

- `OPENAI_MODELS` ile hangi modellerin kullanılabilir olmasını istediğinizi belirleyebilirsiniz.
  - `OPENAI_API_KEY` değeri `user_provided` olarak ayarlandığında → yalnızca bu listeye eklenen modeller kullanılabilir olacaktır
    - ⚠️Yeni modeller otomatik olarak görünmeyecektir; önce onları bu listeye eklemeniz gerekecektir
  - `OPENAI_API_KEY` gerçek API anahtarı değerine ayarlandığında → `OPENAI_MODELS` yorum satırı olarak bırakıldığı sürece, mevcut modelleri öğrenmek için bir API çağrısı yapacaktır; bu da yeni eklenen modelleri de içermelidir.

```bash filename=".env"
OPENAI_MODELS=gpt-5,gpt-5-codex,gpt-5-mini,gpt-5-nano,o3-pro,o3,o4-mini,gpt-4.1,gpt-4.1-mini,gpt-4.1-nano,o3-mini,o1-pro,o1,gpt-4o,gpt-4o-mini
```

**Notlar:**

- Dahili OpenAI varsayılanları, `gpt-5.5`, `gpt-5.5-pro`, `chat-latest`, `gpt-5.4`, `gpt-5.4-pro`, `gpt-5.4-mini`, `gpt-5.4-nano`, `gpt-5.3-codex` ve `gpt-5.2` dahil olmak üzere mevcut öncü kataloğu içerir. Kullanımdan kaldırılan eski varsayılanlar, `OPENAI_MODELS` ile açıkça eklemediğiniz sürece artık dahil edilmemektedir.
- Ekli mesajlar için bir vision modeli seçmeniz gerekmez, çünkü bu işlem arka planda sizin yerinize otomatik olarak gerçekleştirilecektir. Eğer doğrudan bir vision modeli seçmediyseniz, bu model yalnızca vision isteği için kullanılacak ve istek başarıyla tamamlandıktan sonra seçmiş olduğunuz vision olmayan modeli görmeye devam edeceksiniz.
- OpenAI Vision modelleri, ek içermeyen mesajlara izin verir
- OpenAI endpoint, `streamRate`, `headers`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate` ve `titleEndpoint` dahil olmak üzere `librechat.yaml` yapılandırma dosyası aracılığıyla tüm [Shared Endpoint Settings](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings) özelliklerini destekler.
