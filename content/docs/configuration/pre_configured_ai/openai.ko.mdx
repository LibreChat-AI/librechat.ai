OpenAI API 키를 얻으려면 다음을 수행해야 합니다:

- **[https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)**로 이동하세요.
- 계정을 생성하거나 기존 계정으로 로그인하세요
- 계정에 결제 수단을 추가하세요
- https://platform.openai.com/api-keys 로 이동하여 키를 받으세요.
- 다음 환경 변수를 귀하의 키로 설정해야 합니다. 또는 사용자가 직접 자신의 키를 입력하도록 하려면 `user_provided`로 설정할 수 있습니다.

```bash filename=".env"
OPENAI_API_KEY=user_provided
```

- `OPENAI_MODELS`를 사용하여 사용 가능한 모델을 결정할 수 있습니다.
  - `OPENAI_API_KEY`가 `user_provided`로 설정된 경우 → 이 목록에 포함된 모델만 사용할 수 있습니다.
    - ⚠️새로운 모델은 자동으로 표시되지 않습니다. 먼저 이 목록에 추가해야 합니다.
  - `OPENAI_API_KEY`가 실제 API 키 값으로 설정된 경우 → `OPENAI_MODELS`가 주석 처리된 상태로 유지되는 한, 사용 가능한 모델을 확인하기 위해 API 호출을 수행하며, 여기에는 새로 추가된 모델도 포함됩니다.

```bash filename=".env"
OPENAI_MODELS=gpt-5,gpt-5-codex,gpt-5-mini,gpt-5-nano,o3-pro,o3,o4-mini,gpt-4.1,gpt-4.1-mini,gpt-4.1-nano,o3-mini,o1-pro,o1,gpt-4o,gpt-4o-mini
```

**참고:**

- 내부 OpenAI 기본값에는 `gpt-5.5`, `gpt-5.5-pro`, `chat-latest`, `gpt-5.4`, `gpt-5.4-pro`, `gpt-5.4-mini`, `gpt-5.4-nano`, `gpt-5.3-codex` 및 `gpt-5.2`를 포함한 현재 최신 카탈로그가 포함되어 있습니다. 더 이상 사용되지 않는 레거시 기본값은 `OPENAI_MODELS`를 사용하여 명시적으로 추가하지 않는 한 포함되지 않습니다.
- 첨부 파일이 포함된 메시지를 위해 비전 모델을 직접 선택할 필요는 없습니다. 백그라운드에서 자동으로 전환되기 때문입니다. 비전 모델을 명시적으로 선택하지 않은 경우, 해당 모델은 비전 요청에만 사용되며 요청이 성공한 후에는 이전에 선택했던 비전 기능이 없는 모델이 그대로 유지되는 것을 확인할 수 있습니다.
- OpenAI Vision 모델은 첨부 파일이 없는 메시지를 허용합니다
- OpenAI endpoint는 `librechat.yaml` 구성 파일을 통해 `streamRate`, `headers`, `titleModel`, `titleMethod`, `titlePrompt`, `titlePromptTemplate`, `titleEndpoint`를 포함한 모든 [Shared Endpoint Settings](/docs/configuration/librechat_yaml/object_structure/shared_endpoint_settings)을 지원합니다.
