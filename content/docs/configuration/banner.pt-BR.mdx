Exiba mensagens importantes para todos os seus usuários no topo do aplicativo. Perfeito para anúncios, avisos de manutenção ou atualizações.

<Callout type="info" title="Visão Geral Rápida">
  - Apenas **um banner** pode estar ativo por vez
  - Agende banners para aparecer e desaparecer automaticamente
  - Escolha se os usuários podem ou não dispensar o banner
  - Exibir banners para todos ou apenas para usuários logados
</Callout>

---

## Criando um Banner [#creating-a-banner]

Execute este comando para criar ou atualizar um banner:

```bash
npm run update-banner
```

Você será guiado por alguns prompts simples:

```ansi
--------------------------
Update the banner!
--------------------------
Display From (Format: yyyy-mm-ddTHH:MM:SSZ, Default: now):
> 2025-12-02T09:00:00Z

Display To (Format: yyyy-mm-ddTHH:MM:SSZ, Default: not specified):
> 2025-12-31T23:59:59Z

Enter your message (Enter a single dot "." on a new line to finish):
> 🎉 Welcome to LibreChat! Check out our new features.
> .

Is public (y/N):
> n

Is persistable (cannot be dismissed) (y/N):
> n
```

<Callout type="tip" title="O que significam estas opções?">
  - **Exibir De/Até**: Quando o banner deve aparecer e desaparecer. Deixe vazio para "agora" e "para sempre"
  - **Is public**: Mostrar para visitantes que não estão logados (como na página de login)
  - **É persistente**: Se sim, os usuários não podem dispensar o banner — use para avisos importantes
</Callout>

## Excluindo um Banner [#deleting-a-banner]

```bash
npm run delete-banner
```

Você verá o banner atual e será solicitado a confirmar antes de excluir.

## Formato de Data [#date-format]

Use este formato para datas: `yyyy-mm-ddTHH:MM:SSZ`

**Exemplos:**
- `2025-12-25T09:00:00Z` → 25 de dezembro de 2025 às 9:00 (UTC)
- `2025-01-01T00:00:00Z` → 1 de janeiro de 2025 à meia-noite (UTC)

Deixe a data vazia para usar o horário atual ou para não definir uma data de término.
