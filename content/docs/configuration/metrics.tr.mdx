### Genel [#general]

![LibreChat içindeki aktif kullanıcılar](/images/metrics/librechat-metrics-active-users.png)

LibreChat'in aktif kullanımıyla ilgili metrikleri almak için Prometheus veya OpenMetrics uyumlu başka herhangi bir izleme aracını kullanabilirsiniz.
Bu, token kullanımı gibi teknik bilgilerin yanı sıra, toplam veya aktif kullanıcı sayısı gibi son kullanıcılarla ilgili bilgileri de içerir.

LibreChat ayrıca `METRICS_SECRET` yapılandırıldığında `/metrics` adresindeki API sunucusundan yerleşik Prometheus metriklerini dışa aktarır. Bunu `Authorization: Bearer <METRICS_SECRET>` ile kazıyın (scrape). Tarayıcı RUM proxy sonuçları, `endpoint` değerinin `traces`, `logs` veya `unknown` olduğu ve `result` değerinin `success`, `auth_drop`, `auth_error`, `bad_request`, `not_configured`, `collector_4xx`, `collector_5xx`, `collector_error` veya `collector_timeout` olabildiği `rum_proxy_requests_total{endpoint,result}` olarak raporlanır.

Metrik dışa aktarıcı (metrics exporter) [virtUOS/librechat_exporter](https://github.com/virtUOS/librechat_exporter) adresinde mevcuttur.
Bu, LibreChat ile birlikte dağıttığınız ayrı bir araçtır.

### Kurulum [#setup]

Exporter'ı dağıtmak için, gerekli container'ı compose yapılandırmanıza şu şekilde eklemeniz yeterlidir:

```yaml
services:
  metrics:
    image: ghcr.io/virtuos/librechat_exporter:main
    depends_on:
      - mongodb
    ports:
      - '8000:8000'
    restart: unless-stopped
```

İsteğe bağlı olarak dışa aktarıcıyı (exporter) da yapılandırabilirsiniz.
Ancak genellikle varsayılan ayarlar gayet yeterli olacaktır.

```yaml
services:
  metrics:
    environment:
      - MONGODB_URI=mongodb://mongodb:27017/
      - LOGGING_LEVEL=info
```

### Kullanım [#usage]

Artık dışa aktarıcıyı (exporter) Prometheus kazıma (scrape) yapılandırmanıza ekleyebilirsiniz:

```yaml
- job_name: librechat
  static_configs:
    - targets:
        - 'librechat.example.com:8000'
```

Metriklerin kazınması (scraping) başladığında, `librechat_*` metriklerini (örneğin `librechat_registered_users`) arayın.
Dışa aktarıcı (exporter) çeşitli metrikler sağlar.

Grafana panonuzu oluştururken iyi eğlenceler!
