Muestre mensajes importantes a todos sus usuarios en la parte superior de la aplicación. Perfecto para anuncios, avisos de mantenimiento o actualizaciones.

<Callout type="info" title="Resumen rápido">
  - Solo puede haber **un banner** activo a la vez
  - Programar banners para que aparezcan y desaparezcan automáticamente
  - Elija si los usuarios pueden descartar el banner o no
  - Mostrar banners a todos, o solo a los usuarios que han iniciado sesión
</Callout>

---

## Crear un banner [#creating-a-banner]

Ejecuta este comando para crear o actualizar un banner:

```bash
npm run update-banner
```

Se le guiará a través de unas cuantas instrucciones sencillas:

```ansi
--------------------------
Update the banner!
--------------------------
Display From (Format: yyyy-mm-ddTHH:MM:SSZ, Default: now):
> 2025-12-02T09:00:00Z

Display To (Format: yyyy-mm-ddTHH:MM:SSZ, Default: not specified):
> 2025-12-31T23:59:59Z

Enter your message (Enter a single dot "." on a new line to finish):
> 🎉 Welcome to LibreChat! Check out our new features.
> .

Is public (y/N):
> n

Is persistable (cannot be dismissed) (y/N):
> n
```

<Callout type="tip" title="¿Qué significan estas opciones?">
  - **Mostrar desde/hasta**: Cuándo debe aparecer y desaparecer el banner. Déjelo vacío para "ahora" y "siempre"
  - **Es público**: Mostrar a los visitantes que no han iniciado sesión (como en la página de inicio de sesión)
  - **Es persistente**: Si es así, los usuarios no pueden descartar el banner; utilícelo para avisos importantes
</Callout>

---

## Eliminación de un banner [#deleting-a-banner]

```bash
npm run delete-banner
```

Verás el banner actual y se te pedirá que confirmes antes de eliminar.

## Formato de fecha [#date-format]

Utilice este formato para las fechas: `yyyy-mm-ddTHH:MM:SSZ`

**Ejemplos:**
- `2025-12-25T09:00:00Z` → 25 de diciembre de 2025 a las 9:00 AM (UTC)
- `2025-01-01T00:00:00Z` → 1 de enero de 2025 a medianoche (UTC)

Deje la fecha vacía para usar la hora actual o sin fecha de finalización.
