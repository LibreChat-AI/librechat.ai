Zeigen Sie allen Ihren Benutzern wichtige Nachrichten oben in der App an. Perfekt für Ankündigungen, Wartungshinweise oder Updates.

<Callout type="info" title="Kurzer Überblick">
  - Es kann immer nur **ein Banner** gleichzeitig aktiv sein
  - Planen Sie Banner so, dass sie automatisch erscheinen und verschwinden
  - Wählen Sie aus, ob Benutzer das Banner schließen können oder nicht
  - Banner für alle oder nur für angemeldete Benutzer anzeigen
</Callout>

---

## Erstellen eines Banners [#creating-a-banner]

Führen Sie diesen Befehl aus, um ein Banner zu erstellen oder zu aktualisieren:

```bash
npm run update-banner
```

Sie werden durch ein paar einfache Eingabeaufforderungen geführt:

```ansi
--------------------------
Update the banner!
--------------------------
Display From (Format: yyyy-mm-ddTHH:MM:SSZ, Default: now):
> 2025-12-02T09:00:00Z

Display To (Format: yyyy-mm-ddTHH:MM:SSZ, Default: not specified):
> 2025-12-31T23:59:59Z

Enter your message (Enter a single dot "." on a new line to finish):
> 🎉 Welcome to LibreChat! Check out our new features.
> .

Is public (y/N):
> n

Is persistable (cannot be dismissed) (y/N):
> n
```

<Callout type="tip" title="Was bedeuten diese Optionen?">
  - **Anzeige Von/Bis**: Wann das Banner erscheinen und verschwinden soll. Leer lassen für "jetzt" und "für immer"
  - **Ist öffentlich**: Für Besucher sichtbar, die nicht angemeldet sind (wie auf der Anmeldeseite)
  - **Ist persistierbar**: Wenn ja, können Benutzer das Banner nicht schließen — verwenden Sie dies für wichtige Hinweise
</Callout>

---

## Löschen eines Banners [#deleting-a-banner]

```bash
npm run delete-banner
```

Sie sehen das aktuelle Banner und werden vor dem Löschen um eine Bestätigung gebeten.

## Beispiel-Banner [#example-banners]

<Tabs items={['Willkommensnachricht', 'Wartungshinweis', 'Sicherheitswarnung']}>
  <Tabs.Tab>
    Eine einfache Willkommensnachricht, die Benutzer schließen können:

    ```bash
    npm run update-banner "" "" "👋 Welcome to LibreChat!" "false" "false"
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    Ein Hinweis auf geplante Wartungsarbeiten, der nicht ausgeblendet werden kann:

    ```bash
    npm run update-banner "2025-12-20T00:00:00Z" "2025-12-21T06:00:00Z" "⚠️ Scheduled maintenance on Dec 20th, 2-6 AM UTC" "true" "true"
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    Ein dringender Sicherheitshinweis, der für alle sichtbar ist:

    ```bash
    npm run update-banner "" "" "🔒 Please update your password by January 1st" "true" "true"
    ```
  </Tabs.Tab>
</Tabs>

## Datumsformat [#date-format]

Verwenden Sie dieses Format für Daten: `yyyy-mm-ddTHH:MM:SSZ`

**Beispiele:**
- `2025-12-25T09:00:00Z` → 25. Dezember 2025 um 09:00 Uhr (UTC)
- `2025-01-01T00:00:00Z` → 1. Januar 2025 um Mitternacht (UTC)

Lassen Sie das Datum leer, um die aktuelle Zeit oder kein Enddatum zu verwenden.
