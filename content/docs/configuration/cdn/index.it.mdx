<Callout type="info" title="Nota">
  LibreChat supporta l'archiviazione locale, i backend di object storage e la distribuzione tramite CDN. Utilizza un backend di object storage come S3 o Azure Blob Storage per un'archiviazione dei file durevole, quindi aggiungi una CDN come CloudFront quando hai bisogno di link multimediali stabili, edge caching, cookie firmati o URL di download firmati.
</Callout>

## Archiviazione file [#file-storage]

Backend di archiviazione oggetti durevole per caricamenti utente, immagini generate e altri file.

  <Cards.Card icon={<S3Icon />} title="Amazon S3" href="/docs/configuration/cdn/s3" arrow>
    Configura un bucket Amazon S3 come backend di archiviazione oggetti durevole e scalabile.

  <Cards.Card icon={<AzureIcon />} title="Archiviazione Blob Azure" href="/docs/configuration/cdn/azure" arrow>

</Cards>
