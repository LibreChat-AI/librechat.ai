<Callout type="info" title="注記">
  LibreChatは、ローカルストレージ、オブジェクトストレージバックエンド、およびCDNによる配信をサポートしています。耐久性の高いオブジェクトストレージにはS3を使用し、安定したメディアリンク、エッジキャッシング、署名付きCookie、または署名付きダウンロードURLが必要な場合はCloudFrontを追加してください。
</Callout>
