<Callout type="info" title="注意">
  LibreChat 支持本地存储、对象存储后端以及基于 CDN 的分发。使用 S3 进行持久化对象存储，然后在需要稳定的媒体链接、边缘缓存、签名 Cookie 或签名下载 URL 时添加 CloudFront。
</Callout>
