<Callout type="info" title="注意">
  LibreChat 支持本地存储、对象存储后端以及基于 CDN 的分发。请使用 S3 或 Azure Blob Storage 等对象存储后端进行持久化文件存储，然后在需要稳定的媒体链接、边缘缓存、签名 Cookie 或签名下载 URL 时，添加 CloudFront 等 CDN。
</Callout>

## 文件存储 [#file-storage]

用于用户上传、生成图像及其他文件的持久化对象存储后端。

<Cards num={2}>
  <Cards.Card icon={<S3Icon />} title="Amazon S3" href="/docs/configuration/cdn/s3" arrow>
    将 Amazon S3 存储桶设置为持久、可扩展的对象存储后端。
  </Cards.Card>
  <Cards.Card icon={<AzureIcon />} title="Azure Blob Storage" href="/docs/configuration/cdn/azure" arrow>
    使用 Azure Blob Storage 容器作为您的文件存储后端。
  </Cards.Card>
</Cards>

## CDN [#cdn]

用于稳定媒体链接、签名 Cookie 和签名下载 URL 的边缘缓存分发。

<Cards num={2}>
  <Cards.Card icon={<AWSIcon />} title="使用 S3 的 CloudFront" href="/docs/configuration/cdn/cloudfront" arrow>
    通过 CloudFront 为 S3 后端文件提供服务，以实现边缘缓存和签名 URL。
  </Cards.Card>
  <Cards.Card icon={<FirebaseIcon />} title="Firebase CDN" href="/docs/configuration/cdn/firebase" arrow>
    通过由 Google CDN 支持的 Firebase Storage 交付文件。
  </Cards.Card>
</Cards>
