<Callout type="info" title="참고">
  LibreChat은 로컬 스토리지, 객체 스토리지 백엔드 및 CDN 기반 전송을 지원합니다. 내구성 있는 파일 저장을 위해 S3나 Azure Blob Storage와 같은 객체 스토리지 백엔드를 사용하고, 안정적인 미디어 링크, 엣지 캐싱, 서명된 쿠키 또는 서명된 다운로드 URL이 필요한 경우 CloudFront와 같은 CDN을 추가하십시오.
</Callout>

## 파일 저장소 [#file-storage]

사용자 업로드, 생성된 이미지 및 기타 파일을 위한 내구성 있는 객체 스토리지 백엔드입니다.

<Cards num={2}>
  <Cards.Card icon={<S3Icon />} title="Amazon S3" href="/docs/configuration/cdn/s3" arrow>
    내구성이 뛰어나고 확장 가능한 객체 스토리지 백엔드로 Amazon S3 버킷을 설정하세요.
  </Cards.Card>
  <Cards.Card icon={<AzureIcon />} title="Azure Blob Storage" href="/docs/configuration/cdn/azure" arrow>
    Azure Blob Storage 컨테이너를 파일 스토리지 백엔드로 사용하세요.
  </Cards.Card>
</Cards>

## CDN [#cdn]

안정적인 미디어 링크, 서명된 쿠키 및 서명된 다운로드 URL을 위한 엣지 캐시(Edge-cached) 전송.

<Cards num={2}>
  <Cards.Card icon={<AWSIcon />} title="S3를 사용하는 CloudFront" href="/docs/configuration/cdn/cloudfront" arrow>
    엣지 캐싱 및 서명된 URL을 위해 CloudFront를 통해 S3 기반 파일을 제공합니다.
  </Cards.Card>
  <Cards.Card icon={<FirebaseIcon />} title="Firebase CDN" href="/docs/configuration/cdn/firebase" arrow>
    Google의 CDN으로 지원되는 Firebase Storage를 통해 파일을 전달합니다.
  </Cards.Card>
</Cards>
