<Callout type="info" title="Nota">
  O LibreChat oferece suporte a armazenamento local, backends de armazenamento de objetos e entrega via CDN. Utilize um backend de armazenamento de objetos, como S3 ou Azure Blob Storage, para armazenamento de arquivos durável e, em seguida, adicione uma CDN como o CloudFront quando precisar de links de mídia estáveis, cache de borda, cookies assinados ou URLs de download assinadas.
</Callout>

## Armazenamento de Arquivos [#file-storage]

Backends de armazenamento de objetos duráveis para uploads de usuários, imagens geradas e outros arquivos.

<Cards num={2}>
  <Cards.Card icon={<S3Icon />} title="Amazon S3" href="/docs/configuration/cdn/s3" arrow>
    Configure um bucket do Amazon S3 como um backend de armazenamento de objetos durável e escalável.
  </Cards.Card>
  <Cards.Card icon={<AzureIcon />} title="Azure Blob Storage" href="/docs/configuration/cdn/azure" arrow>
    Use contêineres do Azure Blob Storage como seu backend de armazenamento de arquivos.
  </Cards.Card>
</Cards>

## CDN [#cdn]

Entrega com cache na borda para links de mídia estáveis, cookies assinados e URLs de download assinadas.

<Cards num={2}>
  <Cards.Card icon={<AWSIcon />} title="CloudFront com S3" href="/docs/configuration/cdn/cloudfront" arrow>
    Sirva arquivos hospedados no S3 através do CloudFront para cache na borda e URLs assinadas.
  </Cards.Card>
  <Cards.Card icon={<FirebaseIcon />} title="CDN do Firebase" href="/docs/configuration/cdn/firebase" arrow>
    Entregue arquivos através do Firebase Storage utilizando a CDN do Google.
  </Cards.Card>
</Cards>
