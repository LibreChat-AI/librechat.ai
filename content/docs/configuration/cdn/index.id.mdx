<Callout type="info" title="Catatan">
  LibreChat mendukung penyimpanan lokal, backend penyimpanan objek, dan pengiriman berbasis CDN. Gunakan backend penyimpanan objek seperti S3 atau Azure Blob Storage untuk penyimpanan file yang tahan lama, kemudian tambahkan CDN seperti CloudFront jika Anda memerlukan tautan media yang stabil, edge caching, signed cookies, atau signed download URLs.
</Callout>

## Penyimpanan File [#file-storage]

Backend penyimpanan objek yang tahan lama untuk unggahan pengguna, gambar yang dihasilkan, dan file lainnya.

<Cards num={2}>
  <Cards.Card icon={<S3Icon />} title="Amazon S3" href="/docs/configuration/cdn/s3" arrow>
    Siapkan bucket Amazon S3 sebagai backend penyimpanan objek yang tahan lama dan dapat diskalakan.
  </Cards.Card>
  <Cards.Card icon={<AzureIcon />} title="Azure Blob Storage" href="/docs/configuration/cdn/azure" arrow>
    Gunakan kontainer Azure Blob Storage sebagai backend penyimpanan file Anda.
  </Cards.Card>
</Cards>

## CDN [#cdn]

Pengiriman dengan cache tepi untuk tautan media yang stabil, cookie yang ditandatangani, dan URL unduhan yang ditandatangani.

<Cards num={2}>
  <Cards.Card icon={<AWSIcon />} title="CloudFront dengan S3" href="/docs/configuration/cdn/cloudfront" arrow>
    Sajikan file yang didukung S3 melalui CloudFront untuk edge caching dan signed URLs.
  </Cards.Card>
  <Cards.Card icon={<FirebaseIcon />} title="CDN Firebase" href="/docs/configuration/cdn/firebase" arrow>
    Kirimkan file melalui Firebase Storage yang didukung oleh CDN Google.
  </Cards.Card>
</Cards>
