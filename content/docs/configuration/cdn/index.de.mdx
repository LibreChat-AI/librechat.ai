<Callout type="info" title="Hinweis">
  LibreChat unterstützt lokalen Speicher, Objektspeicher-Backends und CDN-gestützte Bereitstellung. Verwenden Sie S3 für dauerhaften Objektspeicher und fügen Sie CloudFront hinzu, wenn Sie stabile Medien-Links, Edge-Caching, signierte Cookies oder signierte Download-URLs benötigen.
</Callout>
