## Créer une application Facebook [#create-a-facebook-application]

- Allez sur le **[Facebook Developer Portal](https://developers.facebook.com/)**

- Cliquez sur « My Apps » dans le menu de l'en-tête

![image](https://github.com/danny-avila/LibreChat/assets/32828263/b75ccb8b-d56b-41b7-8b0d-a32c2e762962)

- Créer une nouvelle application

![image](https://github.com/danny-avila/LibreChat/assets/32828263/706f050d-5423-44cc-80f0-120913695d8f)

- Sélectionnez « Authenticate and request data from users with Facebook Login »

![image](https://github.com/danny-avila/LibreChat/assets/32828263/2ebbb571-afe8-429e-ab39-be6e83d12c01)

- Choisissez "No, I'm not creating a game"

![image](https://github.com/danny-avila/LibreChat/assets/32828263/88b5160a-9c72-414a-bbcc-7717b81106f3)

- Fournissez un `app name` et une `App contact email` puis cliquez sur `Create app`

![image](https://github.com/danny-avila/LibreChat/assets/32828263/e1282c9e-4e7d-4cbe-82c9-cc76967f83e1)

## Configuration de l'application Facebook [#facebook-application-configuration]

- Dans le menu latéral, sélectionnez "Use cases" et cliquez sur "Customize" sous "Authentication and account creation."

![image](https://github.com/danny-avila/LibreChat/assets/32828263/39f4bb70-d9dc-4d1c-8443-2666fe56499b)

-  Ajouter la `email permission`

![image](https://github.com/danny-avila/LibreChat/assets/32828263/dfa20879-2cb8-4daf-883d-3790854afca0)

- Cliquez maintenant sur `Go to settings`

![image](https://github.com/danny-avila/LibreChat/assets/32828263/512213a2-bd8b-4fd3-96c7-0de6d3222ddd)

- Assurez-vous que `Client OAuth login`, `Web OAuth login` et `Enforce HTTPS` sont **activés**.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/3a7d935b-97bf-493b-b909-39ecf9b3432b)

- Ajoutez des `Valid OAuth Redirect URIs` et cliquez sur "Save changes"
    - Exemple pour un domaine : `https://example.com/oauth/facebook/callback`

![image](https://github.com/danny-avila/LibreChat/assets/32828263/ef8e54ee-a766-4871-9719-d4eff7a770b6)

- Cliquez sur `Go back` et sélectionnez `Basic` dans l'onglet `App settings`

![image](https://github.com/danny-avila/LibreChat/assets/32828263/0d14f702-5183-422e-a12c-5d1b6031581b)

- Cliquez sur « Show » à côté de l'App secret.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/9a009e37-2bb6-4da6-b5c7-9139c3db6185)

## Configuration de .env [#env-configuration]

- Copiez l'`App ID` et l'`App Secret` et collez-les dans le fichier `.env` comme suit :

```bash filename=".env"
DOMAIN_CLIENT=https://your-domain.com # use http://localhost:3080 if not using a custom domain
DOMAIN_SERVER=https://your-domain.com # use http://localhost:3080 if not using a custom domain

FACEBOOK_CLIENT_ID=your_app_id
FACEBOOK_CLIENT_SECRET=your_app_secret
FACEBOOK_CALLBACK_URL=/oauth/facebook/callback
```

- Enregistrez le fichier `.env`.

> Remarque : Si vous utilisez Docker, exécutez `docker compose up -d` pour appliquer les modifications de configuration du fichier .env
