## Scadenza della sessione e token di aggiornamento [#session-expiry-and-refresh-token]

- Valori predefiniti: scadenza sessione: 15 minuti, scadenza token di aggiornamento: 7 giorni

    Server-->>Client: Requested data

## Sistema di moderazione automatizzato (opzionale) [#automated-moderation-system-optional]

Il sistema di moderazione automatizzato è abilitato per impostazione predefinita. Utilizza un meccanismo di punteggio per tracciare le violazioni degli utenti. Man mano che gli utenti compiono azioni come accessi eccessivi, registrazioni o invio di messaggi, accumulano punteggi di violazione. Al raggiungimento di una soglia prestabilita, l'utente e il suo indirizzo IP vengono temporaneamente bannati. Questo sistema garantisce la sicurezza della piattaforma monitorando e penalizzando le attività rapide o sospette.

Per configurare il sistema di moderazione, consulta [la guida alla configurazione](/docs/configuration/mod_system).

> *Nota: Se desideri che questo funzioni in modalità di sviluppo, dovrai creare un file chiamato `.env.development` nella directory principale e impostare `DOMAIN_CLIENT` su `http://localhost:3090` o su qualsiasi porta fornita da vite durante l'esecuzione di `npm run frontend-dev`*

## Script di gestione utenti [#user-management-scripts]

### Crea User Script [#create-user-script]

Lo script create-user ti consente di aggiungere utenti direttamente al database, anche quando la registrazione è disabilitata. Ecco come utilizzarlo:

1. Per il `docker-compose.yml` predefinito (se utilizzi `docker compose up` per avviare l'app):
   ```bash
   docker compose exec api npm run create-user
   ```

2. Per il `deploy-compose.yml` (se hai seguito la [Guida Docker per Ubuntu](/docs/remote/docker_linux)):
   ```bash
   docker exec -it LibreChat-API /bin/sh -c "cd .. && npm run create-user"
   ```

3. Per lo sviluppo locale (dalla root del progetto):
   ```bash
   npm run create-user
   ```

Segui le istruzioni per inserire l'email e la password del nuovo utente.

### Script di eliminazione utente [#delete-user-script]

Per eliminare un utente, è possibile utilizzare lo script delete-user:

1. Per il `docker-compose.yml` predefinito (se utilizzi `docker compose up` per avviare l'app):
   ```bash
   docker compose exec api npm run delete-user email@domain.com
   ```

2. Per il `deploy-compose.yml` (se hai seguito la [Guida Docker per Ubuntu](/docs/remote/docker_linux)):
   ```bash
   docker exec -it LibreChat-API /bin/sh -c "cd .. && npm run delete-user email@domain.com"
   ```

3. Per lo sviluppo locale (dalla root del progetto):
   ```bash
   npm run delete-user email@domain.com
   ```

Sostituisci `email@domain.com` con l'email dell'utente che desideri eliminare.
