앱 상단에 모든 사용자에게 중요한 메시지를 표시합니다. 공지사항, 유지보수 알림 또는 업데이트에 적합합니다.

<Callout type="info" title="빠른 개요">
  - 한 번에 **하나의 배너**만 활성화할 수 있습니다
  - 배너가 자동으로 나타나고 사라지도록 예약하세요
  - 사용자가 배너를 닫을 수 있는지 여부를 선택하세요
  - 모든 사용자 또는 로그인한 사용자에게만 배너 표시
</Callout>

---

## 배너 생성하기 [#creating-a-banner]

배너를 생성하거나 업데이트하려면 다음 명령어를 실행하세요:

```bash
npm run update-banner
```

몇 가지 간단한 프롬프트를 통해 안내를 받게 됩니다:

```ansi
--------------------------
Update the banner!
--------------------------
Display From (Format: yyyy-mm-ddTHH:MM:SSZ, Default: now):
> 2025-12-02T09:00:00Z

Display To (Format: yyyy-mm-ddTHH:MM:SSZ, Default: not specified):
> 2025-12-31T23:59:59Z

Enter your message (Enter a single dot "." on a new line to finish):
> 🎉 Welcome to LibreChat! Check out our new features.
> .

Is public (y/N):
> n

Is persistable (cannot be dismissed) (y/N):
> n
```

<Callout type="tip" title="이 옵션들은 무엇을 의미하나요?">
  - **시작/종료 표시**: 배너가 나타나고 사라질 시점입니다. "지금" 및 "항상"으로 설정하려면 비워 두세요.
  - **Is public**: 로그인하지 않은 방문자에게 표시 (로그인 페이지 등)
  - **Is persistable**: 예인 경우, 사용자가 배너를 닫을 수 없습니다. 중요한 공지사항에 사용하세요.
</Callout>

## 배너 삭제하기 [#deleting-a-banner]

```bash
npm run delete-banner
```

현재 배너가 표시되며 삭제하기 전에 확인을 요청받게 됩니다.

## 날짜 형식 [#date-format]

날짜 형식은 `yyyy-mm-ddTHH:MM:SSZ`를 사용하세요.

**예시:**
- `2025-12-25T09:00:00Z` → 2025년 12월 25일 오전 9:00 (UTC)
- `2025-01-01T00:00:00Z` → 2025년 1월 1일 자정 (UTC)

현재 시간을 사용하거나 종료 날짜를 지정하지 않으려면 날짜를 비워 두세요.
