Mostra messaggi importanti a tutti i tuoi utenti nella parte superiore dell'app. Perfetto per annunci, avvisi di manutenzione o aggiornamenti.

<Callout type="info" title="Panoramica rapida">
  - Solo **un banner** può essere attivo alla volta
  - Pianifica la visualizzazione e la scomparsa automatica dei banner
  - Scegli se gli utenti possono ignorare il banner o meno
  - Mostra banner a tutti, o solo agli utenti che hanno effettuato l'accesso
</Callout>

---

## Creazione di un Banner [#creating-a-banner]

Esegui questo comando per creare o aggiornare un banner:

```bash
npm run update-banner
```

Sarai guidato attraverso alcuni semplici prompt:

```ansi
--------------------------
Update the banner!
--------------------------
Display From (Format: yyyy-mm-ddTHH:MM:SSZ, Default: now):
> 2025-12-02T09:00:00Z

Display To (Format: yyyy-mm-ddTHH:MM:SSZ, Default: not specified):
> 2025-12-31T23:59:59Z

Enter your message (Enter a single dot "." on a new line to finish):
> 🎉 Welcome to LibreChat! Check out our new features.
> .

Is public (y/N):
> n

Is persistable (cannot be dismissed) (y/N):
> n
```

<Callout type="tip" title="Cosa significano queste opzioni?">
  - **Visualizza Da/A**: Quando il banner deve apparire e scomparire. Lascia vuoto per "ora" e "per sempre"
  - **È pubblico**: Mostra ai visitatori che non hanno effettuato l'accesso (come nella pagina di login)
  - **È persistente**: Se sì, gli utenti non possono chiudere il banner — da usare per avvisi importanti
</Callout>

---

## Eliminazione di un Banner [#deleting-a-banner]

```bash
npm run delete-banner
```

Vedrai il banner attuale e ti verrà chiesto di confermare prima dell'eliminazione.

  </Tabs.Tab>

## Formato data [#date-format]

Utilizza questo formato per le date: `yyyy-mm-ddTHH:MM:SSZ`

**Esempi:**
- `2025-12-25T09:00:00Z` → 25 dicembre 2025 alle 9:00 (UTC)
- `2025-01-01T00:00:00Z` → 1 gennaio 2025 a mezzanotte (UTC)

Lasciare la data vuota per utilizzare l'ora corrente o nessuna data di fine.
