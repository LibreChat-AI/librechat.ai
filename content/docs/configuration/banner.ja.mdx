アプリの最上部に、すべてのユーザーに対して重要なメッセージを表示します。お知らせ、メンテナンス通知、またはアップデートの告知に最適です。

<Callout type="info" title="クイック概要">
  - 一度にアクティブにできる**バナーは1つだけ**です
  - バナーが自動的に表示・非表示されるようにスケジュールを設定する
  - ユーザーがバナーを非表示にできるかどうかを選択します
  - バナーを全員に表示するか、ログインユーザーのみに表示するかを選択します
</Callout>

---

## バナーの作成 [#creating-a-banner]

このコマンドを実行して、バナーを作成または更新します：

```bash
npm run update-banner
```

いくつかの簡単なプロンプトを通じて案内されます：

```ansi
--------------------------
Update the banner!
--------------------------
Display From (Format: yyyy-mm-ddTHH:MM:SSZ, Default: now):
> 2025-12-02T09:00:00Z

Display To (Format: yyyy-mm-ddTHH:MM:SSZ, Default: not specified):
> 2025-12-31T23:59:59Z

Enter your message (Enter a single dot "." on a new line to finish):
> 🎉 Welcome to LibreChat! Check out our new features.
> .

Is public (y/N):
> n

Is persistable (cannot be dismissed) (y/N):
> n
```

<Callout type="tip" title="これらのオプションはどういう意味ですか？">
  - **Display From/To**: バナーを表示および非表示にする期間。空欄にすると「今すぐ」から「無期限」となります。
  - **Is public**: ログインしていない訪問者（ログインページなど）に表示します
  - **Is persistable**: はいの場合、ユーザーはバナーを非表示にできません。重要な通知に使用してください。
</Callout>

## バナーの削除 [#deleting-a-banner]

```bash
npm run delete-banner
```

現在のバナーが表示され、削除する前に確認を求められます。

## 日付形式 [#date-format]

日付には `yyyy-mm-ddTHH:MM:SSZ` の形式を使用してください。

**例:**
- `2025-12-25T09:00:00Z` → 2025年12月25日 午前9:00 (UTC)
- `2025-01-01T00:00:00Z` → 2025年1月1日 午前0時 (UTC)

現在時刻を使用する場合や終了日を設定しない場合は、日付を空のままにしてください。
