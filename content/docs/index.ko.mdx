<Callout type="info" title="데스크톱 설치 프로그램을 찾으시나요?">

LibreChat은 자체 호스팅 웹 애플리케이션이며, 네이티브 Windows 앱이나 Linux AppImage가 아닙니다. Windows에서는 가장 간단한 로컬 설정을 위해 [Docker Desktop](/docs/local/docker)을 사용하세요. Linux 데스크톱이나 서버의 경우 [Docker](/docs/local/docker), [npm](/docs/local/npm) 또는 [원격 호스팅 가이드](/docs/remote)를 사용하세요.

</Callout>
