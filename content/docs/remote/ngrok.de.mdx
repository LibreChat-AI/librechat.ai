Um ngrok zu verwenden, um Ihren lokalen Server ins Internet zu tunneln, befolgen Sie diese Schritte:

## Registrieren [#sign-up]

1. Gehen Sie zu **[https://ngrok.com/](https://ngrok.com/)** und registrieren Sie sich für ein Konto.

## Docker Installation [#docker-installation]

1. Kopieren Sie Ihr Auth-Token von: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. Öffnen Sie ein Terminal und führen Sie den folgenden Befehl aus: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Windows Installation [#windows-installation]

1. Laden Sie die ZIP-Datei herunter von: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Extrahieren Sie den Inhalt der ZIP-Datei mit 7zip oder WinRar.
3. Führen Sie `ngrok.exe` aus.
4. Kopieren Sie Ihr Auth-Token von: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. Führen Sie im `ngrok.exe`-Terminal den folgenden Befehl aus: `ngrok config add-authtoken <your token>`
6. Falls du dies noch nicht getan hast, starte LibreChat wie gewohnt.
7. Führen Sie im `ngrok.exe`-Terminal den folgenden Befehl aus: `ngrok http 3080`

Sie werden einen Link sehen, der verwendet werden kann, um auf LibreChat zuzugreifen.

## Linux Installation [#linux-installation]

1. Kopieren Sie den Befehl von: **[https://ngrok.com/download](https://ngrok.com/download)** und wählen Sie die **korrekte** Architektur aus.
2. Führen Sie den Befehl im Terminal aus
3. Kopieren Sie Ihr Auth-Token von: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. Führen Sie den folgenden Befehl aus: `ngrok config add-authtoken <your token>`
5. Falls du dies noch nicht getan hast, starte LibreChat wie gewohnt.
6. Führen Sie den folgenden Befehl aus: `ngrok http 3080`

## Mac Installation [#mac-installation]

1. Laden Sie die ZIP-Datei herunter von: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Entpacken Sie den Inhalt der ZIP-Datei mit einer geeigneten Mac-Anwendung wie Unarchiver.
3. Öffnen Sie das Terminal.
4. Navigieren Sie mit dem Befehl `cd` zu dem Verzeichnis, in das Sie ngrok extrahiert haben.
5. Führen Sie ngrok aus, indem Sie `./ngrok` eingeben.
6. Kopieren Sie Ihr Auth-Token von: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. Geben Sie im Terminal, in dem Sie ngrok ausgeführt haben, den folgenden Befehl ein: `ngrok authtoken <your token>`
8. Falls du dies noch nicht getan hast, starte LibreChat wie gewohnt.
9. Geben Sie in dem Terminal, in dem Sie ngrok ausgeführt haben, den folgenden Befehl ein: `./ngrok http 3080`
