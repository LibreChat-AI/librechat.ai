Yerel sunucunuzu internete tünellemek için ngrok kullanmak isterseniz, şu adımları izleyin:

## Kaydol [#sign-up]

1. **[https://ngrok.com/](https://ngrok.com/)** adresine gidin ve bir hesap oluşturun.

## Docker Kurulumu [#docker-installation]

1. Kimlik doğrulama belirtecinizi (auth token) şu adresten kopyalayın: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. Bir terminal açın ve şu komutu çalıştırın: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Windows Kurulumu [#windows-installation]

1. ZIP dosyasını şuradan indirin: **[https://ngrok.com/download](https://ngrok.com/download)**
2. ZIP dosyasının içeriğini 7zip veya WinRar kullanarak çıkartın.
3. `ngrok.exe` dosyasını çalıştırın.
4. Kimlik doğrulama belirtecinizi (auth token) şu adresten kopyalayın: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. `ngrok.exe` terminalinde şu komutu çalıştırın: `ngrok config add-authtoken <your token>`
6. Henüz yapmadıysanız, LibreChat'i normal şekilde başlatın.
7. `ngrok.exe` terminalinde şu komutu çalıştırın: `ngrok http 3080`

## Linux Kurulumu [#linux-installation]

1. **[https://ngrok.com/download](https://ngrok.com/download)** adresinden **doğru** mimariyi seçerek komutu kopyalayın.
2. Komutu terminalde çalıştırın
3. Kimlik doğrulama belirtecinizi (auth token) şu adresten kopyalayın: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. aşağıdaki komutu çalıştırın: `ngrok config add-authtoken <your token>`
5. Henüz yapmadıysanız, LibreChat'i normal şekilde başlatın.
6. aşağıdaki komutu çalıştırın: `ngrok http 3080`

## Mac Kurulumu [#mac-installation]

1. ZIP dosyasını şuradan indirin: **[https://ngrok.com/download](https://ngrok.com/download)**
2. ZIP dosyasının içeriğini Unarchiver gibi uygun bir Mac uygulaması kullanarak çıkarın.
3. Terminal'i açın.
4. `cd` komutunu kullanarak ngrok'u çıkardığınız dizine gidin.
5. `./ngrok` yazarak ngrok'u çalıştırın.
6. Kimlik doğrulama belirtecinizi (auth token) şu adresten kopyalayın: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. ngrok'u çalıştırdığınız terminalde şu komutu girin: `ngrok authtoken <your token>`
8. Henüz yapmadıysanız, LibreChat'i normal şekilde başlatın.
9. ngrok'u çalıştırdığınız terminalde şu komutu girin: `./ngrok http 3080`
