Para usar ngrok para tunelizar tu servidor local a internet, sigue estos pasos:

## Registrarse [#sign-up]

1. Ve a **[https://ngrok.com/](https://ngrok.com/)** y regístrate para obtener una cuenta.

## Instalación con Docker [#docker-installation]

1. Copia tu token de autenticación desde: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. Abra una terminal y ejecute el siguiente comando: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Instalación en Windows [#windows-installation]

1. Descarga el archivo ZIP desde: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Extraiga el contenido del archivo ZIP usando 7zip o WinRar.
3. Ejecuta `ngrok.exe`.
4. Copia tu token de autenticación desde: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. En la terminal de `ngrok.exe`, ejecuta el siguiente comando: `ngrok config add-authtoken <your token>`
6. Si aún no lo has hecho, inicia LibreChat normalmente.
7. En la terminal de `ngrok.exe`, ejecuta el siguiente comando: `ngrok http 3080`

## Instalación en Linux [#linux-installation]

1. Copia el comando desde: **[https://ngrok.com/download](https://ngrok.com/download)** eligiendo la arquitectura **correcta**.
2. Ejecuta el comando en la terminal
3. Copia tu token de autenticación desde: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. ejecute el siguiente comando: `ngrok config add-authtoken <your token>`
5. Si aún no lo has hecho, inicia LibreChat normalmente.
6. ejecuta el siguiente comando: `ngrok http 3080`

## Instalación en Mac [#mac-installation]

1. Descarga el archivo ZIP desde: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Extraiga el contenido del archivo ZIP utilizando una aplicación de Mac adecuada, como Unarchiver.
3. Abre la Terminal.
4. Navegue al directorio donde extrajo ngrok usando el comando `cd`.
5. Ejecuta ngrok escribiendo `./ngrok`.
6. Copia tu token de autenticación desde: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. En la terminal donde ejecutaste ngrok, ingresa el siguiente comando: `ngrok authtoken <your token>`
8. Si aún no lo has hecho, inicia LibreChat normalmente.
9. En la terminal donde ejecutaste ngrok, ingresa el siguiente comando: `./ngrok http 3080`
