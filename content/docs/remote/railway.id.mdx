Railway menyediakan opsi instalasi sekali klik untuk men-deploy LibreChat, membuat prosesnya menjadi lebih sederhana. Berikut adalah cara melakukannya:

## Langkah-langkah [#steps]

### **Kunjungi repositori LibreChat** [#visit-the-librechat-repository]

Buka [repositori LibreChat](https://github.com/danny-avila/LibreChat) di GitHub.

### **Buat akun Railway** [#create-a-railway-account]

[Daftar akun Railway](https://railway.app?referralCode=HI9hWz&utm_medium=integration&utm_source=docs&utm_campaign=librechat) jika Anda belum memilikinya (tautan ini menyertakan kode referral yang mendukung proyek LibreChat).

### **Klik tombol "Deploy on Railway"** [#click-the-deploy-on-railway-button]

<p align="left">
    <a href="https://railway.com/deploy/librechat-official?referralCode=HI9hWz&utm_medium=integration&utm_source=docs&utm_campaign=librechat">
        <img src="https://railway.com/button.svg" alt="Deploy di Railway" height="40"/>
    </a>
</p>

(Tombol tersebut juga tersedia di file README repositori)


### **Konfigurasi variabel lingkungan** [#configure-environment-variables]

Railway akan secara otomatis mendeteksi variabel lingkungan yang diperlukan untuk LibreChat. Tinjau konfigurasi ketiga container tersebut dan klik `Save Config` setelah meninjau masing-masing container.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/4417e997-621c-44b6-8d2d-94d7e4e1a2bf)

Konfigurasi default akan membantu Anda memulai, namun untuk fitur yang lebih canggih, Anda dapat melihat dokumentasi kami mengenai subjek tersebut: [Environment Variables](/docs/configuration/dotenv)

### **Deploy** [#deploy]

Setelah Anda mengisi variabel lingkungan yang diperlukan, klik tombol "Deploy". Railway akan menangani sisanya, termasuk menyiapkan basis data PostgreSQL serta membangun/menyebarkan instans LibreChat Anda.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/d94e20c6-0ae7-42af-8937-7fbd34d63a3b)

### **Akses instans LibreChat Anda** [#access-your-librechat-instance]

Setelah penyebaran berhasil, Railway akan memberikan Anda URL publik tempat Anda dapat mengakses instans LibreChat Anda.

Selesai! Anda telah berhasil men-deploy LibreChat di Railway menggunakan proses instalasi sekali klik. Anda sekarang dapat mulai menggunakan dan menyesuaikan instance LibreChat Anda sesuai kebutuhan.

## Tips Tambahan [#additional-tips]

- Periksa repositori LibreChat secara berkala untuk pembaruan dan terapkan ulang instans Anda untuk mendapatkan fitur terbaru dan perbaikan bug.
- Anda dapat menemukan opsi "redeploy" di Railway setelah Anda masuk dengan mengeklik 3 titik di sebelah kanan tombol "view logs"

Untuk instruksi lebih rinci dan pemecahan masalah, silakan merujuk ke dokumentasi resmi LibreChat dan panduan Railway.
