Om ngrok te gebruiken voor het tunnelen van je lokale server naar het internet, volg je deze stappen:

## Registreren [#sign-up]

1. Ga naar **[https://ngrok.com/](https://ngrok.com/)** en meld je aan voor een account.

## Docker Installatie [#docker-installation]

1. Kopieer je auth token van: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. Open een terminal en voer het volgende commando uit: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Windows-installatie [#windows-installation]

1. Download het ZIP-bestand van: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Pak de inhoud van het ZIP-bestand uit met 7zip of WinRar.
3. Voer `ngrok.exe` uit.
4. Kopieer je auth token van: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. Voer in de `ngrok.exe` terminal het volgende commando uit: `ngrok config add-authtoken <your token>`
6. Als je dat nog niet hebt gedaan, start LibreChat dan normaal op.
7. Voer in de `ngrok.exe` terminal het volgende commando uit: `ngrok http 3080`

Je zult een link zien die kan worden gebruikt om toegang te krijgen tot LibreChat.

## Linux-installatie [#linux-installation]

1. Kopieer het commando van: **[https://ngrok.com/download](https://ngrok.com/download)** en kies de **correcte** architectuur.
2. Voer het commando uit in de terminal
3. Kopieer je auth token van: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. voer het volgende commando uit: `ngrok config add-authtoken <your token>`
5. Als je dat nog niet hebt gedaan, start LibreChat dan normaal op.
6. voer het volgende commando uit: `ngrok http 3080`

## Installatie op Mac [#mac-installation]

1. Download het ZIP-bestand van: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Pak de inhoud van het ZIP-bestand uit met een geschikt Mac-programma zoals Unarchiver.
3. Open de terminal.
4. Navigeer naar de map waar je ngrok hebt uitgepakt met het `cd` commando.
5. Voer ngrok uit door `./ngrok` te typen.
6. Kopieer je auth token van: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. Voer in de terminal waar je ngrok hebt uitgevoerd het volgende commando in: `ngrok authtoken <your token>`
8. Als je dat nog niet hebt gedaan, start LibreChat dan normaal op.
9. Voer in de terminal waar je ngrok hebt uitgevoerd het volgende commando in: `./ngrok http 3080`
