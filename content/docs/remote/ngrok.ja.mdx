ngrokを使用してローカルサーバーをインターネットにトンネリングするには、以下の手順に従ってください：

## サインアップ [#sign-up]

1. **[https://ngrok.com/](https://ngrok.com/)** にアクセスして、アカウントにサインアップしてください。

## Docker インストール [#docker-installation]

1. 以下から認証トークンをコピーしてください: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. ターミナルを開き、以下のコマンドを実行してください: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Windowsへのインストール [#windows-installation]

1. 以下からZIPファイルをダウンロードしてください: **[https://ngrok.com/download](https://ngrok.com/download)**
2. 7zipまたはWinRarを使用して、ZIPファイルの内容を解凍してください。
3. `ngrok.exe` を実行します。
4. 以下から認証トークンをコピーしてください: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. `ngrok.exe` ターミナルで、以下のコマンドを実行してください: `ngrok config add-authtoken <your token>`
6. まだ行っていない場合は、通常通りLibreChatを起動してください。
7. `ngrok.exe` ターミナルで、次のコマンドを実行します: `ngrok http 3080`

LibreChatにアクセスするために使用できるリンクが表示されます。

## Linux インストール [#linux-installation]

1. **[https://ngrok.com/download](https://ngrok.com/download)** から、**正しい**アーキテクチャを選択してコマンドをコピーしてください。
2. ターミナルでコマンドを実行します
3. 以下から認証トークンをコピーしてください: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. 以下のコマンドを実行してください: `ngrok config add-authtoken <your token>`
5. まだ行っていない場合は、通常通りLibreChatを起動してください。
6. 以下のコマンドを実行してください: `ngrok http 3080`

## Mac へのインストール [#mac-installation]

1. 以下からZIPファイルをダウンロードしてください: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Unarchiverのような適切なMac用アプリケーションを使用して、ZIPファイルの内容を解凍してください。
3. ターミナルを開きます。
4. `cd` コマンドを使用して、ngrok を展開したディレクトリに移動します。
5. `./ngrok` と入力して ngrok を実行します。
6. 以下から認証トークンをコピーしてください: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. ngrokを実行したターミナルで、以下のコマンドを入力してください: `ngrok authtoken <your token>`
8. まだ行っていない場合は、通常通りLibreChatを起動してください。
9. ngrokを実行したターミナルで、以下のコマンドを入力してください：`./ngrok http 3080`
