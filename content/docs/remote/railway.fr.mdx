Railway propose une option d'installation en un clic pour déployer LibreChat, rendant le processus encore plus simple. Voici comment vous pouvez le faire :

## Étapes [#steps]

### **Visitez le dépôt LibreChat** [#visit-the-librechat-repository]

Allez sur le [dépôt LibreChat](https://github.com/danny-avila/LibreChat) sur GitHub.

### **Créer un compte Railway** [#create-a-railway-account]

[Créez un compte Railway](https://railway.app?referralCode=HI9hWz&utm_medium=integration&utm_source=docs&utm_campaign=librechat) si vous n'en avez pas déjà un (ce lien inclut un code de parrainage qui soutient le projet LibreChat).

### **Cliquez sur le bouton "Deploy on Railway"** [#click-the-deploy-on-railway-button]

<p align="left">
    <a href="https://railway.com/deploy/librechat-official?referralCode=HI9hWz&utm_medium=integration&utm_source=docs&utm_campaign=librechat">
        <img src="https://railway.com/button.svg" alt="Déployer sur Railway" height="40"/>
    </a>
</p>

(Le bouton est également disponible dans le fichier README du dépôt)


### **Configurer les variables d'environnement** [#configure-environment-variables]

Railway détectera automatiquement les variables d'environnement requises pour LibreChat. Vérifiez la configuration des trois conteneurs et cliquez sur `Save Config` après avoir examiné chacun d'entre eux.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/4417e997-621c-44b6-8d2d-94d7e4e1a2bf)

La configuration par défaut vous permettra de démarrer, mais pour des fonctionnalités plus avancées, vous pouvez consulter notre documentation à ce sujet : [Variables d'environnement](/docs/configuration/dotenv)

### **Déploiement** [#deploy]

Une fois que vous avez renseigné les variables d'environnement requises, cliquez sur le bouton « Deploy ». Railway se chargera du reste, y compris la configuration d'une base de données PostgreSQL ainsi que la construction et le déploiement de votre instance LibreChat.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/d94e20c6-0ae7-42af-8937-7fbd34d63a3b)

### **Accédez à votre instance LibreChat** [#access-your-librechat-instance]

Une fois le déploiement réussi, Railway vous fournira une URL publique où vous pourrez accéder à votre instance LibreChat.

C'est tout ! Vous avez déployé avec succès LibreChat sur Railway en utilisant le processus d'installation en un clic. Vous pouvez maintenant commencer à utiliser et à personnaliser votre instance LibreChat selon vos besoins.

## Conseils supplémentaires [#additional-tips]

- Vérifiez régulièrement le dépôt LibreChat pour les mises à jour et redéployez votre instance pour bénéficier des dernières fonctionnalités et corrections de bugs.
- Vous pouvez trouver l'option "redeploy" dans Railway après vous être connecté en cliquant sur les 3 points à droite du bouton "view logs"

Pour des instructions plus détaillées et le dépannage, consultez la documentation officielle de LibreChat et les guides de Railway.
