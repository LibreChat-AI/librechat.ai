Railway ofrece una opción de instalación con un solo clic para desplegar LibreChat, haciendo el proceso aún más sencillo. Aquí te explicamos cómo puedes hacerlo:

## Pasos [#steps]

### **Visita el repositorio de LibreChat** [#visit-the-librechat-repository]

Ve al [repositorio de LibreChat](https://github.com/danny-avila/LibreChat) en GitHub.

### **Crear una cuenta de Railway** [#create-a-railway-account]

[Regístrate para obtener una cuenta de Railway](https://railway.app?referralCode=HI9hWz&utm_medium=integration&utm_source=docs&utm_campaign=librechat) si aún no tienes una (este enlace incluye un código de referencia que apoya al proyecto LibreChat).

### **Haz clic en el botón "Deploy on Railway"** [#click-the-deploy-on-railway-button]

<p align="left">
    <a href="https://railway.com/deploy/librechat-official?referralCode=HI9hWz&utm_medium=integration&utm_source=docs&utm_campaign=librechat">
        <img src="https://railway.com/button.svg" alt="Desplegar en Railway" height="40"/>
    </a>
</p>

(El botón también está disponible en el archivo README del repositorio)


### **Configurar variables de entorno** [#configure-environment-variables]

Railway detectará automáticamente las variables de entorno requeridas para LibreChat. Revisa la configuración de los tres contenedores y haz clic en `Save Config` después de revisar cada uno de ellos.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/4417e997-621c-44b6-8d2d-94d7e4e1a2bf)

La configuración predeterminada le permitirá comenzar, pero para funciones más avanzadas, puede consultar nuestra documentación sobre el tema: [Environment Variables](/docs/configuration/dotenv)

### **Desplegar** [#deploy]

Una vez que hayas completado las variables de entorno requeridas, haz clic en el botón "Deploy". Railway se encargará del resto, incluyendo la configuración de una base de datos PostgreSQL y la compilación/despliegue de tu instancia de LibreChat.

![image](https://github.com/danny-avila/LibreChat/assets/32828263/d94e20c6-0ae7-42af-8937-7fbd34d63a3b)

### **Acceda a su instancia de LibreChat** [#access-your-librechat-instance]

Después de que la implementación sea exitosa, Railway le proporcionará una URL pública donde podrá acceder a su instancia de LibreChat.

¡Eso es todo! Has implementado con éxito LibreChat en Railway utilizando el proceso de instalación con un solo clic. Ahora puedes comenzar a usar y personalizar tu instancia de LibreChat según sea necesario.

## Consejos adicionales [#additional-tips]

- Comprueba regularmente el repositorio de LibreChat para ver si hay actualizaciones y vuelve a desplegar tu instancia para recibir las últimas funciones y correcciones de errores.
- Puedes encontrar la opción "redeploy" en Railway después de iniciar sesión haciendo clic en los 3 puntos a la derecha del botón "view logs"

Para obtener instrucciones más detalladas y solución de problemas, consulte la documentación oficial de LibreChat y las guías de Railway.
