Untuk menggunakan ngrok dalam melakukan tunneling server lokal Anda ke internet, ikuti langkah-langkah berikut:

## Daftar [#sign-up]

1. Buka **[https://ngrok.com/](https://ngrok.com/)** dan daftar untuk membuat akun.

## Instalasi Docker [#docker-installation]

1. Salin token autentikasi Anda dari: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. Buka terminal dan jalankan perintah berikut: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Instalasi Windows [#windows-installation]

1. Unduh file ZIP dari: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Ekstrak isi file ZIP menggunakan 7zip atau WinRar.
3. Jalankan `ngrok.exe`.
4. Salin token autentikasi Anda dari: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. Di terminal `ngrok.exe`, jalankan perintah berikut: `ngrok config add-authtoken <your token>`
6. Jika Anda belum melakukannya, jalankan LibreChat seperti biasa.
7. Di terminal `ngrok.exe`, jalankan perintah berikut: `ngrok http 3080`

Anda akan melihat tautan yang dapat digunakan untuk mengakses LibreChat.

## Instalasi Linux [#linux-installation]

1. Salin perintah dari: **[https://ngrok.com/download](https://ngrok.com/download)** dengan memilih arsitektur yang **tepat**.
2. Jalankan perintah tersebut di terminal
3. Salin token autentikasi Anda dari: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. jalankan perintah berikut: `ngrok config add-authtoken <your token>`
5. Jika Anda belum melakukannya, jalankan LibreChat seperti biasa.
6. jalankan perintah berikut: `ngrok http 3080`

## Instalasi Mac [#mac-installation]

1. Unduh file ZIP dari: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Ekstrak isi file ZIP menggunakan aplikasi Mac yang sesuai seperti Unarchiver.
3. Buka Terminal.
4. Arahkan ke direktori tempat Anda mengekstrak ngrok menggunakan perintah `cd`.
5. Jalankan ngrok dengan mengetik `./ngrok`.
6. Salin token autentikasi Anda dari: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. Di terminal tempat Anda menjalankan ngrok, masukkan perintah berikut: `ngrok authtoken <your token>`
8. Jika Anda belum melakukannya, jalankan LibreChat seperti biasa.
9. Di terminal tempat Anda menjalankan ngrok, masukkan perintah berikut: `./ngrok http 3080`
