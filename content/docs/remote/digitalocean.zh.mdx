## 第一部分：从零开始： [#part-i-starting-from-zero]

### **1. 在 DigitalOcean 上开始** [#1-get-started-on-digitalocean]

[点击此处](https://www.digitalocean.com/?refcode=4486923fcf00&utm_campaign=Referral_Invite&utm_medium=Referral_Program&utm_source=badge)或点击上方的横幅即可开始。

登录后，您将看到一个[友好的欢迎界面](https://cloud.digitalocean.com/welcome)。

![image](https://github.com/danny-avila/LibreChat/assets/110412045/b7a71eae-770e-4c69-a5d4-d21b939d64ed)

### **a) 导航至 Projects 页面** [#a-navigate-to-the-projects-page]

点击 ["Explore our control panel"](https://cloud.digitalocean.com/projects) 或直接导航至 [Projects page](https://cloud.digitalocean.com/projects)。

在 DigitalOcean 中，服务器实例被称为 **"droplets"**，它们被组织在 **"Projects"** 下。

### **b) 点击 "Spin up a Droplet" 开始设置** [#b-click-on-spin-up-a-droplet-to-start-the-setup]

![image](https://github.com/danny-avila/LibreChat/assets/110412045/6046e8cd-ff59-4795-a29a-5f44ab2f0a6d)

根据您的需求调整这些设置，因为我选择的是能正常工作的最基本/最便宜的选项。

- **选择区域/数据中心：** 离您和您的用户最近的区域
- **选择一个镜像：** Ubuntu 22.04 (LTS) x64
- **选择规格：** 共享 CPU，基础方案
  - CPU 选项：常规，6 美元/月选项（0.009 美元/小时，1 GB 内存 / 1 CPU / 25 GB SSD / 1000 GB 流量）
  - 无需额外存储
- **选择身份验证方法：** 密码选项最简单，但由您自行决定
  - 或者，您可以设置传统的 SSH。
- **推荐：** 添加改进的指标监控和警报（免费）
  - 如果不勾选此项，您或许可以使用每月 4 美元的选项，但目前尚未经过测试。
- **最终确定细节：**
  - 将主机名更改为您喜欢的任何名称，其他所有设置我均保持默认（1 个 droplet，无标签）
  - 最后，点击 "Create Droplet"

![image](https://github.com/danny-avila/LibreChat/assets/110412045/ac90d40e-3ac6-482f-885c-58058c5e3f76)

创建 droplet 后，它现在将启动并显示进度条。

### **2. 访问您的 droplet 控制台** [#2-access-your-droplet-console]

一旦启动完成，**点击该 droplet**，然后点击右侧的 Console 链接以启动控制台。

![image](https://github.com/danny-avila/LibreChat/assets/110412045/47c14280-fe48-49b9-9997-ff4d9c83212c)

![image](https://github.com/danny-avila/LibreChat/assets/110412045/d5e518fd-4941-4b35-86cc-69f8f65ec8eb)

通过这种方式启动 Droplet 控制台是最简单的方法，但如果您在上一步中进行了设置，也可以通过 SSH 进行连接。

为了保持本指南的简洁性，我将采用简单的方式，继续使用 droplet 控制台。如果您感兴趣，这里有一份 [DigitalOcean 官方 SSH 指南](https://docs.digitalocean.com/products/droplets/how-to/connect-with-ssh/)。

### **3. 登录后，请立即创建一个新的非 root 用户：** [#3-once-you-have-logged-in-immediately-create-a-new-non-root-user]

**注意：** 在本指南中，每当你看到大于号/小于号时，都应将其删除。

```bash
# example: adduser danny
adduser <yourusername>
# you will then be prompted for a password and user details
```

完成后，运行以下命令以提升用户权限

```bash
# example: usermod -aG sudo danny
usermod -aG sudo <yourusername>
```

**请通过确认您拥有 sudo 权限来确保您已正确完成此操作：**

```bash
getent group sudo | cut -d: -f4
```

**切换到新用户**

```bash
# example: su - danny
su - <yourusername>
```
