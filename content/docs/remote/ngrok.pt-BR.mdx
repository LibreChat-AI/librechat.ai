Para usar o ngrok para tunelar seu servidor local para a internet, siga estes passos:

## Cadastro [#sign-up]

1. Vá para **[https://ngrok.com/](https://ngrok.com/)** e crie uma conta.

## Instalação com Docker [#docker-installation]

1. Copie seu token de autenticação em: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. Abra um terminal e execute o seguinte comando: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Instalação no Windows [#windows-installation]

1. Baixe o arquivo ZIP em: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Extraia o conteúdo do arquivo ZIP usando o 7zip ou WinRar.
3. Execute o `ngrok.exe`.
4. Copie seu token de autenticação em: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. No terminal do `ngrok.exe`, execute o seguinte comando: `ngrok config add-authtoken <your token>`
6. Se você ainda não fez isso, inicie o LibreChat normalmente.
7. No terminal do `ngrok.exe`, execute o seguinte comando: `ngrok http 3080`

Você verá um link que pode ser usado para acessar o LibreChat.

## Instalação no Linux [#linux-installation]

1. Copie o comando de: **[https://ngrok.com/download](https://ngrok.com/download)** escolhendo a arquitetura **correta**.
2. Execute o comando no terminal
3. Copie seu token de autenticação em: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. execute o seguinte comando: `ngrok config add-authtoken <your token>`
5. Se você ainda não fez isso, inicie o LibreChat normalmente.
6. execute o seguinte comando: `ngrok http 3080`

## Instalação no Mac [#mac-installation]

1. Baixe o arquivo ZIP em: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Extraia o conteúdo do arquivo ZIP usando um aplicativo Mac adequado, como o Unarchiver.
3. Abra o Terminal.
4. Navegue até o diretório onde você extraiu o ngrok usando o comando `cd`.
5. Execute o ngrok digitando `./ngrok`.
6. Copie seu token de autenticação em: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. No terminal onde você executou o ngrok, insira o seguinte comando: `ngrok authtoken <your token>`
8. Se você ainda não fez isso, inicie o LibreChat normalmente.
9. No terminal onde você executou o ngrok, insira o seguinte comando: `./ngrok http 3080`
