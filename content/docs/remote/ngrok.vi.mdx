Để sử dụng ngrok nhằm tạo đường truyền (tunneling) từ máy chủ cục bộ của bạn ra internet, hãy làm theo các bước sau:

## Đăng ký [#sign-up]

1. Truy cập **[https://ngrok.com/](https://ngrok.com/)** và đăng ký tài khoản.

## Cài đặt Docker [#docker-installation]

1. Sao chép auth token của bạn từ: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. Mở terminal và chạy lệnh sau: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Cài đặt trên Windows [#windows-installation]

1. Tải xuống tệp ZIP từ: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Giải nén nội dung của tệp ZIP bằng 7zip hoặc WinRar.
3. Chạy `ngrok.exe`.
4. Sao chép auth token của bạn từ: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. Trong terminal `ngrok.exe`, hãy chạy lệnh sau: `ngrok config add-authtoken <your token>`
6. Nếu bạn chưa thực hiện, hãy khởi động LibreChat như bình thường.
7. Trong terminal `ngrok.exe`, hãy chạy lệnh sau: `ngrok http 3080`

Bạn sẽ thấy một liên kết có thể được sử dụng để truy cập LibreChat.

## Cài đặt trên Linux [#linux-installation]

1. Sao chép lệnh từ: **[https://ngrok.com/download](https://ngrok.com/download)** và chọn kiến trúc **chính xác**.
2. Chạy lệnh trong terminal
3. Sao chép auth token của bạn từ: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. chạy lệnh sau: `ngrok config add-authtoken <your token>`
5. Nếu bạn chưa thực hiện, hãy khởi động LibreChat như bình thường.
6. chạy lệnh sau: `ngrok http 3080`

## Cài đặt trên Mac [#mac-installation]

1. Tải xuống tệp ZIP từ: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Giải nén nội dung của tệp ZIP bằng một ứng dụng Mac phù hợp như Unarchiver.
3. Mở Terminal.
4. Điều hướng đến thư mục nơi bạn đã giải nén ngrok bằng lệnh `cd`.
5. Chạy ngrok bằng cách nhập `./ngrok`.
6. Sao chép auth token của bạn từ: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. Trong terminal nơi bạn đã chạy ngrok, hãy nhập lệnh sau: `ngrok authtoken <your token>`
8. Nếu bạn chưa thực hiện, hãy khởi động LibreChat như bình thường.
9. Trong terminal nơi bạn đã chạy ngrok, hãy nhập lệnh sau: `./ngrok http 3080`
