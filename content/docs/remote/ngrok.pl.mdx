Aby użyć ngrok do tunelowania lokalnego serwera do internetu, wykonaj poniższe kroki:

## Rejestracja [#sign-up]

1. Przejdź do **[https://ngrok.com/](https://ngrok.com/)** i zarejestruj konto.

## Instalacja Docker [#docker-installation]

1. Skopiuj swój token autoryzacyjny z: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. Otwórz terminal i uruchom następujące polecenie: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Instalacja w systemie Windows [#windows-installation]

1. Pobierz plik ZIP z: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Wypakuj zawartość pliku ZIP za pomocą 7zip lub WinRar.
3. Uruchom `ngrok.exe`.
4. Skopiuj swój token autoryzacyjny z: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. W terminalu `ngrok.exe` uruchom następujące polecenie: `ngrok config add-authtoken <your token>`
6. Jeśli jeszcze tego nie zrobiłeś, uruchom LibreChat w normalny sposób.
7. W terminalu `ngrok.exe` uruchom następujące polecenie: `ngrok http 3080`

Zobaczysz link, którego można użyć do uzyskania dostępu do LibreChat.

## Instalacja w systemie Linux [#linux-installation]

1. Skopiuj polecenie z: **[https://ngrok.com/download](https://ngrok.com/download)**, wybierając **właściwą** architekturę.
2. Uruchom polecenie w terminalu
3. Skopiuj swój token autoryzacyjny z: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. uruchom następujące polecenie: `ngrok config add-authtoken <your token>`
5. Jeśli jeszcze tego nie zrobiłeś, uruchom LibreChat w normalny sposób.
6. uruchom następujące polecenie: `ngrok http 3080`

## Instalacja na Macu [#mac-installation]

1. Pobierz plik ZIP z: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Wypakuj zawartość pliku ZIP za pomocą odpowiedniej aplikacji na Maca, takiej jak Unarchiver.
3. Otwórz terminal.
4. Przejdź do katalogu, w którym wypakowano ngrok, używając polecenia `cd`.
5. Uruchom ngrok, wpisując `./ngrok`.
6. Skopiuj swój token autoryzacyjny z: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. W terminalu, w którym uruchomiono ngrok, wprowadź następujące polecenie: `ngrok authtoken <your token>`
8. Jeśli jeszcze tego nie zrobiłeś, uruchom LibreChat w normalny sposób.
9. W terminalu, w którym uruchomiono ngrok, wprowadź następujące polecenie: `./ngrok http 3080`
