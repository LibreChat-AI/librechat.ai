## 注册您的域名 [#registering-your-domain]

### 第一步：选择您的域名 [#step-1-choose-your-domain-name]
- 选择一个符合您品牌形象且令人印象深刻的域名。避免使用复杂的拼写。
- 使用 Cloudflare、Namecheap、GoDaddy 等域名注册商提供的在线工具，检查您心仪的域名是否可用。

### 第 2 步：选择域名注册商 [#step-2-select-a-domain-registrar]
- 根据价格、客户支持、附加功能（如电子邮件、SSL 证书）以及用户评价来评估域名注册商。
- 请特别注意隐私保护服务（WHOIS privacy），因为这有助于保护您的个人信息隐私。

### 第三步：购买并注册域名 [#step-3-purchase-and-register-the-domain]
- 按照域名注册商的购买流程操作，包括提供您的联系信息并完成付款。
- 请仔细阅读服务条款并留意域名的到期日期，以避免意外失效。

## 配置 Cloudflare Tunnels [#configuring-cloudflare-tunnels]

### 第一步：将您的域名添加到 Cloudflare [#step-1-add-your-domain-to-cloudflare]
- 如果您的域名是通过 Cloudflare 购买的，则可以跳过此步骤，因为您的域名已配置为使用 Cloudflare 的名称服务器。
- 登录您的 Cloudflare 账户并添加您的新域名。按照说明将您的域名名称服务器（nameservers）替换为 Cloudflare 提供的名称服务器，这对于激活 Cloudflare 的服务是必要的。

#### 在您的服务器上安装 Cloudflare 的 `cloudflared` [#install-cloudflares-cloudflared-on-your-server]

![image](https://github.com/danny-avila/LibreChat/assets/32828263/82520d5c-8524-415f-82ae-c297bde3288d)

<Callout type="tip" title="提示">
- 为了实现持续运行，请考虑将 `cloudflared` 设置为系统服务。这可以确保隧道在重启或崩溃后依然保持活跃。
</Callout>

- 从 Cloudflare 官方网站下载并安装 [`cloudflared`](https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/downloads/) 工具到您的服务器上。该软件将促进您的域名与内部服务之间的安全连接。
- 使用您的 Cloudflare 账户凭据对 `cloudflared` 进行身份验证，以将其链接到您的域名。

#### 配置您的 Tunnel [#configure-your-tunnel]

- 通过 Cloudflare 控制面板下的 "Tunnels" 部分启动隧道

### 第 6 步：验证隧道 [#step-6-verify-the-tunnel]
- 通过在 Web 浏览器中访问您的域名/子域名来测试连接，确保它能通过 Cloudflare Tunnel 解析到您的服务器且没有错误。
- 直接在 Cloudflare 仪表板的 "Tunnels" 部分监控您的隧道性能和状态。

![image](https://github.com/danny-avila/LibreChat/assets/32828263/2ad741d5-8cdd-46c6-982c-7fa05a5858a8)

## 结论 [#conclusion]
通过保护您的域名注册并配置 Cloudflare Tunnel，您可以增强连接到网络服务的安全性和可靠性。请定期检查并更新您的域名和隧道设置，以适应任何新的需求或网络配置的变化。在维护您的在线状态和安全态势方面保持主动。
