## Mendaftarkan Domain Anda [#registering-your-domain]

### Langkah 1: Pilih Nama Domain Anda [#step-1-choose-your-domain-name]
- Pilih nama domain yang selaras dengan identitas merek Anda dan mudah diingat. Hindari ejaan yang rumit.
- Gunakan alat daring dari registrar domain seperti Cloudflare, Namecheap, GoDaddy, dll., untuk memeriksa apakah nama domain pilihan Anda tersedia.

### Langkah 2: Pilih Registrar Domain [#step-2-select-a-domain-registrar]
- Evaluasi registrar berdasarkan harga, dukungan pelanggan, fitur tambahan (seperti email, sertifikat SSL), dan ulasan pengguna.
- Berikan perhatian khusus pada layanan perlindungan privasi (privasi WHOIS), karena ini membantu menjaga informasi pribadi Anda tetap privat.

### Langkah 3: Beli dan Daftarkan Domain [#step-3-purchase-and-register-the-domain]
- Ikuti proses pembelian di registrar, yang mencakup penyediaan informasi kontak Anda dan penyelesaian pembayaran.
- Baca ketentuan layanan dengan saksama dan catat tanggal kedaluwarsa domain untuk menghindari gangguan yang tidak terduga.

## Mengonfigurasi Cloudflare Tunnels [#configuring-cloudflare-tunnels]

### Langkah 1: Tambahkan Domain Anda ke Cloudflare [#step-1-add-your-domain-to-cloudflare]
- Jika domain Anda dibeli melalui Cloudflare, langkah ini dilewati karena domain Anda sudah dikonfigurasi untuk menggunakan nameserver Cloudflare.
- Masuk ke akun Cloudflare Anda dan tambahkan domain baru Anda. Ikuti petunjuk untuk mengganti nameserver domain Anda dengan yang disediakan oleh Cloudflare, yang diperlukan untuk mengaktifkan layanan Cloudflare.

#### Instal `cloudflared` dari Cloudflare di Server Anda [#install-cloudflares-cloudflared-on-your-server]

![image](https://github.com/danny-avila/LibreChat/assets/32828263/82520d5c-8524-415f-82ae-c297bde3288d)

<Callout type="tip" title="Tips">
- Untuk pengoperasian berkelanjutan, pertimbangkan untuk mengatur `cloudflared` agar berjalan sebagai layanan di sistem Anda. Hal ini memastikan tunnel tetap aktif setelah proses reboot dan crash.
</Callout>

- Unduh dan instal alat [`cloudflared`](https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/downloads/) dari situs resmi Cloudflare ke server Anda. Perangkat lunak ini akan memfasilitasi koneksi aman antara domain Anda dan layanan internal.
- Autentikasi `cloudflared` menggunakan kredensial akun Cloudflare Anda untuk menautkannya ke domain Anda.

### Langkah 6: Verifikasi Tunnel [#step-6-verify-the-tunnel]
- Uji koneksi dengan mengakses domain/subdomain Anda di peramban web, pastikan domain tersebut terhubung ke server Anda melalui Cloudflare Tunnel tanpa kesalahan.
- Pantau performa dan status tunnel Anda secara langsung dari dasbor Cloudflare di bawah bagian "Tunnels".

## Kesimpulan [#conclusion]
Dengan mengamankan pendaftaran domain Anda dan mengonfigurasi Cloudflare Tunnel, Anda memperkuat keamanan dan keandalan koneksi ke layanan jaringan Anda. Tinjau dan perbarui pengaturan domain dan tunnel Anda secara berkala untuk beradaptasi dengan persyaratan baru atau perubahan dalam konfigurasi jaringan. Tetaplah proaktif dalam menjaga kehadiran daring dan postur keamanan Anda.
