Pour utiliser ngrok afin de rediriger votre serveur local vers Internet, suivez ces étapes :

## Inscription [#sign-up]

1. Allez sur **[https://ngrok.com/](https://ngrok.com/)** et créez un compte.

## Installation Docker [#docker-installation]

1. Copiez votre jeton d'authentification depuis : **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. Ouvrez un terminal et exécutez la commande suivante : `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Installation sur Windows [#windows-installation]

1. Téléchargez le fichier ZIP depuis : **[https://ngrok.com/download](https://ngrok.com/download)**
2. Extrayez le contenu du fichier ZIP en utilisant 7zip ou WinRar.
3. Exécutez `ngrok.exe`.
4. Copiez votre jeton d'authentification depuis : **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. In the `ngrok.exe` terminal, run the following command: `ngrok config add-authtoken <your token>`
6. Si vous ne l'avez pas déjà fait, démarrez LibreChat normalement.
7. Dans le terminal `ngrok.exe`, exécutez la commande suivante : `ngrok http 3080`

Vous verrez un lien qui peut être utilisé pour accéder à LibreChat.

## Installation sous Linux [#linux-installation]

1. Copiez la commande depuis : **[https://ngrok.com/download](https://ngrok.com/download)** en choisissant l'architecture **correcte**.
2. Exécutez la commande dans le terminal
3. Copiez votre jeton d'authentification depuis : **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. exécutez la commande suivante : `ngrok config add-authtoken <your token>`
5. Si vous ne l'avez pas déjà fait, démarrez LibreChat normalement.
6. exécutez la commande suivante : `ngrok http 3080`

## Installation sur Mac [#mac-installation]

1. Téléchargez le fichier ZIP depuis : **[https://ngrok.com/download](https://ngrok.com/download)**
2. Extrayez le contenu du fichier ZIP à l'aide d'une application Mac appropriée comme Unarchiver.
3. Ouvrez le terminal.
4. Accédez au répertoire où vous avez extrait ngrok en utilisant la commande `cd`.
5. Exécutez ngrok en tapant `./ngrok`.
6. Copiez votre jeton d'authentification depuis : **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. Dans le terminal où vous avez exécuté ngrok, entrez la commande suivante : `ngrok authtoken <your token>`
8. Si vous ne l'avez pas déjà fait, démarrez LibreChat normalement.
9. Dans le terminal où vous avez exécuté ngrok, entrez la commande suivante : `./ngrok http 3080`
