## Parte I: Começando do Zero: [#part-i-starting-from-zero]

### **1. Comece no DigitalOcean** [#1-get-started-on-digitalocean]

[Clique aqui](https://www.digitalocean.com/?refcode=4486923fcf00&utm_campaign=Referral_Invite&utm_medium=Referral_Program&utm_source=badge) ou no banner acima para começar.

Assim que você estiver logado, você será recebido com uma [tela de boas-vindas agradável](https://cloud.digitalocean.com/welcome).

![image](https://github.com/danny-avila/LibreChat/assets/110412045/b7a71eae-770e-4c69-a5d4-d21b939d64ed)

### **a) Navegue até a página de Projetos** [#a-navigate-to-the-projects-page]

Clique em ["Explore our control panel"](https://cloud.digitalocean.com/projects) ou simplesmente navegue até a [Projects page](https://cloud.digitalocean.com/projects).

Instâncias de servidor são chamadas de **"droplets"** na digitalocean, e elas são organizadas em **"Projects."**

### **b) Clique em "Spin up a Droplet" para iniciar a configuração** [#b-click-on-spin-up-a-droplet-to-start-the-setup]

![image](https://github.com/danny-avila/LibreChat/assets/110412045/6046e8cd-ff59-4795-a29a-5f44ab2f0a6d)

Ajuste essas configurações com base em suas necessidades, já que estou selecionando as opções mínimas/mais baratas que funcionarão.

- **Escolha a Região/Datacenter:** mais próxima de você e de seus usuários
- **Escolha uma imagem:** Ubuntu 22.04 (LTS) x64
- **Escolher tamanho:** CPU Compartilhada, Plano Básico
  - Opções de CPU: Regular, opção de 6 USD/mês (0,009 USD/hora, 1 GB RAM / 1 CPU / 25 GB SSD / 1000 GB de transferência)
  - Sem armazenamento adicional
- **Escolha o método de autenticação:** A opção de senha é a mais fácil, mas a escolha é sua
  - Alternativamente, você pode configurar o SSH tradicional.
- **Recomendado:** Adicione monitoramento de métricas e alertas aprimorados (gratuito)
  - Você talvez consiga utilizar a opção de $4/mês ao não selecionar isto, mas ainda não foi testado.
- **Finalizar Detalhes:**
  - Altere o hostname para o que desejar, deixo todo o resto como padrão (1 droplet, sem tags)
  - Por fim, clique em "Create Droplet"

![image](https://github.com/danny-avila/LibreChat/assets/110412045/ac90d40e-3ac6-482f-885c-58058c5e3f76)

Após criar o droplet, ele será iniciado com uma barra de progresso.

### **2. Acesse o console do seu droplet** [#2-access-your-droplet-console]

Assim que estiver em execução, **clique no droplet** e clique no link Console no lado direito para iniciar o console.

![image](https://github.com/danny-avila/LibreChat/assets/110412045/47c14280-fe48-49b9-9997-ff4d9c83212c)

![image](https://github.com/danny-avila/LibreChat/assets/110412045/d5e518fd-4941-4b35-86cc-69f8f65ec8eb)

Iniciar o console da Droplet desta forma é o método mais fácil, mas você também pode usar SSH se tiver configurado na etapa anterior.

Para manter este guia simples, serei direto e continuarei com o console da droplet. Aqui está um [guia oficial da DigitalOcean para SSH](https://docs.digitalocean.com/products/droplets/how-to/connect-with-ssh/) caso você tenha interesse.

### **3. Assim que você tiver feito login, crie imediatamente um novo usuário não-root:** [#3-once-you-have-logged-in-immediately-create-a-new-non-root-user]

**Nota:** você deve remover os sinais de maior/menor sempre que os encontrar neste guia

```bash
# example: adduser danny
adduser <yourusername>
# you will then be prompted for a password and user details
```

Assim que terminar, execute o seguinte comando para elevar o usuário

```bash
# example: usermod -aG sudo danny
usermod -aG sudo <yourusername>
```

**Certifique-se de ter feito isso corretamente verificando se você possui permissões sudo:**

```bash
getent group sudo | cut -d: -f4
```

**Mude para o novo usuário**

```bash
# example: su - danny
su - <yourusername>
```
