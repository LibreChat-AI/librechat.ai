ngrok을 사용하여 로컬 서버를 인터넷에 터널링하려면 다음 단계를 따르세요:

## 회원가입 [#sign-up]

1. **[https://ngrok.com/](https://ngrok.com/)**로 이동하여 계정을 만드세요.

## Docker 설치 [#docker-installation]

1. 다음 위치에서 인증 토큰을 복사하세요: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
2. 터미널을 열고 다음 명령어를 실행하세요: `docker run -d -it -e NGROK_AUTHTOKEN=<your token> ngrok/ngrok http 80`

## Windows 설치 [#windows-installation]

1. 다음 링크에서 ZIP 파일을 다운로드하세요: **[https://ngrok.com/download](https://ngrok.com/download)**
2. 7zip 또는 WinRar을 사용하여 ZIP 파일의 압축을 푸세요.
3. `ngrok.exe`를 실행하세요.
4. 다음 위치에서 인증 토큰을 복사하세요: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
5. `ngrok.exe` 터미널에서 다음 명령어를 실행하세요: `ngrok config add-authtoken <your token>`
6. 아직 수행하지 않았다면, LibreChat을 평소와 같이 시작하세요.
7. `ngrok.exe` 터미널에서 다음 명령어를 실행하세요: `ngrok http 3080`

LibreChat에 액세스하는 데 사용할 수 있는 링크가 표시됩니다.

## Linux 설치 [#linux-installation]

1. **[https://ngrok.com/download](https://ngrok.com/download)**에서 **올바른** 아키텍처를 선택하여 명령어를 복사하세요.
2. 터미널에서 명령어를 실행하세요
3. 다음 위치에서 인증 토큰을 복사하세요: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
4. 다음 명령어를 실행하세요: `ngrok config add-authtoken <your token>`
5. 아직 수행하지 않았다면, LibreChat을 평소와 같이 시작하세요.
6. 다음 명령어를 실행하세요: `ngrok http 3080`

## Mac 설치 [#mac-installation]

1. 다음 링크에서 ZIP 파일을 다운로드하세요: **[https://ngrok.com/download](https://ngrok.com/download)**
2. Unarchiver와 같은 적절한 Mac 애플리케이션을 사용하여 ZIP 파일의 내용을 추출하세요.
3. 터미널을 엽니다.
4. `cd` 명령어를 사용하여 ngrok을 추출한 디렉토리로 이동하세요.
5. `./ngrok`을 입력하여 ngrok을 실행하세요.
6. 다음 위치에서 인증 토큰을 복사하세요: **[https://dashboard.ngrok.com/get-started/your-authtoken](https://dashboard.ngrok.com/get-started/your-authtoken)**
7. ngrok을 실행한 터미널에서 다음 명령어를 입력하세요: `ngrok authtoken <your token>`
8. 아직 수행하지 않았다면, LibreChat을 평소와 같이 시작하세요.
9. ngrok을 실행한 터미널에서 다음 명령어를 입력하세요: `./ngrok http 3080`
